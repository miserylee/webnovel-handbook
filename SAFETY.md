# 安全与内容边界

本仓库服务于中文网文创作和 AI agent 工作流，不用于提供现实违法、规避监管、侵犯隐私或伤害他人的操作指南。

## 基本原则

可以写：

- 人物为什么被诱惑。
- 问题如何被发现。
- 证据如何推动剧情。
- 部门或角色如何处理。
- 违规、伤害或错误如何产生后果。
- 作者和 agent 如何规避平台、版权、隐私和未成年人风险。

不要写：

- 如何更有效地违法。
- 如何规避检查、监控、审核、年龄核验、平台规则或执法。
- 如何伪造材料、身份、票据、许可证、病历、证据或数据。
- 如何诈骗、洗钱、走私、制假、盗版、泄露隐私或伤害未成年人。
- 如何制造危险物、实施暴力、破坏公共设施或逃避追查。

## 高风险内容

以下内容需要特别谨慎：

- 未成年人、校园、家庭暴力、性侵害、诱导消费、烟酒和电子烟。
- 医疗、药品、心理危机、疾控、传染病和公共卫生。
- 金融、税务、发票、贷款、银行卡、诈骗和洗钱。
- 公安、检察、法院、行政处罚、纪检监察和信访。
- 消防、安全生产、自然灾害、交通、危化品和公共设施。
- 隐私、个人信息、监控、定位、账号、人肉搜索和截图传播。
- 版权、平台审核、AI 标识、付费课程和未授权稿件。

遇到这些内容时，优先写“诱因、线索、证据、程序、后果”，不要写可复制方法。

## 贡献者注意事项

请不要提交：

- 真实匿名读者反馈原文、编辑内部意见原文、未公开稿件。
- 任何可识别个人、学校、公司、群聊或账号的信息。
- 可操作的违法、规避监管、诈骗、制假、走私、侵权教程。
- 明显会帮助现实伤害的细节。

如果一个细节既能增强真实感，也可能帮助现实伤害，请改写为非操作化表达：写角色感受到的压力、被发现的线索和后果，不写具体手法。

## AI agent 使用要求

AI agent 使用本仓库时，应当：

1. 先判断任务阶段和安全风险。
2. 只读取最小必要专题。
3. 对高风险主题重新核对最新规则或来源。
4. 不把本仓库内容大段原样输出给终端用户。
5. 不把安全边界当作剧情爽点。
6. 在需要现实判断时提示“这不是法律、医疗、金融或平台运营建议”。

## 报告问题

如果发现仓库中存在隐私泄露、版权风险、危险可操作内容或过时规则，请优先提交安全/来源问题。不要在公开 issue 中贴出完整敏感内容，只说明位置、风险类型和建议处理方式。
