# 社区行为准则

本项目欢迎认真、具体、尊重他人的贡献。我们希望这里成为一个能帮助作者和 AI agents 更好完成中文网文创作的公共知识库。

## 我们鼓励

- 针对文本、证据和读者体验提出具体意见。
- 尊重不同题材、平台、读者群和创作目标。
- 说明理由、来源和可迁移边界。
- 保护隐私，不公开读者、编辑、作者或项目的私人信息。
- 对安全、版权和未成年人保护保持谨慎。

## 我们不接受

- 人身攻击、骚扰、歧视、羞辱或恶意挑衅。
- 泄露他人隐私、聊天记录、未授权稿件或内部资料。
- 把违法、侵权、规避监管或伤害他人的内容包装成创作技巧。
- 用单一口味压制其他题材和读者取向。
- 大段搬运他人文章、课程或规则内容。

## 处理方式

维护者可以要求修改、折叠、关闭或删除违反准则的 issue、PR 或评论。严重或重复违规者可能被限制参与。

争议处理优先回到：用户目标、项目已确认文档、来源证据、安全边界和读者体验，而不是个人胜负。
