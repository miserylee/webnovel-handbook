# 192. 校园欺凌、未成年人保护、家校沟通、处分转学与心理危机干预链专项

状态：已确认  
日期：2026-06-09  
适用对象：需要描写校园欺凌、学生欺凌、校园暴力、网络欺凌、同伴排挤、家校沟通、班主任处置、学校处分、转学休学复学、未成年人保护、校园安全、心理危机、家长维权、校园舆论和公安/司法介入的 AI agents。

## 1. 本专题解决什么问题

校园欺凌不能写成“主角被欺负，下一章爽快反杀”。现实里的欺凌往往有长期性、群体性、权力差、沉默旁观、家校信息差、证据缺口、未成年人隐私、学校声誉压力和心理后果。高质量校园题材要写出：谁发现，谁保护，谁调查，谁告知监护人，谁处理欺凌者，谁照顾受害学生，证据怎样固定，网络传播怎样下架，处分和转学如何留下后续影响。

本专题把校园欺凌和未成年人保护写成可执行链路：

```text
异常信号 -> 学生求助/同伴报告/教师发现/家长发现 -> 初步安全保护 -> 区分冲突、欺凌、暴力、违法犯罪和心理危机 -> 告知监护人和学校责任人 -> 证据固定 -> 班主任/年级/德育/心理教师/学校欺凌防治机制介入 -> 家校沟通与隐私保护 -> 教育惩戒、纪律处分、转介矫治、公安或司法分流 -> 被欺凌学生心理支持和学习安置 -> 旁观者教育和班级修复 -> 网络传播处置 -> 转学/休学/复学/处分档案和学籍衔接 -> 长期跟踪与台账回写
```

写作原则：校园欺凌不是单一坏学生问题，而是同伴生态、学校制度、家庭监护、网络传播、心理安全和法律边界共同作用的系统。

## 3. 安全边界

本专题只用于小说创作、审稿和设定，不提供法律、心理治疗、危机干预、学校管理或现实维权建议。

禁止写成教程的内容：

- 如何实施、组织、隐藏、升级或规避追责的欺凌、暴力、羞辱、敲诈、孤立、网络攻击或偷拍传播。
- 如何自伤、自杀、诱导他人自伤、威胁自伤或用心理危机操控他人。
- 如何伪造证据、剪辑视频、冒充账号、删除聊天、串供、威胁证人或规避学校/警方调查。
- 如何人肉开盒、公开未成年人姓名照片住址学校班级、组织网暴或群体围堵。
- 如何用私刑、暴力反杀、公开羞辱或极端报复作为“正义解决方案”。

可写方向：早期发现、保护学生、制止传播、证据保全、家校沟通、合理处分、心理支持、同伴修复、制度问责和长期恢复。

## 5. 欺凌、冲突、暴力和心理危机要先分流

| 类型 | 叙事判断 | 处置线 | 常见误写 |
| --- | --- | --- | --- |
| 普通同伴冲突 | 双方力量相对，偶发争执，可沟通修复 | 班主任、调解、规则教育 | 把所有吵架都写成霸凌 |
| 学生欺凌 | 一方利用身体、人数、权力、信息或关系优势，反复或恶意伤害 | 学校防治机制、家校沟通、处分和保护 | 写成受害者自己不坚强 |
| 校园暴力 | 造成明显人身或财产损害，可能有违法犯罪线索 | 学校、医疗、公安、司法分流 | 写成“孩子打闹” |
| 网络欺凌 | 群聊、偷拍视频、匿名投稿、P图、AI合成、造谣扩散 | 平台下架、证据保全、学校和公安接口 | 截图即真相，传播无后果 |
| 性骚扰或性侵风险 | 涉及身体边界、私密影像、性化羞辱 | 高风险，保护隐私，专业和司法介入 | 当成暧昧误会或校园恋爱 |
| 心理危机 | 失眠、恐惧、逃学、失联、自伤风险、极端绝望 | 第105篇、学校心理支持、监护和专业转介 | 写方法细节或用危机换道歉 |

写作提醒：分流不是给角色贴标签，而是决定谁该介入、证据如何保存、隐私如何保护、后果如何落地。

## 6. 早期信号：不要等到最严重时才让学校出现

欺凌叙事有现实感，往往来自细小异常的积累。

可写信号：

- 学生突然不愿上学、频繁请假、成绩下滑、睡眠变差、情绪警觉。
- 校服、书包、文具反复损坏或丢失，饭卡零钱异常消耗。
- 放学绕远路、拒绝坐校车、害怕进厕所或宿舍。
- 手机一响就紧张，群聊里被刻意排除或嘲笑。
- 老师发现座位周围同学突然沉默，点名时有人起哄。
- 家长只看到“孩子不说话”，不知道学校里已有长期信号。

写作技巧：先写一个小异常，再写它如何被大人误读。这样后续爆发时，读者会觉得痛但不突兀。

## 7. 学校角色：每个人职责不同

| 角色 | 叙事功能 | 风险 |
| --- | --- | --- |
| 班主任 | 最早发现、家校沟通、班级管理、谈话记录 | 压事、和稀泥、过度依赖家长私了 |
| 任课老师 | 课堂观察、座位互动、作业变化 | 只看成绩，忽略同伴关系 |
| 年级组/德育处 | 调查、处分、会议、校规执行 | 为校誉压低事件等级 |
| 心理教师 | 情绪评估、支持、转介、危机识别 | 被写成万能治疗师或保密边界不清 |
| 校医/医务室 | 伤情初记、转医建议、身体安全 | 只给创可贴，不留记录 |
| 校长/管理层 | 启动机制、对外沟通、责任问责 | 只在结尾讲话 |
| 家委会 | 家长沟通、舆论压力、资源动员 | 站队、泄露隐私、把调解变审判 |
| 保安/宿管 | 校门、宿舍、监控和巡查 | 发现异常却无反馈链 |

写作规则：学校不能完全消失，也不能一秒解决。真实的戏剧来自职责交接、记录留痕、信息滞后和成年人各自害怕承担责任。

## 8. 家校沟通：不要只写家长冲进办公室

家校沟通是一条信息链，不是单次吵架。

可写节点：

- 第一次电话：班主任措辞含糊，家长听出不对。
- 第一次见面：受害学生坐在角落，欺凌方家长说“孩子之间玩笑”。
- 会议纪要：谁在场、谁说了什么、学校承诺什么、下次什么时候反馈。
- 监护责任：欺凌方家长是否配合教育、赔偿、道歉和后续监督。
- 家长群泄露：本来保密的情况被家长转发，二次伤害发生。
- 多次沟通无果：家长开始收集证据、投诉、转学或报警。

写作提醒：家长不是天然理性保护者，也不是天然反派。有人愤怒失控，有人恐惧校誉，有人缺乏教育能力，有人只想尽快让孩子“恢复正常”。

## 9. 证据链：保护未成年人比赢舆论更重要

证据要有用，但不能让未成年人隐私被二次消费。

证据清单：

- 学校记录：谈话记录、班主任日志、值班记录、处分材料、监控调取记录、家校沟通纪要。
- 身体证据：校医记录、医院病历、伤情照片、物品损坏清单。
- 电子证据：聊天记录、群聊、匿名墙、短视频、平台举报回执、账号信息、转发链。
- 同伴证言：旁观者、同桌、宿舍同学、学生干部、社团成员。
- 环境证据：座位表、监控盲区、厕所走廊、宿舍门禁、校门外路线。
- 心理和学习影响：请假、成绩变化、睡眠、心理支持记录，注意保护隐私。

写作规则：

- 不公开未成年人的全名、照片、班级、家庭住址和心理隐私。
- 群聊截图要保护隐私，并处理截取语境和真假核验。
- 偷拍羞辱视频不应为了“证明”而在剧情里反复播放。
- 证据用于保护和处置，不应变成围观流量。

## 10. 教育惩戒、纪律处分和司法分流

欺凌处置不能只有两种极端：口头道歉或全部开除。现实校园有教育惩戒、纪律处分、转介矫治、公安司法和家庭教育责任等多层路径。

写作规则：

- 教育惩戒应与学生年龄、行为性质、危害程度和教育目的相关，不是老师泄愤。
- 纪律处分要有程序、告知、记录和救济空间，不能靠校长拍桌决定。
- 严重欺凌、伤害、敲诈、侮辱、偷拍传播、寻衅、性侵风险等应进入公安、司法或未成年人保护分流。
- 对欺凌者也要写监护人责任、教育矫治和后续监督，不能只写“坏孩子被打脸”。
- 对被欺凌者要写保护安排，避免处分或调解反而让其再次暴露在同一风险中。

## 11. 转学、休学、复学和学籍档案

很多校园欺凌情节最痛的地方，是受害学生反而被迫离开原本的生活轨道。

可写冲突：

- 家长想立刻转学，孩子害怕“逃跑”标签，也怕新学校追问过去。
- 学校建议“先在家休息”，家长担心学籍、考试和升学。
- 欺凌方家长要求双方都转班，受害方觉得是二次惩罚。
- 处分是否进入档案、是否影响评优升学、是否可申诉，成为双方争点。
- 复学前，班级没有修复，同学只记得“她害别人被处分”。

写作规则：转学休学不是简单换地图，而是学籍、课程、同伴关系、心理恢复、家长工作安排和经济成本共同变化。

## 12. 心理危机：写支持，不写方法

校园欺凌可能引发心理危机，但小说必须遵守第105篇安全边界：不写可模仿自伤自杀方法，不把危机写成解决方案、爱情证明、复仇武器或让他人后悔的奖励。

可写内容：

- 学生出现恐惧、回避、失眠、惊跳、情绪麻木、成绩下滑、拒学等后果。
- 同伴或老师发现异常，选择陪同、通知可信成年人和保护隐私。
- 学校心理教师提供初步支持，并在高风险时转介专业机构或紧急帮助。
- 家长从“你怎么这么脆弱”转为学会倾听和陪伴。
- 恢复是长期过程，可能反复，不是一场道歉就痊愈。

不可写内容：

- 具体自伤自杀方法、地点、工具、步骤或可模仿细节。
- 把自伤当成逼迫学校处理、让霸凌者悔悟、推动恋爱或制造高潮的按钮。
- 把心理危机简单归因于单一事件或单一坏人。
- 用“疯了”“矫情”“玻璃心”作为旁白判断。

## 13. 网络欺凌和平台传播

校园欺凌常常跨出校门，进入群聊、匿名墙、短视频、AI 合成图、游戏语音和粉丝群。

写作规则：

- 线上传播有时间线：谁首发、谁转发、谁截图、谁二创、平台何时下架。
- 群聊、匿名投稿、短视频和 AI 合成内容都要接入第163、174、189篇。
- 学校不能简单说“网上的事不归学校管”，只要和学生、校园关系、安全后果有关，就可能需要协同处理。
- 公开澄清要保护未成年人隐私，不能用更多细节换取网民相信。
- 网络证据要保存原始链接、发布时间、账号、平台回执和删除记录，不能只看一张糊截图。

## 14. 班级修复和旁观者教育

只处理欺凌者和被欺凌者，不处理旁观者，班级生态会继续坏下去。

可写修复动作：

- 班级规则重建：什么是玩笑，什么是边界，如何报告。
- 旁观者教育：起哄、录像、转发和沉默都会产生后果。
- 同伴支持：不要求受害者立刻原谅，但要让安全同伴出现。
- 老师复盘：座位安排、值日管理、宿舍巡查、群聊管理、课间盲区。
- 反报复机制：处分后如何防止“都是你害的”二次伤害。

写作提醒：修复不是大团圆。更真实的是有人道歉，有人装没事，有人转学，有人沉默很久后才说出自己也害怕。

## 15. 类型文里的用法

| 类型 | 用法 | 爽点或痛点 |
| --- | --- | --- |
| 校园现实 | 欺凌识别、家校沟通、班级修复 | 青春不是滤镜，也有制度和伤口 |
| 重生校园 | 主角提前发现信号，改变证据和求助路径 | 爽点来自保护和组织，不来自私刑 |
| 家庭伦理 | 父母的监护失职、过度控制或学习焦虑 | 家长不是旁观者 |
| 律政 | 处分、赔偿、名誉隐私、未成年人保护 | 程序和隐私比热搜更重要 |
| 悬疑 | 群聊、匿名墙、监控盲区、证言矛盾 | 真相藏在同伴生态里 |
| 网络舆论 | 偷拍、AI 合成、短视频扩散、平台下架 | 澄清也可能二次伤害 |
| 心理成长 | 创伤恢复、拒学、复学和同伴支持 | 慢恢复比爽反击更有力量 |
| 教育群像 | 老师、家长、学生、学校管理层各自承担代价 | 成人世界不能缺席 |

## 17. 写作提示词

### 17.1 生成校园欺凌处置场景

```text
请根据第192篇校园欺凌与未成年人保护链，生成一个校园欺凌处置场景。必须包含：异常信号、发现者、初步保护、家校沟通、证据链、学校分流判断、隐私保护和长期余波。不要描写可模仿欺凌或自伤方法，也不要把暴力反杀当解决方案。
```

### 17.2 审稿校园欺凌情节

```text
请审稿以下校园欺凌情节，检查是否存在：欺凌被爽文化、受害者被道具化、学校完全消失、家长只会吵架、证据链缺失、处分无程序、未成年人隐私泄露、心理危机教程化、网络传播无后果、AI 味过重。按“问题-影响-修改建议”输出。
```

### 17.3 设计家校沟通桥段

```text
请设计一个校园欺凌后的家校沟通桥段。要求：班主任、双方家长、年级或德育负责人至少出现三方；写清会议目标、证据、分歧、学校承诺、隐私保护和下次反馈节点。不要让家长吵一架就解决。
```

### 17.4 设计网络欺凌扩散桥段

```text
请设计一个校园网络欺凌扩散桥段。要求：写清首发平台、传播链、受害学生隐私风险、学校和平台处置、证据保存和澄清难点。不得重复展示羞辱内容，不得写人肉或网暴操作。
```

### 17.5 设计复学修复桥段

```text
请设计一个被欺凌学生复学桥段。要求：写清安全安排、同伴支持、班级规则、反报复机制、心理支持和学生本人选择。不要写一场道歉会就让所有创伤消失。
```

## 18. 自查清单

- 是否先联动第105篇控制创伤和自伤安全边界？
- 是否区分普通冲突、欺凌、暴力、违法犯罪线索和心理危机？
- 是否写出发现者、初步保护、通知监护人和隐私保护？
- 是否有学校角色：班主任、年级、德育、心理教师、校医或校长？
- 家校沟通是否有会议目标、记录、承诺和后续反馈？
- 证据是否包括学校记录、身体/医疗记录、电子证据、证人或平台回执？
- 是否避免公开未成年人姓名、照片、班级、住址、心理状态和羞辱材料？
- 教育惩戒和处分是否有程序、比例、记录和救济？
- 转学、休学、复学和学籍档案是否有后果？
- 心理危机是否写支持和转介，不写方法细节或奖励化结果？
- 网络欺凌是否处理平台下架、传播链、澄清和二次伤害？
- 结尾是否回写班级生态、同伴关系、家校信任和长期恢复？

## 19. 最小可用规则

如果时间很紧，至少执行以下 8 条：

1. 先判断是冲突、欺凌、暴力、违法犯罪线索还是心理危机。
2. 先保护学生安全和隐私，再推进调查和处分。
3. 所有欺凌场景必须有学校责任人和监护人链条。
4. 不写可模仿欺凌、自伤、自杀或报复方法。
5. 证据必须有来源、时间、原件或记录，不靠流言定案。
6. 网络欺凌必须处理下架、隐私、传播链和平台回执。
7. 处分、转学、休学、复学必须影响学籍、关系和心理状态。
8. 结尾必须留下长期恢复、班级修复或制度改变，而不是一场道歉结束。
