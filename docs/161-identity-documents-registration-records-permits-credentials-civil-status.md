# 161. 身份文书、户籍档案、证件许可、资格证明与登记记录

适用对象：AI agents、网文作者、审稿 agent、世界观设定 agent、现实题材 agent、古言/权谋 agent、都市职场 agent、校园/家庭 agent、悬疑调查 agent、跨境旅行 agent、连续性维护 agent

适用任务：设计、起草、续写、审稿或修订身份证、户籍、户口、出生登记、死亡登记、婚姻登记、收养登记、改名更名、学籍学历、职业资格、执照许可证、护照签证、通行凭证、居住证、档案、令牌腰牌、路引过所、官凭文牒、宗门身份牌、系统身份、数字 ID、黑户、无证者、冒名顶替、身份冒用、证件丢失、档案调取、登记错误和任何“一个人能否被制度承认”的剧情。

核心问题：AI 写身份系统时常把“身份”写成旁白标签：主角说自己是谁，世界就承认；角色想进城、上学、入职、结婚、继承、开户、报案、坐车、跨境、进宗门、领任务、买房、看病、考试、继承遗产，都不需要证件、档案、登记、审核或见证。结果是世界没有门槛，反转没有证据，冒名顶替没有成本，掉马没有制度后果。

一句话原则：身份文书不是背景资料，而是社会承认机器；它决定角色能否被看见、进入、交易、学习、工作、结婚、继承、通行、受保护、被追责和改变身份。每份关键文书都要有签发者、登记处、适用范围、核验方式、风险后果和台账状态。

重要边界：本篇用于小说创作和审稿，不提供伪造证件、冒用身份、规避边检、规避实名制、非法移民、非法开户、骗取资格、逃避监管等现实操作建议。涉及现实证件、户籍、婚姻、收养、移民、出入境、学历、职业资格、档案和个人信息时，必须查最新官方资料，并联动第 20、111、120、125、144 篇。

## 3. 身份文书是社会承认机器

### 3.1 身份的四层

| 层级 | 含义 | 例子 | 剧情功能 |
|---|---|---|---|
| 自我身份 | 角色知道自己是谁 | 本名、记忆、血脉、前世、性别认同、职业理想 | 内心冲突和自我选择 |
| 关系身份 | 亲友或圈层承认 | 女儿、徒弟、恋人、同桌、队友、义子 | 情感债和关系权限 |
| 社会身份 | 组织或社区承认 | 户籍、学籍、工牌、会员、弟子、军籍 | 行动资格和资源分配 |
| 法定/制度身份 | 由权威登记和证明 | 身份证、护照、婚姻登记、收养登记、职业执照 | 权利、义务、追责和程序结果 |

身份反转好看，是因为这些层级不一致：角色自认是真千金，家人不认，户籍档案不支持，亲子鉴定又出现新证据。每一层都可能制造冲突。

### 3.2 身份文书的七个功能

1. 进入：进城、进校、进宗门、进公司、进副本、过边检。
2. 领取：领取工资、补贴、任务、药品、养老金、奖学金、系统奖励。
3. 交易：开户、签约、买房、租房、抵押、继承、纳税。
4. 关系：结婚、离婚、收养、监护、认亲、户主、师承、组织成员。
5. 保护：未成年人保护、劳动保护、医疗服务、社会保障、难民/无国籍保护。
6. 追责：报案、传唤、处罚、失信、通缉、问责、取消资格。
7. 记忆：历史档案、族谱、学籍、病历、战功册、门派谱牒、系统日志。

如果身份文书只在掉马那一刻出现，前文又从未限制行动，读者会觉得作者临时补丁。

## 4. 文书生命周期

一份有效文书不是凭空出现的，它至少有生命周期。

```text
申请/登记 -> 材料 -> 审核 -> 签发/记录 -> 使用 -> 核验 -> 更新/变更 -> 挂失/注销/失效 -> 存档/追责
```

### 4.1 每个阶段都能写戏

| 阶段 | 可写冲突 | 常见爽点 | 风险 |
|---|---|---|---|
| 申请 | 材料不齐、见证人缺失、身份不明 | 主角补齐关键证据 | 流程水章 |
| 审核 | 排队、管辖、联网核对、档案冲突 | 对手卡流程被反制 | 官员工具化 |
| 签发 | 印章、编号、回执、公开名单 | 身份终于被承认 | 一句话生效无凭证 |
| 使用 | 进门、领钱、考试、继承 | 权限打开新地图 | 证件万能通吃 |
| 核验 | 信息不一致、过期、冒用、系统故障 | 破绽暴露或反杀 | 核验随作者改 |
| 变更 | 改名、婚姻、迁户、转学、换证 | 身份重构 | 台账忘更新 |
| 失效 | 注销、吊销、挂失、封存、通缉 | 角色失去行动资格 | 没有实际后果 |
| 存档 | 档案调取、历史追溯、证据链 | 旧真相浮出 | 档案万能答案 |

### 4.2 文书要有适用范围

证件不是无限钥匙。

- 学生证能进图书馆，不等于能进档案室。
- 宗门腰牌能入山门，不等于能进内库。
- 护照能证明国籍和身份，不等于自动获得入境许可。
- 结婚证能证明婚姻状态，不等于自动解决财产争议。
- 房契/产权证明能证明权利，但实际占有还受门锁、租约、法院、家族和地方势力影响。
- 系统身份卡能进副本，不等于现实世界承认。

规则：每份凭证都要写清“能做什么”和“不能做什么”。

## 5. 现实身份系统的写作转化

### 5.1 出生、死亡与生命事件登记

UNICEF 和联合国 Legal Identity 资料强调，出生登记是法律身份的重要起点；民事登记和生命统计系统记录出生、死亡、婚姻、离婚等重大事件。小说中这可以转化为：

- 孩子是否被社会承认。
- 年龄、继承、上学、医疗、监护和婚姻是否有证明。
- 失踪、假死、死亡登记是否造成财产和婚姻后果。
- 重生/穿越者的身体身份与社会记录冲突。
- 古代户籍册、族谱、里甲、宗门名册承担类似功能。

风险：不要把“没有出生登记/黑户/无国籍”写成轻松梗。这通常意味着教育、医疗、旅行、工作、保护和追责都受影响。

### 5.2 户籍、居住和地址

户籍、居住证、地址登记、宗门籍贯、家族族谱、城市居民册等决定一个人属于哪里。

可写冲突：

- 主角有能力，但没有本地户籍或居住证明，无法入学/买房/领补贴。
- 逃亡者不敢登记地址，导致无法获得服务。
- 假千金的户籍和亲缘记录冲突。
- 古言女子婚后户籍、族谱、嫁妆和居住权改变。
- 宗门弟子被除名后，腰牌失效，洞府和任务资格被收回。

### 5.3 婚姻、离婚和收养登记

婚礼仪式不等于制度生效；仪式、见证、文书和登记共同决定后果。

写法：

- 结婚：谁有资格登记，材料是否齐，婚姻状态是否冲突，是否联网核验，是否有跨境/跨族规则。
- 离婚：协议、财产、子女、冷静期或架空对应程序，不能只靠一句“休书”解决所有现代问题。
- 收养：登记机关、当事人、证明材料、孩子权益、户籍迁移、亲子关系生效。
- 古代/架空：婚书、族谱、媒证、宗门谱牒、神誓、契约都要有见证和后果。

### 5.4 学籍、学历和资格

学校、考试和职业资格是身份系统的重要分支。

可写冲突：

- 冒名顶替上学。
- 学籍档案被压住，无法转学或毕业。
- 职业资格吊销，角色不能继续执业。
- 玄幻宗门任务只认功勋册，不认口头承诺。
- 职场角色能力强，但没有执照，不能签字或上台。

### 5.5 护照、签证和跨境身份

跨境情节要注意：护照、签证、入境许可、边检、居留身份、难民/无国籍等现实规则复杂且会变化。

写作原则：

- 不写现实可操作规避教程。
- 不把边检写成随便糊弄。
- 不把无证者、难民、无国籍者写成犯罪标签。
- 重点写等待、材料、身份不确定、语言、费用、亲属关系、保护需求和风险。

## 6. 身份缺失、错误和冒用

### 6.1 身份缺失的后果

角色无证、黑户、被注销、档案丢失、系统查无此人，会带来持续后果：

- 不能上学或考试。
- 不能合法工作或领工资。
- 不能开户、买票、住店、看病、结婚、继承。
- 报案和求助困难。
- 容易被剥削、威胁或冒名。
- 被组织排除在资源分配外。

这不是“自由人设”，而是行动成本。

### 6.2 身份错误的后果

- 名字错、出生日期错、性别/籍贯错、父母信息错、婚姻状态错。
- 学籍和身份证信息不一致。
- 护照和签证信息不一致。
- 档案里有处分、旧案、假死亡、错误亲属关系。
- 旧系统没同步，新系统无法核验。

身份错误是很好的低烈度冲突，因为它会让角色在每一次办事时被迫解释。

### 6.3 冒名顶替和身份冒用

身份冒用是高风险情节，只写叙事后果，不写操作教程。

有效写法：

- 受害者失去机会、财产、资格或名声。
- 冒用者要持续维护材料、口供、关系和核验漏洞。
- 证据链包含登记时间、照片/画像、指纹/血脉/灵魂、见证人、档案流转、资金流、通信记录。
- 揭穿后有法律、组织、家族、舆论和关系后果。

禁止：详细教读者如何伪造、绕过核验、规避实名或冒用他人证件。

## 7. 档案、数据库和记录链

档案不是万能答案。档案也会有权限、缺页、错录、封存、篡改、管辖和时间延迟。

### 7.1 档案五问

```text
谁建立：医院、学校、单位、官府、宗门、法院、系统、家族。
谁能看：本人、监护人、单位、人事、执法、长老、黑客、内鬼。
记录什么：身份、成绩、病历、奖惩、任务、婚姻、财产、出入境、死亡。
如何改：申请更正、补材料、上级审批、法院裁判、宗门会议、系统任务。
如何错：录入错误、同名混淆、档案丢失、故意篡改、旧系统迁移失败。
```

### 7.2 档案调取要有成本

- 时间：排队、审批、等待调档。
- 权限：不是谁想看就能看。
- 风险：留下查询记录，暴露调查方向。
- 完整性：旧档案可能缺页、霉损、烧毁、封存。
- 解释权：档案内容需要懂专业术语、旧文书、缩写和制度背景。

## 8. 题材用法

### 8.1 都市现实

重点：身份证、户口、居住证、学籍、社保、医保、劳动合同、职业资格、房产证、银行卡、婚姻登记、出生证明。

规则：现实题材不能让角色“靠人情一句话”绕过所有实名、资格和记录。若确实绕过，要写风险、关系成本和不可复制性。

### 8.2 校园/青春

重点：学籍、转学、考试报名、处分档案、奖学金、毕业证、户籍学区、未成年人监护。

规则：校园身份不是“学生”两个字，而是学籍、班级、年级、监护、成绩和处分记录共同构成。

### 8.3 古言/宅斗/权谋

重点：户籍册、族谱、婚书、契书、路引、过所、官凭、诰命、户帖、地契、奴籍、军籍、度牒。

规则：真实朝代差异很大，不能凭印象混写。架空可做功能性制度，但必须清楚谁签发、谁核验、谁执行。

### 8.4 玄幻仙侠

重点：宗门腰牌、命灯、魂牌、弟子籍、功勋册、入城令、秘境资格、契约印记、血脉谱牒。

规则：超凡身份凭证也要有适用范围、识别边界、伪造成本和失效后果。身份牌不能既当地图、钱包、通讯器、钥匙又无任何限制。

### 8.5 悬疑推理

重点：死亡登记、失踪档案、假身份、旧户籍、病历、学校档案、出入境记录、酒店登记、车辆记录。

规则：登记记录可以提供线索，但要有来源、权限、时间戳、可篡改点和误读空间。

### 8.6 商战/职场

重点：营业执照、行业许可、法人身份、资质证书、员工档案、竞业协议、信用记录。

规则：公司不是靠名片运转。许可证、法人、合同签字权、资质等级和备案都能决定生意能不能做。

### 8.7 末世/废土

重点：旧身份失效、新营地登记、配给卡、通行牌、感染证明、居住资格、技能认证。

规则：文明崩塌不代表身份消失，而是旧身份和新身份系统冲突。谁发配给卡，谁就拥有权力。

### 8.8 星际/赛博

重点：数字身份、生物识别、芯片、星际护照、殖民地户籍、公司公民身份、数据隐私。

规则：数字身份越强，越要写数据错误、权限锁定、监控、被注销、被复制和身份恢复的代价。

## 9. 常见失败模式与修法

| 失败模式 | 读者体感 | 修法 |
|---|---|---|
| 身份旁白化 | 主角说是谁，世界就信 | 加签发者、凭证、核验、适用范围和失败后果 |
| 证件万能 | 一块令牌通吃全部权限 | 拆分门禁、任务、财务、档案、通行和身份权限 |
| 没有登记后果 | 结婚、收养、死亡、改名像口头约定 | 写仪式后文书、登记、见证和台账更新 |
| 冒名无成本 | 假身份太轻松 | 加材料维护、关系口供、核验风险和受害者后果 |
| 档案万能答案 | 一查档案全部真相 | 加权限、缺页、错录、封存、调档时间和误读 |
| 现实制度胡写 | 出戏和高风险 | 查最新官方资料，只转化为剧情约束 |
| 古代制度混搭 | 朝代感崩 | 架空则功能自洽，真实朝代则查证 |
| 无证者浪漫化 | 漂泊自由但无现实成本 | 写教育、医疗、通行、工作和保护困难 |
| 改名没后果 | 多 agent 连续性崩 | 回写证件名、常用名、旧名、公开范围和关联记录 |
| 身份揭示无余波 | 掉马爽一下就没了 | 写资格、关系、财产、问责、舆论和安全后果 |
| 平台实名随便绕 | 合规风险 | 不写可操作规避教程，只写风险和后果 |
| 文书只做道具 | 纸张出现一次就消失 | 建生命周期：申请、签发、使用、核验、变更、失效、存档 |

## 10. 审稿检查问题

### 10.1 身份系统可信度

1. 角色当前身份由谁承认？自我、亲友、组织、政府、宗门、系统是否一致？
2. 有哪些凭证证明这个身份？证件、档案、见证人、登记记录、令牌、血脉、日志？
3. 凭证在哪里有效，在哪里无效？
4. 核验方式是什么，失败后会发生什么？
5. 身份变更后，称呼、权限、财产、关系、记录是否更新？

### 10.2 剧情功能

1. 身份文书是否真正改变角色行动资格？
2. 证件缺失、错误、过期、丢失、被注销是否产生持续后果？
3. 档案调取是否有权限、时间、成本和风险？
4. 掉马、认亲、结婚、继承、入学、入职是否有制度链，不只靠台词？
5. 是否有台账字段记录证件状态和下次使用点？

### 10.3 安全边界

1. 是否涉及伪造证件、身份冒用、非法出入境、规避实名、骗取资格？
2. 如果涉及，是否只写叙事后果，不写现实操作步骤？
3. 是否涉及未成年人、收养、婚姻、亲子、个人信息隐私？
4. 是否查证现实法律、政策、流程或使用架空功能性制度？
5. 是否避免把无证者、难民、无国籍者、移民或被登记错误者污名化？

## 11. 台账字段

```text
身份事项编号：
角色：
身份事项：出生 / 死亡 / 婚姻 / 收养 / 改名 / 户籍 / 学籍 / 职业资格 / 通行 / 出入境 / 组织身份 / 产权 / 其他
登记名 / 常用名 / 化名 / 旧名：
凭证名称与编号：
签发/登记者：
适用范围：
核验方式：
当前状态：有效 / 过期 / 丢失 / 错误 / 被封存 / 被注销 / 冒用中 / 待补办 / 有争议
公开范围：谁知道、谁不知道、谁误会
关联证据：档案、见证人、照片/画像、生物识别、血脉、灵魂印记、系统日志、通信记录
行动资格：能进入哪里、能领取什么、能签什么、能继承什么、能被谁追责
风险后果：
本次变化：
下次回收点：
禁写边界：不提供现实伪造、冒用、规避核验、非法通行教程
```

最小回写：出生、死亡、婚姻、离婚、收养、改名、户籍迁移、学籍变化、资格取得/吊销、证件丢失/补办、身份公开/注销都必须写入台账。

## 12. 可直接给 AI agent 的提示词

### 12.1 身份文书场景设计提示词

```text
请设计一场身份文书/证件/登记/资格相关场景。不要让角色一句话证明身份。先填写身份文书八线卡：身份事项、凭证名称、签发/登记者、适用范围、核验方式、角色状态、当前目标、风险后果和台账回写。正文必须让证件、档案、登记、审核或核验改变角色行动资格，并写清核验失败后的代价。
```

### 12.2 身份反转审稿提示词

```text
请审稿这段身份反转/冒名顶替/认亲/掉马情节，重点检查：身份层级是否清楚，自我身份、关系身份、社会身份和制度身份是否冲突；证据链是否有来源和核验；冒名是否有维护成本；揭示后是否改变资格、财产、关系、舆论和追责。请按“问题 -> 读者体感 -> 结构修法 -> 需查证来源 -> 台账回写”输出。
```

### 12.3 架空身份系统提示词

```text
请为这个架空世界设计一套身份系统。不要只给名词。请说明谁登记、谁签发、凭证是什么、如何核验、在哪里有效、如何变更、如何失效、无证者会怎样、身份错误如何纠正、伪造/冒用会带来什么后果。输出要能直接用于章节中的进城、入学、入宗门、领任务、结婚、继承、通行、审判和档案调取场景。
```

## 13. 来源与吸收方式

- UN Legal Identity Agenda 与联合国民事登记/生命统计资料：吸收点是法律身份通常通过出生、死亡、婚姻、离婚等民事登记和授权机构签发证明来确认；迁移为“身份不是旁白标签，而是登记、证书和权威承认链”。
- UNICEF Birth registration：吸收点是出生登记是儿童第一份法律身份证明，影响教育、医疗、保护和年龄证明；迁移为“没有出生记录会影响角色被制度看见和保护”。
- World Bank ID4D：吸收点是身份系统与社会保护、健康、金融、治理、性别、数字服务和数据保护相关；迁移为“身份凭证能打开资源，也会制造排除、隐私和监控风险”。
- UNHCR Statelessness：吸收点是无国籍和缺乏身份证明会影响国籍确认、旅行证件、保护和服务获得；迁移为“无证/无国籍/身份不明不是自由设定，而是高成本生活状态”。
- 中国《中华人民共和国居民身份证法》：吸收点是居民身份证用于证明身份、保障权益、便利社会活动并维护社会秩序，也涉及个人信息保护和违法责任；迁移为“现代中国题材中身份证是行动资格和风险边界，不是随口报号码”。
- 中国《中华人民共和国户口登记条例》：吸收点是户口登记涉及出生、死亡、迁出迁入等人口管理事项；迁移为“户籍不是背景词，而是入学、居住、迁移、亲属和社会服务的制度入口”。
- 中国《婚姻登记条例》及婚姻登记全国通办政策解读：吸收点是婚姻登记机关核对证件、材料、身份和婚姻状况信息，虚假材料会有责任；迁移为“婚礼仪式不等于制度婚姻，结婚/离婚必须有登记、材料和后果”。
- 中国《中国公民收养子女登记办法》：吸收点是收养关系依法办理登记，收养人需提交户口簿、身份证、婚姻和抚养能力等材料；迁移为“收养不是一句认亲，必须有登记、孩子权益、户籍和监护后果”。
- 中国《中华人民共和国护照法》与国家移民管理相关资料：吸收点是护照用于出入国境和在国外证明国籍、身份；迁移为“跨境剧情要区分身份证明、旅行证件、签证/入境许可和边检核验”。
- 本知识库第 31、111、120、125、144、147、149、152、157、158 篇：将制度层、事实查证、证据链、身份揭示、官场文书、仪式身份转换、通行凭证、住所权属、学籍和亲子登记重组为本篇的身份文书执行链。

## 14. 最小执行规则

如果时间不够，AI agent 至少执行以下 8 条：

1. 身份必须区分自我身份、关系身份、社会身份和制度身份。
2. 关键身份必须有凭证、登记者、适用范围、核验方式和失败后果。
3. 出生、死亡、婚姻、收养、改名、迁户、入学、入职、通行、继承都要有文书或登记链。
4. 证件缺失、错误、过期、注销、封存、丢失和被冒用必须产生持续后果。
5. 档案和数据库不能万能，要有权限、缺页、错录、封存、调取时间和误读空间。
6. 身份揭示后必须改变资格、财产、关系、舆论、问责或安全状态。
7. 现实证件、户籍、婚姻、收养、移民、学历、职业资格和个人信息必须查最新官方资料。
8. 不提供现实伪造证件、冒用身份、规避核验、非法通行或规避实名制教程。
