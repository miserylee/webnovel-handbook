# 154. 卫生清洁、洗浴如厕、衣物清洗、气味污秽、经期体面与公共卫生

状态：已确认

适用对象：AI agents、网文作者、审稿 agent、连续性维护 agent、生活细节 agent、医疗/末世/古言/逃亡/校园/都市/种田题材 agent

核心问题：AI 写长篇时常让人物像没有身体：连日赶路不洗不换也没人有气味，战后血污自动消失，古代宅院和末世基地没有厕所和污水，女性角色的经期和隐私完全失踪，穷困角色永远保持干净体面，餐馆经营只写香味不写清洁，疫病灾荒没有水源和垃圾压力。卫生清洁不是低俗细节，而是身体、尊严、阶层、健康、安全、劳动和社会秩序的基础系统。

---

## 1. 一句话原则

已确认：卫生不是尴尬细节，而是“水从哪来、污物去哪、身体如何保持体面、谁承担清洁劳动、缺清洁会带来什么风险”的叙事系统。

AI agent 只要写到长期居住、旅行、逃亡、战斗、病伤照护、餐饮经营、宿舍同住、古代宅院、末世基地、校园生活、经期、衣物、气味或公共空间，就必须检查：

1. 水源、厕所、洗浴、换衣和垃圾处理是否存在。
2. 清洁条件如何影响体面、健康、行动、关系和阶层。
3. 污渍、气味、血污、泥污、污水、脏衣和身体状态是否需要回写台账。

## 3. 卫生清洁十线卡

AI agent 写长期生活、旅程、基地、宅院、餐馆、宿舍、战后、病房、灾荒或末世场景前，先填这张卡。

```text
场景/地点：宅院、出租屋、宿舍、客栈、军营、基地、船上、秘境、医院、餐馆、野外等。
水源：井水、河水、雨水、自来水、灵泉、储水罐、净水器、供水车、系统资源。
饮水处理：能否直接喝，是否需要烧开、过滤、净化、符箓、药剂、检测。
洗浴条件：澡堂、浴桶、浴室、河边、擦洗、公共浴室、无条件清洗。
如厕条件：厕所、净房、茅厕、夜壶、恭桶、马桶、临时坑位、组织公共设施。
污物去向：污水、粪污、厨余、垃圾、血污、脏绷带、药渣、兽粪如何处理。
衣物清洗：换洗衣物、洗衣地点、晾晒、干燥、除味、修补、谁来洗。

台账回写：脏衣、血污、气味、缺水、垃圾、经期、伤口清洁、住所卫生状态。
```

## 4. 水源是第一卫生规则

没有水，就没有稳定清洁。

AI agent 写水源时要回答：

- 水从哪里来。
- 谁控制水源。
- 水能否饮用，能否洗澡，能否洗衣。
- 缺水时谁优先。
- 水源被污染、断供、冻结、抢占或封锁后，人物怎么改变行动。

可用剧情压力：

- 客栈只剩一桶热水，队伍内部决定谁先洗。
- 末世基地饮水优先给伤员，衣物只能晒不能洗。
- 宗门灵泉只供内门使用，外门弟子只能去山下挑水。
- 古代宅院的井被封，厨房、洗漱和药煎全受影响。

风险：不要把“净水”“消毒”“过滤”写成现实可执行教程。需要专业或现实规则时查官方资料，正文只写叙事效果。

## 5. 洗浴与身体体面

洗浴不是福利场景，也不是自动擦除状态的按钮。

### 5.1 洗浴的叙事功能

- 恢复体面：从血污、泥污、汗味、狼狈中重新获得见人资格。
- 暴露伤痕：洗去外物后，旧伤、新伤、纹身、烙印、印记出现。
- 关系边界：谁能递水，谁能守门，谁必须回避，谁被允许照护。
- 阶层差：热水、香皂、澡堂、仆役、浴室、干净换洗衣物都是资源。
- 安全风险：洗澡时防备低，容易被偷袭、偷听、发现身份或缺少逃跑准备。

### 5.2 写法边界

- 不把洗浴写成低俗凝视或身体消费。
- 不用未经同意的偷看、闯入、强行洗浴当浪漫糖点。
- 涉及未成年人、权力差、创伤、羞辱、强迫清洁时，必须联动第 20、105、115 篇。

## 6. 如厕、污物与垃圾：可以不细写，但不能不存在

商业网文不需要频繁写如厕细节，但长期生活场景不能完全没有污物处理逻辑。

### 6.1 应该交代的情况

- 末世、逃荒、军营、船上、秘境、基地、荒野长途。
- 疫病、灾后、洪水、旱灾、缺水、围城、难民安置。
- 古言宅院、客栈、监牢、病房、厨房、餐馆经营。
- 悬疑线索依赖气味、污水、垃圾、血污、脏绷带、弃置物。
- 人物被羞辱、失去体面、被剥夺隐私或被迫处于恶劣环境。

### 6.2 写法原则

- 以功能为主，少写猎奇细节。
- 重点写限制、气味、隐私、劳动、路线、制度和后果。
- 脏污可以制造压迫，但不要消费受害者尊严。

## 7. 衣物清洗与换洗连续性

衣物不会自己变干净。

AI agent 要记录：

- 角色有几套换洗衣物。
- 脏衣、血衣、制服、礼服、伪装服是否清洗、丢弃或藏起。
- 晾晒地点会不会暴露身份。
- 雨天、冬天、潮湿、缺水时衣物如何干。
- 谁负责洗衣，清洁劳动是否体现阶层或关系。
- 气味、血迹、药味、烟味、香粉是否成为线索。

例：逃亡三天后，主角不能仍然“衣袂如雪”。如果仍然干净，必须有设定支撑：法衣自净、备用衣物、客栈洗衣、清洁符、队友照护，但这些都要有成本或限制。

## 8. 气味不是五感装饰，而是信息

气味很适合做低成本但强代入的线索。

可用功能：

- 身份线索：药味、香粉、烟草、油烟、铁锈、兽腥、墨味、消毒水。
- 时间线索：新鲜血味和旧血味、刚洗过的衣物、潮湿霉味、饭菜变馊。
- 空间线索：下水道、垃圾堆、药房、后厨、马厩、病房、洗衣房。
- 情绪线索：熟悉的洗衣皂味代表归属，刺鼻消毒味代表危险。
- 追踪线索：动物、灵宠、追踪术、猎人、怪物依靠气味。

风险：不要每个场景都写气味；气味一旦成为线索，就要能改变判断或被后文回收。

## 9. 经期、身体隐私与尊严

经期/月事属于身体和卫生条件的一部分，不应被猎奇化、羞辱化或完全抹除。

适用场景：

- 长途旅行、逃亡、军营、宿舍、校园、古言宅院、末世基地、贫困生活、身体虚弱、礼仪场合、战斗任务。

可写压力：

- 是否有干净用品、热水、隐私、换洗条件、疼痛照护、污物处理。
- 是否被规矩、贫困、旅途、监禁、任务期限压缩选择。
- 谁提供帮助，谁尊重边界，谁借此羞辱或控制。
- 主角是否因为身体状态调整计划，而不是硬撑到像没身体。

安全边界：

- 不写成低俗看点。
- 不用经期羞辱制造爽点。
- 不把身体隐私暴露当作轻浮玩笑。
- 涉及未成年人、强迫、羞辱、创伤、医疗判断时联动第 20、105、131 篇。

## 10. 清洁劳动与阶层

“干净”常常是别人劳动的结果。

可写维度：

- 谁烧水、挑水、倒夜壶、洗衣、擦地、收拾餐具、处理垃圾。
- 主人是否看见这些劳动。
- 仆役、杂役、外门弟子、保洁、护士、后厨、清洁工、宿管、物业如何构成信息网络。
- 清洁劳动者能看到别人忽略的痕迹：血迹、药渣、纸团、账单、泥印、掉发、香粉。
- 主角主动做清洁，可以体现羞耻、独立、照护、赎罪、伪装或阶层跨越。

风险：不要让清洁劳动只服务主角体面；劳动者也有观察、判断、利益和安全边界。

## 11. 公共卫生与群体空间

公共卫生不是现代背景才有的问题。任何群居空间都有卫生秩序。

适用场景：

- 军营、学校、宗门、客栈、船舱、监牢、难民营、末世基地、医院、餐馆、矿区、工坊、宫廷、城池。

可写压力：

- 水源污染、垃圾堆积、厕所不足、蚊虫鼠患、拥挤、通风差、污水倒灌。
- 疫病谣言、隔离、清洁规定、巡查、罚款、驱逐、资源优先级。
- 谁有权制定卫生规则，谁承担执行，谁被污名化。
- 反派可以通过水源、垃圾、后厨、病房和清洁流程制造问题，但不要写现实可模仿的危害教程。

## 12. 题材用法

### 12.1 古言/宅斗/宫廷

重点：净房、浴桶、热水、仆役、衣物浆洗、熏香、月事隐私、门第体面、厨房卫生、祠堂礼仪。

规则：宅院不是只有花园和厅堂。厨房、洗衣处、井、净房、下人房、后门和杂役路线都能成为信息通道。

### 12.2 玄幻/仙侠

重点：清洁术、净尘诀、自洁法衣、灵泉、洞府污物、兽宠排泄、丹房卫生、秘境水源。

规则：清洁法术可以存在，但必须有消耗、品级、限制或社会差异。不要让“法术清洁”抹掉所有身体和阶层问题。

### 12.3 都市/校园/职场

重点：租房卫生、合租公共区、宿舍厕所、洗衣机、通勤汗味、办公室体面、外卖垃圾、物业和邻居。

规则：现代读者对生活卫生很敏感。合租、宿舍、职场和医院等场景要有真实但克制的卫生摩擦。

### 12.4 末世/逃荒/战争

重点：净水、厕所、垃圾、血污、伤口清洁、衣物干燥、营地卫生、虫鼠、气味暴露。

规则：末世不是只缺食物和武器。清洁和污物处理会直接影响安全、士气、尊严和群体秩序。

### 12.5 美食/经营

重点：后厨清洁、食材保存、餐具、手部卫生、垃圾、鼠虫、顾客信任、监管检查。

规则：美食文不能只写香味和夸赞。真正的餐饮经营需要看见清洁和稳定出品背后的劳动。

## 13. 常见失败模式

- 无身体人物：长期行动后不洗不换也无气味、无污渍、无不适。
- 自动清洁世界：血污、泥污、汗味、脏衣、垃圾和污水自动消失。
- 卫生猎奇化：为了冲击读者而堆脏污细节，没有叙事功能。
- 经期消失：所有女性角色长期旅行、战斗、校园、逃亡都没有任何身体安排。
- 经期羞辱化：把身体隐私当笑点、爽点、低俗看点或惩罚工具。
- 贫穷不脏也不累：底层角色资源匮乏，却永远干净体面，生活成本不存在。
- 豪门干净无劳动：府邸、豪宅、宗门永远整洁，却没人烧水、洗衣、清扫、处理垃圾。
- 餐馆只香不洁：后厨、餐具、储存和卫生完全缺席。
- 末世只缺武器：水、厕所、垃圾、伤口清洁、衣物干燥和气味暴露都不影响生存。
- 清洁法术万能：一个净尘诀解决所有卫生、身份、线索和阶层问题。

## 14. 审稿检查问题

AI 审稿或修订时，逐条检查：

- 这个场景的水从哪里来，污物到哪里去？
- 人物近期是否有洗浴、换衣、洗手、净水、处理血污/泥污/脏衣的机会？
- 清洁条件是否与角色阶层、地点、季节、旅程、职业和资源一致？
- 气味、污渍、脏衣、垃圾是否有叙事功能，还是猎奇堆砌？
- 谁承担清洁劳动？劳动者是否有观察、利益和边界？
- 经期、身体隐私、洗浴、如厕、照护是否被尊重，而非低俗化或羞辱化？
- 餐饮、医疗、病房、营地、基地、宿舍、客栈是否有基本卫生逻辑？
- 玄幻清洁术、法衣自净、灵泉净化是否有消耗、限制和阶层差？
- 卫生状态是否影响行动、体面、健康风险、关系、线索或社会评价？
- 涉及现实公共卫生、医疗、消毒、防疫、经期健康、食品安全、饮水安全时，是否需要查官方资料并避免建议化？

## 15. AI 写作提示词片段

### 15.1 建立卫生清洁系统

```text
请为这个场景建立卫生清洁系统。
要求：写清水源、饮水处理、洗浴条件、如厕条件、污物去向、衣物清洗、身体隐私、清洁劳动者、气味线索、卫生风险和台账回写。不要写猎奇细节，只写会改变行动、关系、体面或安全的要点。
```

### 15.2 修订无身体人物

```text
请检查这段剧情是否把人物写得像没有身体。
如果角色经历了赶路、战斗、逃亡、病伤、劳动、经期、雨雪、泥泞、酷暑或长期同住，请补上清洁条件如何影响衣物、气味、隐私、体面、健康风险或下一步行动。
```

### 15.3 写卫生不水的生活流

```text
请写一段生活流清洁场景。
要求：不能只是洗澡洗衣做家务；必须通过清洁动作暴露一个关系变化、线索、阶层差、照护动作、空间秘密或下一步计划。
```

### 15.4 设计末世/军营/客栈公共卫生压力

```text
请设计一个群居空间的公共卫生压力。
要求：写清水源、厕所、垃圾、洗衣、伤口清洁、人员分工、规则执行、违反后果、谣言或疾病恐慌、以及主角必须做出的选择。不要提供现实危害或防疫教程。
```

## 17. 安全与真实性边界

- 本专题只用于小说创作，不提供医疗、公共卫生、防疫、消毒、饮水处理、食品安全、经期健康或职业卫生建议。
- 涉及疾病、传染、伤口处理、消毒、防疫、经期异常、精神创伤、未成年人、性化凝视、强迫洗浴、身体羞辱时，必须联动第 20、105、131 篇。
- 不写可模仿的投毒、污染水源、传播疾病、规避卫生监管、处理危险废弃物或伤害他人的操作步骤。
- 不把身体隐私、经期、如厕、洗浴、污秽或失禁写成低俗笑点、羞辱爽点或猎奇看点。
- 现实餐饮、医院、学校、宿舍、租房、公共卫生、饮水和食品安全规则会变化；涉及具体规则必须查最新官方资料。
- 架空世界可以自设清洁法术、净水符、灵泉、自动清洁阵法，但规则一旦建立，就要持续执行并进入台账。

## 18. 来源与吸收方式

- WHO Water, Sanitation and Hygiene / Drinking-water 资料：吸收饮水、卫生设施和公共卫生基础条件会影响健康、尊严和社会运行的视角，用于水源、厕所和污物处理规则。
- CDC Handwashing and hygiene 资料：吸收手部卫生和清洁习惯对疾病预防和日常安全的重要性，用于餐饮、医疗、校园、旅途和群居场景。
- UNICEF WASH 与 menstrual hygiene 资料：吸收水、卫生设施、隐私、月经卫生、尊严和学校/社区生活之间的关系，用于经期体面和身体隐私边界。
- WHO menstrual health 资料：吸收“经期健康不只是卫生用品问题，还涉及尊严、信息、隐私、服务和社会支持”的视角，用于避免经期消失、猎奇化和羞辱化。
- Writers Helping Writers、Reedsy、Writer's Digest 等感官与 setting 写作资料：吸收气味、污渍、环境细节应服务人物、冲突、沉浸和线索的写作原则。
- 本知识库第 30、31、71、105、131、137、143、145、149、152、153 篇：整合生活细节、地域风俗、感官落地、敏感安全、医疗边界、美食卫生、天气季节、服饰气味、旅程洗浴、住所动线和身体资源，形成“水源 -> 清洁条件 -> 身体体面 -> 劳动分配 -> 风险后果 -> 台账回写”的执行链。

## 19. 最小执行规则

已确认：AI agent 写任何长期生活、旅途、战后、群居、餐饮、病伤、末世或身体隐私相关内容，最少必须检查“三件事”：

1. 水、洗、排、换、污物处理是否有基本逻辑。
2. 清洁条件是否改变体面、隐私、行动、关系、健康风险或线索。
3. 脏衣、气味、血污、经期、缺水、垃圾和卫生状态是否需要进台账。

如果三件事都没有，卫生描写就是空白或猎奇，应压缩、删除或重构。
