## 2. 推荐链路

### 2.1 写一场高压动作戏

```text
118/121/123 场景类型 -> 32 段落动作顺序 -> 07 连续性台账 -> 必要时 safety-research
```

动作戏先保证空间、目标、阻力、失败代价和结果状态，再考虑句子漂亮。

### 2.2 写一场对峙交易戏

```text
122 谈判筹码 -> 156 非语言潜台词 -> 04 人物对白 -> 43 反馈优先级按需修订
```

谈判不是互相解释立场，而是用筹码、误导、试探和让步改变局面。

### 2.3 写一场生活质感戏

```text
137/143/145/150/152/153/154 目标生活细节 -> 11 人味增强 -> 32 段落体验
```

生活细节只保留能暴露人物状态、关系压力、阶层差异或下一步选择的部分。

### 2.4 写一场关系爆点戏

```text
115/127/128/129 关系场景 -> 25 情绪线 -> 37 角色声音 -> 156 肢体沉默潜台词
```

关系爆点不靠喊情绪，要写出谁先退、谁不肯说、谁用动作保留尊严。

### 2.5 写一场调查推理戏

```text
120 线索证据 -> 141 裁判/审理按需 -> 26 信息控制 -> 63 伏笔读者记忆
```

推理场景必须让读者能回看线索，不能靠作者临时补设定。

## 3. 提取格式

读取场景专题时，只带走这些内容：

- 场景目标：本场人物想得到什么，失败会失去什么。
- 空间与物件：哪些位置、道具、身体状态会改变行动。
- 信息差：谁知道、谁误判、谁隐瞒、谁在试探。
- 互动结构：进攻、防守、让步、反制、误会、沉默或转移话题。
- 结果状态：关系、证据、资源、身体、名声、情绪或台账发生了什么变化。
- 常见 AI 误写：过度解释、动作不可拍、爽点没落袋、生活细节无功能。

## 4. 常见误用

- 不要把场景专题当素材清单；每个细节都必须服务场景目标和结果状态。
- 不要为了“真实”把路程、饭菜、服装、钱价和卫生细节全写出来；只写能压到人物的细节。
- 不要写动作戏只写招式名；读者要能看见位置、节奏、受伤、误判和代价。
- 不要写谈判戏只写大道理；筹码、底线和可执行后果比观点更重要。
- 不要把生活场景写成过场水；低强度场景也要改变关系、信息、情绪或下一步选择。
