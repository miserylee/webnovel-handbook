# 153. 睡眠疲劳、休息恢复、作息节律、精神消耗与身体资源连续性

状态：已确认

适用对象：AI agents、网文作者、审稿 agent、连续性维护 agent、动作/战斗/逃亡/修炼/日常/医疗/都市职业题材 agent

核心问题：AI 写长篇时常把人物写成永动机：连续赶路、战斗、熬夜、调查、修炼、照顾病人、开会、上班、逃亡、爆更或情绪崩溃后，第二天仍然判断清晰、体力满格、语气稳定、伤势不影响行动。这样会削弱真实感，也会让高压剧情失去代价。商业级长篇需要把睡眠、疲劳、休息和精力恢复当成身体资源系统，而不是“睡了一觉就全好了”的清零按钮。

---

## 1. 一句话原则

已确认：休息不是跳过剧情，疲劳不是形容词，睡眠不是回血按钮；它们都是角色行动资格、判断质量、情绪稳定、关系照护和后续代价的连续性资产。

AI agent 只要写到熬夜、守夜、连战、赶路、闭关、训练、夜班、照护、惊吓、噩梦、失眠、休整或“第二天”，都必须检查：

1. 角色睡了多久、在哪睡、睡得是否安全。
2. 睡眠/休息前后，体力、注意力、情绪、伤病和判断有何差额。
3. 休息是否改变关系、信息整理、下一步决定或台账状态。

## 3. 身体资源七线卡

AI agent 在写高压行动、连战、夜戏、逃亡、训练、疗伤、闭关、赶路和低强度休整前，先填这张卡。

```text
当前时间：白天/夜晚/黎明/深夜/连续第几天。
上次有效睡眠：多久前，睡了多久，睡眠质量如何。
当前身体资源：体力、精神、饥饿、疼痛、伤病、药效、灵力/异能/专注力。
当前疲劳来源：赶路、战斗、守夜、熬夜、情绪冲击、工作、修炼、照护、恐惧、饥饿、伤痛。
疲劳表现：反应慢、记错、易怒、走神、手抖、判断冒进、沉默、感官迟钝、过度敏感。
恢复条件：安全地点、时间、食物、水、药、照护、轮班、保暖/降温、隔音、心理安全。
恢复后差额：体力恢复多少，伤痛是否减轻，情绪是否稳定，信息是否整理，下一步决定是否改变。
台账回写：睡眠不足、伤病、药效、精神消耗、守夜轮班、身体透支、恢复状态。
```

## 4. 睡眠不是“黑屏跳过”：四种叙事功能

### 4.1 生理恢复

用于恢复体力、反应、专注、伤病承受力和情绪稳定。

写法：不必每次详细写睡眠过程，但要在高压后保留恢复差额。比如“睡了四个时辰，醒来时手还在抖，但终于能分辨昨夜谁说了谎”。

风险：一觉满血会让战斗、逃亡、伤病和情绪压力失去代价。

### 4.2 信息整理

睡前/醒后适合整理线索、复盘对话、意识到遗漏、改变判断。

写法：不要写成大段旁白总结，可以让角色在半梦半醒、洗脸、换药、收拾行李时突然把两个细节连上。

风险：所有推理都在睡醒后“灵光一现”，会显得作者代替角色思考。

### 4.3 关系照护

睡眠是关系最容易显形的场景：谁守夜、谁让床、谁盖被子、谁不敢睡、谁在门外坐一夜。

写法：用具体照护动作替代空泛关心。

风险：把监视、控制、无处可逃、未经同意的同床共处写成浪漫，会触发关系和安全风险。

### 4.4 脆弱暴露

睡着意味着防备降低。适合写秘密暴露、噩梦、梦话、夜袭、被照顾、信任测试。

写法：脆弱暴露要有后果，不能只用来撒糖或吓人。

## 5. 疲劳表现：不要只写“很累”

疲劳最好通过判断和动作表现，而不是反复写“疲惫”。

可见表现：

- 反应延迟：晚半拍才听懂，错过对手动作。
- 注意力缩窄：只盯着最近的出口、食物、床、灯、伤口。
- 记忆出错：忘记锁门、漏看名单、把时间记错。
- 情绪阈值降低：更易怒、沉默、哭、笑场、迁怒、过度防御。
- 身体信号：眼睛发涩、头痛、手抖、脚步虚、胃里发空、动作变慢。
- 风险偏好改变：太疲惫时可能冲动，也可能过度保守。
- 社交能力下降：不想解释、说话短、误会别人、失去耐心。

审稿问题：这段疲劳是否改变了选择？如果没有，只是装饰词。

## 6. 作息节律与昼夜压力

昼夜不是背景色，而是行动成本。

### 6.1 夜晚

夜晚带来：视线差、困意、宵禁、安保变化、街道空、危险增加、秘密行动机会、睡眠被打断。

写法：夜戏要考虑“谁本该睡觉，谁为什么醒着，醒着的代价是什么”。

### 6.2 黎明

黎明适合写崩溃后的冷静、追逃后的第一口气、秘密行动的截止、城市重新运转。

写法：黎明不是自动治愈，常常是“必须面对白天后果”。

### 6.3 夜班/轮班/守夜

轮班制度本身能制造关系和信息差。

```text
谁第一班，谁最后一班？
谁偷偷没睡？
谁在交接时隐瞒了异常？
谁因为困倦误判了声音？
谁被安排守最危险的时段？
```

### 6.4 时差/长期作息错位

现代、科幻、异世界、多地图、跨时区剧情中，时差和作息错位能解释判断迟钝、情绪烦躁和沟通失败。

## 7. 休息章不等于水章

低强度休整场景要有状态增量。至少完成以下一种：

- 身体增量：伤痛减轻、体力恢复、药效过去、副作用显现。
- 信息增量：整理线索、交换情报、发现遗漏、做出计划。
- 关系增量：照护、道歉、试探、同住边界、守夜分工。
- 资源增量：补给、修装备、买药、换衣、修阵、安排轮班。
- 情绪增量：从惊惧到稳定，从麻木到崩溃，从硬撑到承认需要帮助。
- 风险增量：敌人追近、住处暴露、休息被打断、睡眠不足导致失误。

如果休息章删掉后只剩“他们休息了一夜”也能接上，就应压缩。

## 8. 熬夜、通宵与过劳代价

熬夜可以写，但不能免费。

可用代价：

- 第二天判断变差，错过细节。
- 情绪失控，关系误伤。
- 战斗/考试/谈判/审判反应变慢。
- 身体旧伤复发。
- 灵力/异能/精神力恢复不足。
- 需要队友接管，形成信任或羞耻。
- 对手利用主角疲劳设局。

都市、娱乐圈、职场、医生、警察、程序员、主播、作家、学生等题材中，熬夜和过劳尤其要谨慎处理，不能把长期透支包装成单纯励志。

## 9. 修仙/玄幻中的休息与恢复

玄幻仙侠可以用灵力、丹药、打坐、闭关、阵法恢复，但仍要有规则。

```text
恢复方式：睡眠、打坐、灵泉、丹药、阵法、功法、契约、系统。
恢复对象：体力、灵力、神识、伤口、精神污染、寿元、法器冷却。
恢复速度：多久恢复多少？是否受伤势、环境、境界影响？
恢复代价：灵石、丹药、时间、灵脉、记忆、副作用、暴露风险。
恢复限制：不能连续使用、需要安全环境、会被打断、只恢复灵力不恢复心理。
后文后果：下一战可用几成力，旧伤是否留下，敌人是否知道恢复方式。
```

风险：打坐一晚全恢复会让连战、伤病、秘境、追逃和资源经济全部失效。

## 10. 梦、噩梦与睡眠质量

普通梦和噩梦不等同于第 133 篇的幻境/精神空间，但也需要功能。

可用功能：

- 创伤回响：不是解释创伤，而是显示角色还没真正安全。
- 线索重排：梦里出现白天忽略的细节，但醒后仍需验证。
- 关系触发：被噩梦惊醒后谁在，谁不在，谁被拒绝。
- 身体状态：发烧、疼痛、药效、精神污染影响梦境。
- 预兆/预知：若梦具有超自然信息，必须联动第 133、135 篇并设规则。

风险：不要用噩梦反复卖惨；不要把梦当万能剧透；不要让梦醒后前文失去价值。

## 11. 题材用法

### 11.1 动作/战斗/追逃

重点：体力、守夜、连战、疼痛、反应速度、撤离后崩溃。

规则：逃亡队伍速度取决于最累的人。追逃必须有轮休、补给、误判和崩点，否则逃方像无限体力。

### 11.2 悬疑/刑侦/调查

重点：熬夜查案、信息过载、误判、咖啡因依赖、证据整理。

规则：疲劳可以制造错误线索和人际摩擦，但关键推理不能只靠睡醒灵光一现。

### 11.3 医疗/照护/病弱角色

重点：照护者疲劳、病人睡眠质量、康复期、长期陪护。

规则：医疗细节归第 131 篇；本篇只管理睡眠、陪护、恢复差额和照护关系。不要提供现实诊疗建议。

### 11.4 校园/职场/都市

重点：早八、夜班、通勤、考试、项目赶工、直播熬夜、写作爆更。

规则：现实过劳、睡眠不足、职业安全和未成年人健康要谨慎，不把透支身体当无代价燃点。

### 11.5 修仙/升级/训练

重点：闭关、打坐、神识疲劳、丹药恢复、突破后虚弱、修炼窗口。

规则：修炼不等于永远不用睡。即使设定里睡眠需求减少，也要说明用什么机制替代，以及替代机制的限制。

## 12. 常见失败模式

- 永动机主角：连战连夜后仍满状态。
- 一觉清零：伤病、创伤、关系裂痕、精神消耗睡一觉全没。
- 疲劳只当形容词：写了“很累”，但选择、动作、判断不变。
- 休整水章：吃饭睡觉聊天，没有身体、关系、信息、资源或风险增量。
- 夜戏无困意：所有人深夜随叫随到，没人问为什么醒着。
- 守夜无制度：队伍长期野外行动却没有轮班、交接和疲劳代价。
- 丹药万能恢复：恢复资源无限，没有副作用、冷却、成本和限制。
- 噩梦卖惨：反复写噩梦，但不改变角色、关系或行动。
- 照护者不累：长期照护病人或伤员的人没有睡眠不足和情绪压力。
- 过劳浪漫化：把持续透支包装成单纯努力、爱情或忠诚证明。

## 13. 审稿检查问题

AI 审稿或修订时，逐条检查：

- 角色上次有效睡眠是什么时候？是否符合当前行动强度？
- 疲劳有没有改变反应、判断、情绪、身体动作或社交方式？
- 休整场景是否产生至少一种状态增量？
- 睡眠/休息后，是否有明确状态差额，而不是全清零？
- 守夜、夜班、闭关、连战、逃亡是否有轮班、补给和恢复安排？
- 伤病、药效、灵力、精神力、旧伤和心理压力是否与休息状态一致？
- 是否把过劳、长期失眠、照护疲劳或未成年人熬夜写成无代价励志？
- 噩梦、梦境、睡醒灵感是否有验证和后果？
- 如果休息章删除后只需一句“他们睡了一觉”即可接上，是否应压缩？
- 涉及真实睡眠障碍、精神健康、医疗、职业安全、未成年人健康时，是否需要查专业资料并避免建议化？

## 14. AI 写作提示词片段

### 14.1 建立身体资源状态

```text
请为当前章节建立身体资源状态表。
要求：列出主要角色上次有效睡眠、当前疲劳来源、体力/精神/伤病/饥饿/药效/灵力状态、可见疲劳表现、下一步行动限制和需要回写台账的身体状态。不要写医疗建议，只写叙事连续性。
```

### 14.2 修订永动机主角

```text
请检查这段剧情是否把主角写成永动机。
如果角色连续熬夜、赶路、战斗、调查、照护或修炼，请补上疲劳如何改变判断、动作、情绪或关系；如果不需要展开，请用 1-3 句压缩说明休息和恢复差额。
```

### 14.3 写休整场景

```text
请写一段低强度休整场景。
要求：不能只是睡觉吃饭聊天；至少包含一个身体恢复差额、一个关系动作、一个信息整理或资源补给，并用结尾接到下一步明确行动。
```

### 14.4 写守夜轮班

```text
请设计一段野外/基地/病房/宗门任务中的守夜轮班。
要求：写清轮班顺序、交接信息、谁没睡够、谁隐瞒异常、疲劳导致的误判或关系变化，以及第二天状态差额。
```

## 16. 安全与真实性边界

- 本专题只用于小说创作，不提供医学、心理健康、睡眠治疗、用药、职业安全或未成年人健康建议。
- 涉及失眠、睡眠障碍、梦魇、创伤、精神健康、药物、昏迷、医疗恢复、过劳风险时，必须联动第 20、105、131 篇，并避免诊疗化、教程化和浪漫化。
- 涉及现实职业安全、夜班、疲劳驾驶、医疗值班、未成年人熬夜、极限训练、连续工作等内容时，必须查最新官方或专业资料。
- 幻想世界可以设定打坐、丹药、灵力循环、系统休眠替代睡眠，但必须写清限制、代价和台账状态。

## 17. 来源与吸收方式

- CDC Sleep and Sleep Disorders 资料：吸收睡眠不足会影响健康、清醒度和日常功能的基础判断，用于避免角色长期缺觉却无代价。
- NIH / NINDS “Brain Basics: Understanding Sleep”：吸收睡眠与大脑、记忆、身体功能和昼夜节律相关的基础视角，用于“睡眠不是黑屏”的叙事规则。
- NHLBI “Sleep Deprivation and Deficiency”：吸收睡眠不足会影响注意力、反应、情绪、判断和安全的资料，用于熬夜、夜班、守夜和连战后的状态差额。
- NIGMS / NIH Circadian Rhythms 资料：吸收生物钟、光暗周期和昼夜节律影响睡眠与身体功能的基础视角，用于夜戏、倒时差、闭关和异世界昼夜设定。
- NIOSH / CDC 关于疲劳和工作安全资料：吸收疲劳会影响警觉、反应和安全的资料，用于职业、战斗、追逃、夜班和照护桥段。
- Reedsy、Writer's Digest、Writing Excuses 等写作节奏资料：吸收高压后的反应/余波、场景节奏和读者呼吸空间必须继续服务剧情的原则，用于防止休息章水化。
- 本知识库第 57、62、71、85、89、118、121、131、133、143 篇：整合时间压缩、低强度防水、身体感官、压力波形、内心过滤、动作消耗、追逃疲劳、伤病恢复、梦境规则和昼夜环境，形成“睡眠 -> 疲劳 -> 恢复 -> 状态差额 -> 台账回写”的执行链。

## 18. 最小执行规则

已确认：AI agent 写任何疲劳或休息相关内容，最少必须检查“三件事”：

1. 角色上次有效睡眠和当前疲劳来源。
2. 疲劳是否改变判断、动作、情绪或关系。
3. 休息后是否留下身体、信息、资源、关系或风险差额，并回写台账。

如果三件事都没有，疲劳就是形容词，休息就是水章，应压缩、删除或重构。
