# 159. 残障、慢性病、神经差异、辅助器具与无障碍环境

适用对象：AI agents、网文作者、审稿 agent、人物塑造 agent、敏感内容审稿 agent、医疗/康复题材 agent、校园/职场/家庭线 agent、玄幻架空设定 agent、连续性维护 agent

适用任务：设计、起草、续写、审稿或修订残障角色、慢性病角色、病弱但长期生活的角色、神经差异角色、视障/听障/行动障碍/言语障碍/认知障碍角色、轮椅/拐杖/义肢/助听器/白杖/屏幕阅读器/字幕/手语/服务动物等辅助系统、无障碍环境、公共服务、学校/职场便利、照护关系、残障歧视、信息可及性、超凡治愈后遗症、架空世界的无障碍设计和任何“身体/感知/认知差异与环境互动”的情节。

核心问题：AI 写残障、慢性病和神经差异时常在三个极端之间摆动：把角色写成可怜对象，把残障写成励志装置，把治愈当作唯一圆满；或者反过来，把残障写成超能力代价、反派标签、喜剧笑点、恋爱救赎和病弱美学。商业级长篇需要的是：人物仍然是完整的人，障碍也真实影响日常、选择、关系、资源和环境；故事冲突不只来自身体本身，也来自不友好的空间、制度、沟通和他人态度。

一句话原则：残障不是人物缺陷或剧情惩罚，而是身体/感知/认知状态与环境、制度、资源、他人态度共同作用后形成的生活条件；写作重点不是“让角色恢复正常”，而是让角色在真实限制、辅助系统和社会关系中拥有目标、选择、尊严和后果。

重要边界：本篇用于小说创作和审稿，不提供医疗、康复、法律或心理健康建议。涉及现实疾病、残障评定、药物、治疗、康复训练、心理危机、无障碍法律、服务动物、教育/就业权益时，必须查最新官方资料，并按需要联动第 20、105、111、131 篇。

## 2. 残障与无障碍十线卡

AI agent 写任何残障、慢性病、神经差异或辅助器具情节前，先填这张卡。

```text
角色身份：主角 / 配角 / 反派 / 家人 / 同事 / 同学 / 导师 / 照护者 / 其他
差异类型：行动 / 视力 / 听力 / 言语 / 认知 / 神经差异 / 慢性疼痛 / 慢性疲劳 / 精神健康 / 内脏/免疫/代谢 / 多重障碍 / 架空设定
状态来源：先天 / 后天 / 战伤 / 疾病 / 年龄 / 超凡代价 / 诅咒 / 未公开 / 不重要
角色自我关系：自豪 / 接纳 / 适应中 / 愤怒 / 隐瞒 / 借此争取权益 / 不愿被定义 / 阶段变化
当前目标：学习 / 工作 / 战斗 / 恋爱 / 逃亡 / 照护别人 / 证明能力 / 休息 / 求助 / 维权 / 隐藏身份
身体/感知/认知限制：疼痛、疲劳、行动范围、感官过载、沟通方式、记忆/执行功能、发作/波动、风险触发
辅助系统：轮椅、拐杖、义肢、支具、助听器、白杖、字幕、手语、读屏、药物、日程软件、服务动物、照护者、法术/科技辅助
环境障碍：台阶、窄门、噪音、强光、字幕缺失、复杂表格、无电梯、交通、卫生间、排队、信息不透明、歧视态度
社会位置：家庭、学校、职场、宗门、军队、贵族、底层、公共服务、网络舆论、法律/制度支持
披露与隐私：谁知道、谁不知道、谁误会、谁越界询问、谁需要知道才能提供支持
本场变化：行动资格、关系边界、资源消耗、身份公开、权益获得/受损、辅助器具状态、台账回写
```

最小规则：只要写残障或辅助器具，就必须回答“角色想做什么、环境怎么阻拦或支持、辅助系统如何改变行动资格、角色是否保有选择权”。

## 3. 不只写“身体有问题”，要写“环境如何制造或减轻障碍”

CDC 和 WHO 资料都强调，残障者参与社会生活时遇到的障碍包括物理、沟通、态度和制度等多层障碍。迁移到小说中，残障不是只在角色身体里，也在世界如何回应角色。

### 3.1 七类障碍

| 障碍类型 | 小说表现 | 可用冲突 | 错误写法 |
|---|---|---|---|
| 物理障碍 | 台阶、窄门、无电梯、碎石路、船舱梯、马车高度 | 进不去、绕路、求助、迟到、暴露身份 | 只写“他坐轮椅”，环境完全配合 |
| 沟通障碍 | 无字幕、无手语、口罩遮挡、嘈杂、复杂术语、读写困难 | 信息迟到、误解、被排除 | 全员自动懂、自动翻译 |
| 感官障碍 | 强光、噪音、人群、气味、触碰、视觉混乱 | 过载、撤离、寻找安静空间 | 把过载写成矫情 |
| 态度障碍 | 怜悯、嘲笑、过度保护、怀疑能力、强行帮忙 | 尊严受损、冲突、边界谈判 | “好心人”越界无后果 |
| 制度障碍 | 请假、考试、招聘、门禁、排队、医保、残障证明、宗门考核 | 争取便利、权利、例外条件 | 制度随主角光环消失 |
| 经济障碍 | 辅助器具昂贵、维护、交通、照护、药物、康复费用 | 资源选择、债务、家庭压力 | 工具无限供应无成本 |
| 时间/能量障碍 | 慢性疲劳、疼痛、恢复期、药效、预约、休息窗口 | 放弃活动、重排任务、求助 | 永远有体力硬撑 |

### 3.2 环境也可以成为爽点

有效爽点不是“残障被治好”，而可以是：

- 角色提前研究路线，反过来利用无障碍通道突围。
- 团队终于按他的沟通方式行动，任务效率提升。
- 反派低估轮椅/义肢/助听器/读屏/记忆辅助的功能，被反制。
- 主角不再用硬撑证明自己，而是成功调动资源。
- 一个公共空间被改造后，配角第一次能平等参与。

这种爽点比“奇迹站起来”更有尊重，也更有长篇后劲。

## 4. 人物不是残障标签

### 4.1 人物三层目标

残障角色和任何角色一样，需要三层目标。

```text
外部目标：这章要完成什么？考试、谈判、逃脱、告白、查案、打工、修炼、照护别人。
关系目标：想被谁尊重、相信、放过、理解、不要过度保护？
自我目标：想证明、拒绝证明、保护隐私、接受帮助、争取便利、改变规则？
```

如果角色的唯一功能是“提醒主角善良”或“让主角被治愈”，就是工具化。

### 4.2 不要只有两种情绪

残障角色不只会自卑或坚强。可写的情绪包括：

- 对环境障碍的烦躁。
- 对越界帮助的厌烦。
- 对自己熟练适应的自豪。
- 对他人低估的冷静利用。
- 对反复解释需求的疲惫。
- 对真正被理解的轻松。
- 对失去旧能力的哀悼。
- 对新身份、新社群、新工具的亲近。
- 对未来的普通焦虑。

### 4.3 “治愈”不是唯一结局

在玄幻、科幻、神医、系统文中，治愈能力很常见，但必须问：

- 角色是否自愿？
- 治愈是否会抹去身份、技能、关系和适应经验？
- 治愈有无代价、限制、失败风险和后续恢复？
- 世界是否把“不接受治愈”的人视为有问题？
- 如果治愈是终局奖励，是否暗示残障人生不完整？

更好的选择：部分改善、辅助升级、环境改造、权利获得、团队适配、角色自主选择，而不是统一“恢复正常”。

## 5. 辅助器具不是装饰，是行动系统

WHO 资料把辅助技术视为帮助人在认知、沟通、听力、行动、自我照护和视觉等方面维持或改善功能的产品、系统和服务。迁移到小说中，辅助器具不是外观点缀，而是行动资格。

### 5.1 辅助器具六件事

```text
器具类型：轮椅 / 拐杖 / 白杖 / 助听器 / 人工耳蜗 / 义肢 / 支具 / 读屏 / 字幕 / 手语 / 记忆辅助 / 服务动物 / 魔法器具 / 机甲接口
获得方式：购买、配发、定制、继承、战利品、宗门炼制、维修替换、非法改装
适配程度：合身、磨合中、旧型号、不合适、临时借用、过度消耗
维护成本：电量、零件、清洁、调试、训练、药物、保养、修理、费用
社会反应：被尊重、被盘问、被抢夺、被当武器、被当弱点、被当装饰
剧情功能：进入空间、沟通、战斗、工作、学习、隐蔽、暴露、回忆、身份、反制
```

### 5.2 辅助器具常见写法

| 器具 | 有效功能 | 风险 |
|---|---|---|
| 轮椅 | 移动、速度、路线选择、空间判断、疲劳管理 | 当成悲惨符号或随便抬人 |
| 拐杖/支具 | 平衡、疼痛管理、节奏、被迫慢下来 | 需要时消失 |
| 白杖 | 探路、空间信息、身份信号、行人互动 | 当魔法雷达或纯装饰 |
| 助听器/人工耳蜗 | 信息获取、噪音压力、维护、电量、环境适配 | 戴上就听力完全普通化 |
| 字幕/读屏 | 信息可及、工作学习、网络生活 | 忘记数字无障碍 |
| 义肢 | 行动恢复、适配、磨损、疼痛、身份感 | 一装上就无限强 |
| 服务动物 | 任务辅助、独立出行、安全提醒、社会误解 | 当普通宠物或万能保镖 |
| 记忆/日程辅助 | 执行功能、药物、约会、任务拆解 | 写成“懒”或“笨” |
| 架空法器/科技 | 替代现实器具、制造题材爽点 | 没有成本和维护 |

### 5.3 器具损坏要有后果

辅助器具损坏不是普通装备坏了。它可能让角色无法出门、无法沟通、错过考试/任务、疼痛加重、暴露隐私或被迫依赖不信任的人。

写法：

- 损坏前预置用途。
- 损坏时写角色如何快速评估替代方案。
- 后果进入台账：维修、费用、信任、出行计划、心理压力。
- 不把抢走轮椅、助听器、药物、白杖当轻松笑点。

## 6. 慢性病和波动性状态

慢性病、慢性疼痛、慢性疲劳和许多神经/精神状态不是一场伤病，而是长期资源管理。

### 6.1 能量配额

```text
今天可用能量：高 / 中 / 低 / 崩溃边缘
已消耗：通勤、上课、战斗、社交、噪音、疼痛、情绪冲突、治疗、失眠
必须保留：回家、吃饭、用药、照护别人、任务关键动作
可放弃：聚会、争辩、训练、无效社交、非必要证明
透支后果：疼痛、发作、睡眠债、情绪失控、任务失败、关系误会
```

### 6.2 波动性要写进节奏

同一个角色今天能做到的事，明天不一定能做到。这不是设定错误，而是许多长期状态的真实特征。小说中要用台账维护：

- 触发因素：天气、睡眠、压力、食物、药物、噪音、战斗、经期、感染、人群。
- 缓解条件：安静、休息、药物、热敷、辅助器具、可信照护者、退出空间。
- 代价选择：硬撑去任务，还是保留能量等关键时刻。

风险：如果波动只在作者需要虐时出现，读者会觉得操控；如果波动从不影响行动，就只是标签。

## 7. 神经差异与认知差异

ASAN 关于神经多样性的资料强调，脑相关差异的人应被社会接纳和纳入，而不是只被视为需要修正。迁移到小说中：自闭症、ADHD、学习障碍、认知差异、精神健康状态等不能被统一写成“天才怪人”“社恐萌点”“麻烦制造者”或“反派不共情”。

### 7.1 可写维度

- 感官：光、声音、触感、气味、人群、服饰标签、食物口感。
- 沟通：直白、延迟回应、需要文字、需要明确规则、不读暗示。
- 执行功能：开始任务、切换任务、时间感、整理步骤、记忆提醒。
- 注意力：高度专注、易分心、兴趣驱动、环境干扰。
- 社交能量：聚会后耗竭、需要退出空间、偏好小范围关系。
- 规则需求：流程、预告、可预测性、例外处理。
- 遮蔽/伪装：为了适应环境长期压抑反应，事后崩溃或疲惫。

### 7.2 不要把神经差异写成单一能力包

错误：他是自闭症，所以不会撒谎、没有感情、数学天才。她有 ADHD，所以永远闯祸但很可爱。

改法：给具体人物具体组合：注意力、感官、社交、执行功能、兴趣、表达方式、家庭支持、环境便利、压力反应各不相同。

### 7.3 便利不是作弊

学校、职场、宗门考试、团队任务中的合理便利可以包括：

- 明确流程和截止时间。
- 文字说明替代口头临时通知。
- 安静空间。
- 字幕或转录。
- 任务拆分。
- 休息窗口。
- 辅助软件。
- 更清楚的反馈标准。

小说冲突可以来自别人误以为这是“特殊待遇”，但叙事立场不能默认歧视者正确。

## 8. 无障碍场景怎么写

### 8.1 场景检查

```text
入口：能否进入？门宽、台阶、电梯、坡道、门槛、交通、安检。
路线：能否移动？地面、拥挤、照明、噪音、标识、座位、逃生路径。
信息：能否知道？字幕、手语、读屏、清晰文字、颜色对比、公告方式。
互动：能否发言？麦克风、文字渠道、翻译、等待时间、社交规则。
卫生与休息：能否上厕所、洗漱、换药、坐下、避开刺激、充电。
安全：紧急时谁通知、怎么撤离、辅助器具和服务动物是否被保护。
后果：障碍造成迟到、误会、疼痛、暴露、费用、关系冲突还是制度改变？
```

### 8.2 场景示例

- 酒楼二楼没有电梯：不是“他被抬上去”一句解决，而是提前订位、换包间、被店家推诿、朋友是否尊重他的选择。
- 宗门试炼只按跑山速度排名：行动障碍弟子被制度排除，主角可以挑战规则或选择不同证明方式。
- 宫宴没有手语/字幕/近侍解释：听障角色被排除于信息战之外，或反过来利用他人以为她听不见。
- 星际战舰控制台没有触觉反馈：视障军官必须依靠定制界面，战斗中系统损坏就是真实危机。
- 校园公告只贴在高处小字栏：视力障碍学生错过信息，后果不该被写成“你怎么不自己看”。

## 9. 照护关系与边界

照护可以是爱，也可以是控制。关键是角色是否保有自主权、隐私和退出空间。

### 9.1 好照护的行动

- 先问需求，而不是直接上手。
- 记住对方偏好和禁忌。
- 在公开场合保护隐私。
- 不把照护当恩情账本。
- 支持对方做决定，即使决定不完美。
- 承认自己也会疲惫，建立可持续分工。
- 协助争取环境改变，而不是要求对方永远适应。

### 9.2 坏照护的风险

- 以爱之名控制出行、社交、工作和恋爱。
- 代替角色说话。
- 抢夺辅助器具或药物。
- 公开病史和隐私。
- 把照护辛苦变成要求服从的理由。
- 把对方的依赖写成恋爱甜点。

### 9.3 恋爱线边界

残障角色可以恋爱、欲望、拒绝、主动追求和被追求。但亲密关系不能只建立在“病人/照护者”结构上。

检查：

- 两人是否有照护以外的共同目标和吸引？
- 残障角色是否能拒绝、发脾气、选择、离开？
- 伴侣是否尊重辅助器具、休息、疼痛、感官边界和隐私？
- 爱情是否被写成治愈替代品？

## 10. 题材用法

### 10.1 都市/职场

重点：通勤、电梯、办公室动线、会议字幕、招聘歧视、请假、远程办公、无障碍厕所、同事态度。

规则：职场冲突不只来自能力，也来自制度是否允许角色平等发挥能力。不要把“不给便利”写成磨炼。

### 10.2 校园/青春

重点：座位、考试便利、体育课、同伴玩笑、宿舍、校园坡道、老师沟通、家长态度。

规则：学生不是被保护起来就完事。要写学习权、同伴关系、隐私、参与感和自主选择。

### 10.3 古言/宫廷/宅斗

重点：礼法、出行、婚姻价值、继承、下人照护、药材、无障碍缺失、家族态度。

规则：不要用“残疾所以不配”当世界观默认价值而不审视。即使时代残酷，也要写出人物主体和制度压迫。

### 10.4 玄幻/仙侠/武侠

重点：断肢重生、经脉损伤、灵根缺陷、义肢法器、轮椅机关、神识感知、宗门考核。

规则：超凡治愈不能一键抹除残障经验；辅助法器也要有灵力消耗、维护、适配和社会反应。

### 10.5 末世/战争/逃亡

重点：药物、辅助器具损坏、路况、撤离、感染风险、队友分工、照护资源。

规则：不要用“拖后腿”标签处理残障角色。真正的紧张来自资源分配、路线设计、团队信任和环境风险。

### 10.6 悬疑/推理

重点：证词可及、感官证据差异、监控死角、读屏记录、助听器电量、行动路线。

规则：残障不是“制造不可靠证人”的廉价理由。感官差异可以提供独特线索，也会有可验证边界。

### 10.7 喜剧/轻松日常

重点：误会、生活技巧、无障碍吐槽、朋友笨拙学习。

规则：笑点可以来自环境荒谬、制度僵硬、朋友好心办坏事，但不能来自羞辱身体、发作、沟通方式或辅助器具。

### 10.8 科幻/赛博/机甲

重点：脑机接口、义体、辅助 AI、可访问界面、增强与监控、数据隐私。

规则：科技不等于自动平等。谁能买得起、谁维护、谁拥有数据、谁能远程关停，都是冲突来源。

## 11. 常见失败模式与修法

| 失败模式 | 读者体感 | 修法 |
|---|---|---|
| 残障只为卖惨 | 工具化、冒犯 | 给角色目标、关系、能力、喜好、缺点和长期台账 |
| 奇迹治愈当大圆满 | 暗示残障人生不完整 | 改成自主选择、环境改善、辅助升级或部分恢复有代价 |
| 励志模板 | 像宣传稿 | 写具体障碍、疲惫、技巧、失败和普通欲望 |
| 反派残障化 | 把身体差异当道德缺陷 | 分离身体状态和道德选择，避免用畸形/疯癫暗示邪恶 |
| 辅助器具装饰化 | 假 | 写适配、维护、成本、损坏后果和行动功能 |
| 无障碍消失 | 世界像自动为主角让路 | 写台阶、信息、交通、厕所、噪音、制度和他人态度 |
| 慢性病只在虐点出现 | 作者操控感强 | 建能量配额、触发因素、缓解方式和波动台账 |
| 神经差异天才化 | 刻板 | 拆成感官、沟通、执行功能、社交能量和支持系统 |
| 越界帮助被赞美 | 不舒服 | 让角色表达边界，让帮助者学习先问需求 |
| 照护恋爱化 | 权力不平等 | 保留拒绝权、隐私、外部支持和照护以外的吸引 |
| 拿残障开玩笑 | 强毒点 | 改成环境/制度/误会笑点，不笑身体和辅助器具 |
| 现实政策胡写 | 出戏和风险 | 查最新官方资料，只把事实转成叙事约束 |

## 12. 审稿检查问题

### 12.1 人物主体性

1. 角色是否有残障以外的目标、关系、技能、缺点和欲望？
2. 残障是否只用于让主角善良、让读者同情或制造反派恐怖？
3. 角色是否保有选择权、拒绝权、隐私和行动策略？
4. 是否把治愈、站起来、变“正常”当唯一圆满？
5. 是否有残障社群、同伴经验或自我认同，而不是只有孤立痛苦？

### 12.2 环境与制度

1. 场景是否写出物理、沟通、感官、态度、制度、经济或能量障碍？
2. 无障碍设施、信息格式、交通、卫生间、紧急撤离是否影响行动？
3. 学校、职场、宗门、军队、宫廷或公共服务是否有便利/排斥机制？
4. 环境改善是否成为实际剧情结果，而不是口号？
5. 他人的帮助是否先问需求，并尊重角色边界？

### 12.3 辅助系统

1. 辅助器具是否有来源、适配、维护、成本和损坏后果？
2. 辅助系统是否改变行动资格，而不是只做外观？
3. 角色是否会熟练使用自己的工具，而不是永远等别人帮助？
4. 服务动物、手语、助听器、读屏、义肢等现实内容是否查证？
5. 架空辅助法器/科技是否有资源消耗和失效边界？

### 12.4 敏感边界

1. 是否把残障、慢性病、神经差异、精神状态当成笑点或道德缺陷？
2. 是否出现强制治疗、强制披露、抢夺辅助器具、羞辱、虐待或控制？
3. 是否涉及自伤、心理危机、医疗风险、药物、康复训练或法律权益？
4. 如涉及，是否联动第 20、105、111、131 篇？
5. 是否避免现实医疗建议、剂量、操作步骤和错误法律结论？

## 13. 台账字段

```text
角色：
差异/障碍类型：行动 / 视力 / 听力 / 言语 / 认知 / 神经差异 / 慢性病 / 精神健康 / 多重障碍 / 架空
状态来源与公开范围：
角色自我关系：接纳、隐瞒、适应中、维权、阶段变化
当前功能影响：移动、沟通、感官、执行功能、疼痛、疲劳、睡眠、社交、工作/学习
辅助系统：器具、药物、软件、服务动物、照护者、法器/科技、环境改造
维护成本：费用、电量、维修、清洁、训练、预约、耗材、替代方案
环境障碍：空间、信息、交通、制度、态度、经济、能量
合理便利/支持：
禁写边界：不嘲笑、不强制治愈、不公开隐私、不教程化医疗、不抢夺器具当笑点
本次场景变化：
下次回收点：
```

最小回写：辅助器具损坏/升级、状态恶化/缓解、身份公开、便利获得/被拒、环境改造、照护关系变化、长期限制变化都必须写入台账。

## 14. 可直接给 AI agent 的提示词

### 14.1 残障角色设计提示词

```text
请设计一个残障/慢性病/神经差异角色。不要把残障写成卖惨、励志、反派标签或等待治愈的缺陷。先填写残障与无障碍十线卡：差异类型、角色自我关系、当前目标、功能影响、辅助系统、环境障碍、社会位置、披露隐私、本场变化。角色必须有残障以外的目标、关系、技能、缺点和欲望；场景冲突至少包含一项环境或制度障碍，并让角色通过选择、策略或支持系统推进剧情。
```

### 14.2 无障碍场景改写提示词

```text
请把这段场景改得更有无障碍真实感。不要只写“他坐着轮椅/她听不见/他很病弱”。请检查入口、路线、信息、互动、卫生与休息、安全和后果，补出至少三项障碍或支持：台阶/电梯、字幕/手语/读屏、噪音/强光、交通、座位、卫生间、排队、他人态度、制度规则。结尾要让环境改变人物的行动、关系、成本或下一步计划。
```

### 14.3 残障敏感审稿提示词

```text
请审稿这段残障、慢性病或神经差异描写，重点检查：是否工具化、卖惨、励志模板、奇迹治愈唯一圆满、反派残障化、拿身体差异开玩笑、辅助器具装饰化、照护关系越界、现实医疗/法律胡写。请按“问题 -> 读者体感 -> 结构修法 -> 需查证来源 -> 台账回写”输出，不要只润色句子。
```

## 15. 来源与吸收方式

- WHO《Global report on health equity for persons with disabilities》与 WHO disability / assistive technology 资料：吸收点是残障与健康公平、参与机会、辅助技术和环境支持密切相关；迁移为“残障不是单一医学标签，而是功能、环境、服务和社会参与共同构成的生活条件”。
- CDC Disability Inclusion 资料：吸收点是残障包容需要识别和移除物理、沟通、态度等障碍；迁移为“小说场景要写环境如何制造或减轻障碍”。
- UN《Convention on the Rights of Persons with Disabilities》：吸收点是残障者享有平等权利、无障碍、独立生活、个人行动能力、教育、就业和社会参与等权利；迁移为“残障角色不是慈善对象，而是权利主体”。
- ADA.gov 关于有效沟通、服务动物和无障碍主题资料：吸收点是沟通辅助、服务动物、公共服务和合理便利需要尊重个人选择与功能需求；迁移为“辅助系统不是装饰，公开空间要处理沟通与进入权”。
- 中国《中华人民共和国残疾人保障法》与《中华人民共和国无障碍环境建设法》：吸收点是残疾人在政治、经济、文化、社会和家庭生活中享有平等权利，无障碍环境建设保障残疾人、老年人平等、充分、便捷地参与社会生活；迁移为“现代中国题材必须把权利、设施、信息无障碍和社会参与写进制度背景”。
- 中国政府网/中国残联关于辅助技术和辅助器具定义资料：吸收点是辅助技术包括装置、服务、策略和训练；迁移为“辅助器具不是单件装备，而是获得、适配、训练和维护系统”。
- ASAN 关于神经多样性和自我倡导的资料：吸收点是脑相关差异者应被接纳和纳入，信息应有 plain language / Easy to Read 等可及形式；迁移为“神经差异不是天才/怪人模板，而是沟通、感官、执行功能、环境和自我倡导的组合”。
- BookTrust / Writer's Digest 等关于残障角色写作资料：吸收点是残障角色应多维、真实、有自己的关系和故事功能，不应只作为文学装置；迁移为“角色先是人，残障是生活条件之一”。

## 16. 最小执行规则

如果时间不够，AI agent 至少执行以下 8 条：

1. 残障角色必须有残障以外的目标、关系、技能、缺点和欲望。
2. 场景至少写出一项环境、沟通、制度、态度、经济或能量障碍。
3. 辅助器具必须有功能、适配、维护、成本和损坏后果。
4. 慢性病和神经差异要写波动、触发、缓解和能量管理，不只写标签。
5. 不把治愈当唯一圆满，不把残障当道德缺陷、笑点、反派标记或恋爱救赎工具。
6. 帮助和照护必须尊重角色选择、隐私和拒绝权。
7. 现实医疗、康复、法律、无障碍和服务动物内容必须查最新官方资料。
8. 重大状态变化、辅助系统变化、身份公开、便利获得/被拒和环境改造都要回写台账。
