# 人物与对话

人物塑造的目标不是“资料卡很完整”，而是读者在场景中能分辨：这个人想要什么，怕什么，在压力下会怎么说、怎么动、怎么选择。

## 1. 重要人物必须有可见差异

不要只用姓名、职业、身份标签区分人物。重要人物至少要在以下维度有差异：

- 欲望：他主动追求什么。
- 压力：什么会让他急、撒谎、妥协、攻击或逃避。
- 羞耻或恐惧：他不想让别人看见什么。
- 利益优先级：他第一眼关注钱、地位、安全、证据、感情、职责、速度、面子、控制、公平还是生存。
- 说话习惯：长句还是短句，客气还是直接，爱不爱讽刺，是否回避重点，是否喜欢威胁或找借口。
- 行动习惯：沉默、抢话、查数据、安抚、甩锅、讲笑话、讨价还价、离场。
- 压力反应：变尖锐、变温柔、变大声、变小声、变滑、变诚实、变冷酷。

写作时不要把这些都展示给读者。作者和 AI 需要知道它们，是为了在场景中选出两三个最有用的反应。

## 2. 人物卡应记录什么

推荐人物卡：

```markdown
## 人物名

- 身份/位置：
- 当前欲望：
- 当前压力：
- 恐惧或羞耻：
- 利益优先级：
- 说话习惯：
- 行动习惯：
- 与主角关系：
- 对主角的误解或真实看法：
- 弧线变化：
- 首次登场要让读者记住的动作或选择：
```

人物卡不要写成“善良、坚强、聪明”这类抽象词堆。要尽量写成可执行表现：

- 不写：他很谨慎。
- 改写：重要问题不会当场表态，会先问“谁还知道这件事”。
- 不写：她很有控制欲。
- 改写：别人还没说完，她已经把记录顺序改成对自己有利的版本。

## 3. 对话要有议程和遮掩

对话不是互相传递信息。每个说话的人最好都在做一件事：

- 争取某个结果。
- 隐藏某个事实。
- 保护某段关系。
- 试探对方知道多少。
- 转移责任。
- 抢先定义事件。
- 安抚场面。
- 逼对方表态。

对话自检：

- 这句话如果删掉，场景压力会不会下降？
- 这句话只是作者解释设定，还是人物为了自己的目的而说？
- 同一句信息能不能用更符合人物身份的方式说？
- 对方是否回答了问题背后的压力，而不是字面问题？

示例思路：

- 直白说明：“愿池记录显示你拿了她的灵根。”
- 带议程说法：“长老若说记录有误，就请当场写进问愿册，日后仙盟来查，也省得我们各执一词。”

后者不仅给信息，还逼对方承担公开记录的风险。

## 4. 群戏要用反应顺序区分人物

群戏最怕“所有人同时震惊、同时沉默、同时站队”。更有效的做法是安排反应顺序：

- 谁第一时间要证据。
- 谁先安抚受害者或装作安抚。
- 谁抢先定义事件。
- 谁威胁主角闭嘴。
- 谁算利益。
- 谁怕旧账翻出来。
- 谁沉默观察。
- 谁表面站队，实际留后路。

来自 `许愿池修仙` 的可迁移经验：

- 宗门维护小师妹不能写成集体降智。
- 普通弟子维护的是宗门天才红利。
- 大师兄维护的是秩序和自我清白。
- 长老维护的是愿池收益和代价转嫁制度。
- 师尊维护的是更高层的飞升棋局。

这类分层能让群体压迫更可信，也能让主角反击有更多角度。

## 5. 反派和工具人也要社会真实

短期对手可以功能化，但不能完全纸片化。

可用动机：

- 求生存。
- 求晋升。
- 怕承担责任。
- 维护脸面。
- 忠于组织。
- 嫉妒。
- 赚钱。
- 避险。
- 坚信某套秩序。

反派不必聪明，但要有自我解释。他可以错，但不要只为了让主角爽而蠢。

工具人风险：

- 男主只负责解释规则。
- 朋友只负责捧主角。
- 反派只负责说恶毒台词。
- 路人只负责震惊。
- 编辑、客户、长老、警察、老师只负责给流程盖章。

修正方法：

- 给工具人一个和主角目标不完全一致的小利益。
- 给他一个不完美反应：迟疑、误解、怕担责、先顾自己。
- 给他一个能改变局面的微动作：记录顺序、压住半页账、抢走证据、假装听不懂、把话题引向程序。

## 6. 主角必须持续做选择

主角可以被压迫、被动开局、暂时不知道真相，但不能长期被系统、男主、外挂、命运或旁白推着赢。

主角主动性可以表现为：

- 选择什么时候说出证据。
- 诱导对手亲口承认。
- 用规则逼对方公开记录。
- 放弃短期胜利换更大证据。
- 承担使用外挂的代价。
- 在两个都不完美的选择中做取舍。
- 故意让对手局部维稳成功，再抓更大破绽。

自检问题：

- 本章主角有没有一个可见选择？
- 这个选择失败会有什么后果？
- 如果删掉主角，本章是否仍会自动发生？如果会，主角太弱。

## 7. 人物自检清单

起草或修订后检查：

- 不看名字，读者能否凭台词和动作区分关键人物？
- 每个重要人物进入场景时是否带着自己的目的或压力？
- 是否有人只是在替作者交代信息？
- 群戏是否出现全员同步理解、同步沉默、同步震惊？
- 对手是否有能自洽的动机和自保动作？
- 主角是否仍在做决定，而不是被规则和解释推着走？
- 是否有一个小动作、眼神、停顿、记录动作或站位变化承担了人物塑造？
