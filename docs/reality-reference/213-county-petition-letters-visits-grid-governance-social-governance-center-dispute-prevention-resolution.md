# 213 县域信访、12345、综治中心、网格治理、矛盾纠纷预防化解与群众诉求分流链专项

状态：已确认  
更新日期：2026-06-09  
适用对象：负责县域现实题材、基层干部、乡村振兴、社区群像、征收补偿、物业邻里、欠薪、环保投诉、村务公开、热线工单、信访接待、网格员走访和基层矛盾化解剧情的 AI agents。

## 1. 本篇解决什么问题

AI 写“群众来找政府”“打 12345”“上访”“网格员发现问题”“综治中心调处”时，最常出现五种失真：

- 把 12345、信访、调解、行政复议、诉讼、纪检举报、报警混成一个万能“找领导”。
- 把群众诉求写成喊得越凶越有理，缺少事实、证据、主管部门和法定路径。
- 把基层干部写成只会压事、拖延或维稳的脸谱反派。
- 把综治中心、网格员写成万能侦探和万能协调员，什么部门都能替代。
- 把矛盾化解写成一次谈话就“皆大欢喜”，没有回访、履行、复发和长期台账。

本篇把基层诉求治理写成一条可执行链：

```text
风险信号/群众诉求出现
-> 第一入口接住：网格员、村居、12345、信访窗口、部门窗口、调解组织、公安或法院
-> 判断诉求性质：咨询、投诉、举报、信访、民事纠纷、行政争议、涉法涉诉、纪检监察、紧急警情
-> 明确主管部门和办理路径
-> 登记、分流、转办、交办、督办或告知法定途径
-> 核实事实、走访当事人、查看材料、组织调解或依法办理
-> 反馈结果、签收意见、救济告知、复查回访
-> 积案化解、源头治理、舆情回应和长篇台账回写
```

一句话：群众诉求戏的爽点不在“谁闹赢了”，而在主角把混成一团的委屈、利益、证据和程序拆开，让每一类问题回到能解决它的路径上。

## 3. 当前规则底座

### 3.1 信访工作条例

《信访工作条例》是写信访事项的重要底座。它强调畅通和规范群众诉求表达、利益协调、权益保障通道，同时也意味着信访不是替代行政复议、行政诉讼、仲裁、司法执行或纪检监察的万能捷径。

写作影响：

- 信访事项要有登记、受理、转送、办理、答复、复查复核或告知路径。
- 涉法涉诉事项要尊重司法程序，不能写成“找信访领导改判”。
- 信访人有表达诉求的通道，也有依法、理性、有序表达的边界。

### 3.2 12345 热线

2021 年国务院办公厅推动地方政务服务便民热线归并优化；2025 年国务院办公厅又发文进一步规范和提升 12345 热线服务。12345 是倾听群众和企业诉求、分办督办和提升治理效能的重要窗口，但不是所有事项的直接裁判者。

写作影响：

- 12345 接诉后要分类、分办、督办、反馈和评价。
- 对应通过诉讼、仲裁、纪检监察、行政复议、政府信息公开、信访等程序解决的事项，应告知相应法定途径。
- 12345 工单可以制造剧情压力：超期、重复投诉、部门推诿、回访不满意、数据暴露治理短板。

### 3.3 信访工作法治化

2024 年起，官方公开信息持续强调推进信访工作法治化，关键是把群众诉求放进依法分类、转送交办、督办问责、实质化解和秩序维护的流程中。

可转化为小说规则：

```text
预防在前
-> 受理规范
-> 办理有责
-> 监督追责
-> 维护秩序
-> 回访和源头治理
```

### 3.4 基层治理与网格服务

基层治理不是一个部门单独完成。村居、网格员、人民调解组织、司法所、派出所、社区卫生、民政、住建、市场监管、城管、人社、法院、检察、信访部门、12345 平台等，都可能在不同节点进入。

写作影响：

- 网格员负责发现、记录、转介、协助，不是警察、法官、心理医生和民政干部的集合体。
- 综治中心在不少地方承接矛盾纠纷一站式接收、多元调处、部门联动等功能，但具体设置和权限高度地方化，写作时不要全国一刀切。
- 多元化解的核心是分层递进：能调解的先调解，行政争议走复议诉讼，违法犯罪进公安司法，救助照护进民政和公共服务。

## 6. 12345、信访和综治中心怎么区分

### 6.1 12345：接诉、分办、督办、反馈

适合写：

- 停水停电、垃圾未清、路灯坏、窗口服务、占道扰民、政策咨询。
- 多部门推诿导致工单反复流转。
- 回访不满意让基层重新核查。
- 热线数据暴露某个小区、街区、行业的重复问题。

不适合写：

- 直接裁判合同赔偿。
- 替代法院执行。
- 处理正在诉讼或应通过特定法定程序解决的事项。

### 6.2 信访：表达诉求、依法办理、监督推动

适合写：

- 长期得不到答复的历史遗留问题。
- 群众对机关单位处理意见不服。
- 多次投诉后需要依法登记、转送、督办和答复。
- 积案化解中需要重新核查事实、解释政策、联动救助或纠错。

不适合写：

- 一句“我要上访”让所有法律程序失效。
- 用集体围堵、拉横幅、堵门作为正义爽点。
- 让信访干部直接改判、撤销处罚或分配补偿款。

### 6.3 综治中心：多元调处和部门协同

适合写：

- 邻里纠纷、物业矛盾、征地补偿外溢、欠薪群体、婚姻家庭冲突、校园周边矛盾。
- 多部门一站式接收、分流、会商和跟踪。
- 人民调解、行政调解、司法调解、法律咨询、心理疏导、救助转介之间的衔接。

不适合写：

- 综治中心替代所有主管部门。
- 一个主任拍板解决土地、社保、刑事、执行和医疗所有问题。
- 把复杂矛盾压成“签个承诺书就散”。

## 7. 网格员写法

网格员是基层叙事里的“第一触点”，但不是万能角色。

可以写的职责：

- 走访独居老人、困境儿童、残障家庭、出租屋、商户和重点区域。
- 发现安全隐患、家庭困难、邻里冲突、环境卫生问题并上报转介。
- 协助通知、登记、宣传政策、收集材料、跟进回访。
- 在矛盾升级前，把“小事”带到社区、村委会、司法所、派出所或主管部门。

不要写的越界：

- 独自调查刑事案件。
- 直接决定低保、处罚、赔偿、拆迁补偿。
- 随意公开居民隐私、家庭矛盾、健康和财务信息。
- 以“基层需要”为名强行进入住宅或扣押财物。

## 8. 常见剧情结构

### 8.1 热线工单推动小问题变成制度问题

```text
居民反复打12345投诉夜市噪声
-> 工单转街道和城管
-> 现场发现不仅是噪声，还有垃圾、停车和摊位外溢
-> 商户说夜市带动就业
-> 居民说孩子睡不着
-> 多部门会商调整时段、清运和摊位边界
-> 回访满意度仍分裂
-> 主角意识到治理不是消灭冲突，而是重新排序利益
```

### 8.2 信访积案不是一句道歉能解决

```text
老人多年反映征地补偿历史问题
-> 村里口头解释多次无果
-> 信访窗口登记旧材料
-> 乡镇、自然资源、村委会、司法所共同核查档案
-> 发现原始公示、会议记录和家庭分配都有缺口
-> 能纠错的纠错，不能重来的解释政策并补救救助
-> 老人未必完全满意，但终于知道问题卡在哪里
```

### 8.3 网格员发现风险，避免小事拖成大事

```text
网格员走访发现独居老人家中断水断药
-> 先联系家属和社区卫生服务
-> 转介民政临时救助和家庭医生
-> 发现背后是子女赡养矛盾
-> 司法所和村居组织调解
-> 后续回访确认药、饭、水电和联系机制
```

### 8.4 群体诉求要拆成多个问题

```text
业主群集体到街道反映开发商延期交房
-> 表面是集体维权
-> 实际包括合同违约、监管账户、房屋质量、子女入学、按揭压力、物业前期服务
-> 12345 和信访只能分流督办
-> 住建、法院、金融、教育、物业管理分别处理不同部分
-> 主角把“所有人一起喊”拆成可推进的清单
```

## 9. 人物写法

| 角色 | 表层目标 | 深层压力 | 可写复杂性 |
| --- | --- | --- | --- |
| 信访人 | 要说法、要答复、要补偿 | 长期被忽视、证据缺失、家庭压力、旧账缠身 | 诉求可能合理，表达方式可能失控 |
| 12345 接线员 | 准确登记和安抚 | 话务量、情绪压力、分类准确率 | 她不是决策者，却常接住第一波怒气 |
| 网格员 | 发现和转介问题 | 辖区大、任务碎、熟人社会 | 最懂人情，也最怕隐私和越权 |
| 村居干部 | 稳住现场、推动处理 | 资源有限、亲缘压力、群众不信任 | 既可能拖延，也可能真的被夹在中间 |
| 信访干部 | 登记、转送、督办、答复 | 积案压力、重复访、程序规范 | 不是万能领导，更像流程和情绪的交汇点 |
| 主管部门工作人员 | 实质办理 | 法律边界、预算、证据、历史包袱 | 有些问题不是不想办，是没有事实或权限 |
| 调解员 | 让双方能谈 | 人情、法律、情绪都要照顾 | 调解成功也要履行和回访 |
| 主角 | 识别路径、推动闭环 | 夹在群众期待和制度边界之间 | 价值在拆清问题，而不是替所有人拍板 |

## 10. 商业级冲突抓手

### 10.1 把“旧账”写出来

信访和基层积案常常不是今天才发生。它可能跨过换届、搬迁、离婚、老人去世、企业倒闭、档案缺失。旧账越久，越要写材料断裂和情绪积累。

### 10.2 把“谁有权限”写出来

主角再能干，也不能替法院改判、替部门发证、替财政拨款、替开发商赔钱。限制越清楚，推动越有质感。

### 10.3 把“部分解决”写出来

现实矛盾很少一次全清：可以先解决急难愁盼，再告知长期路径；先调解当前冲突，再推动制度修补；先救助老人吃饭看病，再处理赡养纠纷。

### 10.4 把“回访”写出来

回访是基层治理的戏剧宝藏。它可以发现：

- 签了调解协议但没履行。
- 群众表面满意，回家又被亲戚施压。
- 部门答复合法，但群众听不懂。
- 一个工单解决后，暴露出整个小区的同类问题。

### 10.5 把“隐私和尊严”写出来

信访、救助、家庭矛盾、心理危机、未成年人、残障和贫困都涉及隐私。不要把群众的身份证号、住址、病情、家庭丑事和求助经历当舆论素材。

## 11. 写作禁区与安全边界

不得提供以下现实操作：

- 如何伪造投诉、信访、证明、判决、病历、村务公示或历史档案。
- 如何组织围堵、冲击机关、堵路、非法聚集、暴力抗法或威胁工作人员。
- 如何利用信访、12345、媒体曝光敲诈勒索或恶意施压。
- 如何人肉、网暴、泄露信访人、举报人、基层干部、未成年人或困难家庭隐私。
- 如何绕过行政复议、行政诉讼、仲裁、司法执行等法定程序取得不当结果。
- 如何制造虚假舆情、剪辑视频、冒充群众或冒充部门工作人员。

可以写：

- 合法、理性、有序表达诉求。
- 基层干部依法分类、转办、督办、调解和回访。
- 群众从情绪表达转向证据和路径。
- 错误处理被纠正，部门推诿被督办，合理诉求得到实质推动。

## 12. 审稿检查问题

写完相关剧情后，逐项检查：

- 这件事是咨询、投诉、举报、信访、民事纠纷、行政争议、涉法涉诉、纪检监察还是警情？
- 第一入口是谁？12345、信访窗口、网格员、村居、司法所、派出所、法院还是主管部门？
- 诉求有没有事实和证据？证据能证明什么，不能证明什么？
- 有没有明确主管部门和法定路径？有没有把信访写成万能捷径？
- 是否写清登记、分流、核实、办理、答复、督办、回访或复查复核？
- 网格员、综治中心、村居干部有没有越权？
- 群众诉求是否既有情绪也有边界？是否把“闹得大”写成唯一有效方式？
- 基层干部是否被脸谱化？有没有写资源有限、权限有限和程序约束？
- 是否保护信访人、举报人、未成年人、老人、困难家庭和基层工作人员隐私？
- 结尾有没有留下履行、复发、回访、制度修补或关系余波？

## 13. 可直接套用的桥段提示词

### 13.1 12345 重复投诉

```text
请写一场县城 12345 重复投诉推动基层治理的小副本。要求写清投诉人诉求、工单分办、主管部门推诿点、现场核实、群众情绪、办理答复、回访结果和制度修补。不要让热线客服直接裁判结果，也不要把群众写成无理取闹。
```

### 13.2 信访积案化解

```text
请设计一场信访积案化解戏。必须写清历史材料、原处理意见、当前诉求、主管部门、信访登记、依法分类、核查过程、可解决和不可解决的部分、解释沟通、救助或调解接口、回访余波。不要让信访替代法院、复议或执行。
```

### 13.3 网格员发现矛盾

```text
请写一场网格员走访发现家庭或邻里风险的现实戏。要求写清发现信号、隐私保护、上报转介、村居/司法所/派出所/民政等分工、当事人情绪和后续回访。不要让网格员越权侦查、处罚或强行处理。
```

## 14. 一句话原则

县域信访和基层治理戏的商业价值，不在“把麻烦群众按下去”，也不在“让主角一句话替群众赢”；它真正好看的地方，是主角让无序的情绪找到路径，让分散的部门形成闭环，让被忽视的人在规则里重新被看见。
