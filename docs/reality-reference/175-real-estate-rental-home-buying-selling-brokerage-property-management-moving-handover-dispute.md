# 175. 房地产租赁、买房卖房、中介合同、物业纠纷与搬家交割链专项

状态：已确认  
适用对象：AI agents、小说作者、审稿代理、连续性维护代理  
最后更新：2026-06-09

## 1. 本篇解决什么问题

房子不是“主角有钱后随手买下”的奖杯，也不是“穷人交不起房租”的单一压力。现代中文网文里，租房、合租、买房、卖房、二手房、新房、房贷、首付、中介、产权、网签备案、不动产登记、物业、小区、验房、搬家、押金退还、水电燃气交割、维修基金、噪音漏水、邻里纠纷和房屋查封抵押，都是能改变人物行动资格、家庭权力、婚恋关系、现金流、身份归属和现实压力的制度链。

本篇用于让 AI 在写作或审稿时，把房屋交易和居住交割写成一条可追踪的链条：

```text
居住/交易目的 -> 房源来源 -> 权属核查 -> 中介/平台 -> 看房 -> 议价 -> 合同 -> 定金/押金/首付 -> 贷款/资金监管 -> 网签备案/登记 -> 交房验房 -> 水电燃气物业交割 -> 搬家入住/退租 -> 争议处理 -> 台账回写
```

本篇重点解决以下失真：

| 失真类型 | 读者体感 | 修正方向 |
|---|---|---|
| 买房像买奶茶 | 金钱和制度压力消失 | 补首付、贷款、税费、资格、网签、登记、交割 |
| 租房只写押金 | 居住不真实 | 补合同、付款周期、维修、转租、合租、退租验房 |
| 中介像万能 NPC | 行业链浅 | 补房源核验、带看、居间合同、佣金、信息披露和跳单争议 |
| 产权一笔带过 | 悬疑和家庭伦理失真 | 补共有、抵押、查封、继承、婚前婚后、登记簿和证书 |
| 物业只会收钱 | 小区社会消失 | 补物业服务、业主、公共维修、门禁停车、噪音漏水和业委会 |
| 搬家像瞬移 | 状态变化无质感 | 补钥匙、门禁、水电燃气、快递地址、家具家电、清洁和损坏 |
| 纠纷靠吵赢 | 程序和证据缺失 | 补合同、聊天、转账、房屋照片、维修记录、投诉调解和诉讼 |

## 2. 与既有专题的分工

- 第 152 篇“家、房屋、住所、房产与私密空间”负责房子作为生活空间、地盘归属和私密边界。本篇负责房子作为租赁、买卖、交割和争议对象的制度链。
- 第 150 篇“金钱、货币、价格、收入与生活成本”负责房租、房价和购买力感。本篇处理房租押金、首付贷款、税费佣金、物业费和交割账单。
- 第 165 篇“银行贷款、信用、保险、税票与资金合规”负责贷款、征信、资金流水、抵押和金融合规。本篇只处理房产交易中这些节点如何进入叙事。
- 第 161 篇“身份文书、登记、档案与许可”负责证件和登记的通用规则。本篇聚焦不动产登记、产权证、网签备案和购房资格。
- 第 173 篇“快递物流、仓储配送与签收退货”负责搬家快递和物品流转。本篇负责房屋钥匙、门禁、水电燃气、家具家电和搬家交割状态。
- 第 166 篇“报警接处警与案件程序”可联动非法侵入、诈骗租房、强占房屋、查封执行、损坏财物和邻里警情。
- 第 164 篇“灾害事故与应急疏散”可联动房屋漏水、燃气、火灾、危房、装修事故和物业责任。

## 3. 安全与合规边界

本篇只用于小说创作、审稿和现实可信度建设，不提供法律、投资、税务、贷款、购房资格、避税、骗贷、规避限购、虚假交易、伪造材料或强制清退的现实建议。

AI 不应输出以下内容：

- 不写伪造购房资格、假离婚购房、阴阳合同避税、骗取贷款、骗提公积金、虚假流水、规避限购限贷的操作步骤。
- 不写伪造产权证、不动产登记材料、租赁合同、授权委托书、收据发票或中介资质的方法。
- 不指导强行换锁、断水断电赶租客、暴力清退、非法进入他人住宅、私装监控、窃取门禁数据。
- 不把租房诈骗、黑中介、房屋查封、隐瞒抵押、非法群租写成可模仿教程。
- 不把家庭暴力、经济控制、强迫同住、监控囚禁包装成甜宠或“霸总保护”。

安全写法可以包括：

- 角色核验产权、查看合同、保存转账和聊天记录、走正规登记/备案/贷款/资金监管流程。
- 发生纠纷时通过协商、平台投诉、行业主管部门、消费者组织、人民调解、仲裁或诉讼处理。
- 写违法交易时只写风险、被发现、证据链和后果，不写规避方法。
- 涉及真实城市限购、税费、公积金和住房保障政策时，正式写作前必须查当地最新规则。

## 4. 十二行房产/租赁场景卡

写任何租房、买房、卖房、中介、物业或搬家交割场景前，AI 先补齐这十二行：

```text
1. 目的：租住、合租、买房自住、卖房套现、投资、结婚、离婚分割、继承、搬迁、逃离、学区/通勤？
2. 房源：新房、二手房、出租房、公寓、城中村、老小区、保障房、公租房、商住、别墅、宅基地/架空？
3. 权属：产权人是谁？共有吗？抵押、查封、租赁占用、继承争议、婚前婚后、授权委托是否清楚？
4. 渠道：中介、平台、熟人、开发商、房东直租、公司宿舍、政府保障、拍卖、法院执行？
5. 价格账：房租、押金、佣金、首付、贷款、税费、物业费、维修基金、水电燃气、搬家费？
6. 看房验房：位置、楼层、采光、噪音、漏水、墙裂、家电、家具、门窗、消防、车位、学位、邻里？
7. 合同：租赁合同、买卖合同、居间合同、补充协议、违约责任、付款节点、交付标准？
8. 资金路径：转账对象、资金监管、贷款审批、定金/订金、发票收据、押金退还、退款争议？
9. 登记/备案：租赁备案、网签、预售许可、不动产登记、抵押登记、户籍/居住证/学籍关联？
10. 交割：钥匙、门禁、物业、水电燃气、家具家电清单、维修记录、快递地址、停车位、宠物规则？
11. 争议证据：合同、聊天、转账、房屋照片视频、验房清单、维修单、物业记录、邻居证言？
12. 后续代价：搬家、通勤、现金流、关系变化、学业/工作、信用、诉讼、被执行、生活安全感？
```

## 5. 租房链：不是交押金就能住

可信租房链通常是：

```text
找房 -> 看房 -> 核验房东/房源 -> 议价 -> 签租赁合同 -> 支付押金和租金 -> 交钥匙门禁 -> 水电燃气/物业交接 -> 入住 -> 维修和邻里 -> 退租验房 -> 押金结算
```

| 节点 | 可写冲突 |
|---|---|
| 找房 | 平台照片失真、虚假低价、距离地铁远、房源已租 |
| 看房 | 中介催促、合租室友不在、房东隐瞒漏水噪音 |
| 核验 | 房东是否产权人、转租是否授权、二房东风险 |
| 合同 | 租期、付款周期、押金、违约、维修责任、转租限制 |
| 入住 | 钥匙、门禁、电表水表、网络、家具家电清单 |
| 维修 | 热水器、空调、漏水、墙皮、下水、谁付钱 |
| 合租 | 公共区域、卫生、访客、宠物、作息、性别边界 |
| 退租 | 提前退租、找替租、清洁费、损坏扣押金 |
| 押金 | “正常磨损”和“损坏”争议，证据在入住照片和清单 |

写法提醒：租房最能暴露人物处境。一个人搬进出租屋时，真正改变他的不是“有屋可住”，而是通勤、现金流、邻里、同住人、门锁、安全感和可支配时间。

## 6. 买房卖房链：房子是最大额的连续性资产

可信买卖链通常是：

```text
需求确认 -> 资格/预算 -> 房源筛选 -> 产权核查 -> 看房 -> 议价 -> 签约 -> 定金/首付 -> 贷款审批 -> 网签备案/资金监管 -> 缴税过户/不动产登记 -> 交房验房 -> 物业水电交割 -> 装修/入住/出租
```

| 节点 | 可写冲突 |
|---|---|
| 资格预算 | 限购、贷款资格、征信、首付、家庭出资 |
| 房源筛选 | 学区、通勤、楼龄、车位、物业、邻里、噪音 |
| 产权核查 | 共有权人、抵押、查封、租约、继承、离婚分割 |
| 价格谈判 | 中介话术、卖家急售、买家压价、市场变化 |
| 定金/首付 | 定金罚则、订金误解、资金来源、父母出资 |
| 贷款审批 | 征信、收入流水、银行评估、利率和放款时间 |
| 网签/备案 | 防止一房二卖、交易锁定、政策变化压力 |
| 税费 | 契税、增值税、个税、中介费、维修资金等地区差异 |
| 过户登记 | 登记簿、证书、抵押登记、领证时间 |
| 交房 | 户口迁出、租客腾退、家具家电、钥匙和物业欠费 |

买房戏不要只写“刷卡买下”。即使主角有钱，也要处理产权、税费、登记、交割和后续维护。越豪气的买房，越需要现实链条托住爽感。

## 7. 中介和经纪：信息接口，不是万能 NPC

房地产经纪人常承担信息整理、带看、撮合、合同协助和交易风险提示，但他们也有佣金动机和信息不对称。

| 中介节点 | 可写内容 |
|---|---|
| 房源发布 | 真房源、假房源、低价引流、图片美化 |
| 带看 | 路线安排、筛选客户、催促决策、避开缺陷时间段 |
| 信息披露 | 房屋权属、抵押、查封、租约、瑕疵、周边规划 |
| 佣金 | 谁付、何时付、跳单、独家委托、服务范围 |
| 居间合同 | 服务内容、费用、违约、保密、推荐房源记录 |
| 议价 | 两头传话、制造紧迫、隐瞒底价、促成交易 |
| 风险提示 | 产权核验、资金监管、交易税费、贷款进度 |
| 争议 | 黑中介、吃差价、虚假承诺、佣金纠纷、平台投诉 |

写法原则：中介可以帮角色省时间，也可以制造误导。不要让中介随口保证法律后果；真正关键的权属、登记、贷款、税费和政策，最好由相应机构、合同或查询结果支撑。

## 8. 不动产登记、产权和证据链

房屋权属不是“拿着钥匙就是我的”。可信产权线至少区分：

| 概念 | 叙事意义 |
|---|---|
| 不动产登记簿 | 权利归属核心证据，优先于口头说法 |
| 不动产权证书 | 权利证明，但也要与登记簿一致 |
| 共有 | 夫妻、父母子女、兄弟姐妹、合伙购房会影响处分权 |
| 抵押 | 房贷、经营贷款、债务担保，影响出售和过户 |
| 查封 | 司法执行风险，买卖可能被卡住 |
| 租赁占用 | 买卖不等于立刻能住，原租约和腾退要处理 |
| 继承 | 遗嘱、法定继承、继承人意见和登记变更 |
| 婚姻关系 | 婚前婚后、共同还贷、父母出资，家庭伦理强钩子 |
| 授权委托 | 代签、代卖、代收款必须核验授权 |

悬疑和家庭伦理中，房产证、登记簿、抵押记录、查封信息、购房合同、转账流水、委托书、遗嘱、结婚离婚材料和聊天记录都能形成证据链。

## 9. 新房、预售和交付：样板间不等于到手房

新房线特别适合写期待落差和群体维权。

| 节点 | 可写冲突 |
|---|---|
| 宣传广告 | 学区、地铁、配套、绿化、物业、样板间夸大 |
| 预售许可 | 能否合法预售，何时交付，资金监管 |
| 认购和定金 | 冲动认购、销售话术、退定争议 |
| 商品房买卖合同 | 面积、交付标准、违约责任、精装标准 |
| 贷款 | 银行审批、利率变化、放款延迟 |
| 延期交房 | 开发商资金压力、业主群、违约金 |
| 验房 | 空鼓、漏水、裂缝、门窗、层高、精装质量 |
| 收房 | 物业费起算、维修资金、钥匙、资料交接 |
| 产权证 | “交房即交证”与办证延迟、地方差异 |

样板间、沙盘和销售承诺不要自动等于合同内容。商业级写法要让角色学会区分“广告、销售话术、合同条款、实物交付和登记结果”。

## 10. 物业、小区和业主关系

物业不是只收物业费的反派，也不是万能管家。它是小区公共事务的接口之一。

| 物业节点 | 可写内容 |
|---|---|
| 物业费 | 收费标准、欠费、催缴、服务不满 |
| 门禁安保 | 外卖快递、访客、车辆、陌生人、监控调取 |
| 公共维修 | 电梯、屋顶、外墙、管道、消防、维修资金 |
| 停车位 | 产权车位、租赁车位、临停、堵车位 |
| 噪音装修 | 装修时间、扰民、投诉、楼上楼下冲突 |
| 漏水渗水 | 楼上楼下、公共管道、维修责任、损失赔偿 |
| 宠物 | 牵引、粪便、叫声、禁养、邻里群舆论 |
| 业主大会/业委会 | 更换物业、公共收益、维修资金使用、投票争议 |
| 社区/居委会 | 调解、通知、老人儿童、公共安全和基层治理 |

写小区要有“小社会”感：物业、保安、保洁、业主群、业委会、居委会、邻居、快递员、外卖员都可能成为信息节点和冲突节点。

## 11. 搬家和交割：状态变化要落到物件和账单

搬家不是黑屏跳转。可信搬家链：

```text
通知/解约 -> 打包 -> 预约搬家公司 -> 退租验房 -> 水电燃气结清 -> 押金/尾款 -> 新房交钥匙 -> 家具家电清点 -> 地址变更 -> 快递/证件/通勤/邻里更新
```

| 交割项 | 可写作用 |
|---|---|
| 钥匙门禁 | 谁还持有，是否换锁，门禁权限是否注销 |
| 水电燃气 | 读表、欠费、过户、充值卡、智能表 |
| 家具家电 | 清单、损坏、折旧、归属、带走还是留下 |
| 网络宽带 | 迁移、销户、押金、设备归还 |
| 物业停车 | 欠费、车位、门禁、访客权限 |
| 快递地址 | 旧地址泄露、包裹错送、证件信件滞留 |
| 清洁维修 | 开荒保洁、墙面孔洞、油烟、猫砂味 |
| 邻里告别 | 楼下阿姨、门卫、邻居群、最后一次碰面 |
| 情感物件 | 旧照片、钥匙扣、门牌、坏灯、划痕、植物 |

搬家最有价值的戏，是角色发现自己不只是换了地址，而是换了生活半径、关系网、身份感和安全感。

## 12. 纠纷处理：合同、证据和程序

常见纠纷链：

```text
问题出现 -> 沟通协商 -> 留存证据 -> 平台/中介/物业介入 -> 主管部门/消费者组织/社区调解 -> 仲裁或诉讼 -> 执行/修复/搬离 -> 舆论余波
```

| 纠纷 | 证据 |
|---|---|
| 押金不退 | 合同、押金收据、转账、入住/退租照片、维修清单 |
| 房屋漏水 | 照片视频、维修记录、物业记录、邻居证言、损失清单 |
| 虚假房源 | 平台页面、聊天、带看记录、付款记录、房源状态 |
| 中介佣金 | 居间合同、带看确认、聊天记录、成交房源一致性 |
| 一房二租/二卖 | 合同、付款、登记/备案、交付时间、房屋占有状态 |
| 贷款失败 | 银行通知、合同约定、征信/收入材料、责任归属 |
| 延期交房 | 合同、开发商通知、业主群、现场照片、官方通报 |
| 物业服务 | 收费公示、服务合同、维修记录、投诉记录、业主表决 |
| 强制清退 | 合同、通知、报警记录、换锁断水电证据 |
| 房屋查封 | 登记查询、法院文书、交易时间、付款证据 |

写纠纷不要只靠角色吵架。越现实的纠纷，越要让合同条款、钱款路径、照片视频、时间节点和第三方记录发声。

## 13. 类型文中的高价值用法

| 类型 | 用法 |
|---|---|
| 都市言情 | 合租边界、租房退押金、买房署名、共同还贷、搬家同居 |
| 家庭伦理 | 父母出首付、房产证加名、兄弟姐妹继承、老人房产被控制 |
| 重生赚钱 | 不能只靠知道房价涨，要写资格、资金、税费、贷款和政策风险 |
| 职场现实 | 通勤租房、公司宿舍、离职搬家、社保公积金与购房资格 |
| 商战创业 | 办公室租赁、商铺转让、仓库租金、物业和消防 |
| 悬疑刑侦 | 空房、钥匙、门禁、物业监控、房屋查封、假房东 |
| 校园青春 | 宿舍退宿、考研租房、合租室友、家长出钱控制 |
| 娱乐圈 | 明星住址泄露、豪宅偷拍视频、物业安保、房产代持 |
| 乡村现实 | 宅基地、拆迁安置、进城买房、返乡建房，需另查地方政策 |
| 末世灾变 | 安全屋、物业崩溃、空置房、物资仓库、居住权重排 |

## 14. 审稿坏味道与修复办法

| 坏味道 | 为什么削弱商业感 | 修复办法 |
|---|---|---|
| 主角有钱直接买房 | 爽点没现实托底 | 加资格、看房、产权、付款、贷款、税费、登记 |
| 租房只有押金 | 生活空 | 加合同、钥匙、水电、维修、退租、同住人 |
| 中介全知全能 | 行业浅 | 加房源核验、佣金动机、信息披露和合同边界 |
| 房产证当万能答案 | 证据单薄 | 加登记簿、共有、抵押、查封、付款和合同 |
| 物业只会坏 | 脸谱化 | 加服务范围、收费公示、公共维修和业主关系 |
| 押金纠纷靠吵赢 | 不可信 | 加入住照片、清单、合同、转账和第三方调解 |
| 搬家一笔带过 | 状态变化浪费 | 加打包、搬家公司、地址变更、旧物和新通勤 |
| 父母出钱无后果 | 家庭权力缺失 | 加出资证明、署名、控制、情感债和继承风险 |
| 漏水噪音无证据 | 邻里纠纷弱 | 加物业记录、照片视频、维修单和邻居证词 |
| 现实政策一概而论 | 城市差异失真 | 标注地方政策需查最新规则，避免编造限购税费 |

## 17. 写作前自查清单

- 房子在本场戏里是什么：住所、资产、负债、筹码、战利品、牢笼、证据还是情感记忆？
- 是租、买、卖、继承、抵押、查封、搬离还是退租？制度链是否匹配？
- 房源来源和权属是否清楚？是否有中介、平台、房东、共有权人或抵押权人？
- 钱款是否有路径：押金、房租、定金、首付、贷款、税费、物业费、中介费、维修资金？
- 合同里是否有租期、付款、维修、违约、交付标准、家具家电和补充协议？
- 是否写了看房/验房：采光、噪音、漏水、墙裂、门窗、家电、邻里、通勤？
- 是否处理登记/备案/网签/不动产登记/抵押登记等关键节点？
- 搬家交割是否有钥匙门禁、水电燃气、物业、家具清单、快递地址和旧物处理？
- 纠纷是否有合同、聊天、转账、照片视频、维修单、物业记录和第三方介入？
- 是否避免伪造材料、骗贷、规避限购、阴阳合同、避税、暴力清退等可执行内容？

## 18. 最小规则

1. 第 152 篇管房子作为生活空间，第 175 篇管房子作为交易、租赁、交割和争议对象。
2. 每个重要房屋事件都要有“房源、权属、合同、钱款、登记、交割、证据、后果”。
3. 钥匙不等于产权，房产证不等于完整事实，住进去不等于纠纷结束。
4. 租房必须写押金、合同、维修、退租和生活状态；买房必须写资格、资金、登记、税费和交付。
5. 中介、物业、银行、登记机构、开发商、房东、租客、邻居都是制度接口，不是纯 NPC。
6. 房产爽点必须落到现金流、行动半径、关系权力、身份安全感和长篇资产台账。
7. 房屋最有价值的戏，常常发生在看房路上、合同桌前、银行电话、物业前台、验房现场、搬家纸箱和退押金争执里。
