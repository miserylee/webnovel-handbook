## 1. 本专题解决的 AI 写作问题

AI 常把检察院写成一个模糊的“更高级司法部门”，于是出现：

1. 检察官像刑警一样到处秘密侦查，忽略侦查主体、提前介入和补充侦查边界。
2. 公安抓人后直接判刑，跳过审查逮捕、审查起诉、证据审查和不起诉可能。
3. 检察院提起公诉被写成“已经定罪”，忽略法院审判才是定罪裁判节点。
4. 认罪认罚被写成“签字换轻判”，缺少自愿性、真实性、值班律师、量刑建议和法院审查。
5. 不起诉被写成“放人无事”，忽略法定不起诉、酌定不起诉、证据不足不起诉、训诫、赔偿、行政处罚移送和被害人感受。
6. 公益诉讼被写成检察官替主角打民事官司，忽略国家利益、社会公共利益、诉前程序和行政机关履职督促。
7. 未成年人案件被写成一罚了之，忽略教育、感化、挽救、隐私保护、附条件不起诉、帮教和被害未成年人保护。
8. 检察监督被写成“检察院命令法院改判”，忽略抗诉、检察建议、监督程序和权力边界。

本专题把检察院写成“刑事追诉 + 法律监督 + 公益保护 + 未成年人综合保护”的制度节点。

## 2. 总原则：检察院不是公安升级版，也不是法院替身

县域检察院的戏剧价值，不在于检察官替主角亲自破案或替法院判案，而在于：

- 对侦查机关移送的案件进行证据和法律审查。
- 决定是否批准逮捕、是否起诉、是否退回补充侦查或不起诉。
- 在法院审判中支持公诉，并对诉讼活动是否合法进行监督。
- 对行政机关未依法履职、公共利益受损、生态环境破坏、食品药品安全、国有财产保护等场景发出公益诉讼检察建议或提起公益诉讼。
- 对未成年人案件实行特殊保护和综合帮教。
- 对民事、行政、刑事执行等诉讼活动进行法律监督。

标准刑事链条：

```text
案发报警或线索发现
-> 公安受案、立案、侦查、采取强制措施
-> 需要逮捕时提请检察院审查批准逮捕
-> 检察院审查证据、社会危险性和羁押必要性
-> 批准逮捕、不批准逮捕或要求继续侦查
-> 公安侦查终结后移送审查起诉
-> 检察院全面审查事实、证据、罪名、程序和量刑情节
-> 听取犯罪嫌疑人、辩护人、被害人及诉讼代理人意见
-> 退回补充侦查、自行补充侦查、认罪认罚、公开听证或不起诉审查
-> 提起公诉、不起诉或移送处理
-> 法院开庭，检察官出庭支持公诉
-> 法院裁判
-> 检察机关视情况监督刑事执行、审判活动和生效裁判
```

标准公益诉讼链条：

```text
公共利益受损线索
-> 检察机关发现或受理线索
-> 初步调查和评估是否属于法定公益保护领域
-> 立案调查
-> 调取资料、询问、现场勘查、检测鉴定或听证
-> 行政公益诉讼诉前检察建议，督促行政机关依法履职
-> 行政机关整改、回复、复查
-> 未依法履职或整改不到位时提起行政公益诉讼
-> 民事公益诉讼中支持适格主体起诉或依法提起诉讼
-> 判决、修复、赔偿、整改和社会治理回写
```

AI agent 写检察院桥段时，先问三件事：

- 这是刑事追诉、法律监督、公益诉讼、未成年人保护，还是控告申诉？
- 检察院处在案子的哪个程序节点？
- 这个节点能作出什么法律处理，不能越权做什么？

## 3. 法律与制度底座

### 3.1 人民检察院组织法：检察院是国家法律监督机关

人民检察院组织法确认人民检察院的设置、职权和办案组织。其核心职能包括：对刑事案件审查批准或者决定逮捕，决定是否提起公诉并支持公诉，依法提起公益诉讼，对诉讼活动、生效法律文书执行、监狱和看守所执法活动等实行法律监督。

写作迁移：

- 检察院的主轴是法律监督和公诉，不是万能侦探社。
- 县级检察院通常贴近基层案件，但仍受上级检察院领导和业务监督。
- 检察官可以提前介入重大、疑难、复杂案件，引导侦查取证和监督侦查活动，但不等于接管公安侦查。

### 3.2 刑事诉讼法和刑事诉讼规则：批捕、起诉和监督有程序

刑事诉讼法规定公检法分工负责、互相配合、互相制约。人民检察院刑事诉讼规则细化检察机关办理刑事案件的管辖、强制措施、案件受理、审查逮捕、审查起诉、出庭公诉、未成年人特别程序和刑事诉讼法律监督。

写作迁移：

- 批捕审查不是“公安说抓就抓”，检察院要审证据、事实、罪名和社会危险性。
- 审查起诉不是走过场，可能出现退回补充侦查、不起诉、变更罪名、追诉漏犯漏罪或提出量刑建议。
- 检察院发现侦查违法、审判违法、执行违法，可以依法提出纠正意见、检察建议、抗诉等，但不能随意命令法院改判。

### 3.3 认罪认罚从宽：不是认了就自动轻判

两高三部关于认罪认罚从宽制度的指导意见强调适用范围、条件、权益保障、值班律师、量刑建议、程序选择和法院审查。

写作迁移：

- “认罪”要承认主要犯罪事实，“认罚”要愿意接受处罚和程序安排。
- 检察院可提出量刑建议，但法院仍要审查事实、证据、法律适用和认罪认罚自愿性。
- 被害人谅解、赔偿、退赃、社会危害性、初犯偶犯、未成年人等会影响从宽幅度。
- 严重暴力、重大敏感、社会影响恶劣案件要慎重把握，不能写成花钱买轻判。

### 3.4 民事、行政检察监督：不是第三审，但能监督生效裁判和程序违法

民事诉讼监督规则、行政诉讼监督规则规定，当事人对法院生效裁判、调解书、审判程序或执行活动存在法定监督情形时，可以向检察机关申请监督；检察机关也可以依职权发现监督线索，依法提出再审检察建议、抗诉、检察建议或推动行政争议实质性化解。

写作迁移：

- 检察监督不是“输了官司就找检察院重打一次”。
- 它关注生效裁判是否确有错误、审判程序是否违法、执行活动是否合法、虚假诉讼线索是否存在。
- 行政检察监督可以推动行政争议实质性化解，但不是替行政机关作决定。

### 3.5 公益诉讼：保护公共利益，不是私人复仇工具

公益诉讼办案规则规定，检察机关办理公益诉讼案件，是通过依法履行检察权，督促行政机关履职，支持适格主体依法行使公益诉权，维护国家利益和社会公共利益。

常见领域：

- 生态环境和资源保护。
- 食品药品安全。
- 国有财产保护。
- 国有土地使用权出让。
- 英烈权益保护。
- 未成年人保护。
- 安全生产、个人信息保护、军人地位和权益保障等法律明确领域。

写作迁移：

- 公益诉讼不是检察官帮主角要赔偿，而是保护不特定多数人或国家、社会公共利益。
- 行政公益诉讼通常先有诉前检察建议，给行政机关依法履职、整改和回复空间。
- 民事公益诉讼可由法律规定的机关和组织提起，检察机关可支持起诉或依法提起诉讼。
- 高质量写法要写出线索发现、调查核实、检察建议、整改复查、起诉条件和治理后果。

### 3.6 未成年人检察：惩治也是挽救，保护不是纵容

未成年人检察工作强调对涉罪未成年人的教育、感化、挽救，对侵害未成年人犯罪依法严惩，同时推动家庭、学校、社会、网络、政府、司法“六大保护”协同。

写作迁移：

- 涉罪未成年人案件要注意隐私保护、分案处理、合适成年人到场、法律援助、社会调查、附条件不起诉、考察帮教和心理支持。
- 侵害未成年人案件要注意被害人保护、强制报告、证据固定、避免二次伤害、司法救助和家庭教育指导。
- 附条件不起诉不是“没事了”，而是有考验期、帮教、义务、评估和可能撤销的制度安排。

## 6. 程序节点写法

### 6.1 审查逮捕：抓人不是结束，而是最严厉强制措施前的审查

低质写法：

> 公安把人抓了，检察院马上批捕，反派彻底完了。

高质写法：

> 卷宗送到检察院时，承办检察官没有先看热搜，而是把证据分成三堆：能证明盗窃事实的，能证明金额的，能证明社会危险性的。监控里的人影模糊，赃物去向还没查清，嫌疑人却有前科且曾威胁证人。批捕与否，卡在“证据够不够”和“是否必须羁押”之间。

可写要点：

- 讯问犯罪嫌疑人。
- 听取辩护律师意见。
- 审查是否有逮捕必要。
- 对证据不足或社会危险性不足案件不批捕。
- 要求公安继续侦查、补强证据或变更强制措施。

### 6.2 审查起诉：案件从“破了”变成“能不能起诉”

低质写法：

> 公安把案子交给检察院，检察院直接起诉。

高质写法：

> 公安卷宗里有十七份询问笔录，检察官却在付款时间线上发现一个空洞：最关键的转账发生在所谓诈骗话术之前。她退回补充侦查，不是替嫌疑人开脱，而是知道这个空洞如果带上法庭，会变成整案的裂缝。

审查内容：

- 犯罪事实是否清楚。
- 证据是否确实、充分。
- 罪名适用是否准确。
- 是否遗漏犯罪嫌疑人、犯罪事实或量刑情节。
- 侦查活动是否合法。
- 是否需要退回补充侦查或自行补充侦查。
- 是否适用认罪认罚、速裁、简易程序或不起诉。

### 6.3 不起诉：不是“没事”，而是法定后果

不起诉类型可以产生很强戏剧张力：

- 法定不起诉：依法不应追究刑事责任。
- 酌定不起诉：犯罪情节轻微，依照刑法不需要判处刑罚或免除刑罚。
- 证据不足不起诉：补充侦查后仍不符合起诉条件。
- 附条件不起诉：主要适用于符合条件的未成年人案件。

写作抓手：

- 被害人认为“不起诉就是包庇”，检察官需要释法说理。
- 嫌疑人以为不起诉就没有社会后果，结果仍要赔偿、道歉、接受行政处罚或帮教。
- 主角作为律师或记者发现，不起诉有时是防错案的底线，有时也会刺痛被害人的公平感。

### 6.4 认罪认罚和量刑建议：效率、悔罪和公平感的三角

高质写法要同时写三层：

- 犯罪嫌疑人是否真正承认主要事实。
- 被害人损失是否赔偿、是否谅解。
- 检察院量刑建议是否与罪责刑相适应。

可写冲突：

- 嫌疑人只认轻的，不认关键情节。
- 家属拿赔偿压被害人谅解。
- 值班律师提醒签署具结书的后果。
- 检察官在从宽和公众公平感之间谨慎拿捏。
- 法院审查后认为量刑建议明显不当，要求调整或依法裁判。

### 6.5 出庭支持公诉：检察官不是旁白，是控方诉讼角色

庭审中检察官的功能：

- 宣读起诉书。
- 出示证据并质证。
- 讯问被告人，询问证人、鉴定人。
- 发表公诉意见和量刑建议。
- 回应辩护意见。
- 发现法庭审理中程序问题时依法履行监督职责。

不要写成：

- 检察官直接宣布有罪。
- 检察官替法官裁判。
- 检察官用情绪演说替代证据。
- 检察官无视辩护权或被害人权利。

### 6.6 公益诉讼：公共利益受损也能成为剧情主线

县域适合写的公益诉讼场景：

- 河道被养殖废水污染，行政机关长期未履职。
- 校园周边食品安全隐患多次投诉无果。
- 非法采砂、毁林、占用耕地造成生态损害。
- 历史文物、烈士纪念设施、古树名木被破坏。
- 养老机构、无障碍设施、个人信息泄露等涉及不特定多数人权益。

高质链条：

```text
群众举报或检察官履职发现线索
-> 初查确认可能损害公共利益
-> 立案调查
-> 现场勘查、检测、调取行政处罚记录、询问相关人员
-> 组织公开听证或专家论证
-> 向行政机关发出诉前检察建议
-> 行政机关制定整改方案并回复
-> 检察院复查整改效果
-> 整改不到位时提起公益诉讼
-> 判决、修复、赔偿、制度整改和基层治理回写
```

关键：公益诉讼是“公共利益”和“依法履职”的故事，不是主角私人索赔的捷径。

### 6.7 未成年人检察：把保护写具体，不把保护写成纵容

高质量未检桥段常见链条：

```text
校园欺凌、盗窃、聚众斗殴或侵害未成年人案件发生
-> 公安侦查或学校、家庭、强制报告线索进入司法
-> 检察机关审查未成年人身份、年龄、家庭、教育、犯罪情节和被害情况
-> 对涉罪未成年人考虑是否起诉、附条件不起诉、帮教、心理疏导和家庭教育指导
-> 对被害未成年人保护隐私、避免重复询问和二次伤害
-> 联动学校、民政、妇联、团委、社工、法律援助和心理服务
-> 考察期、回访、救助或依法严惩侵害未成年人犯罪
```

不要写成：

- 未成年人犯罪一律轻轻放过。
- 未成年人犯罪一律从重报复。
- 检察官用道德说教替代法定程序。
- 被害未成年人隐私被剧情消费。

## 8. 高质量桥段示例

### 8.1 伤害案批捕审查：被害人愤怒和证据标准发生冲突

低质写法：

> 检察官听说受害者很惨，马上批准逮捕。

高质写法：

```text
夜市打架造成轻伤
-> 公安提请批准逮捕
-> 检察官讯问嫌疑人，听取律师意见
-> 卷宗显示监控缺失关键十秒，证人说法互相矛盾
-> 被害人家属在检察院门口情绪激烈
-> 检察官要求补强先动手、工具来源、伤情因果关系和社会危险性证据
-> 可能批捕、也可能不批捕并要求继续侦查或变更强制措施
```

商业价值：不是削弱被害人，而是让“愤怒”撞上“不能错捕”的制度底线。

### 8.2 电诈案件审查起诉：不是抓到卡主就能结案

```text
县城老人被骗二十万元
-> 公安抓获出售银行卡的年轻人
-> 案件移送审查起诉
-> 检察官审查其是否明知、获利多少、参与层级、是否有上线证据
-> 要求补充侦查聊天记录、资金流水、取现视频和同案人供述
-> 对认罪认罚者提出量刑建议
-> 对上线漏犯追诉或另案处理
-> 被害人追赃挽损和公诉庭审并行
```

关键：电诈案件不能把“卡主”写成所有罪责的终点，也不能把追赃写成检察院私下讨债。

### 8.3 污染河道公益诉讼：没有单一主角，公共利益就是受害者

```text
村民发现河水发臭
-> 多次投诉后问题反复
-> 检察院公益诉讼部门发现线索
-> 调取环保执法记录、排污许可、监测数据和河道照片
-> 组织听证，邀请行政机关、村民代表、专家参加
-> 发出诉前检察建议，督促行政机关依法履职
-> 企业整改、清理污染、补做治理设施
-> 检察院复查仍有偷排迹象
-> 提起公益诉讼或继续监督整改
-> 结尾回写村民信任、招商口碑和基层治理压力
```

关键：公益诉讼的爽点不是“检察官替村民骂企业”，而是公共利益终于有了程序化保护者。

### 8.4 未成年人附条件不起诉：保护不是免罪牌

```text
初二学生参与盗窃手机
-> 公安移送审查起诉
-> 检察官审查年龄、犯罪情节、退赔、认罪悔罪、家庭监护和学校表现
-> 听取被害人、辩护人、法定代理人意见
-> 认为符合条件，作出附条件不起诉决定
-> 设置考察期、帮教计划、社区服务、心理辅导和家庭教育指导
-> 学生一度违反考察要求
-> 检察官、社工、学校重新评估
-> 最终根据考察表现决定是否起诉
```

关键：附条件不起诉要有条件、有考察、有帮教、有可能撤销，不能写成“未成年所以没事”。

### 8.5 民事虚假诉讼监督：检察院不是帮输家翻盘，而是维护司法秩序

```text
债务人离婚后突然被前妻起诉分割财产
-> 债权人怀疑二人通过虚假诉讼转移房产
-> 法院判决已生效，执行受阻
-> 债权人向检察院申请民事诉讼监督
-> 检察官调取婚姻、交易、资金流水、诉讼材料和执行记录
-> 发现诉讼时间、债务爆发和财产转移高度重合
-> 依法提出检察建议或抗诉
-> 法院再审或纠错
```

关键：这类桥段要写“监督司法活动和打击虚假诉讼”，不要写成检察院替主角开后门。

## 10. 安全边界

允许写：

- 批捕、审查起诉、公诉、公益诉讼、检察监督和未成年人保护的概念性程序。
- 检察官如何通过证据标准、法律适用、程序合法性和公共利益进行审查。
- 侦查瑕疵、证据不足、被害人不满、舆论压力和司法审慎之间的戏剧冲突。
- 违法侦查、虚假诉讼、污染环境、侵害未成年人和规避执行被监督纠正的后果。

不要写成教程：

- 如何串供、翻供、毁灭证据、威胁证人或规避批捕起诉。
- 如何伪造谅解书、赔偿凭证、公益诉讼线索、检测报告或未成年人帮教材料。
- 如何买通检察官、干扰听证、操纵量刑建议或泄露案卷秘密。
- 如何通过公益诉讼牟取私人利益。
- 如何泄露未成年人姓名、照片、学校、家庭住址、伤害细节等可识别隐私。
- 如何规避刑事执行、社区矫正、附条件不起诉考察或检察监督。

反派可以实施违法行为，但文本必须呈现其法律风险、证据痕迹和后果，不提供可复制步骤，也不把规避司法写成值得模仿的聪明。

## 11. 审稿检查问题

1. 这个检察院桥段是否明确是刑事追诉、法律监督、公益诉讼、未成年人检察还是控告申诉？
2. 是否区分公安侦查、检察审查起诉和法院审判？
3. 批捕是否写出证据审查、社会危险性和羁押必要性，而不是公安抓了就捕？
4. 审查起诉是否写出阅卷、讯问、听取意见、补侦、不起诉或起诉选择？
5. 提起公诉是否被误写成已经定罪？
6. 认罪认罚是否写出自愿性、真实性、律师参与、被害人意见、量刑建议和法院审查？
7. 不起诉是否写出类型、理由、后果、被害人救济和后续处理？
8. 公益诉讼是否围绕国家利益或社会公共利益，而不是主角私人索赔？
9. 公益诉讼是否写出线索、调查、诉前检察建议、整改复查和起诉条件？
10. 未成年人案件是否写出隐私保护、法律援助、社会调查、帮教、家庭教育指导和被害人保护？
11. 民事、行政、刑事执行监督是否避免写成“检察院命令法院改判”？
12. 是否存在串供、伪证、规避起诉、买通检察官、泄露案卷或泄露未成年人隐私被奖励的问题？
