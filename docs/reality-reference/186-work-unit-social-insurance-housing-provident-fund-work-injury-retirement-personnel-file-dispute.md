# 186. 单位、社保、公积金、工伤、退休、人事档案与职场保障纠纷链专项

状态：已确认  
日期：2026-06-09  
适用对象：需要描写单位用工、社会保险、医保断缴、养老保险缴费年限、公积金缴存/贷款、工伤认定、劳动能力鉴定、退休年龄、养老金、离职社保转移、人事档案、政审、职称、工龄和职场保障纠纷的 AI agents。

## 1. 本专题解决什么问题

第162篇已经处理招聘、入职、劳动合同、工资、绩效、离职和劳动争议的职场生命周期。本篇专门下钻到“职场保障账户”：一个角色在单位里留下的记录，怎样影响他生病、受伤、买房、失业、生育、退休、换城市、考编、评职称和去世后的家庭资金线。

现实题材里，角色不是只拿工资。他还有社保缴费记录、公积金账户、工伤认定材料、人事档案、离职证明、退休年龄、养老金资格、医保结算、失业金、工龄争议和单位经办人的一摞表格。

本专题把职场保障写成可执行链路：

```text
用工关系形成 -> 劳动合同/入职材料 -> 社保公积金登记 -> 缴费基数和缴费月份 -> 工资个税与工资条 -> 人事档案或工作记录归集 -> 发生风险事件（生病/生育/失业/工伤/购房/退休/死亡） -> 经办申请和材料提交 -> 单位、人社、医保、公积金、档案机构或法院分流 -> 待遇、贷款、资格、赔偿或退休结果 -> 欠缴补缴/档案更正/劳动争议/行政投诉 -> 人物生活成本和关系余波 -> 台账回写
```

写作原则：社保、公积金和人事档案不是背景福利，而是角色未来几十年能不能治病、买房、退休、证明身份和抵御风险的制度暗线。

## 3. 安全边界

本专题只用于小说创作、审稿和设定，不提供法律、社保、公积金、退休、人事档案或工伤办理建议。

禁止写成教程的内容：

- 如何规避社保、公积金缴存、伪造劳动关系、虚构工资基数或骗取待遇。
- 如何骗提公积金、骗取医保、工伤保险、失业保险、生育保险或养老金。
- 如何伪造劳动合同、工资条、考勤、病历、工伤材料、档案材料、离职证明或退休材料。
- 如何恶意拖延工伤认定、威胁员工私了、隐瞒事故、篡改监控或销毁安全记录。
- 如何非法截留个人档案、扣押社保卡、拒开离职证明或利用档案报复员工。

可写方向：揭示制度压力、保护劳动者、展示单位经办失误或恶意、推动证据固定、呈现待遇资格和生活成本后果。

## 5. 五险一金：不要写成一句“公司给交”

五险一金在小说里至少有六个叙事功能：生病、养老、失业、工伤、生育、买房。每一项都可能在角色最需要时才显露问题。

| 项目 | 剧情功能 | 常见冲突 | 写作提醒 |
| --- | --- | --- | --- |
| 基本养老保险 | 退休时间、缴费年限、养老金、跨地转移 | 单位欠缴、工龄争议、最低缴费年限不足 | 退休不是到岁数自动领钱 |
| 基本医疗保险 | 住院结算、异地就医、断缴压力 | 离职断缴、单位欠缴、参保地和就医地不同 | 医保和商业保险分开写 |
| 工伤保险 | 工作事故、职业病、工亡、停工留薪 | 单位不申报、私了、外包派遣责任切割 | 工伤先看工作原因、材料和认定流程 |
| 失业保险 | 被裁、失业金、临近退休衔接 | 主动离职和非本人意愿中断就业差异 | 不要写成任何离职都能领 |
| 生育保险 | 产检、生育医疗费、生育津贴、产假 | 断缴、岗位歧视、女职工保护 | 接入第158篇孕产专题 |
| 住房公积金 | 买房贷款、租房提取、离职封存、城市资格 | 少缴、未缴、断缴、骗提、贷款资格失败 | 强城市差异，需查当地规则 |

写作技巧：不要一次性解释全部制度。让角色在具体事件中发现问题：医院窗口显示医保不能结算、房产中介说公积金连续缴存不够、工伤认定窗口要求材料、退休窗口发现档案年龄和身份证年龄不一致。

## 6. 缴费基数、断缴和补缴：最适合制造现实压迫

社保公积金的戏剧性常常不在“有没有交”，而在“按什么工资交、交了几个月、中间断没断、欠缴能不能补、补上后是否影响待遇”。

可写节点：

- 员工工资一万五，社保却按最低基数缴，短期到手多，长期保障缩水。
- 公司口头承诺入职当月缴社保，实际试用期三个月都没缴。
- 角色离职换城市，中间断缴两个月，买房资格、医保待遇或生育待遇被卡住。
- 公司经营困难，工资照发但社保欠缴，员工到医院才发现账户异常。
- 老员工退休前核算待遇，发现早年单位漏缴或档案工龄缺一段。

检查问题：

- 缴费主体是谁？本单位、派遣公司、外包公司、个人灵活就业，还是空白？
- 缴费基数与工资条、银行流水、个税记录是否一致？
- 断缴发生在哪个月，和住院、买房、生育、退休、离职是否撞上？
- 补缴是单位漏缴、个人补缴、历史遗留，还是被角色误以为“花钱就能补”？
- 故事城市是否需要查地方政策？公积金和医保尤其不能全国一刀切。

## 7. 工伤认定与劳动能力鉴定

工伤情节不能只写“老板赔钱”或“员工闹事”。真实的戏剧链条是：事故发生、救治、证明劳动关系、申报工伤、认定、治疗、停工留薪、劳动能力鉴定、待遇支付、复查或争议。

工伤事故叙事链：

```text
工作任务/工作场所/工作时间或相关外出 -> 事故/职业病线索 -> 急救和病历 -> 单位报告 -> 工伤认定申请 -> 劳动关系和事故证据 -> 工伤认定决定 -> 治疗和停工留薪 -> 劳动能力鉴定 -> 工伤待遇或赔偿 -> 复查/争议/仲裁诉讼 -> 职业和家庭余波
```

常见高张力场景：

- 外卖骑手、货车司机、工地工人、仓库分拣员、直播助理、夜班保安都在“是不是劳动关系”上被切割。
- 公司让受伤员工签“自愿放弃工伤”或“按人道补偿了结”，员工家属发现数额远低于应有保障。
- 工伤认定需要工资、考勤、工作安排和事故现场证据，公司却说他是临时帮忙。
- 伤情稳定后，角色以为事情结束，才知道还要劳动能力鉴定，鉴定等级改变长期待遇。
- 未参保不等于没有责任，反而可能把待遇责任压回单位。

证据清单：

- 劳动合同、工牌、考勤、排班、工作群通知、工作任务单、工资流水。
- 事故现场监控、同事证言、报警记录、急救记录、安全培训记录。
- 病历、诊断证明、检查报告、职业病诊断材料。
- 工伤认定申请、受理通知、认定决定、劳动能力鉴定申请和结论。
- 单位沟通记录、补偿协议、离职协议、仲裁材料。

## 8. 退休年龄、缴费年限和养老金：2025年后不能再用老桥段

截至 2026 年，现实写作必须注意：从 2025 年 1 月 1 日起，中国已启动渐进式延迟法定退休年龄改革，并配套弹性退休制度。不能再笼统写“男 60、女 50/55 到点就退”，除非故事时间明确在旧制度下，或角色属于特定历史背景。

写作规则：

- 退休年龄要看出生年月、性别、原岗位或身份类型、法定退休年龄对照和弹性退休规则。
- 领取基本养老金通常还涉及最低缴费年限，不是“年龄到了自动有养老金”。
- 弹性提前或弹性延迟退休有条件和程序，不能写成角色一句话决定。
- 退休返聘、超过法定退休年龄继续工作、劳务关系和劳动关系边界要谨慎处理。
- 特殊工种、因病或非因工致残、工伤、灵活就业、机关事业单位、企业职工之间差异很大，具体剧情要查最新和地方规则。

可写冲突：

- 角色准备按旧年龄退休，窗口查询后发现法定退休年龄已随出生年月延后。
- 女职工因岗位身份不同，退休年龄判断引发单位和个人争议。
- 父亲想弹性延迟退休多缴几年，儿女担心他的身体扛不住。
- 临近退休员工被裁，失业保险、养老缴费和退休时间连锁影响家庭安排。
- 老员工发现档案出生日期和身份证出生日期不一致，退休核定被卡。

## 9. 人事档案：都市文里常被忽略的暗门

人事档案在普通民企日常里似乎没存在感，但一到考编、国企事业单位招聘、政审、职称评审、党员材料、参军、出国境、退休待遇核定、工龄认定，就会突然变成关键证据。

写作规则：

- 流动人员人事档案通常应由公共就业和人才服务机构或经授权机构管理，不是随便放在角色抽屉里。
- 档案不是简历，不能随便改；它强调真实性、严肃性、转递和保密。
- 档案材料可能包括学历、工作经历、党团材料、职称、奖惩、工资工龄、政审材料等。
- 档案丢失、缺材料、年龄不一致、学历经历矛盾、原单位不转档，会影响考编、评职称和退休。
- 私企老板扣押“档案”来威胁员工时，要区分真实人事档案、公司内部员工档案和离职证明。

高张力场景：

- 主角考上事业单位，政审时发现档案还在十年前的学校人才中心。
- 退休窗口说档案里最早招工表缺失，老员工三十年前的工龄认定不了。
- 角色档案年龄比身份证大一岁，影响退休时间和待遇。
- 原单位拒绝配合转档，主角的新单位录用卡在最后一步。
- 人事外包公司弄丢材料，HR 只说“我们系统里查不到”。

## 10. 公积金：买房线和职场线的暗扣

公积金适合连接职场和房产剧情。它不是单纯“公司福利”，而是账户、缴存比例、缴存基数、连续缴存、贷款额度、提取用途、封存转移和地方规则的组合。

写作规则：

- 公积金贷款额度和资格通常具有城市差异，不能全国套一个数字。
- 连续缴存、账户状态、缴存基数、夫妻双方账户都会影响买房节奏。
- 离职、换城市、封存、转移接续，会让买房计划突然卡住。
- 少缴公积金短期提高到手工资，长期影响贷款额度和住房选择。
- 骗提公积金、虚假租房、假材料贷款等不能写成教程，只能作为风险或违法后果处理。

可写桥段：

- 房产中介算完贷款，说“你公积金断过，首付要多准备二十万”。
- 公司按最低基数缴公积金，员工买房时才知道额度少得可怜。
- 离职后账户封存，新城市还没接续，婚房计划被迫延期。
- 配偶发现对方一直没有公积金，才知道所谓“稳定工作”其实是外包。

## 11. 医保、生育和失业：生活风险不等人

这三类保障适合写“制度问题在最脆弱时爆发”。

### 11.1 医保

- 住院前后最能暴露参保地、断缴、异地就医、单位欠缴和待遇等待问题。
- 医保账户、职工医保、居民医保、商业保险和医疗救助不要混写。
- 家属以为“有工作就有医保”，窗口却显示欠缴或暂停待遇，是强现实打击。

### 11.2 生育

- 生育保险与产假、工资、生育津贴、产检和女职工保护联动。
- 断缴、试用期怀孕、入职体检、调岗、绩效考核、产假返岗都能成为剧情冲突。
- 接入第158篇时，要写身体、家庭分工和职场压力，不要只写制度。

### 11.3 失业

- 被裁、合同到期不续、公司倒闭、非本人意愿中断就业，和主动裸辞的后果不同。
- 失业保险不仅是钱，也可能连接临近退休、再就业培训和家庭现金流。
- 失业金争议适合写“离职原因措辞”：解除、辞退、协商一致、个人原因，都会影响后续叙事。

## 12. 单位经办人：制度戏里的关键配角

很多职场保障情节不是老板亲自作恶，而是 HR、财务、人事外包、社保专员、档案管理员、项目经理之间的信息差。

可写配角：

- HR：知道规则，但要在老板成本和员工权益之间夹着。
- 财务：负责缴费和工资，可能拖延、压款、按最低基数操作。
- 社保专员：掌握申报窗口和历史记录，是现实悬疑里的“数据库守门人”。
- 档案管理员：不起眼，却能决定政审、职称和退休核定能否通过。
- 工会或员工代表：可以推动协商，也可能只是摆设。
- 人事外包公司：让责任链变长，适合写推诿和材料丢失。

叙事技巧：让配角说具体话，而不是空泛法条。例如“你不是没交，是三月欠缴挂在系统里”“这份离职证明写个人原因，你再想领失业金就麻烦了”。

## 13. 纠纷处理与责任分流

职场保障纠纷要先判断争议性质，不要所有问题都写成劳动仲裁。

分流规则：

- 劳动关系、工资、解除、经济补偿、未缴社保导致劳动争议：接入劳动仲裁和第162篇。
- 社保登记、缴费记录、待遇经办、基金安全：接入社保经办机构、人社部门和社会保险法链。
- 医保待遇、异地结算、医保账户、医疗费用：接入医保部门和第169篇。
- 公积金开户、缴存、提取、贷款、单位少缴未缴：接入住房公积金管理中心和房产专题。
- 工伤认定、劳动能力鉴定、工伤待遇：接入人社、劳动能力鉴定委员会、医疗和仲裁诉讼链。
- 人事档案接收、转递、查借阅、证明、政审：接入档案管理服务机构、公共就业人才服务机构或用人单位人事部门。
- 伪造材料、骗取待遇、侵占基金、恶意诈骗：接入公安、刑事和金融合规专题。

证据清单：

- 劳动合同、入职登记、offer、员工手册、岗位说明、工牌、门禁、考勤。
- 工资条、银行流水、个税记录、报销单、奖金规则。
- 社保缴费记录、参保证明、欠缴记录、补缴材料、社保卡、医保结算单。
- 公积金缴存明细、贷款申请、提取材料、账户封存和转移记录。
- 工伤事故报告、病历、诊断证明、监控、证人、认定决定、劳动能力鉴定结论。
- 人事档案存档证明、转递单、政审函、职称材料、学历材料、招工表、离职证明。
- 微信群、邮件、电话录音、会议纪要、投诉受理回执、仲裁申请和裁决。

## 14. 类型文里的用法

| 类型 | 用法 | 爽点或痛点 |
| --- | --- | --- |
| 都市现实 | 社保断缴、买房公积金、退休被卡 | 生活不是崩在大事，而是崩在窗口一句“不符合” |
| 职场文 | HR、财务、老板、员工围绕成本和保障博弈 | 制度细节打脸口头承诺 |
| 工厂/工地文 | 工伤认定、外包派遣、劳动能力鉴定 | 受伤后的尊严和长期生计 |
| 家庭伦理 | 父母退休、养老金、医保、工亡抚恤 | 家庭现金流和照护责任 |
| 律政 | 劳动仲裁、社保补缴、工伤待遇、公积金争议 | 证据和程序逆转弱势局面 |
| 重生逆袭 | 提前补齐社保档案、避免工伤私了、守住公积金贷款资格 | 现实经验带来的安全感爽点 |
| 悬疑 | 档案年龄、社保记录、工伤事故时间线 | 制度记录暴露谎言 |
| 买房婚恋 | 公积金、社保连续缴纳、收入证明、贷款资格 | 爱情被缴费记录和城市规则拷问 |

## 16. 写作提示词

### 16.1 生成社保断缴场景

```text
请根据第186篇职场保障与档案纠纷链，生成一个社保断缴导致现实压力爆发的都市场景。必须包含：角色劳动关系、单位经办人、断缴月份、触发事件、窗口反馈、证据材料、家庭或职场后果。不要提供现实补缴教程。
```

### 16.2 审稿工伤情节

```text
请审稿以下工伤情节，检查是否存在：劳动关系不清、事故时间地点任务不清、病历和事故证据缺失、单位责任切割过于方便、工伤认定和劳动能力鉴定混淆、待遇结果过快、违法私了写成爽点、AI 味过重。按“问题-影响-修改建议”输出。
```

### 16.3 设计退休被卡桥段

```text
请设计一个退休办理被卡住的桥段。要求：体现2025年后渐进式延迟退休背景、缴费年限或档案材料问题、窗口经办反馈、家庭现金流压力、一个能推动亲情或旧单位责任线的伏笔。不要把退休写成到年龄自动领钱。
```

### 16.4 设计公积金买房冲突

```text
请设计一个公积金影响买房计划的桥段。要求：写清缴存基数、连续缴存、账户状态或城市规则中的一个问题；让配偶或家人误以为“有工作就能贷”；通过一张缴存明细或贷款预审结果制造冲突。
```

### 16.5 设计人事档案暗线

```text
请设计一个人事档案影响考编、职称或退休的暗线。要求：档案位置、缺失材料、转递流程、经办人反馈和角色旧经历之间形成因果；不要把档案写成普通简历或电脑文档。
```

## 17. 自查清单

- 是否和第162篇分工清楚，没有重复写普通招聘辞退？
- 是否写清劳动关系或用工关系，而不是默认所有工作都有完整保障？
- 是否拆分社保五险和公积金，而不是一句“五险一金齐全”？
- 是否写出缴费基数、缴费月份、断缴欠缴或转移接续？
- 是否让社保/医保/公积金问题在具体事件中爆发？
- 工伤是否有事故、劳动关系、病历、申报、认定、鉴定和待遇链？
- 退休是否考虑2025年后的渐进式延迟退休和最低缴费年限？
- 人事档案是否有位置、转递、材料和用途，而不是普通简历？
- 是否区分社保经办、人社、医保、公积金中心、档案机构、劳动仲裁和法院？
- 是否避免输出骗保、骗提公积金、伪造材料、规避社保的操作方法？
- 是否回写角色的医疗、买房、退休、失业、家庭现金流或职业前途后果？

## 18. 最小可用规则

如果时间很紧，至少执行以下 8 条：

1. 先确认劳动关系和单位主体。
2. 社保要写参保地、缴费月份、缴费基数和断缴情形。
3. 公积金要写账户状态、缴存基数和城市规则差异。
4. 工伤要写事故、病历、劳动关系证据、认定和鉴定。
5. 退休要写出生年月、法定退休年龄变化、缴费年限和档案材料。
6. 人事档案要写存放位置、转递和具体用途。
7. 纠纷要分流到人社、医保、公积金、档案机构、仲裁或法院，不要一锅端。
8. 结尾必须回写待遇、贷款、退休、档案、家庭现金流或职业机会的变化。
