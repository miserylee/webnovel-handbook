# 168. 驾驶车辆、道路交通、交通事故、责任认定与车险理赔专项

状态：已确认  
适用对象：AI agents、小说作者、审稿代理、连续性维护代理  
最后更新：2026-06-09

## 1. 本篇解决什么问题

车辆和道路交通不是“从 A 到 B”的背景板，而是现代都市、家庭伦理、职场通勤、悬疑破案、校园青春、乡村返乡、商战物流和末世逃亡里高频出现的制度化行动场。商业级小说里，一场车戏通常同时承载人物关系、时间压力、证据链、责任归属、金钱损失和后续舆论。

本篇用于让 AI 在写作或审稿时，把驾驶与事故写成一条可追踪的链条：

```text
出行目的 -> 驾驶资格 -> 车辆状态 -> 道路环境 -> 交通规则 -> 驾驶行为 -> 风险触发 -> 事故现场 -> 报警/救助 -> 责任认定 -> 保险理赔 -> 维修/赔偿/处罚 -> 后续行动资格 -> 台账回写
```

本篇重点解决以下失真：

| 失真类型 | 读者体感 | 修正方向 |
|---|---|---|
| 角色随便开车 | 像纸片人移动，没有现实摩擦 | 补驾驶证、准驾车型、车况、停车、路况 |
| 车祸只为制造误会 | 狗血但不可信 | 补现场救助、报警、证据、责任认定、保险和赔偿 |
| 交警只负责训话 | 制度链断裂 | 明确接警、现场处置、事故调查、违法处理和责任文书 |
| 富人撞车直接私了 | 风险被写轻 | 区分轻微财损、伤亡事故、酒驾逃逸、刑事/行政/民事后果 |
| 保险万能兜底 | 金钱逻辑空心 | 区分交强险、商业险、免赔/除外、定损、修车、追偿 |
| 行车记录仪万能 | 悬疑太平 | 记录范围、角度、时间、损坏、剪辑、云端和道路监控都有局限 |

## 2. 与既有专题的分工

- 第 149 篇“旅行路线、交通工具、旅店关卡、地图物流与旅程成本”解决长距离出行、路线和旅程成本。本篇解决车辆上路后的资格、规则、事故和理赔。
- 第 164 篇“火灾灾害、应急疏散、救援事故与灾后余波”解决灾害级事故和应急体系。本篇解决道路交通事故这个高频日常事故类型。
- 第 165 篇“银行贷款、信用保险、税票与金融合规”解决金融与保险合规。本篇只处理车险在交通事故中的叙事用法。
- 第 166 篇“报警接处警、公安办案、侦查措施、拘留逮捕与案件程序”解决公安程序总链。本篇聚焦交警、道路违法、事故认定和交通事故证据链。
- 第 163 篇“数字账号、密码隐私、网络安全、黑客数据与电子证据”可补充行车记录仪、车机数据、手机定位、平台订单、道路摄像头等电子证据边界。

## 3. 安全与合规边界

本篇只用于小说创作、审稿和世界观可信度建设，不提供现实法律意见、保险理赔意见、逃避执法方法或危险驾驶技巧。

AI 不应输出以下内容：

- 不写酒驾、醉驾、毒驾、无证驾驶、套牌、遮挡号牌、逃避测速、逃避检查、肇事逃逸的操作方法。
- 不写伪造事故、骗保、篡改行车记录仪、毁灭证据、买分卖分、冒名顶替处理违法的可执行步骤。
- 不把“私了”写成伤亡事故、酒驾事故、逃逸事故的安全出口。
- 不鼓励读者模仿危险驾驶、飙车、闯红灯、逆行、占用应急车道等行为。
- 需要描写违法行为时，只写人物动机、风险、后果和制度反应，不写可复制技巧。

安全写法可以包括：

- 车辆靠边、人员避险、救助伤者、报警、通知保险、保护现场、配合调查。
- 角色因违法驾驶付出行政、民事、刑事、职业、家庭和心理代价。
- 事故线索通过合法调取、公开监控、行车记录、证人证言、车辆痕迹和保险材料逐步拼合。

## 4. 十行场景卡

写任何驾驶或交通事故场景前，AI 先补齐这十行：

```text
1. 出行目的：角色为什么此刻必须移动？迟到、追人、逃离、送医、接孩子、运货、约会、取证？
2. 驾驶者：谁开车？是否有驾驶证？准驾车型是否匹配？是否疲劳、饮酒、服药、情绪失控？
3. 车辆身份：车是谁的？私家车、公车、租车、网约车、货车、摩托、电动车、共享车？是否登记、投保、年检、保养？
4. 道路空间：城市主干道、高速、乡道、学校门口、小区地库、停车场、雨夜山路、施工路段？
5. 交通规则：此处的限速、信号灯、让行、停车、车道、应急车道、非机动车道规则是什么？
6. 风险触发：超速、分心、抢灯、追尾、盲区、路怒、疲劳、雨雪、视线遮挡、行人突然横穿、车辆故障？
7. 现场后果：人伤、财损、堵车、二次事故、围观、舆论、迟到、错过关键事件？
8. 证据来源：行车记录仪、道路监控、停车场监控、目击者、刹车痕、车辆损伤、手机定位、平台订单、酒精检测、维修记录？
9. 制度介入：交警、急救、消防、保险公司、道路救助基金、单位、学校、平台、媒体？
10. 后续代价：责任认定、扣分罚款、暂扣/吊销、赔偿、保险拒赔或追偿、刑责、职业处分、亲密关系裂痕？
```

## 5. 驾驶资格：不要让角色“会开”就等于“能合法开”

### 5.1 机动车驾驶证

中国大陆现实语境中，驾驶机动车通常必须依法取得机动车驾驶证。小说写作时至少确认三件事：

| 问题 | 写作意义 | 常见剧情用法 |
|---|---|---|
| 角色是否有证 | 决定违法风险和事故后果 | 未成年偷开车、失业司机证件逾期、外地返乡换证 |
| 准驾车型是否匹配 | 决定“会开小车”不能随便开客车、货车、摩托或特种车 | 男主临时开货车救人，结果承担更高风险 |
| 证件状态是否正常 | 决定被查、理赔、职业资格 | 实习期、扣满分、暂扣、逾期未审验、被公告停止使用 |

AI 写驾驶资格时，不要只写“他掏出驾照”。更有效的是把证件和人物处境绑定：

- 新手司机的犹豫、倒车困难、夜路焦虑，能制造生活感。
- 职业司机的准点压力、平台评价、疲劳驾驶风险，能制造职业压迫。
- 家庭角色的“谁来开车”，能暴露权力关系和责任分配。
- 悬疑角色的证件异常，能成为身份线索。

### 5.2 年龄、健康与状态

驾驶不是只看技能，还看身体和心理资源。AI 应检查：

- 是否疲劳：熬夜加班、长途返乡、夜班代驾、货运连续排班，会提高事故风险。
- 是否分心：手机消息、导航争吵、直播、孩子哭闹、车机弹窗，能自然触发失误。
- 是否饮酒或药物影响：只写风险和后果，不写逃避检测办法。
- 是否情绪失控：路怒、追车、争吵后开车、失恋后飙车，都应付出合理代价。
- 是否存在视力、听力、肢体或慢性病限制：可与第 159 篇残障/慢病专题联动，避免污名化。

## 6. 车辆身份：车本身也是角色关系和证据节点

一辆车至少有六层身份：

| 层级 | 需要确认 | 叙事作用 |
|---|---|---|
| 所有人 | 车登记在谁名下 | 暴露财产关系、挂靠关系、夫妻矛盾、公司控制 |
| 使用人 | 实际谁在开 | 区分所有权、使用权和责任承担 |
| 登记状态 | 号牌、行驶证、注册/转让/抵押/注销 | 悬疑身份、债务纠纷、二手车陷阱 |
| 技术状态 | 年检、保养、刹车、轮胎、灯光、电池 | 事故成因、产品质量、维修店责任 |
| 保险状态 | 交强险、商业险、是否过期 | 理赔张力和家庭财务压力 |
| 数据状态 | 行车记录仪、车机系统、定位、充电记录 | 证据链和隐私冲突 |

### 6.1 常见车辆类型的叙事差异

| 类型 | 典型压力 | 常见误区 |
|---|---|---|
| 私家车 | 停车难、保养、剐蹭、家庭成员共用 | 把车写成无限可用，不考虑油电、停车、保险 |
| 公司车 | 用车审批、司机职责、事故报备、单位责任 | 私人挪用后果写轻 |
| 租车/共享车 | 押金、租赁协议、车况交接、异地还车 | 事故后只写赔钱，不写平台和保险链 |
| 网约车/出租车 | 平台订单、评分、乘客纠纷、路线争议 | 忽略平台记录和职业资格压力 |
| 货车/客运车 | 载重、排班、疲劳、货损、营运资质 | 把大车当小车开，忽略盲区和制动距离 |
| 摩托/电动车 | 头盔、非机动车道、充电、改装、地方管理差异 | 不区分机动车、非机动车和地方规则 |
| 新能源车 | 续航、充电桩、热失控风险、车机数据 | 只写高科技，不写补能和数据留痕 |

### 6.2 车况细节让场景更“活”

商业级场景不必堆砌参数，但要选一个能推动剧情的车况细节：

- 雨刷老化，暴雨夜视线一片糊。
- 胎压报警被忽视，长途高速埋雷。
- 后备箱堆满搬家纸箱，倒车镜视野被遮挡。
- 新能源车剩余电量不够，角色在服务区争抢充电桩。
- 公司车行车记录仪被统一上传，角色以为没人知道，实际留下证据。
- 车内儿童安全座椅缺失，家庭责任冲突被放大。

## 7. 道路空间：不同道路带来不同风险和节奏

| 场景 | 主要风险 | 适合制造的剧情 |
|---|---|---|
| 城市主干道 | 信号灯、并线、公交站、外卖骑手、早晚高峰 | 迟到、追赶、轻微剐蹭、路怒冲突 |
| 小区/地库 | 视线盲区、倒车、老人儿童、物业监控 | 家庭矛盾、邻里纠纷、隐蔽证据 |
| 学校/医院门口 | 人流密集、停车混乱、情绪焦虑 | 接送孩子、急诊、亲子压力 |
| 高速公路 | 车速高、疲劳、应急车道、二次事故 | 长途返乡、追车、逃亡、重大事故 |
| 乡村道路 | 无路灯、机非混行、农用车、牲畜、路面破损 | 返乡、扶贫、乡土悬疑、夜路惊险 |
| 山路/隧道/桥梁 | 视线受限、落石、弯道、风雨雪雾 | 悬疑伏击、灾害预兆、心理压迫 |
| 施工路段 | 临时改道、标志不清、车道变窄 | 误入、堵车、责任争议 |
| 停车场 | 低速碰撞、监控盲区、车位纠纷 | 都市生活梗、羞辱反击、偷拍视频线索 |

道路空间要服务人物和节奏。不要每次都写“车流如织”，要写这条路为什么在今天、此刻、对这个人变得危险。

## 8. 高风险驾驶行为：只写风险、动机与后果

AI 可以描写风险行为，但要避免写成教程。

| 行为 | 叙事动机 | 合理后果 |
|---|---|---|
| 超速 | 追赶、迟到、逃避、炫技 | 反应距离不足、责任加重、罚款记分、事故升级 |
| 饮酒后驾驶 | 面子、侥幸、代驾迟到、情绪失控 | 被查、事故、行政/刑事风险、职业声誉崩塌 |
| 疲劳驾驶 | 货运排班、长途返乡、熬夜加班 | 打瞌睡、偏离车道、追尾、同伴劝阻无效 |
| 分心驾驶 | 手机、导航、争吵、直播、孩子哭 | 错过信号、刹车晚、证据被聊天记录反证 |
| 路怒追车 | 被别车、被挑衅、面子受损 | 碰撞、围观偷拍视频、社交媒体发酵 |
| 逆行/闯灯 | 抢时间、道路不熟、恶意逃避 | 事故责任、监控证据、舆论谴责 |
| 超载/违规载人 | 省钱、赶工、平台压单 | 伤亡扩大、单位或平台责任、家庭赔偿压力 |
| 肇事逃逸 | 恐惧、酒驾、无证、车内有秘密 | 后果显著加重、身份暴露、心理崩溃 |

写法原则：风险行为不是爽点本身，而是人物弱点、制度压力和剧情代价的放大镜。

## 9. 交通事故现场链

一场可信的交通事故不只是一声巨响。AI 应按“现场安全 -> 人员救助 -> 证据保护 -> 制度介入 -> 后续处置”写。

| 环节 | 可写内容 | 不宜偷懒写法 |
|---|---|---|
| 事故瞬间 | 声音、惯性、玻璃、气囊、刹车、身体反应 | “车一撞，人就晕，醒来已在医院”反复使用 |
| 二次风险 | 后车、夜间、弯道、雨雪、高速、油电风险 | 所有人站在车道中央吵架却没人避险 |
| 人员救助 | 伤情、急救、旁人报警、亲属慌乱 | 角色先吵输赢，完全不管伤者 |
| 现场保护 | 车辆位置、散落物、刹车痕、视频、目击者 | 角色随便挪车后仍轻松认定责任 |
| 交警到场 | 询问、勘查、测试、调取监控、疏导交通 | 交警凭一句话立刻判定所有复杂事故 |
| 保险介入 | 报案、定损、拖车、修理、赔付争议 | “保险全赔”一句带过所有金钱问题 |
| 余波 | 误工、住院、修车、换车、赔偿谈判、心理阴影 | 事故当天结束，后续无人受影响 |

轻微财损事故可以快写，但涉及伤亡、酒驾、逃逸、公共交通、营运车辆、未成年人、老人、舆论热点时，必须拉长制度链。

## 10. 责任认定与证据：不要让真相只靠主角嘴硬

交通事故责任往往来自多源证据拼合。AI 审稿时要问：谁说了算，证据在哪里，证据是否完整？

### 10.1 常见证据来源

| 来源 | 能证明什么 | 局限 |
|---|---|---|
| 行车记录仪 | 车前/车后画面、时间、速度感、对方行为 | 角度有限、夜间不清、断电损坏、只拍车外 |
| 道路监控 | 信号灯、车道、路口全景 | 调取需要程序，可能有盲区或保存期限 |
| 停车场/小区监控 | 低速碰撞、肇事车辆进出 | 物业管理、清晰度和覆盖范围不稳定 |
| 目击者 | 行人、乘客、店主、保安的观察 | 记忆偏差、立场偏向、视角不完整 |
| 车辆痕迹 | 碰撞角度、刹车、灯光、损伤位置 | 需要专业勘验，不能凭肉眼随便下结论 |
| 手机/平台数据 | 通话、导航、订单、定位、消息时间 | 涉及隐私和调取边界 |
| 酒精/药物检测 | 驾驶状态 | 不应写规避方法，只写检测和后果 |
| 维修/保养记录 | 车况、故障、责任争议 | 可能造假或记录缺失，可作为悬疑点 |

### 10.2 责任认定的叙事用法

责任认定不是“谁有钱谁赔”，也不是“谁哭得惨谁无责”。它可以成为剧情推进器：

- 主角以为自己无责，行车记录仪却显示他最后一秒分心。
- 反派以为删掉车机数据就安全，停车场摄像头拍到车辆进入时间。
- 女主想替家人顶包，但准驾车型、手机定位和伤情时间线对不上。
- 富二代试图私了，伤者后续诊断加重，事故性质升级。
- 职业司机被平台时限逼到疲劳驾驶，责任从个人扩展到管理制度。

## 11. 交强险、商业车险与理赔链

### 11.1 交强险不是“万能赔偿”

中国大陆现实语境中，机动车交通事故责任强制保险是机动车上路的基础强制责任保险，核心是保障道路交通事故中本车人员、被保险人以外的受害人在责任限额内得到赔偿。写作时要注意：

- 交强险具有强制投保属性，未投保会影响车辆登记、检验和上路合法性。
- 交强险的保障对象、赔偿限额和责任边界有限，不等于所有损失都由保险公司兜底。
- 车上人员、被保险人、本车损失、商业车损、精神损害、停运损失等要根据具体保险和法律关系处理，不要一概写成“保险赔”。
- 保险公司赔付后可能存在追偿、责任争议或合同除外情形，适合制造后续压力。

### 11.2 商业车险的叙事张力

商业车险来自合同约定。AI 可以把它用于制造现实摩擦：

| 环节 | 剧情张力 |
|---|---|
| 投保时 | 为省保费没买足、保险快到期、车辆用途写错 |
| 出险时 | 是否及时报案、现场是否改变、驾驶人是否符合条款 |
| 定损时 | 4S 店、维修厂、保险员、车主之间的金额争议 |
| 修车时 | 配件等待、代步车、误工、营运车停运损失 |
| 赔付后 | 责任方追偿、保费上涨、家庭财务压力、单位问责 |

### 11.3 道路交通事故社会救助基金

道路交通事故社会救助基金适合用于伤亡事故中的“先救命、后追偿”叙事，但不能写成普通商业保险。它更适合以下情境：

- 肇事车辆未投保交强险或逃逸，受害人抢救费用一时无法落实。
- 受害人身份暂不明或家庭陷入困难，需要制度性垫付和后续追偿。
- 医院、交警、基金管理机构、责任人之间形成文件和时间压力。

用法提醒：救助基金增强制度真实感，但不要把它写成主角随口申请、秒到账、无追偿的万能金库。

## 12. 后果链：交通事故的价值在“余波”

交通事故最大的商业叙事价值，往往不在撞击瞬间，而在后续余波。

| 后果类型 | 可写内容 |
|---|---|
| 人身后果 | 住院、复健、创伤后反应、照护压力、残障适应、死亡善后 |
| 财产后果 | 修车、折旧、拖车、停车费、货损、营运停摆 |
| 行政后果 | 罚款、记分、暂扣、吊销、学习考试、车辆扣留 |
| 民事后果 | 医疗费、误工费、护理费、赔偿谈判、诉讼、执行 |
| 刑事风险 | 严重伤亡、危险驾驶、交通肇事、逃逸等高风险情节 |
| 职业后果 | 网约车停运、货车司机失业、公职人员处分、明星塌房 |
| 家庭后果 | 谁陪护、谁出钱、谁隐瞒、谁被责怪、婚恋关系裂开 |
| 舆论后果 | 短视频传播、人肉、平台热搜、品牌公关、粉丝控评 |

AI 写事故余波时，至少保留一个“现实账单”：时间账、钱账、证据账、情感账或职业账。

## 13. 弱势交通参与者

道路不是只有汽车。商业级现实感来自对行人、骑行者、老人、儿童、残障人士和职业骑手的可见性。

### 13.1 行人、老人和儿童

- 学校门口、医院门口、菜市场、小区门口是高密度冲突点。
- 老人反应慢、儿童突然奔跑、照护者分心，都能制造事故风险，但不能把弱势群体写成天然麻烦制造者。
- 儿童事故会迅速牵动家庭、学校、物业、司机、保险和舆论。

### 13.2 外卖骑手、快递员与平台时限

- 骑手不是“突然窜出来”的工具人，应补平台算法、订单超时、罚款、天气、装备、家庭收入压力。
- 事故责任可以落在具体行为上，但角色塑造要避免职业污名化。
- 平台数据、订单时间、路线轨迹和配送压力能成为证据链。

### 13.3 电动车、摩托车和自行车

- 不同地区对电动自行车、电动摩托车、改装车、牌照和头盔的管理可能不同。
- 写作时不要笼统称“电瓶车全无责”或“电动车一定违法”。
- 这类车辆在视觉盲区、夜间照明、雨天刹车、路口抢行中很有戏剧张力。

## 14. 类型文中的高价值用法

| 类型 | 用法 |
|---|---|
| 都市言情 | 车内密闭空间、接送、代驾、停车场误会、事故陪护、保险账单考验关系 |
| 家庭伦理 | 谁买车、谁开车、谁赔钱、老人孩子出行、夫妻名下财产与责任 |
| 职场商战 | 公司车、公务行程、司机背锅、货运延误、品牌事故公关 |
| 悬疑刑侦 | 逃逸车辆、套牌、车机数据、事故伪装、监控盲区、保险受益人 |
| 校园青春 | 新手驾驶、共享电动车、校门口事故、家长接送、驾考压力 |
| 娱乐圈 | 酒驾塌房、保姆车事故、粉丝围堵、热搜舆论、商业代言违约 |
| 乡村返乡 | 夜路、农用车、乡道、救护距离、亲戚借车、邻里调解 |

## 15. 人物塑造：开车方式就是性格外化

| 开车细节 | 可能透露的人物 |
|---|---|
| 每次上车先调座椅、看后视镜 | 谨慎、有秩序、受过训练 |
| 永远抢黄灯、贴车尾 | 控制欲强、急躁、侥幸心理 |
| 车内干净到没有一张纸 | 克制、洁癖、隐藏生活痕迹 |
| 副驾堆满发票和药盒 | 疲惫、生活失控、长期照护压力 |
| 开车时不停接工作电话 | 被职场吞没，事故风险正在累积 |
| 开旧车但保养极好 | 经济有限但自尊和责任感强 |
| 开豪车却不懂基本规则 | 阶层特权、被照顾惯了、反差喜剧 |
| 出事故第一反应护住乘客 | 关系优先级和真实情感暴露 |

## 16. 审稿坏味道与修复办法

| 坏味道 | 为什么削弱商业感 | 修复办法 |
|---|---|---|
| “他开车飞驰而去”反复出现 | 移动没有代价 | 加入时间、路线、停车、堵车、证据留痕 |
| 车祸只为让男女主同住医院 | 情节工具感强 | 加责任认定、家属到场、赔偿、工作耽误 |
| 交警一眼判全责 | 程序过快 | 加现场勘查、询问、监控和文书时间 |
| 主角酒驾仍被美化 | 价值风险高 | 改为代驾、劝阻、后果承担或反派行为 |
| 行车记录仪拍到一切 | 悬疑失去张力 | 设置角度盲区、断电、夜间眩光、时间戳争议 |
| 保险“全赔” | 金钱逻辑虚 | 明确交强险、商业险、定损、免赔/除外、追偿 |
| 肇事逃逸后轻松洗白 | 读者反感 | 强化法律后果、心理代价、受害者视角 |
| 大货车像小轿车一样灵活 | 物理不可信 | 补盲区、载重、制动距离、职业排班 |
| 小区事故无人调监控 | 生活常识断裂 | 加物业、保安、门禁、监控保存期限 |
| 事故后没有任何后续 | 戏剧资源浪费 | 写复诊、修车、保险员、赔偿谈判、舆论二次发酵 |

## 18. 可直接给 AI 的写作提示词

### 18.1 生成交通事故场景

```text
请为中文商业网文生成一场交通事故场景。先补齐：出行目的、驾驶者资格和状态、车辆身份、道路空间、风险触发、现场后果、证据来源、交警/急救/保险介入、责任争议、后续代价。事故描写要有感官细节和制度余波，但不要提供危险驾驶、逃避执法、骗保或毁灭证据的操作方法。场景必须推动人物关系或主线悬念，不能只为狗血。
```

### 18.2 审稿交通事故可信度

```text
请审查以下章节中的驾驶、道路交通和事故描写。按“驾驶资格、车辆状态、道路空间、风险行为、现场处置、证据链、责任认定、保险理赔、后续余波、安全边界”逐项指出不可信之处，并给出商业级修复建议。不要只润色句子，要补齐制度链和人物代价。
```

### 18.3 改写轻飘车戏

```text
请把以下“开车赶路/撞车/剐蹭”片段改写得更真实。要求：保留原剧情目的，增加一个道路空间压力、一个车辆或驾驶状态细节、一个证据留痕、一个后续成本，并让人物性格通过开车方式外化。不要扩写成法律科普。
```

### 18.4 悬疑取证提示词

```text
请围绕一场疑似伪装成交通事故的案件设计证据链。证据来源必须合法且多源，包括行车记录仪、道路监控、目击者、车辆痕迹、手机或平台数据、维修记录中的若干项。请说明每项证据能证明什么、有什么局限、会如何误导侦查。不要提供伪造事故、逃避侦查或毁灭证据的步骤。
```

## 19. 写作前自查清单

- 角色为什么必须此刻出行，而不是随便移动？
- 谁开车？他是否有合法驾驶资格和匹配准驾车型？
- 车是谁的？车辆是否登记、投保、保养，车内是否有关键物件？
- 这条路为什么在此刻危险？天气、时间、道路、车流、人流有什么压力？
- 风险行为来自人物性格、外部压力还是制度压迫？
- 事故后第一反应是否符合人身安全和救助逻辑？
- 证据来自哪里？有没有盲区、保存期限、调取程序或隐私边界？
- 责任认定是否有过程，而不是一句话判定？
- 保险是否区分交强险和商业险？有没有定损、维修、赔付或追偿争议？
- 后续是否至少留下一个时间账、钱账、证据账、情感账或职业账？
- 是否避免了危险驾驶、逃避执法、骗保、毁灭证据的可执行描写？

## 20. 最小规则

1. 每一场车戏都要有“目的、驾驶者、车辆、道路、风险、证据、后果”。
2. 驾驶资格、车辆登记、保险状态和道路规则，是现实感的四根骨头。
3. 交通事故不要只写撞击，要写救助、报警、证据、责任、理赔和余波。
4. 行车记录仪、道路监控、车机数据都不是万能真相，只是证据链的一环。
5. 违法驾驶可以成为人物缺陷或反派行为，但必须有后果，不能被美化成爽点。
6. 交强险负责基础强制责任保险，商业险按合同，救助基金是特殊救助机制，不要混成“保险全赔”。
7. 交通事故最有价值的戏，常常发生在医院走廊、交警队、修理厂、保险电话、家属争吵和舆论发酵里。
