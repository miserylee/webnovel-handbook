# 219 县域文物保护、非遗传承、古建筑古村落、考古发现、博物馆与文旅开发保护链专项

状态：已确认  
更新时间：2026-06-09  
适用对象：县域现实题材、乡村振兴题材、家族宅院题材、民俗非遗题材、文旅开发题材、悬疑调查题材、年代与地方志题材。

## 1. 本专题解决什么问题

AI 写县域文物、非遗和古村开发时，最容易把“保护对象”写成“爽文道具”：

- 把祖宅、祠堂、古桥、老街写成主角随意拆改的私产。
- 把地下发现的陶片、墓葬、古钱币写成主角偷偷挖、偷偷藏、偷偷卖的发财线。
- 把考古写成盗墓冒险，把文物鉴定写成江湖捡漏，把博物馆写成随便借展和私人交易的仓库。
- 把非遗写成“穿个戏服、开个直播、摆个摊”的视觉标签，忽略传承人、保护单位、活态传承和商业化压力。
- 把古村古镇开发写成“老板来了就拆、网红来了就火”，缺少规划、审批、消防、安全、居民生活和保护边界。

本专题的目标不是替代法律文本，而是给 AI agent 一套小说创作可执行的规则：先识别保护对象，再写权责链，再写利益冲突，最后写人物选择。

## 2. 一句话原则

涉及文物、非遗、古建古村、博物馆和文旅开发时，小说不能只问“这个东西值多少钱、能不能火”，必须先问“它属于什么保护对象、谁有权处理、发现后必须怎么处置、商业利用会伤害谁”。

## 3. 标准叙事链

```text
土地/老宅/祠堂/古桥/老街/作坊/民俗/技艺/传说
-> 判断是否涉及文物、不可移动文物、地下文物、历史建筑、传统村落、非遗代表性项目、代表性传承人、博物馆馆藏
-> 查登记、认定、公布、保护范围、建设控制地带、保护规划、传承记录
-> 日常使用、修缮、展示、传承、培训、经营或文旅开发
-> 触发冲突：施工发现、拆改装修、商业包装、事故损坏、权属争议、流量炒作、盗掘倒卖、居民利益分配
-> 第一反应：停止破坏性行为、保护现场、报告主管部门、固定证据、安抚群众、隔离安全风险
-> 部门介入：文物、文旅、住建、自然资源、公安、市场监管、消防、乡镇、村委会、博物馆或专家组
-> 调查评估：文物认定、考古调查勘探、保护修缮方案、非遗保护计划、展陈或利用方案、违法责任判断
-> 处理结果：保护、修缮、迁建限制、延期施工、补偿协调、行政处罚、刑事移送、传承扶持、文旅整改
-> 回写人物和村庄：谁保住了记忆，谁失去短期利益，谁被迫承认祖辈历史，谁学会在保护和生计之间谈判
```

## 4. 当前可用的制度底座

### 4.1 文物保护法的小说转化

已确认规则：新修订《中华人民共和国文物保护法》自 2025 年 3 月 1 日起施行。写当代情节时，AI 应按这一时间点处理。

创作转化：

- 建设工程、农业生产、宅基地整理、修路、挖沟、河道清淤中发现疑似文物，正确戏剧动作是“停、护、报”，不是“挖、藏、卖”。
- 地下埋藏文物不是发现者的私人财产；主角不能因为“祖传”“自家地里挖出来”就自由处置。
- 文物价值不能只写钱，还要写历史、艺术、科学价值，以及地方身份记忆。
- 不可移动文物、古建筑、古墓葬、古遗址、石刻、近现代重要史迹等，可能与房屋、土地、宗族和村集体利益交叉。
- 文物保护不是“谁有情怀谁说了算”，要落到保护范围、建设控制地带、修缮方案、考古调查、专家论证和责任追究。

风险：

- 如果主角带头私挖、私藏、倒卖，而叙事又奖励他，会让故事站到违法行为一边。
- 如果把文物部门写成只会阻挠发展，会削弱现实可信度；更好的写法是让部门、村民、开发者和传承人各有合理压力。
- 如果把“古董发财”当主线，很容易滑向盗墓、倒卖、鉴宝爽文，脱离县域现实题材。

检查问题：

- 发现文物后的第一反应是否是保护现场并报告？
- 主角是否把文物当成私人物品交易？
- 施工方、乡镇、村委会、文物部门、公安是否都有各自行动？
- 是否写出了延期施工、补偿、舆情、群众误解、专家评估等现实成本？

### 4.2 非遗法的小说转化

非物质文化遗产不是“古风装饰”，而是活态传承：技艺、仪式、口头传统、表演艺术、民俗活动、传统知识与实践，以及承载这些内容的人和社区。

创作转化：

- 非遗重点不是“摆拍好看”，而是“谁会、怎么学、靠什么活、如何传给下一代”。
- 代表性项目、保护单位、代表性传承人、传习所、展演活动、补助培训，都可以成为现实情节支点。
- 非遗商业化可以写，但要写边界：过度简化、乱改仪式、低价压榨传承人、把学徒当流量工具，都会制造矛盾。
- 年轻人加入非遗线时，不要只写“回乡治愈”，还要写收入、审美差异、市场渠道、师徒关系、家庭反对和时间成本。

风险：

- 把非遗写成“老人守旧、年轻人直播一下就救活”，会显得轻飘。
- 把传承人写成纯工具人，会失去人物温度。
- 把民俗仪式完全改成商业表演，会让读者感觉作者不尊重地方文化。

检查问题：

- 这一项非遗具体依附于哪些人、场地、季节、材料、禁忌和社区关系？
- 传承人为什么愿意或不愿意教？
- 年轻角色能带来什么新渠道，又会造成什么变形风险？
- 文旅公司、村集体和传承人之间的钱如何分？

### 4.3 历史文化名城名镇名村、传统村落和古建筑

古村、老街、古桥、祠堂、老宅、戏台、牌坊、古井、码头，不只是“场景氛围”，可能牵涉不可移动文物、历史建筑、传统村落保护、风貌控制、消防安全和居民权益。

创作转化：

- 老宅是否能改民宿，不能只看房主愿不愿意，还要看房屋性质、保护等级、结构安全、消防、修缮方案和周边风貌。
- 古村开发不是“刷白墙、挂灯笼、开咖啡店”；更真实的矛盾是保护原真性、改善居住条件、引入游客、安置老人、收益分配之间拉扯。
- 传统村落保护既要防止拆真建假，也要防止“只许观赏不许生活”，把本地居民挤出自己的村庄。
- 古建筑修缮不是普通装修，不能随便换梁、刷漆、开洞、贴瓷砖、装巨大招牌。

风险：

- 把保护写成绝对浪漫，会忽略漏雨、潮湿、老人摔倒、消防不达标等真实生活难题。
- 把开发写成绝对邪恶，会失去商业级故事的复杂度；更有张力的是“开发有必要，但方式错了”。
- 把老宅全部归某个主角单独决定，会忽略宗族、共有产权、村集体、租户和历史保护限制。

检查问题：

- 这栋老宅或这条老街有没有登记、挂牌、保护范围或风貌要求？
- 居民生活需求是否被写入，而不是只服务游客镜头？
- 开发方案有没有消防、安全、污水、停车、人流承载等现实问题？
- 是否出现“拆真古董、建假古镇”的讽刺或反思？

### 4.4 考古发现与博物馆

考古不是寻宝，博物馆不是仓库。县域故事里，考古和博物馆更适合承担“地方记忆被重新确认”的功能。

创作转化：

- 修路、建厂、河道治理、矿山治理、棚改、宅基地整理，都可能触发地下文物发现或考古调查。
- 博物馆可以承担收藏、保护、研究、展示、教育功能，也可以成为地方身份、亲情记忆、案件线索和公共舆论的聚点。
- 展览线要注意来源、登记、保管、借展、保险、运输、修复、说明牌、讲解词和公众教育。
- 可以写基层博物馆经费紧、库房小、专业人员少、展陈老旧，但不要把它写成随便卖文物、随便收赃物、随便外借真品。

风险：

- 考古情节如果提供盗掘步骤、探墓技巧、规避监管方法，会越界。
- 博物馆如果被写成“主角想拿就拿”，会极不真实。
- 把专家写成万能鉴宝机器，会削弱专业可信度。

检查问题：

- 文物来源是否清楚？
- 收藏、展示、借用、修复有没有手续？
- 考古发现是否由有资质单位和主管部门处理？
- 这个发现改变了谁对家族、村庄或地方历史的理解？

## 5. 常见角色与真实压力

| 角色 | 想要什么 | 怕什么 | 可写冲突 |
| --- | --- | --- | --- |
| 老宅房主/宗族长辈 | 修房、保祖业、体面 | 房塌、没钱、被外人抢话语权 | 不愿挂牌、又怕祖屋毁在自己手里 |
| 年轻返乡者 | 做民宿、做品牌、做直播 | 投入打水漂、被说不懂规矩 | 想商业化，却被保护红线和长辈禁忌拦住 |
| 非遗传承人 | 有徒弟、有尊严、有收入 | 技艺断代、被廉价表演化 | 接不接受网红团队改编 |
| 村干部 | 保稳定、争项目、促增收 | 群众误解、舆情、项目延期 | 一边催开发，一边被要求保护原真性 |
| 文物/文旅干部 | 守底线、做利用、控风险 | 被骂拖后腿、出事故担责 | 面对招商压力仍要求停工评估 |
| 开发商/运营商 | 快速改造、形成卖点 | 审批慢、成本高、舆论翻车 | 把古村当布景，引发居民和传承人反弹 |
| 博物馆馆员 | 保护藏品、做展览、做教育 | 经费少、人手少、文物受损 | 用小展览撬动地方认同 |
| 公安/市场监管 | 打击盗掘倒卖、管交易市场 | 证据链不足、线索断 | 文物线与诈骗、赝品、非法交易交织 |
| 普通村民 | 改善住房、分到收益、生活方便 | 被赶出村、游客扰民、祖坟祖屋被动 | 支持开发但反对“只让外人赚钱” |

## 7. 高质量剧情用法

### 7.1 工地挖出东西

低质写法：主角半夜下坑，挖出宝贝，一夜暴富。  
高质写法：挖机铲出陶片后，工头想赶工，村民围观拍视频，村干部怕项目停摆，年轻主角先拉警戒线、劝停机器、联系乡镇和文物部门；之后考古评估导致工期延期，补偿、舆情、招商压力一起上来。

可写爽点：主角不是“偷宝贝爽”，而是“在所有人都想糊弄过去时守住底线，并找到保护与项目调整的方案”。

### 7.2 祖宅改民宿

低质写法：女主回乡继承祖宅，刷白墙、买软装、开业爆红。  
高质写法：祖宅疑似历史建筑，族叔有份额，老人还住一间，屋顶漏水，消防不达标，运营方想拆掉旧门槛方便游客拍照；女主必须先查权属和保护状态，再找工匠做最小干预修缮，最后用家族故事和地方工艺做低密度经营。

可写爽点：不是豪装砸钱，而是“懂规矩、懂材料、懂人心”。

### 7.3 非遗直播爆红

低质写法：年轻人拍个短视频，非遗传承人瞬间全国爆火。  
高质写法：直播带来订单，也带来低价仿品、游客打扰、表演时间被压缩、徒弟只学镜头不学基本功；传承人想拒绝商业化，年轻人想保住收入，两人争吵后重做产品线：体验课、定制品、展演、学徒培养分开设计。

可写爽点：不是“流量万能”，而是“让流量服从技艺和人”。

### 7.4 传统村落开发

低质写法：外地公司入驻，一周变网红古镇，全村发财。  
高质写法：公司想统一门头、拆掉杂物间、把老人迁到新楼；村民担心游客扰民、租金分配不公，文物和住建部门要求保留街巷肌理、控制招牌和消防通道。主角组织村民议事，最后把开发范围、居民通道、非遗空间、收益分成、修缮基金写进协议。

可写爽点：不是“招商压倒一切”，而是“谈出一套让村庄还能活着的开发”。

### 7.5 家里翻出老物件

低质写法：主角拿去古玩市场捡漏。  
高质写法：老人从箱底翻出旧器物和一段家族秘密，主角先找博物馆或文物部门咨询，确认是否属于文物或有重要地方史价值；家族成员围绕“捐、藏、展、讲不讲旧事”争执，最后通过展览说明牌修复一段被误解的家族关系。

可写爽点：不是“卖了多少钱”，而是“被遮住的历史终于有名字”。

## 8. 与既有专题的分工

- 与 `195` 县域文旅、乡村旅游和民宿开发专题：`195` 管产业、游客、运营和乡村项目；本篇管文物、非遗、古建古村的保护红线与活态传承。
- 与 `194` 县域土地征收、承包地和宅基地专题：`194` 管土地权属、征收、补偿和宅基地审批；本篇管土地施工中发现文物、古村保护范围和建设控制冲突。
- 与 `152` 住房、宅基地、房屋继承专题：`152` 管房屋居住、继承、买卖、修缮纠纷；本篇管老宅一旦涉及古建筑、历史建筑或传统村落后的特殊限制。
- 与 `185` 殡葬、公墓和祖坟专题：`185` 管殡葬秩序和祖坟伦理；本篇管古墓葬、墓葬文物和保护对象，不写盗墓爽点。
- 与 `120` 刑事侦查、`138` 逻辑谜题专题：这些专题管案件结构和推理；本篇负责把文物犯罪、盗掘倒卖、赝品诈骗等情节控制在合法叙事与安全边界内。

## 9. 禁写与降风险清单

AI agent 写作时，应避免下列内容：

- 详细描写盗掘古墓、探测定位、开墓取物、规避巡查、销赃倒卖的可操作步骤。
- 把私挖、私藏、倒卖文物写成被叙事奖励的成功路径。
- 写主角以“自家地里挖出来”“祖上传下来”为由随意出售疑似文物。
- 写无资质团队私自考古、私自修复、私自迁移不可移动文物，并把结果写成正义。
- 把非遗传承人压成流量工具，不写收入、尊严、师徒和社区关系。
- 把传统村落改造成假古镇，不写居民生活、消防安全、修缮成本和原真性损失。
- 把博物馆写成随意买卖藏品、随意出借真品、随意让主角带走文物的地方。

如果剧情必须出现违法行为，应让叙事立场清楚：这是风险、罪错或冲突来源，不是教程和奖励。

## 10. 审稿检查问题

1. 这段剧情里的“老物件、老房子、老技艺、老村落”是否被识别为可能的保护对象？
2. 发现疑似文物后，人物有没有停止破坏、保护现场、报告，而不是私自处理？
3. 文物、文旅、住建、公安、消防、乡镇、村委会、博物馆等主体是否有合理分工？
4. 非遗是否写出了传承人、学徒、社区、材料、季节、收入和商业化边界？
5. 古村开发是否同时写了保护、生活、生计、游客体验和安全，而不是只写网红装修？
6. 文物或非遗的价值是否超越金钱，能改变人物对家族、地方和自身身份的理解？
7. 是否出现盗掘、倒卖、私藏等危险内容？如果出现，叙事是否明确否定并承担后果？
8. 结局是否不仅解决项目，还让村庄记忆、人物关系或地方公共利益发生变化？

## 11. 可复用提示词

### 11.1 起草提示词

```text
请按“县域文物非遗保护链”写这一场：先判断保护对象，再写发现/认定/登记或报告动作，再写村民、干部、部门、开发方、传承人之间的利益冲突。不要把文物写成主角私人物品，不要写盗掘或倒卖教程。重点写保护与生计之间的谈判，以及这个保护对象如何改变人物对家族或地方的理解。
```

### 11.2 审稿提示词

```text
请审查这段文物/非遗/古村开发剧情是否存在以下问题：私挖私藏被奖励、考古写成盗墓、老宅随便拆改、非遗只剩摆拍、古村开发不写居民生活、博物馆手续缺失、部门职责混乱。请按“问题-读者体感-修订建议-可替换情节”输出。
```

### 11.3 人物深化提示词

```text
请为这位传承人/老宅房主/博物馆馆员/村干部设计三层压力：制度压力、生计压力、情感压力。要求每层压力都能在一个具体场景中表现出来，并且不能把人物写成单纯阻碍主角的人。
```

## 12. 写作提醒

本专题的商业价值不在“古董值钱”，而在“旧物、旧宅、旧技艺把人重新拽回历史和共同体”。

好的县域文物非遗线，应该让读者同时看到三件事：

- 眼前利益为什么诱人。
- 保护底线为什么不能退。
- 人物为什么愿意为一个看似不会说话的旧物、旧屋、旧技艺付出代价。
