## 1. 本专题解决的 AI 写作问题

AI 写职校、技校和实习时容易犯这些错：

1. 把职业教育写成“差生收容所”，忽略职业教育与普通教育同等重要的类型教育定位。
2. 把中职、职高、技校、高职、技师学院、职业培训机构混成一个模糊的“技校”。
3. 把专业选择写成家长一句“去学护理/汽修/电商”，缺少招生、学籍、课程、实训、资助、升学和就业。
4. 把校企合作写成学校把学生送进厂，忽略合作协议、培养方案、岗位匹配、管理责任和学生权益。
5. 把学生实习写成廉价劳动力，缺少认识实习、岗位实习、实习协议、报酬、安全、保险、投诉和禁止事项。
6. 把实习单位写成随便安排夜班流水线，忽略未成年学生、专业相关性、工作时间、休息休假、报酬和安全生产边界。
7. 把中职资助写成“穷学生被施舍”，忽略免学费、国家助学金、奖学金、资助卡、信息系统和隐私尊严。
8. 把职业技能证书写成毕业自动附赠，忽略技能等级、职业资格、证书核验和岗位真实能力。
9. 把就业推荐写成包分配，忽略企业面试、岗位适配、薪资社保、试用期、晋升和学历歧视。

本篇把职业教育写成“身份选择、技术训练、家庭期待、学校管理、企业用工、学生权益和县域产业”共同作用的成长系统。

## 2. 总原则：职业教育不是失败人生，实习不是廉价用工

职业教育的戏剧张力，不在“考不上高中所以人生完了”，而在：

- 普通教育评价体系和技能成长路径之间的冲突。
- 家庭短期赚钱压力和学生长期职业发展的冲突。
- 学校招生规模、专业建设、企业需求和学生个人兴趣之间的错位。
- 技能训练需要真实场景，但真实场景不能把学生变成廉价劳动力。
- 县域产业需要人，学生也需要尊严、技能、证书、升学通道和合法权益。

标准链条：

```text
初中毕业或转学转轨压力
-> 家庭、老师和学生判断普高/中职/技校/外出打工/复读等路径
-> 招生宣传、学校资质、专业设置、学籍和资助政策确认
-> 入学、课程学习、文化课、专业课、实训室和技能竞赛
-> 免学费、助学金、奖学金、住宿生活费和家庭现金压力
-> 校企合作或专业实训基地进入学习计划
-> 认识实习或岗位实习安排
-> 学校、实习单位、学生三方协议和安全教育
-> 实习岗位、报酬、保险、工作时间、专业匹配和指导老师管理
-> 实习问题：岗位不符、强制实习、扣费、拖欠报酬、夜班高危、受伤、投诉
-> 技能证书、毕业、升学、就业推荐或返乡创业
-> 学历、技能、家庭关系、县域产业和自我评价回写
```

AI agent 写作时，先问：

- 这是中职、职业高中、技工学校、高职，还是校外培训机构？
- 学生当前是招生入学、在校学习、实训、实习、升学、就业，还是维权？
- 这个“实习”是认识实习还是岗位实习？是否有协议、报酬、保险和管理责任？
- 冲突来自家庭压力、学校管理、企业用工、学生权益、专业错配，还是社会偏见？

## 3. 法律与政策底座

### 3.1 职业教育法：职业教育是与普通教育具有同等重要地位的教育类型

2022 年修订的职业教育法明确，职业教育包括职业学校教育和职业培训，强调立德树人、德技并修、产教融合、校企合作、面向市场、促进就业、面向实践、强化能力。职业教育不是普通教育的失败出口，而是培养技术技能人才的类型教育。

写作迁移：

- 不要把职校学生天然写成差生、混混或廉价工人。
- 职业教育要写“专业、技能、实训、职业道德、就业创业、升学贯通”，而不是只写纪律差。
- 家庭和社会偏见可以存在，但文本不能默认偏见正确。
- 职业学校教育与职业培训要区分：学历职业学校有学籍、课程、毕业和资助；培训机构只提供培训服务或技能提升，不等同学历。

### 3.2 国家职业教育改革实施方案：职业教育从规模扩张转向质量和类型教育

国家职业教育改革实施方案强调现代职业教育体系、产教融合、校企合作、1+X 证书制度试点、职业院校质量提升和技能人才培养。

写作迁移：

- 高质职教桥段不只写“学一门手艺”，要写专业群、产业需求、技能标准、证书、竞赛、实训设备和企业评价。
- 职业教育有升学通道和终身学习接口，不是一次分流决定终身阶层。
- 产业变化会改变专业冷热：新能源车、护理养老、电商直播、智能制造、文旅民宿、低空经济、现代农业都能进入县域职教叙事。

### 3.3 校企合作促进办法：合作要有协议、机制和权益保障

职业学校校企合作促进办法明确，校企合作是职业学校和企业通过共同育人、合作研究、共建机构、共享资源等方式实施的合作活动。合作不只是安排实习，也包括专业建设、课程开发、教师实践、学生就业创业、技术研发和社会服务。

写作迁移：

- 校企合作不能只写“学校和工厂有关系”。要写合作协议、专业培养方案、实训基地、企业导师、教师下企业、岗位需求和质量评价。
- 企业不是慈善，也不是单纯剥削者。它要人、要技能、要稳定；学校要教学质量和就业率；学生要技能、报酬、安全和未来。
- 好冲突来自三方目标不一致：学校要完成实习计划，企业要订单产能，学生想学真技能或准备升学。

### 3.4 职业学校学生实习管理规定：实习是教育教学活动，不是普通派工

2021 年修订的职业学校学生实习管理规定明确，职业学校学生实习包括认识实习和岗位实习，是按照专业培养目标和人才培养方案安排的实践性教育教学活动。规定要求学校、实习单位和学生签订实习协议，明确实习内容、岗位、时间、地点、报酬、责任保险、管理责任和权益保障，并设置禁止性要求。

写作迁移：

- 实习必须和专业培养相关，不能随便把学生送去与专业无关的流水线。
- 岗位实习不是普通劳动合同，但也不能因此剥夺学生安全、报酬、休息和申诉权利。
- 实习报酬、支付周期、是否扣费、是否经过第三方转发、是否强制实习、是否收押金/管理费，都能成为剧情冲突。
- 未成年学生、高危岗位、夜班、加班、单独顶岗、强制指定实习单位等，要特别谨慎描写并体现风险和监管。

### 3.5 中职资助：免学费和助学金不是施舍，而是教育公平工具

学生资助资金管理和中等职业教育资助政策覆盖中职免学费、国家助学金、奖学金等。资助申请、审核、公示、资助卡发放、系统数据和资金监管都可能进入县域教育叙事。

写作迁移：

- 贫困学生上中职不是“被同情”，而是依法享受资助政策。
- 资助材料、家庭经济困难认定、公示隐私、助学金到账、班主任通知和学生自尊，能制造很细的情绪戏。
- 学校或工作人员挤占挪用、虚报冒领、截留资助资金，应进入财政、教育、审计、纪检或司法链，不写成“小误会”。

### 3.6 学籍、技能证书和就业：毕业不是自动有工作

职业学校学生有学籍、课程、考核、实习和毕业要求。技能等级证书、职业资格和毕业证各自功能不同。就业推荐、校园招聘、企业面试、试用期和劳动合同又是下一条链。

写作迁移：

- 毕业证、技能等级证、职业资格、培训结业证不要混用。
- “包就业”要拆成推荐、面试、录用、试用、劳动合同、工资社保和岗位晋升，而不是包分配。
- 职校学生也可以升学、专升本、考技能证、进厂、进服务业、返乡创业或转行，不要只有进厂一条路。

## 6. 程序节点写法

### 6.1 初中后选择：不是一句“考不上就去技校”

低质写法：

> 他没考上高中，只能去技校，人生完了。

高质写法：

```text
中考成绩出来
-> 班主任分别介绍普高、职高、中专、技校、民办学校和外出打工风险
-> 家里希望孩子早点赚钱
-> 孩子想学新能源汽车维修，但亲戚说“修车没出息”
-> 学校开放日展示实训车间、升学班和企业合作基地
-> 主角第一次不是被分数定义，而是被迫问自己愿不愿意认真学一门技能
```

### 6.2 招生宣传：专业不是愿望清单

招生桥段至少写清：

- 学校是中职、职业高中、技工学校、民办培训机构，还是高职院校。
- 是否有办学资质、招生计划、学籍和收费项目。
- 专业名称、课程、实训条件、升学方向和就业方向。
- “包就业”“高薪就业”“升本科”是否被夸大。

高质冲突：

> 招生老师说电商专业“毕业就是主播运营”，学生入学后发现课程从商品拍摄、客服、数据表格和仓库打包开始。她不满，老师却把直播后台打开给她看：镜头前的一分钟，背后是七个人的流程。

### 6.3 校企合作：协议、培养方案和真实岗位都要出现

```text
县职校和新能源零部件厂合作开订单班
-> 学校展示合作协议和培养方案
-> 企业提出岗位技能要求
-> 专业课教师发现现有设备落后
-> 学生期待毕业进厂，家长担心被绑定
-> 企业师傅进校上课，学生去车间认识实习
-> 真问题出现：企业想提前要人，学校坚持不能压缩课程和安全教育
```

关键：合作不是贴牌，冲突来自育人目标和生产目标的拉扯。

### 6.4 岗位实习：三方协议是底线材料

岗位实习桥段至少保留：

- 实习单位名称和岗位。
- 是否与专业相关。
- 学校、学生、实习单位协议。
- 实习时间、地点、内容、报酬和支付方式。
- 指导教师、企业导师或带教师傅。
- 责任保险和安全教育。
- 投诉渠道和突发情况处理。

低质写法：

> 学校把全班送进厂，大家打工到毕业。

高质写法：

```text
护理专业学生被安排到养老机构实习
-> 学校召开实习说明会，发放实习协议和安全告知
-> 学生发现实际岗位更多是生活照护，不是临床护理
-> 企业说养老照护也是专业内容
-> 指导老师要求重新拆分岗位职责和学习目标
-> 实习报酬、夜班、心理压力和老人突发情况成为真正考验
```

### 6.5 实习权益：学生不是正式员工，但不是没权利

常见红线和风险：

- 强制安排到与专业无关岗位。
- 不签实习协议或协议内容空泛。
- 收取实习押金、管理费、就业服务费等。
- 扣押身份证、学生证或证件。
- 拖欠或克扣实习报酬。
- 安排过度加班、危险岗位、夜班或不符合未成年人保护要求的工作。
- 通过中介层层转派，学校、企业和第三方责任不清。
- 实习事故后互相推责。

写作规则：不要教学生如何规避管理，也不要教学校企业如何规避规定；只写风险、证据和后果。

### 6.6 资助：到账的一笔钱，可能撑住一个学期

```text
中职学生申请国家助学金
-> 班主任提醒准备家庭经济困难材料
-> 学生怕被同学知道家里情况
-> 学校按程序认定、公示必要信息并保护隐私
-> 助学金通过资助卡到账
-> 家里想拿这笔钱还债，学生想买实训工具
-> 一笔小钱牵出家庭权力、尊严和继续读下去的可能
```

关键：资助不是“可怜人设”，而是让角色有继续学习的现实条件。

### 6.7 技能证书和就业：证书不是外挂，岗位才是检验

```text
学生考下技能等级证书
-> 校招企业仍要求现场操作
-> 她在实训室能完成，在企业节拍下却紧张出错
-> 师傅没有嘲笑，只让她重新按安全规范做一遍
-> 证书证明她入门，岗位让她知道熟练还要时间
```

写作重点：技能成长要通过手、眼、时间、错误和反馈呈现，不能靠“拿证”瞬间升级。

## 8. 高质量桥段示例

### 8.1 中考后被分流的女孩学会选择

```text
中考成绩未达普高线
-> 亲戚饭桌上说她“只能去职校”
-> 班主任带她和母亲参加职业学校开放日
-> 她看到护理、养老、康复、幼保和电商直播专业的实训室
-> 发现每个专业都有不体面的辛苦，也有被需要的技能
-> 她选择养老服务专业，不是认输，而是因为奶奶失能后她第一次知道“会照护”也是本事
```

### 8.2 订单班不是包分配

```text
县职校开设智能制造订单班
-> 企业承诺优先录用优秀毕业生
-> 学生家长误以为进班就等于包工作
-> 学校说明课程、考核、实习和企业面试要求
-> 主角发现自己理论一般但实操很稳
-> 最后进入企业不是靠关系，而是靠实训记录、师傅评价和面试表现
```

### 8.3 实习单位把学生当廉价工，班主任顶住压力

```text
电商专业学生被安排到仓库打包
-> 岗位长期重复，与专业学习目标不符
-> 实习报酬由第三方拖延发放
-> 学生在群里抱怨，家长以为学校卖学生
-> 班主任收集实习协议、考勤、岗位说明和学生反馈
-> 学校要求企业调整岗位、补发报酬并建立整改台账
-> 主角第一次发现维权不是喊口号，而是把材料递到能处理的位置
```

### 8.4 资助名单公示刺痛自尊

```text
家庭困难学生申请中职助学金
-> 班里要收材料
-> 她不愿让同学知道父亲失业、母亲长期病痛
-> 班主任只公示必要信息，提醒同学不得传播家庭隐私
-> 助学金到账后，她买了实训鞋和一本工具书
-> 父亲沉默很久，说这钱先别还债，先让你读下去
```

### 8.5 技能竞赛改变一个学生的县城坐标

```text
汽修专业学生平时文化课弱
-> 在新能源车实训课上第一次超过同学
-> 老师推荐他参加技能竞赛
-> 备赛需要晚自习后练习，家里却催他周末打零工
-> 企业师傅指出他动作快但安全步骤不稳
-> 决赛中他没有最炫操作，却靠标准流程拿到企业面试机会
```

## 10. 安全边界

允许写：

- 职业教育选择、专业学习、实训、校企合作、学生实习、资助和就业衔接的概念性流程。
- 学校、企业、学生、家长和主管部门之间的利益冲突。
- 违规实习、虚假招生、资助冒领、拖欠实习报酬、实习事故和证书造假的风险与后果。

不要写成教程：

- 如何强迫学生实习、规避实习协议、克扣实习报酬、收取押金或管理费。
- 如何伪造学籍、资助材料、实习记录、技能证书、职业资格或毕业证。
- 如何通过中介转派学生、隐藏高危岗位、规避安全生产和未成年人保护要求。
- 如何替考作弊、买证挂证、冒领助学金或套取财政资助。
- 如何压制学生投诉、威胁家长、删改考勤或销毁事故记录。

反派可以实施违法违规行为，但文本必须呈现其风险、证据痕迹和后果，不提供可复制步骤，也不把压榨职校生写成商业智慧。

## 11. 审稿检查问题

1. 学校类型是否明确：中职、职高、普通中专、技工学校、技师学院、高职，还是培训机构？
2. 是否把职业教育写成失败标签，而不是一种教育类型和技能成长路径？
3. 专业选择是否写出课程、实训、就业、升学、家庭压力和学生兴趣？
4. 招生宣传是否有学校资质、学籍、收费、资助、专业和就业承诺边界？
5. 校企合作是否有协议、培养方案、企业导师、实训基地或岗位需求？
6. 学生实习是否区分认识实习和岗位实习？
7. 岗位实习是否有三方协议、专业匹配、报酬、保险、安全教育和管理责任？
8. 是否把实习学生写成廉价工，或让违规实习无后果？
9. 资助是否写出申请、审核、公示必要信息、发放和隐私尊严？
10. 技能证书、职业资格、学历证书和培训结业证是否被混用？
11. 就业推荐是否被误写成包分配？是否进入面试、合同、工资社保和试用期？
12. 是否避免强迫实习、克扣报酬、伪造学籍资助、替考作弊和证书造假教程化？
