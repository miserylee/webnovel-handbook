# 165. 银行账户、贷款信用、保险理赔、税务发票与金融合规专项

状态：已确认

适用范围：当 AI agent 需要设计、起草、审稿或修订都市现实、家庭伦理、商战创业、职场公司、重生赚钱、豪门财产、悬疑调查、娱乐圈合同、事故赔偿、医疗理赔、房车交易、网店经营、非法集资、金融诈骗、账户冻结、资金流水、贷款逾期、征信污点、保险投保/理赔、发票报销、税务稽查、公司财务、反洗钱审查和任何“钱进入制度后产生后果”的剧情时，使用本文件。

核心目标：让钱不再只是“到账了、赔了、破产了、暴富了”的数字按钮，而是形成可追踪的制度链：

`资金来源 -> 身份核验 -> 账户载体 -> 交易记录 -> 合同/票据 -> 信用评价 -> 还款/赔付/纳税 -> 风控审查 -> 争议处理 -> 现实后果 -> 台账回写`

一句话规则：钱一旦进入现代制度，就会留下账户、合同、流水、征信、票据、税务、保险和监管痕迹；这些痕迹既能制造爽点，也能制造限制、证据和余波。

---

## 1. 本专题解决什么问题

AI 写金融桥段常见坏味道：

1. 到账按钮化：大额资金说转就转，没人问来源、账户、限额、税务、合同和反洗钱。
2. 贷款按钮化：主角缺钱就立刻贷款，或反派一句话让银行断贷，没有审批、资信、担保、抵押和还款压力。
3. 征信空气化：逾期、担保、共同债务、信用卡和贷款记录不影响后续买房、创业、入职或合作。
4. 保险万能化：事故后一句“保险赔了”，不写投保前提、责任范围、免赔、材料、调查和争议。
5. 发票装饰化：报销、发票、合同和做账只是职场笑话，不影响成本、税务、证据和公司风险。
6. 税务消失：奖金、稿费、直播收入、股权分红、店铺利润、房产交易和赔偿收入都不处理纳税或申报问题。
7. 现金流无压力：企业只看利润，不写账期、应收账款、垫资、工资、税费、库存和贷款到期。
8. 金融诈骗弱智化：骗局靠受害者突然变蠢，不写信任、权威、收益诱惑、信息不对称和羞耻沉默。
9. 反洗钱和监管消失：异常大额资金、多人拆分转账、虚假交易、壳公司和跨境资金没有任何风控与痕迹。
10. 金融违法教程化：为了写真实，细写逃税、洗钱、骗保、套现、虚开发票、骗贷、非法集资和规避监管方法。

本专题把钱写成“资源 + 权利 + 义务 + 记录 + 风险”的复合资产。

## 3. 安全边界：只写小说可信度，不写金融违法教程

金融、税务、保险和反洗钱内容具有现实风险。AI agent 必须遵守：

1. 不输出洗钱、逃税、骗税、骗保、骗贷、虚开发票、套现、非法集资、内幕交易、操纵市场、规避风控和转移赃款的操作步骤。
2. 不提供现实投资、贷款、保险、税务或法律建议。
3. 不替真实平台、银行、保险公司、税务机关或监管机关编造最新规则、额度和处理时限；需要精确规则时查最新官方来源。
4. 不把金融诈骗写成可复用话术、脚本、引流路径或规避识别方法。
5. 可以写概念性叙事：身份核验、账户冻结、材料补充、风险审查、保险调查、税务问询、律师/会计师介入、监管通报、法院审理。
6. 涉及真实个人、公司、平台、金融机构和真实案例改编时，联动第 20、105、111 篇，避免诽谤、隐私泄露和未证实指控。
7. 架空世界也要有等价规则：钱庄票号、灵石账簿、商会信用、担保契约、税赋贡纳、镖局保价、宗门赏罚和天道契约都要留下痕迹。

可写的是“钱如何改变人物选择、关系、信用、权利和风险”。

不可写的是“现实中如何违规获利或逃避责任”。

## 4. 金融制度十线卡

设计资金桥段时，至少检查十条线。

| 线 | 要问什么 | 叙事功能 | 常见坏味道 |
| --- | --- | --- | --- |
| 来源线 | 钱从工资、经营、借款、赔偿、赠与、投资、违法所得还是未知来源来 | 合法性、动机、风险 | 钱凭空出现 |
| 身份线 | 谁开户、谁签约、谁受益、谁担保、谁纳税 | 责任归属、身份错认 | 账户等于本人 |
| 账户线 | 钱进入银行卡、对公账户、支付账户、现金、保单、证券账户还是账外 | 可追踪性、冻结、争议 | 所有钱都像现金 |
| 流水线 | 交易时间、金额、对手方、备注、频率和异常 | 证据链、反洗钱、悬疑 | 流水只在结尾空降 |
| 合同线 | 是否有借款、保单、发票、订单、收据、协议 | 权利义务和争议边界 | 口头说赔就赔 |
| 信用线 | 征信、逾期、担保、负债、信用卡和历史记录 | 后续资格和身份压力 | 欠债无后果 |
| 票税线 | 发票、报销、收入申报、税费、账务处理 | 职场/经营真实感 | 利润等于现金 |
| 风控线 | 银行、平台、保险、税务、监管如何识别异常 | 阻力、误伤、调查 | 大额转账无人问 |
| 争议线 | 谁拒付、拒赔、追债、质疑账目或申请复核 | 冲突升级 | 资金问题秒解决 |
| 余波线 | 钱改变信用、名声、关系、资格、诉讼和台账 | 长篇连续性 | 付款后清零 |

最小规则：每笔关键钱都要回答“谁的钱、从哪来、进哪去、凭什么、留下什么记录、后续谁承担”。

## 5. 账户与资金流水：钱在哪里，谁能动

现代资金不只是“现金”。不同账户有不同限制和后果。

### 5.1 常见资金载体

| 载体 | 可写内容 | 限制与后果 |
| --- | --- | --- |
| 个人银行账户 | 工资、借款、转账、生活支出、赔偿 | 实名、流水、冻结、限额、风控 |
| 对公账户 | 公司收入、货款、工资、税费、报销 | 公司治理、审批、财务记录、审计 |
| 支付账户 | 网店、直播打赏、平台提现、消费记录 | 平台规则、风控、账号安全 |
| 信用卡/授信 | 透支消费、分期、短期周转 | 账单、逾期、利息、征信 |
| 现金 | 私下交易、古早年代、逃避记录的诱惑 | 运输风险、保管风险、证明困难 |
| 证券/基金账户 | 投资收益、亏损、持仓、交易记录 | 市场风险、适当性、监管 |
| 保单现金价值 | 退保、贷款、受益安排 | 合同限制、损失、受益人争议 |
| 账外资金 | 小金库、回扣、灰色收入 | 高风险、证据和合规冲突 |

### 5.2 流水的叙事价值

资金流水可以证明或制造：

- 主角确实在关键时间收到钱。
- 反派资金来源与公开身份不符。
- 家庭成员长期转账照护老人。
- 公司“利润很好”，但现金流断裂。
- 恋爱关系里的金钱边界和控制欲。
- 诈骗资金经过多层账户，受害者很难追回。
- 事故赔偿到账但分配引发亲属冲突。
- 旧恩情通过多年小额汇款被重新发现。

检查问题：这笔钱是否有账户、时间、对手方和后续用途，而不是单纯数字？

## 6. 贷款、债务与征信：借钱不是拿资源包

贷款桥段要写“资格、审批、成本、还款和信用后果”。

### 6.1 贷款五要素

| 要素 | 小说表达 |
| --- | --- |
| 借款人 | 谁承担还款义务，是否共同借款、担保或家庭债务 |
| 用途 | 买房、经营、消费、周转、医疗、学费、救急 |
| 资信 | 收入、负债、征信、职业稳定性、资产和历史记录 |
| 增信 | 抵押、质押、担保、共同还款、保证金 |
| 还款 | 期限、利息、分期、逾期、展期、催收、诉讼 |

### 6.2 征信和信用后果

征信不是惩罚标签，而是后续资格系统：

- 逾期影响房贷、车贷、经营贷。
- 共同借款和担保会影响家庭关系。
- 父母要求子女做担保，可能变成亲情债。
- 主角想创业，但历史逾期导致融资困难。
- 反派用“帮你贷款”控制受害者。
- 主角替别人还款，不等于信用记录自然消失。

### 6.3 债务冲突

可用冲突：

- 借款合同签了，钱被亲人拿走。
- 公司账期拖延，主角只能垫资发工资。
- 担保人被追责，真正用钱的人失踪。
- 债务人有钱不还，债权人缺证据。
- 高利、套路贷、非法催收引发安全风险。
- 角色宁愿失去机会，也不再为家人担保。

检查问题：这笔债务是否改变人物信用、关系、现金流和未来资格？

## 7. 信用卡、分期与消费贷：小钱也能变成大压力

信用卡和消费贷适合写现代生活压力，但不能只当“随便刷”。

### 7.1 可写压力

- 最低还款让短期压力下降，长期成本上升。
- 分期消费掩盖真实负债。
- 多平台借款让角色自我感觉还有余地。
- 家人不知道角色账单已经滚大。
- 直播打赏、追星、游戏充值、医美分期和培训贷制造消费陷阱。
- 年轻角色因信用记录受损，影响租房、入职或贷款。

### 7.2 叙事边界

不要写具体套现、骗贷、逃避催收或规避风控方法。

可以写：

- 账单日和还款日制造倒计时。
- 催收通知和亲友知情制造羞耻压力。
- 主角通过协商、节流、卖资产、换工作、求助正规机构来止损。
- 反派用债务控制关系。

检查问题：消费贷是否写出心理压力、生活选择和信用后果，而不是单纯数字上涨？

## 8. 保险投保与理赔：保险不是自动赔钱机器

保险剧情要把“合同、责任、材料、调查、争议和赔付”拆开。

### 8.1 保险六问

1. 投的是什么：人身险、健康险、意外险、车险、财产险、责任险、农业险、商业险。
2. 谁投保：本人、家属、公司、学校、活动主办方、物业、工厂。
3. 谁是被保险人和受益人：身份、关系和利益冲突。
4. 事故是否在责任范围内：合同约定、除外责任、等待期、免赔额等概念性限制。
5. 理赔材料有哪些：事故证明、医疗资料、财产损失证明、报警记录、合同票据等。
6. 争议在哪里：事故性质、告知义务、资料缺失、责任比例、受益人争议、骗保嫌疑。

### 8.2 理赔流程的戏剧价值

保险理赔可以制造：

- 家属争夺受益人资格。
- 公司说买了保险，员工发现保障不足。
- 灾后材料被烧毁，赔付变难。
- 保险调查发现事故并非意外。
- 主角为弱势角色补齐材料，建立信任。
- 反派试图骗保，留下时间线和资金痕迹。
- 保险公司拒赔不一定是反派，也可能是合同边界或证据不足。

### 8.3 安全边界

不写骗保方案、伪造事故、伪造材料、规避调查步骤。

可写调查发现矛盾、材料补充、协商、投诉、诉讼和情绪余波。

检查问题：这次理赔是合同权利、证据争议、家属冲突，还是作者一句话发钱？

## 9. 税务、发票与报销：利润不等于到手钱

税务和发票是现代经营、职场、直播和创业题材的可信度核心。

### 9.1 常见税务/票据场景

| 场景 | 可写冲突 |
| --- | --- |
| 工资奖金 | 个税、绩效奖金、年终奖、补贴和到手差异 |
| 稿费版权 | 平台代扣、版权分成、合同价和税后收入 |
| 直播电商 | 销售额、退款、平台扣点、税费、佣金、发票 |
| 公司报销 | 发票合规、审批流程、虚假报销、垫付款 |
| 店铺经营 | 进货票据、销售收入、成本、库存、税费 |
| 房产交易 | 契税、个税、增值税、手续费、贷款和过户成本 |
| 股权分红 | 分红税费、公司账面利润和现金分配 |
| 捐赠赔偿 | 票据、用途、公示、受赠人资格和争议 |

### 9.2 发票的叙事功能

发票不是纸片，它可以：

- 证明交易真实发生。
- 证明货物、服务、时间和金额。
- 暴露虚假报销或利益输送。
- 成为税务风险和职场处分证据。
- 让主角发现账目和实际经营不符。
- 让普通员工陷入“领导让你报”的灰色压力。

### 9.3 经营利润拆分

经营文不要写“今天营业额十万，所以赚十万”。至少考虑：

`营业收入 -> 平台/渠道扣点 -> 原料/进货成本 -> 人工 -> 房租水电 -> 物流仓储 -> 税费 -> 退款售后 -> 贷款利息 -> 可分配现金`

检查问题：这笔收入到手前，被哪些成本、税费、账期和合同义务切走了？

## 10. 现金流、账期与企业财务：公司死于没钱，不一定死于亏损

商战和创业线最常见错误是只写利润，不写现金流。

### 10.1 现金流压力来源

| 压力 | 剧情效果 |
| --- | --- |
| 应收账款拖延 | 客户欠款，账面盈利但发不出工资 |
| 库存压货 | 货卖不掉，现金被锁死 |
| 预付款不足 | 接大单需要先垫资 |
| 银行抽贷/断贷 | 周转链断裂，合作方恐慌 |
| 工资社保税费 | 员工生存线和法律责任 |
| 平台结算周期 | 销售额高，但到账慢 |
| 退款售后 | 爆单后反而亏现金 |
| 供应商账期 | 上下游信任和议价能力 |
| 审计和风控 | 大额异常资金被卡，项目停摆 |

### 10.2 财务角色不要工具人化

财务、会计、出纳、审计、税务顾问、法务和银行客户经理都可以产生戏剧：

- 财务拒绝老板不合规报销。
- 会计发现账实不符。
- 出纳被迫夹在亲戚公司和制度之间。
- 审计发现关联交易。
- 银行客户经理提醒贷款条件变化。
- 税务顾问让主角放弃“看似省钱”的高风险做法。

检查问题：公司冲突是否有现金流、账期、票税、审批和合规阻力，而不是老板拍桌？

---

## 11. 反洗钱、异常交易与金融风控

大额资金和异常资金流会触发风控、反洗钱或内部审查。小说不用写具体监测规则，但要保留“异常会被记录和询问”。

### 11.1 异常资金桥段

可写：

- 主角突然收到巨额转账，账户被要求核实来源。
- 公司多笔资金绕行关联账户，引发审计疑问。
- 反派用慈善、咨询费、采购款掩盖利益输送。
- 家庭成员频繁小额转账，背后是长期控制或照护。
- 警方/银行/平台冻结涉案账户，误伤无辜角色。
- 角色拿不出资金来源证明，合作方取消签约。

不可写：

- 如何拆分转账逃避监测。
- 如何搭壳公司洗钱。
- 如何购买、出售、出租银行卡或账户。
- 如何伪造贸易背景和发票链。
- 如何规避银行、平台、税务或监管审查。

### 11.2 风控的叙事作用

风控可以是阻力，也可以保护角色：

- 阻止被骗资金继续转走。
- 让主角无法立即动用一笔可疑钱。
- 逼角色解释资金来源，暴露秘密。
- 让反派以为钱到手，实际被冻结。
- 让清白角色因关联账户被卷入调查。

检查问题：异常资金是否留下风控、问询、冻结、审计或解释成本？

## 12. 金融诈骗、非法集资与投资亏损

金融骗局的重点不是“受害者笨”，而是信任、贪念、恐惧、身份背书和信息不对称。

### 12.1 可信诱因

| 诱因 | 可写场景 |
| --- | --- |
| 熟人信任 | 亲戚、同学、同事、导师、粉丝群推荐 |
| 权威背书 | 假冒机构、明星站台、场地豪华、合同包装 |
| 高收益诱惑 | 保本高息、内部名额、限时认购、稳赚不赔 |
| 群体压力 | 大家都投了，不投显得不合群 |
| 恐惧焦虑 | 养老、医疗、孩子教育、房价焦虑 |
| 早期返利 | 前期小额到账增强信任 |
| 羞耻沉默 | 受害者怕被骂，不敢及时报警或求助 |

### 12.2 安全写法

可写：

- 角色如何被信任和压力击中。
- 主角如何识别“收益与风险不匹配”。
- 钱如何进入对方账户并留下合同/转账/聊天证据。
- 受害者如何保存材料、报警、投诉、诉讼或集体维权。
- 家庭关系如何因投资亏损破裂或重建。

不可写：

- 可复制的诈骗话术、拉新脚本、资金盘结构、洗钱路径和逃避监管方法。

检查问题：金融骗局是否写出信任机制、损失后果和维权路径，而不是嘲笑受害者？

## 13. 家庭、婚恋与金钱边界

金融制度常把亲情和爱情拉进硬后果。

### 13.1 高压场景

- 婚前债务和婚后共同还款。
- 父母要求孩子贷款或担保。
- 情侣共同买房但只写一方名字。
- 家庭成员拿老人银行卡或养老金。
- 亲戚借钱多年不还，没有借条。
- 保险受益人不是现任配偶。
- 离婚时发现对方隐匿债务或转移财产。
- 赡养、医疗、养老和保险支出压垮家庭。

### 13.2 写法

金钱边界不是道德说教，要落到制度动作：

- 谁签字。
- 谁转账。
- 谁还款。
- 谁担保。
- 谁拿发票和合同。
- 谁被写成受益人。
- 谁的征信受影响。
- 谁在法庭、银行、保险公司或税务机关面对后果。

检查问题：这段家庭冲突是否有可见金融动作，而不是只靠吵架？

## 14. 古代、玄幻和架空世界迁移

没有现代银行，也有金融制度。

### 14.1 古代/架空等价物

| 现代概念 | 古代/玄幻等价物 |
| --- | --- |
| 银行账户 | 钱庄户头、票号、商会账簿、宗门库房 |
| 转账流水 | 银票、账簿、印鉴、押运记录、镖局回执 |
| 贷款 | 典当、借契、商会周转、宗门预支、灵石贷 |
| 抵押担保 | 地契、田产、铺面、族人作保、法器质押 |
| 征信 | 商誉、族谱信誉、宗门功过簿、黑名单 |
| 保险 | 镖局保价、海商互保、宗门抚恤、行会赔偿 |
| 税务 | 田赋、商税、关税、贡纳、宗门供奉 |
| 发票票据 | 契书、收据、牙帖、官府文书、账房凭证 |
| 反洗钱 | 查赃银、来历不明财物、商路盘查、官府稽核 |

### 14.2 迁移规则

架空金融也要有：

- 记账人。
- 凭证。
- 信用评价。
- 违约代价。
- 争议处理者。
- 暴力或法律执行力。
- 风险扩散。

检查问题：这个世界的钱是如何被记录、证明、追索和执行的？

## 15. 类型文用法

### 15.1 都市家庭伦理

可用：房贷压力、担保翻车、老人保险、医疗费、离婚财产、亲戚借钱、养老金被挪用。

重点：钱要改变关系边界和生活选择。

### 15.2 商战创业

可用：现金流断裂、融资失败、税务问询、客户账期、供应商追款、财务审计、保险理赔。

重点：公司不是利润数字，而是现金、合同、票据、工资、税费和信用的组合。

### 15.3 重生赚钱

可用：先知投资、房产交易、股票基金、创业起盘、债务规避。

重点：不能让主角凭未来记忆无风险暴富；要写账户、税费、交易限制、资金来源、风险和社会后果。

### 15.4 娱乐圈文娱

可用：片酬税务、代言违约赔偿、演出保险、粉丝集资、工作室报销、合同分成。

重点：名气必须落到账、税、合同、票据和舆论风险。

### 15.5 悬疑刑侦

可用：异常转账、保单受益人、骗保疑云、洗钱线索、虚假发票、公司小金库。

重点：资金证据要交叉验证，不靠一张流水定罪。

### 15.6 古言权谋

可用：赈灾银、嫁妆账、铺面收益、私库、盐税、商会借贷、账房反水。

重点：账簿、印鉴、见证人、官府文书和执行力决定钱的可信度。

| 坏味道 | 读者体感 | 修复方式 |
| --- | --- | --- |
| 大额钱秒到账 | 太假，像游戏金币 | 写资金来源、账户、限额、审核、税费和风控 |
| 缺钱就贷款 | 资源包 | 写资信、用途、担保、抵押、审批、还款和逾期 |
| 欠债无后果 | 轻飘 | 写征信、催收、担保人、诉讼、亲情压力和未来资格 |
| 保险自动赔 | 合同不存在 | 写投保、责任范围、材料、调查、拒赔/赔付争议 |
| 公司利润等于现金 | 经营像儿戏 | 写账期、应收、库存、工资、税费、贷款和平台结算 |
| 发票只是笑话 | 职场空 | 写报销、票据、税务、虚假交易和审计风险 |

| 金融诈骗靠蠢 | 冒犯受害者 | 写信任、权威、高收益诱惑、群体压力和羞耻沉默 |
| 资金证据一锤定音 | 作者作弊 | 写流水来源、合同、发票、聊天、证人和资金用途交叉验证 |
| 金融犯罪教程化 | 安全风险 | 只写概念、后果、调查和责任，不写操作步骤 |

## 17. 审稿检查清单

审稿时逐条检查：

1. 关键资金是否有来源、账户、时间、对手方和用途。
2. 是否区分个人账户、对公账户、支付账户、现金、保单和投资账户。
3. 大额或异常资金是否留下风控、问询、冻结、审计或解释成本。
4. 贷款是否有借款人、用途、资信、担保、利息、还款和逾期后果。
5. 征信、担保、共同债务和信用卡是否影响后续资格。
6. 保险是否写清投保人、被保险人、受益人、责任范围、材料和争议。
7. 理赔是否不是自动到账，而是有调查、材料、时差和情绪后果。
8. 公司经营是否区分营业额、利润和现金流。
9. 是否处理账期、应收账款、库存、工资、税费、贷款和平台结算。
10. 发票、报销、合同和票据是否承担证据或税务功能。
11. 税务是否影响到手收入、经营利润、分红、房产交易或直播电商收入。
12. 金融诈骗是否来自可信的信任和压力机制，而不是受害者降智。
13. 资金证据是否有交叉验证，而不是一张流水解决全部争议。
14. 是否避免洗钱、逃税、骗保、骗贷、虚开发票、套现、非法集资等可模仿步骤。
15. 金融事件是否改变人物关系、信用、职业、名声、合同、诉讼或未来机会。
16. 是否把资金状态、债务、保单、票据、税务、信用和争议写入台账。

## 20. 外部资料吸收点

本专题吸收外部资料时，只转化为小说可信度和安全边界。

- 中国人民银行关于个人银行账户分类管理的资料：个人银行账户和支付账户存在身份核验、分类管理和支付限额等制度设计；迁移为“账户不是无限口袋，身份、限额、风控会改变资金行动”。
- 中国《征信业管理条例》：征信业务涉及信用信息采集、整理、保存、加工和提供，信息主体有查询和异议权；迁移为“信用记录是后续资格和冲突来源”。
- 中国《个人贷款管理办法》《商业银行互联网贷款管理暂行办法》：贷款需要用途、授信、风险管理、合同和贷后管理；迁移为“贷款是资格、合同、还款和信用后果，不是资源包”。
- 中国《中华人民共和国保险法》、国家金融监督管理总局金融消费者保护资料：保险基于合同、投保告知、保险责任、理赔材料和争议处理；迁移为“保险理赔不是自动发钱，而是证据和合同边界”。
- 中国《中华人民共和国税收征收管理法》《个人所得税法》《增值税法》、国家税务总局发票资料：纳税、发票和税务管理会影响收入、经营利润、报销和证据；迁移为“利润和到手钱之间隔着税费、票据和合规”。
- 中国《反洗钱法》及中国人民银行反洗钱监管资料：金融机构需要客户身份识别、交易记录保存和可疑交易报告；迁移为“异常资金会留下审查、冻结、问询和解释成本”。
- 国家金融监督管理总局消费者权益保护资料：金融消费者保护强调适当性、信息披露、投诉处理和风险提示；迁移为“金融机构不是绝对反派，也有规则、风控和争议处理路径”。
- OECD/World Bank 关于金融消费者保护和金融素养资料：金融消费者在信息不对称和复杂产品面前需要清晰披露和保护；迁移为“金融诈骗与误售要写权威、信任、误解和信息差”。

## 21. 最小执行规则

如果 AI agent 只能记住五条：

1. 关键钱必须有来源、账户、交易记录、合同/票据和后续用途。
2. 贷款不是资源包，要写资信、担保、利息、还款、逾期和征信后果。
3. 保险不是自动赔钱，要写投保、责任范围、材料、调查、争议和赔付余波。
4. 经营收入不是到手现金，要扣除账期、成本、工资、税费、平台结算、贷款和退款。
5. 金融桥段只写叙事可信度、风险和后果，不写洗钱、逃税、骗保、骗贷、虚开发票、套现、非法集资等可模仿步骤。
