状态：已确认

适用对象：负责乡村现实、种田文、返乡创业、合作社、家庭农场、粮食生产、农忙抢收、农业社会化服务、农机手、农机合作社、农机事故、农业补贴和县域农业现代化剧情的 AI agents。

## 1. 本篇解决什么问题

AI 写农业生产时，容易把农机写成“收割机开进来，粮食自动进仓”的爽点道具，或者把返乡创业写成“买几台机器就能带全村致富”。这会漏掉真正好看的现实张力：

- 农作物成熟窗口很短，天气一变，机器、人工、烘干、运输都会排队。
- 小农户未必买得起大机器，需要农机合作社、服务组织、跨区机手或生产托管。
- 农机购置补贴不是主角随手领钱，涉及机具目录、申请核验、公示兑付、违规风险。
- 农机作业不是无风险劳动，涉及登记、驾驶、检验、维修、道路转移、田间安全和事故处理。
- 机收质量会影响真实收益，损失率、倒伏、含水量、烘干能力和结算争议都能入戏。

本篇把县域农机写成一条可执行的叙事链：

```text
种植计划/农忙节点
-> 地块条件、作物成熟度和天气窗口
-> 判断农机需求：耕、种、管、收、烘干、转运、秸秆处理
-> 自有机具/农机合作社/农业社会化服务组织/跨区机手/生产托管
-> 作业预约、价格、面积、质量标准和风险约定
-> 机具登记、驾驶资格、维修保养、油料和运输通行
-> 田间作业、机收减损、安全防护和现场调度
-> 结算、损失争议、农机事故、保险或赔偿
-> 补贴、报废更新、维修升级、服务评价和长篇台账回写
```

一句话：农机戏的核心不是“机器替人干活”，而是机器把土地、天气、钱、技术、安全和乡村关系压缩到一个农忙窗口里。

## 3. 当前规则底座

### 3.1 农业机械化促进法

《中华人民共和国农业机械化促进法》确认农业机械化不是个人炫技，而是国家支持、科技推广、财政金融政策、农机服务和农业现代化的一部分。

写作影响：

- 农机进入小说时，要同时影响产量、成本、劳动强度、时间窗口和农户组织方式。
- 县级农业农村部门、农技农机推广机构、合作社、服务组织、机手和农户都可以成为角色。
- 机械化要因地制宜：山区、小田块、丘陵、稻田、麦田、果园、大棚和畜牧场需要的机具不同。

### 3.2 农业机械安全监督管理条例

《农业机械安全监督管理条例》覆盖农业机械生产、销售、维修、使用操作和安全监督管理。拖拉机、联合收割机等重要农业机械还涉及登记、检验、驾驶证件和安全监管。

写作影响：

- 不要让任何人“会开车就能上收割机”。
- 农机安全不是可有可无的旁白，而会影响人身伤害、财产损失、保险赔偿和行政责任。
- 农机事故既可能发生在田间作业，也可能发生在转移、装卸、道路运输和维修环节。

### 3.3 2024—2026 年农机购置与应用补贴

截至 2026-06-09，农机购置与应用补贴可按《2024—2026年农机购置与应用补贴实施意见》理解。该政策支持农民和农业生产经营组织购置使用先进适用农业机械，同时强调公平公正、优机优补、严惩违规。

写作影响：

- 补贴不是“买了就自动到账”。要有申请、核验、公示、兑付和监管。
- 补贴可以制造现实冲突：名额、目录、价格、发票、经销商、虚假购机、套补、二手翻新、机具闲置。
- 主角若做农机项目，必须面对“买得起、用得上、修得起、赚得回、查得清”。

### 3.4 农业社会化服务和生产托管

农业社会化服务把小农户和现代农业连接起来，常见形式包括耕、种、防、收、烘干、秸秆处理、生产托管等。它不是单纯雇机器，而是一种服务合同和组织方式。

写作影响：

- 小农户可以不买机器，而是购买单环节、多环节或全程托管服务。
- 服务组织的信誉、排班、作业质量、收费、赔偿和验收都能形成剧情。
- 合作社、村集体、家庭农场、农业公司、农机大户之间会发生资源竞争。

## 5. 农机不是一种机器，而是一组服务关系

### 5.1 自有农机

适合写家庭农场、农机大户、返乡创业者。

冲突点：

- 买机贷款、维修成本、油料、折旧、淡季闲置。
- 自家田用完后要不要外出接活。
- 机器越大，越依赖连片地、道路和维修能力。
- 亲戚邻居想插队，服务价格不好开口。

### 5.2 农机合作社

适合写村庄群像和合作社治理。

冲突点：

- 机器是谁出资买的，补贴算谁的，收益怎么分。
- 谁优先服务：成员、脱贫户、村干部亲戚、老客户还是出价高的人。
- 司机、维修、油料、保险和事故责任谁承担。
- 合作社账目是否公开，机器是否闲置或被个人长期占用。

### 5.3 农业社会化服务组织

适合写现代农业、产业服务和小农户连接。

冲突点：

- 服务合同写得漂亮，实际作业质量不稳定。
- 全程托管后，农户担心失去对土地的掌控感。
- 服务组织追求效率，农户关心自家地块细节。
- 项目补贴、验收、服务面积和农户满意度之间可能不一致。

### 5.4 跨区机手

适合写“三夏”“三秋”、公路通行、抢收和旅途群像。

冲突点：

- 机手按成熟期南北流动，路线排错就耽误全季收入。
- 高速通行、运输车辆、跨区作业证、维修和住宿都可能成为阻点。
- 外地机手和本地农机服务组织抢单，价格战和排队矛盾出现。
- 极端天气来临前，谁先收、谁等一等，会变成人情和利益冲突。

## 6. 农忙窗口怎么制造商业张力

农忙不是背景音，它是倒计时。

可写压力：

- 天气预报显示暴雨还有 36 小时。
- 小麦倒伏，越晚收损失越大。
- 全村只有两台能下田的大喂入量收割机。
- 烘干机排队，粮食含水量高，晒场不够。
- 机手疲劳，机器需要保养，但农户在田埂上催。
- 年轻人外出打工，老人不会联系服务组织，只能等村干部帮忙。

不要写成：

- 主角一声令下，全县机器都到位。
- 下雨前所有地块都能完美收完。
- 农户只关心产量，不关心价格、损失、烘干、运输和结算。

## 7. 机收减损：把“颗粒归仓”写成可见冲突

机收减损不是口号。小说里可以转化为这些细节：

- 作物成熟度不一，早收影响产量，晚收增加掉粒或霉变风险。
- 机器型号、保养状态、驾驶技术、地块平整度和倒伏情况影响损失。
- 农户站在田边看漏收、破碎、抛撒，会直接影响信任和结算。
- 服务合同可约定作业质量，但现实验收常靠经验、样方、照片和双方认可。

安全边界：不要写农机具体操作和危险调整步骤。可以写“技术人员检查、维修人员处理、机手按规范调整”，不要写可模仿的拆装、改装和冒险操作。

## 8. 农机购置补贴怎么写

### 8.1 正常路径

可写成：

```text
农户或服务组织判断需求
-> 查询补贴目录和适用机型
-> 经销商报价和售后承诺
-> 自筹资金、贷款或合作社出资
-> 购机、开票、申请补贴
-> 机具核验、公示、兑付
-> 实际使用、维修、收益和后续监管
```

### 8.2 冲突和风险

可写：

- 主角发现大家都想买同一类机器，但本地真正缺的是烘干或丘陵小型机。
- 经销商夸大补贴和收益，农户贷款买机后接不到活。
- 合作社购机时账目不透明，成员怀疑补贴被截留。
- 二手翻新机冒充新机、虚假购机、套取补贴等违规线索被查。

禁写：

- 伪造购机发票、虚假申请、倒卖补贴机具的操作方法。
- 如何规避核验、公示和监管。

## 9. 农机事故和安全边界

农机事故可以发生在：

- 田间作业：人员靠近、儿童围观、倒车盲区、地块边沟、夜间作业。
- 道路转移：拖拉机、收割机、农用运输车辆与机动车、行人、电动车冲突。
- 装卸运输：装车固定、超宽超高、转场路线、坡道和吊装风险。
- 维修保养：机器未停稳、带病作业、私自改装、疲劳抢修。
- 烘干仓储：电气、火灾、粉尘、通风和粮食堆放风险。

可写流程：

```text
事故发生
-> 先救人、报警或求医、保护现场
-> 通知机主、服务组织、村干部和相关主管部门
-> 固定作业记录、照片视频、证人、合同和机具信息
-> 事故调查或责任认定
-> 医疗、赔偿、保险、调解或诉讼
-> 安全培训、机器检修、服务合同和下一季规则调整
```

禁止提供：

- 危险驾驶、超限运输、违规改装、绕开检查、事故逃避和伪造现场的方法。
- 具体维修拆装、排障和危险操作步骤。

## 10. 高价值场景

### 10.1 暴雨前抢收

```text
天气预报提示强降雨
-> 全村小麦集中成熟
-> 外地跨区机手还堵在高速服务区或维修点
-> 村里老人排不上队，合作社成员要求优先
-> 主角按地块成熟度、倒伏风险、老人劳动力和烘干能力重排顺序
-> 有人不满，但最后保住关键地块，也留下人情裂缝
```

戏剧点：公平不是平均排队，而是在有限时间里排序风险。

### 10.2 合作社买机风波

```text
合作社准备买大型联合收割机
-> 成员争论是买大机、烘干机还是小型丘陵机
-> 经销商承诺收益，银行愿意贷款
-> 老农问：我们的小地块进得去吗，坏了谁修
-> 补贴申请、公示和收益分配成为矛盾核心
```

戏剧点：先进设备不等于适合本地。

### 10.3 机收质量争议

```text
机手赶时间作业
-> 农户发现田里漏收和破碎较多
-> 机手说地块倒伏、泥深、时间紧
-> 双方拿照片、视频、服务约定和现场样点说话
-> 村干部和农机技术人员介入
-> 结算从吵架变成质量和风险分担谈判
```

戏剧点：一亩地的损失，就是农户一季的情绪账。

### 10.4 农机事故改变家庭

```text
夜间抢收中发生农机伤人事故
-> 先救人送医，机器停在地头
-> 机主担心误工和赔偿，伤者家属担心手术费
-> 合同、驾驶资格、现场照明、疲劳作业、保险成为证据
-> 事故处理不只是赔钱，还改变合作社下一季安全规则
```

戏剧点：农忙的速度和人的安全发生硬碰撞。

### 10.5 跨区机手的路线选择

```text
机手从南往北赶麦收
-> 前一地拖欠作业款，后一地催他进场
-> 运输车手续、油价、维修件和天气预报一起压过来
-> 他不是单纯“外地人”，而是带着全家一年收入在路上赌博
```

戏剧点：跨区作业是一种流动职业，也是一条风险很高的生产链。

## 11. 人物表

| 小农户 | 快点把地收完 | 一季收入、天气、劳动力不足 | 以为机器来了就必须先收自己家 |

| 合作社理事长 | 提高服务能力 | 补贴、贷款、成员公平、账目公开 | 被怀疑拿机器做人情 |
| 返乡主角 | 用机械化改变村庄 | 技术理想和本地地块现实冲突 | 把大设备当万能现代化 |
| 农机经销商 | 卖机和售后 | 市场竞争、补贴政策、库存 | 夸大收益引发纠纷 |
| 农业农村干部 | 保障农忙和政策落地 | 天气、舆情、补贴监管、事故风险 | 被误解为只会开会 |
| 老农 | 保住自家粮食 | 不懂手续、怕欠人情、怕新技术 | 被年轻人误解为保守 |
| 维修师傅 | 尽快修好机器 | 配件、时间、野外环境、安全 | 被催到冒险维修 |

## 12. 审稿检查问题

写完农机相关剧情后，逐项检查：

- 这场戏发生在耕、种、管、收、烘干、转运还是秸秆处理？
- 作物、季节和天气窗口是否明确？有没有倒计时压力？
- 地块条件是否适合所写机具？有没有把山区小田块写成大平原？
- 服务主体是谁：自有农机、合作社、农机大户、跨区机手还是社会化服务组织？
- 价格、排班、结算和质量标准有没有写清？
- 机具登记、驾驶证件、检验、保险、维修和安全边界有没有进入剧情？
- 农机购置补贴是否写成自动到账？有没有申请、核验、公示、兑付和监管？
- 机收减损是否有可见冲突，而不是只喊“颗粒归仓”？
- 农机事故是否先救人、保护现场、固定证据，再处理责任和赔偿？
- 是否避免了危险驾驶、违规改装、维修拆装、骗补套补和逃避事故责任的操作细节？

## 13. 可直接套用的桥段提示词

### 13.1 暴雨前抢收

```text
请写一场县域小麦暴雨前抢收戏。必须写清天气窗口、地块成熟度、农机排班、跨区机手或合作社资源、老人和小农户压力、烘干或运输瓶颈、机收减损和结算余波。不要让主角一声令下全县机器自动到位。
```

### 13.2 农机合作社购机争议

```text
请设计一场农机合作社购置新设备的会议戏。要求写清本地地块条件、作物需求、补贴申请、贷款压力、机具适配、收益预测、成员公平和账目公开。不要把农机购置补贴写成随手领钱，也不要提供骗补操作。
```

### 13.3 农机事故处理

```text
请写一场农忙夜间农机事故后的处理戏。必须写清先救人、报警或送医、现场保护、机具和驾驶资格、作业合同、证人、保险、调解或诉讼接口，以及事故对下一季安全规则的影响。不要写危险操作、违规改装或逃避责任方法。
```

## 14. 一句话原则

县域农机戏的商业级写法，是让读者看见一台机器背后有季节、地块、天气、贷款、补贴、维修、机手、老人、合作社和安全责任；机械化不是让农业变轻松，而是把农业的风险换了一种更集中的形态。
