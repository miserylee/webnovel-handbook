# 31. 地域、时代与社会风俗质感

状态：已确认。用途：给 AI agents 在开书、设定、起草、修订和审稿时使用，解决“古代像现代换衣服、城市乡村像同一个背景、异世界没有生活制度、地域只剩地名、风俗只剩猎奇标签”的问题。

本专题联动 `docs/storycraft/28-worldbuilding-setting-exposition.md`、`docs/reality-reference/30-profession-industry-life-detail-credibility.md`、`docs/market-serialization/13-genre-patterns.md`、`docs/safety-research/20-content-compliance-risk-review.md` 和 `docs/market-serialization/29-reader-expectation-genre-promise-selling-point.md`。

## 1. 核心原则：背景必须改变人物行动

地域、时代和风俗不是背景板。它们必须改变人物能做什么、不能做什么、需要付出什么成本、如何获取信息、如何与他人相处。

AI 写作时使用这条链路：

```text
地域/时代/风俗约束 -> 人物身份与欲望 -> 场景选择 -> 冲突成本 -> 可见细节 -> 读者体感/剧情后果
```

如果故事换到另一个城市、另一个时代、另一个族群或另一个世界，剧情几乎不需要改，那么当前背景就是装饰。

## 2. 三个概念不要混用

| 名称 | 解决的问题 | 常见误用 |
|---|---|---|
| 地域 | 人生活在什么空间里，气候、地形、交通、口味、产业和熟人网络如何影响行动 | 只写地名、景点和方言梗 |
| 时代 | 人生活在什么时间秩序里，技术、信息速度、制度、材料、货币和价值观如何限制选择 | 古人像现代人，九十年代人用智能手机式思维 |
| 社会风俗 | 人如何婚丧嫁娶、拜访送礼、称呼避讳、处理面子、宗族、邻里、信仰和节令 | 写成猎奇展示或固定刻板印象 |

背景质感不是“多查资料”，而是让读者感到：这个人只能在这个地方、这个时代、这套人情秩序里这样行动。

## 3. 背景三问

每次设定地域、时代或风俗前，AI 必须先回答：

1. 如果换一个地方或时代，主角今天遇到的困难会怎样改变？
2. 这个背景给主角带来什么限制、机会、成本或误会？
3. 读者会从哪几个动作、物件、称谓、路线、节令或人情反应中感知它？

三问答不出，说明背景还没有进入剧情。

## 4. 六层资料：不要只查朝代和地名

| 层级 | 资料问题 | 正文投放方式 |
|---|---|---|
| 时间层 | 年代、季节、节令、作息、宵禁、农时、推荐期、赛季 | 倒计时、等待、误工、错过、节日压力 |
| 空间层 | 气候、地形、街巷、村落、城门、河道、楼层、交通 | 路线、遮挡、拥挤、绕行、逃跑、相遇 |
| 制度层 | 法律、礼法、户籍、阶层、门禁、学校、宗门、平台规则 | 权限、证件、审批、名额、惩罚 |
| 生产层 | 当地产业、职业结构、物资来源、价格、运输、税费 | 钱、货、工、账、材料、供应链 |
| 生活层 | 饮食、服饰、住房、取暖、通讯、娱乐、卫生、身体负担 | 动作、物件、感官、生活成本 |
| 人情层 | 亲族、邻里、师门、同乡、面子、称呼、送礼、禁忌 | 求人、站队、误会、欠账、舆论 |

一场戏至少选两层进入正文。只写空间层会像旅游介绍；只写制度层会像论文；只写生活层容易变流水账。

## 5. 地域质感：从“景点”转为“行动地图”

地域不是名胜清单。读者真正感到地点存在，是因为人物被地点影响。

可用维度：

- 气候：雨季、风沙、潮湿、干冷、酷热、海风、雾气如何影响动作。
- 地形：山路、河道、城中村、胡同、老小区、宗门山门、港口、草原如何改变路线。
- 交通：步行、骑马、公交、地铁、船、驿站、飞舟、传送阵各有什么成本和限制。
- 产业：当地人靠什么赚钱，主角能从哪里获得资源。
- 熟人网络：谁认识谁，陌生人为什么显眼，流言如何扩散。
- 地方口味：食物、气味、店铺、集市、夜生活和节令活动如何成为场景入口。

写法：不要说“这是一个江南小城”，而要让雨水、窄巷、青石路、茶馆账本、熟人目光和船期共同影响主角的行动。

## 6. 时代质感：先查技术和信息速度

时代最容易写错的不是服装，而是信息速度、材料条件和制度边界。

AI 处理时代背景时，至少确认：

- 通讯：消息靠手机、电话、书信、驿站、飞鸽、宗门令牌还是口口相传？多久到？会被谁截断？
- 交通：人和货物移动需要多久，路上有什么风险，谁有资格使用快速通道？
- 材料：衣服、门窗、纸张、兵器、药物、照明、食物保存是否符合时代？
- 钱与计量：货币、票据、账本、银钱、灵石、积分、粮票如何流通？
- 身份制度：户籍、官职、学籍、宗门身份、婚姻身份、家族排行如何限制行动？
- 舆论传播：消息在宫廷、村镇、校园、直播间、宗门、市井如何传播？

硬规则：时代错位会直接破坏读者信任。可以架空，但要在项目设定中说明架空规则，而不是把历史常识写错。

## 7. 社会风俗：让规矩制造剧情

风俗不是民俗展览，而是人和人之间的默认规则。

可用类型：

| 风俗类型 | 剧情功能 |
|---|---|
| 称呼与礼数 | 显示亲疏、尊卑、阵营和冒犯 |
| 婚丧嫁娶 | 制造家族压力、身份变化、财产和名声问题 |
| 节日与集会 | 提供公开场景、群像反应、交易和冲突机会 |
| 饮食与禁忌 | 暴露地域、阶层、亲密度和冒犯风险 |
| 送礼与人情 | 制造求人、欠账、站队和腐败/合规风险 |
| 宗族与邻里 | 制造舆论、庇护、排斥和熟人压力 |
| 信仰与仪式 | 提供价值观、恐惧、禁区和群体行动理由 |

写法：不要解释“当地有重男轻女的陋习”，而要写女主拿着录取通知书回村，亲戚第一句话问她弟弟彩礼够不够。风俗通过具体人情压力出现。

## 8. 普通人生活坐标

商业网文可以爽，但爽点最好从真实生活压力里弹出来。

普通人生活坐标包括：

- 住在哪里：房租、院落、宿舍、客栈、宗门杂役房、城中村、老小区、宅院偏房。
- 怎么吃饭：自己做、食堂、路边摊、酒楼、军粮、辟谷、灵食、外卖、配给。
- 怎么挣钱：工资、接单、田地、铺子、赏金、宗门任务、平台分成、家族供养。
- 怎么移动：步行、借车、骑马、船票、地铁、飞舟、传送名额。
- 怎么联系：敲门、托人带话、书信、电话、群聊、符信、榜单公告。
- 怎么求助：家人、邻居、师兄、同事、平台运营、官府、帮派、宗门执事。
- 怎么被看见：闲话、公告、榜单、热搜、宗门贡献榜、村口大喇叭、街坊眼神。

写一场戏前，先选一个生活坐标进入冲突，场景就不会漂浮。

## 9. 地图不是地理，路线才是剧情

地图的价值不是画得大，而是让角色行动有成本。

AI 设计地点时问：

- 主角从 A 到 B 要多久？中途会遇到谁？
- 路线被封、天气变化、宵禁、船期、传送名额不足会怎样改变计划？
- 哪些地方是公共空间，适合打脸、审判、表白、揭露和群像反应？
- 哪些地方是私密空间，适合秘密、误会、交易和亲密关系变化？
- 哪些地方有权力门槛，必须靠身份、钱、人情或证件进入？

如果一张地图不能生成相遇、阻碍、追逐、藏匿、公开见证或资源分布，它就只是装饰图。

## 10. 方言、称谓与口音：少量、准确、可读

方言和称谓可以增加地域感，也最容易变成噪音或冒犯。

用法：

1. 用少量高识别词，不要整段方言影响阅读。
2. 优先写语序、称呼、礼貌距离、语气习惯，而不只改字音。
3. 同一地域内部也有人群差异，不要所有人一个腔。
4. 方言可用于亲疏变化：角色紧张、愤怒、撒娇或回到熟人圈时更容易漏出口音。
5. 真实族群、地域和阶层口音不要写成嘲笑对象。

## 11. 风俗不能只当猎奇

写不熟悉的地域、民族、宗教、国家、阶层或历史时期时，AI agents 必须谨慎。

硬规则：

- 不把真实群体写成单一性格或落后标签。
- 不把禁忌、仪式、服饰、饮食和信仰当奇观消费。
- 不把现实文化挪用成自己原创设定而不做变形或说明。
- 不编造真实历史、民族、宗教或地域的负面事实。
- 不用“当地风俗”合理化侵害、歧视、违法或平台高风险内容。
- 如果资料不足，改为架空融合文化，并记录“候选/待确认”。

## 12. 不同题材的背景质感重点

| 题材 | 背景重点 | 常见失败 |
|---|---|---|
| 古言/历史 | 礼法、称谓、财产、出行、婚丧、官民关系、物件时代 | 现代人换古装，制度不产生压力 |
| 年代文/民国/近现代 | 信息速度、票证、单位、交通、物价、家庭结构、时代情绪 | 用现代消费逻辑写过去生活 |
| 都市地域文 | 城市结构、通勤、租房、产业、地方口味、熟人/陌生人边界 | 所有城市都像同一个 CBD |
| 乡土/现实 | 土地、亲族、邻里、集市、婚丧、人情、基层制度 | 只写贫穷苦难，不写具体生活和能动性 |
| 玄幻/仙侠 | 宗门生活、坊市、资源流通、门规、地形禁区、身份等级 | 只有境界和秘境，没有社会生活 |
| 悬疑/规则怪谈 | 禁区、地方传闻、公共空间、历史旧账、风俗误读 | 只靠恐怖氛围，不让地点生成线索 |
| 末世/灾变 | 资源点、避难所、路线、气候、信息断裂、群体秩序 | 地图像游戏副本，没有生存成本 |
| 科幻/未来 | 技术普及程度、阶层差异、城市基础设施、身体与信息边界 | 科技只当皮肤，不改变日常行为 |
| 异国/异文化 | 当地生活逻辑、法律制度、称谓礼貌、饮食交通、视角边界 | 旅游明信片化，靠地标和刻板印象代替生活 |
| 轻喜剧/日常 | 小区、学校、公司、村镇、街坊、家庭聚会、地方习惯 | 段子可搬到任何地方，背景无贡献 |

## 13. 背景质感三件套

快速增强一场戏时，优先补：

```text
一个地方规则 + 一个公共/私密空间 + 一个社会成本
```

例：

- 古言宅院：请安规矩 + 偏厅屏风后 + 被长辈当众冷落的名声成本。
- 都市老小区：电梯长期坏 + 楼道转角 + 邻居群里被截图传播的舆论成本。
- 玄幻坊市：入市身份牌 + 灵药摊前 + 买错材料导致任务失败的资源成本。
- 乡土现实：红白事随礼规矩 + 村口小卖部 + 不随礼就被亲族议论的人情成本。
- 末世避难所：水票限额 + 公共水房 + 插队引发冲突的安全成本。

## 14. 场景激活方法

背景进入正文的方式，不是单独说明，而是被视角人物激活。

```text
视角人物需求 -> 背景阻力/资源 -> 具体动作 -> 周围人反应 -> 后果
```

示例：

- 主角想见人，但宵禁已到，城门落锁，必须找守门小吏欠人情。
- 女主想拒婚，但族谱、嫁妆账本和长辈席位让她不能只说“不”。
- 主角想直播翻盘，但当地夜市信号差，商户怕被曝光，平台审核还卡着标题。
- 修士想买药，但坊市只认功勋牌，散修只能找中间人加价。

## 15. AI 写作硬性禁令

1. 禁止用大段背景百科替代场景压力。
2. 禁止只换地名，不改变人物行动逻辑。
3. 禁止古人、旧时代人物、异世界居民都用现代城市人的信息速度和价值判断。
4. 禁止拿真实地域、民族、宗教、职业或阶层做刻板笑料。
5. 禁止方言堆砌到影响阅读或变成嘲讽。
6. 禁止历史、制度、风俗硬伤进入目标读者熟悉的题材。
7. 禁止为了爽点完全无视礼法、法律、交通、通讯、材料和人情成本。
8. 禁止把所有地方都写成“热闹街市、豪华宫殿、现代办公室、荒凉村庄”四种模板。

## 16. 地域/时代/风俗资料卡

```text
【作品/章节】
【背景类型】现实地域 / 历史时代 / 架空王朝 / 异世界 / 未来科幻 / 乡土 / 都市 / 宗门 / 副本
【目标读者熟悉度】低 / 中 / 高
【资料来源】亲历 / 采访 / 官方资料 / 地方志/史料 / 行业报道 / 纪录片 / 同题材作品 / 社区经验 / AI 推测
【来源可信度】已确认 / 候选 / 待确认
【时间层】年代、季节、节令、作息、宵禁、农时、赛季
【空间层】气候、地形、街巷、村落、建筑、交通
【制度层】法律、礼法、户籍、阶层、门禁、宗门/平台规则
【生产层】产业、职业、物资、价格、运输、税费
【生活层】饮食、服饰、住房、通讯、娱乐、卫生、身体负担
【人情层】亲族、邻里、师门、同乡、面子、称呼、禁忌
【本章三件套】地方规则 + 空间 + 社会成本
【可用冲突】
【可用回报】
【合规/尊重风险】
【正文投放方式】动作 / 对话 / 物件 / 路线 / 节令 / 人情反应 / 后果
```

## 17. 审稿流程

1. 标出所有地域、时代、风俗、历史、族群、宗教、地方生活和社会制度信息。
2. 判断每一项是否改变人物行动、成本、信息速度、关系或后果。
3. 删除只展示资料、不服务场景的背景段落。
4. 给空背景补“地方规则 + 空间 + 社会成本”。
5. 检查时代错位：技术、材料、称谓、货币、交通、通讯、礼法是否一致。
6. 检查地域错位：气候、路线、产业、口味、熟人网络是否有可见差异。
7. 检查风俗风险：是否刻板化、猎奇化、冒犯真实群体或合理化高风险内容。
8. 会影响后续的地点、路线、节令、制度和人情规则，写入设定或连续性台账。

## 18. 快速自查问题

- 如果把这场戏搬到另一个城市、时代或世界，是否必须重写？
- 背景给主角制造了什么具体阻力或机会？
- 读者能否从动作、物件、路线、称谓或人情反应感到地点存在？
- 有没有时代错位的技术、材料、货币、通讯或价值判断？
- 方言、风俗、族群和地域表达是否尊重、准确、可读？
- 这章有没有一个地方规则、一个空间和一个社会成本？
- 背景是在推进主承诺，还是只让文字显得有资料？

## 19. 来源与迁移说明

- 起点社区《作者入门——新手写作指导》强调故事背景要明确，阶级结构、法律道德和人际关系等软环境要合理，场景要符合情节需要，细节要精心设计。本文将其迁移为“六层资料”和“背景必须改变行动”。
- 起点社区《写小说经常遇到的情节问题》强调写作前应找齐时代背景和社会背景，避免常识性时代错位。本文将其迁移为“时代质感”和“时代错位审稿”。
- 起点社区《起点现实短篇组半年成绩小结》提到现实题材覆盖行业发展、市井生活、个体成长、婚姻教育、职场故事、女性情感、民族题材等，并强调好故事藏在现实生活中。本文将其迁移为“普通人生活坐标”和“乡土/现实背景重点”。
- Writer's Digest《5 Tips for Writing Setting as a Character》强调设定可像角色一样承载情绪和故事负荷，季节、感官、温度和地点特有细节会影响人物。本文将其迁移为“地域质感”和“场景激活方法”。
- Reedsy《What Is the Setting of a Story?》把设定拆为时间、环境和具体地点，并强调地点应影响人物生活、帮助或阻碍目标，且要关注独特细节和读者方向感。本文将其迁移为“地图不是地理，路线才是剧情”。
- Writer's Digest《The Top 10 Elements of Setting In a Story》强调 setting 不只是背景，还包括天气、季节、日期、时间、氛围和意义。本文将其迁移为“六层资料”和“背景质感三件套”。
- Writer's Digest《Infusing Your Fiction With Real Culture the Reader Can Feel》强调文化不应只停留在标签和装饰，而要通过日常、价值、仪式和人物反应让读者感到。本文将其迁移为“社会风俗”和“风俗不能只当猎奇”。
