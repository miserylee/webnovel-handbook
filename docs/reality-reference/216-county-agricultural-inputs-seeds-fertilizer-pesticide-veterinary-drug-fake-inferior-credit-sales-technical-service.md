## 1. 本篇解决什么问题

AI 写农业生产时，常把“买种子、买化肥、买农药、买饲料”当成一句过场。但在县域现实里，农资是上游命门：一袋种子、一瓶药、一车肥、一张赊账单，可能决定农户一季收入、合作社信用、农产品质量、邻里关系和监管处罚。

常见失真：

- 农资店像杂货铺，谁都能卖种子农药兽药，不写资质、标签、票据、批号和进货台账。
- 农户买错农资后，剧情只写“老板赔钱”，不写鉴定、检测、因果关系和使用条件。
- 假劣农资只写反派故意造假，不写经销链、厂家业务员、合作社统一采购、赊销和农技推荐责任。
- 为了真实，错误写出农药、兽药、肥料的具体配比、用量、混用和规避检测方法，形成危险教程。

本篇把县域农资写成一条可执行链：

```text
种植/养殖计划
-> 农资需求：种子、化肥、农药、兽药、饲料、农膜、基质或设备耗材
-> 采购渠道：农资店、供销社、合作社统一采购、厂家业务员、电商平台、熟人代购
-> 核对资质、标签、批号、登记证号、适用范围、保质期、票据和台账
-> 农技建议、赊销、人情担保和使用记录
-> 出苗、长势、病虫害、肥害药害、畜禽异常或产量波动
-> 留样、包装、发票、照片、地块记录、天气和使用记录
-> 农业农村、市场监管、公安、司法调解、检测鉴定或诉讼
-> 赔偿、退换、行政处罚、刑事移送、信用崩塌和下一季台账回写
```

一句话：农资戏的核心不是“买东西”，而是农业风险在播种前就已经开始分配了。

## 3. 当前规则底座

### 3.1 种子法

《中华人民共和国种子法》用于理解种质资源、品种审定或登记、种子生产经营、标签、质量、监督管理和法律责任。写种子纠纷时，不能只写“种子不好”，要写清品种、批次、标签、购买渠道、种植条件、发票和田间表现。

写作影响：

- 种子可能有品种适应性、区域适宜性、纯度、发芽率、标签和包装问题。
- 农户出苗差不一定都是假种子，也可能是天气、地温、水分、播种深度、病虫害或管理问题。
- 要写鉴定和因果争议，而不是一口咬定老板赔。

### 3.2 农药管理条例

《农药管理条例》覆盖农药登记、生产、经营、使用、监督管理和法律责任。农药是高风险农资，小说只写资质、标签、记录、风险和后果，不写具体用法、剂量、混配或规避检测方法。

写作影响：

- 农药经营通常不是普通小卖部生意，应写经营许可、进货查验、销售台账和适用作物风险。
- 药害纠纷要写时间、地块、天气、作物阶段、产品批号、使用记录和是否混用等事实，但不要给出操作细节。
- 禁限用、过期、标签不明、夸大效果和虚假农技推荐都可入戏。

### 3.3 肥料登记和肥料风险

肥料、土壤调理剂、微生物肥料、叶面肥等常以“增产”“抗病”“修复土壤”等话术进入农户视野。肥料登记、标签、质量和宣传会影响农户判断。

写作影响：

- 肥害、烧苗、长势异常、土壤问题和产量下降，常常需要排除天气、水分、品种、病虫害和用量条件。
- 肥料争议适合写“专家现场看地、取样、比对同批次、查购买票据和施用记录”。
- 不要写具体施肥配方、剂量和混用教程。

### 3.4 兽药、饲料和养殖投入品

《兽药管理条例》《饲料和饲料添加剂管理条例》可作为养殖投入品底座。养殖线里，兽药、饲料、添加剂、疫病、出栏和检测常常连在一起。

写作影响：

- 兽药和饲料问题可能影响畜禽健康、死亡、出栏时间、食品安全和合作社信誉。
- 不写具体用药、剂量、配方或违规添加方法。
- 写证据：进货票据、批号、养殖日志、兽医处方或建议、死亡记录、检测结果、同批次情况。

### 3.5 农业技术推广法和农技服务

农技员、农资店老板、厂家业务员、合作社技术员都可能给建议，但身份和责任不同。农业技术推广是公共服务和经营服务的交界地带，特别适合写信任、能力和利益冲突。

写作影响：

- 公益农技员通常偏技术指导，不应被写成卖货提成角色。
- 农资店或厂家业务员推荐产品时，有销售利益，需要写清身份和承诺边界。
- 技术建议要转化成“谁说的、基于什么判断、有没有书面记录、是否夸大效果”。

## 5. 农资店不是普通小卖部

农资店连接的是村庄信用、生产风险和监管责任。

可写功能：

- 春耕前赊种子、化肥、农药，秋后收粮款再结账。
- 店老板记得每家地块、作物、欠账和往年病虫害。
- 厂家业务员带新产品来试用，承诺增产或包效果。
- 农技员下乡时，农户把问题带到店里一起问。
- 一旦出事，农资店会被所有熟人围住，账本变成证据。

高质量写法：

- 店老板不一定是反派，可能是被上游厂家、经销商、赊账压力和乡邻关系夹住。
- 农户不一定“贪便宜”，可能是现金流断裂，只能接受赊销和熟人推荐。
- 真正的冲突常在“谁也不想害人，但票据、标签和责任都不完整”。

## 6. 种子纠纷怎么写

种子戏最适合写“希望从播下去那天开始，风险也一起埋进土里”。

可写流程：

```text
农户选品种
-> 农资店或合作社推荐
-> 核对标签、适宜区域、批号和购买票据
-> 播种后出苗差或性状异常
-> 排查天气、地温、水分、播种方式、病虫害和管理因素
-> 保存包装、剩余种子、田间照片和同批次情况
-> 农业农村部门、专家或检测鉴定介入
-> 协商赔偿、行政处理或诉讼
```

常见冲突：

- 农户说“同样地块，隔壁家的苗好好的”。
- 店老板说“今年倒春寒，不能全怪种子”。
- 合作社统一采购后，几十户同时出问题，村里压力暴增。
- 厂家业务员承诺“包产量”，事后说那只是宣传话术。

不要写：

- 如何伪造种子标签、检测报告或留样。
- 如何违法销售未审定、假冒、套牌种子。

## 7. 肥害和药害怎么写

肥害、药害、烧苗、落叶、畸形、减产，适合做强情绪冲突，但必须写因果复杂性。

可写证据：

- 购买凭证、产品批号、剩余样品、包装标签。
- 使用时间、作物生育阶段、天气、地块水分、相邻田块差异。
- 农资店或业务员的书面建议、微信群语音、宣传单。
- 同批次其他农户是否出现类似异常。
- 农业技术人员现场查看、取样、检测或鉴定。

安全边界：

- 可以写“疑似药害”“疑似肥害”“技术人员判断需要检测”。
- 不写具体药剂、剂量、混配、施用方法或规避检测。
- 不把一场异常直接等同违法，先排除天气、病虫害、管理和品种因素。

## 8. 兽药、饲料和养殖线

养殖农资戏适合写资金压力和食品安全前端。

可写链条：

```text
养殖户赊饲料或兽药
-> 店家或业务员推荐方案
-> 畜禽出现异常、死亡、停食或出栏受阻
-> 保存饲料袋、兽药包装、进货票据和养殖日志
-> 联系兽医、农业农村部门或检测机构
-> 判断是疫病、饲养管理、饲料质量、兽药问题还是复合原因
-> 赔偿、退货、召回、监管处罚或刑事线索
```

注意：

- 兽药和饲料不要写具体使用、配方、添加和违规操作细节。
- 出栏、检测、残留、动物源性食品安全要联动第 197 篇。
- 疫病防控、无害化处理和动物防疫可另行联动相关畜牧专题，若未覆盖，应先查资料。

## 9. 农资赊销：县域商业的隐形债务

农资赊销很适合写人情与现金流。

常见模式：

- 春天赊种肥药，秋后卖粮还账。
- 农资店老板替老客户垫货，靠熟人信用维持生意。
- 合作社统一采购，年底从成员分红或粮款里扣。
- 农户遭遇灾害或价格下跌后，还不上农资账，下一季买不到货。

可写冲突：

- 农资质量出问题，农户拒绝还赊账。
- 店老板说自己也欠上游货款，赔不起全村损失。
- 村干部出面协调，发现账本既是债务证据，也是人情地图。
- 赊账被平台金融、合作社代扣或供应链贷款替代，旧人情变成新合同。

## 10. 假劣农资和农资打假

假劣农资不是普通消费纠纷，因为损失发生在整季生产之后。

可写监管动作：

- 春耕备耕、三夏、秋冬种等重点时段开展农资检查。
- 查种子、农药、肥料、兽药、饲料标签、资质、台账和进货渠道。
- 对问题产品抽样检测、溯源上游、责令下架、行政处罚或移送公安。
- 发布农资打假案例和消费警示。

不要写成：

- 主角靠直播骂一顿就能定性假货。
- 农业农村部门和市场监管部门随意互相替代。
- 一袋产品出问题，所有同类产品自动违法。

## 11. 农技服务：信任比药更贵

农技建议常常是剧情里的关键证据。

区分三类人：

| 角色 | 主要身份 | 风险 |
| --- | --- | --- |
| 公益农技员 | 技术推广、培训、现场指导 | 被误解为销售背书，或建议被断章取义 |
| 农资店老板 | 销售者兼经验顾问 | 推荐带销售利益，口头承诺难证明 |
| 厂家业务员 | 推产品和维护渠道 | 夸大效果、试验田包装、售后切割 |

好写法：

- 主角不靠“神农技”立刻治好全村，而是让农户学会记录、留样、看标签、问清责任。
- 农技服务不是万能诊断，很多问题需要时间、检测、对照和排除。
- 技术信任一旦破裂，比一季损失更难修复。

## 12. 高价值场景

### 12.1 合作社统一采购种子翻车

```text
合作社为压低成本统一采购新品种
-> 厂家业务员承诺适合本地
-> 出苗后几十户长势异常
-> 农户围到合作社门口
-> 理事长拿不出完整会议记录和采购比价
-> 农业农村部门、检测鉴定和上游厂家一起进入
-> 赔偿之外，合作社信用被重写
```

戏剧点：统一采购本来是效率，出事后就变成共同风险。

### 12.2 农资赊账引爆人情

```text
农户春天赊肥料和农药
-> 夏天疑似药害减产
-> 农户拒绝还账，农资店老板催收
-> 账本里牵出亲戚担保、上游货款和去年欠账
-> 村干部调解时发现，争议不只是质量，而是谁承担这一季失败
```

戏剧点：农资账是农业版“人情金融”。

### 12.3 短视频农技推荐惹祸

```text
返乡青年看短视频买网红肥料
-> 没核对登记、适用范围和本地土壤条件
-> 长势异常后找平台客服和本地农资店吵
-> 主角把网络宣传、订单、标签、地块记录和检测串起来
-> 农户第一次意识到“网上种田”和自家地不是一回事
```

戏剧点：新渠道带来新信任，也带来新证据链。

### 12.4 兽药饲料问题压垮养殖户

```text
养殖户赊饲料扩大规模
-> 畜禽连续异常，死亡和停食加重
-> 饲料商说是管理问题，养殖户说同批次都有问题
-> 兽医、农业农村部门和检测进入
-> 出栏延期、贷款、家庭关系和食品安全压力一起爆发
```

戏剧点：养殖失败不是一栏死畜，而是一整套现金流崩塌。

### 12.5 农资打假专项检查

```text
春耕前县里开展农资检查
-> 小农资店紧张整理台账
-> 大经销商暗中换包装或清库存
-> 主角发现某批种子标签和进货票据对不上
-> 检查不是打脸爽点，而是保护农户一季生产
```

戏剧点：农资打假真正保护的是还没发生的损失。

## 13. 写作禁区与安全边界

不得提供以下内容：

- 农药、兽药、肥料的具体用法、剂量、混配、替代、增强效果或规避检测方法。
- 制售假劣种子、农药、肥料、兽药、饲料的操作流程。
- 伪造标签、登记证号、批号、检测报告、进货票据或合格证明的方法。
- 规避农资检查、抽样、台账、许可和监管的技巧。
- 投毒、污染、违法添加、非法使用禁限用药物的教程。
- 伪造损失、骗赔、恶意投诉和敲诈农资店的方法。

可以写：

- 农户如何保存包装、票据、照片和记录。
- 农业农村、市场监管、公安、检测鉴定和调解诉讼如何进入。
- 生产损失、信用破裂、赔偿协商、行政处罚和刑事追责。
- 合法农技推广、农资风险提示、留样记录和购前核验意识。

## 14. 审稿检查问题

写完农资相关剧情后，逐项检查：

- 农资类型是什么：种子、化肥、农药、兽药、饲料还是其他投入品？有没有混写规则？
- 采购渠道是谁：农资店、供销社、合作社、厂家业务员、电商平台还是熟人代购？
- 卖方身份和资质有没有写清？有没有标签、批号、票据和台账？
- 农技建议是谁给的？是公益服务、销售推荐还是厂家承诺？
- 异常结果是什么：不出苗、死苗、药害肥害、病虫害失控、畜禽死亡、检测不合格还是产量下降？
- 有没有排除天气、地块、管理、病虫害、品种适应性等其他原因？
- 证据是否足够：剩余产品、包装、发票、赊账本、照片、日志、同批次情况和检测鉴定？
- 赔偿是否直接跳结论？有没有协商、检测、监管、调解或诉讼路径？
- 农资赊销有没有改变人情、债务和下一季生产资格？
- 是否避开了农药兽药使用教程、制假售假、规避检测和骗赔操作？

## 15. 可直接套用的桥段提示词

### 15.1 种子纠纷

```text
请写一场县域合作社统一采购种子后出苗异常的纠纷戏。必须写清采购渠道、品种标签、批号、票据、适宜区域、天气和播种条件、同批次农户情况、农业农村部门或专家鉴定、合作社信用和赔偿余波。不要写伪造标签、检测或非法售种方法。
```

### 15.2 农资赊账

```text
请设计一场农资店赊账引发的人情和质量纠纷。要求写清农户现金流、赊账本、农资店上游货款、疑似肥害或药害、证据留存、村干部调解、检测鉴定和下一季生产压力。不要提供农药化肥具体用法或骗赔方法。
```

### 15.3 农资打假

```text
请写一场春耕前农资打假专项检查。必须写清检查对象、标签批号、进货票据、销售台账、农户投诉、抽样检测、下架整改、行政处罚或移送线索，以及农户一季生产风险。不要把检查写成随意抄店，也不要写制假售假细节。
```

## 16. 一句话原则

县域农资戏的商业价值，是让读者看见一季庄稼的命运从买种买肥那天就开始下注；农资不是货架上的商品，而是农户、店主、厂家、合作社、农技员、监管和土地共同承担的一场风险分配。
