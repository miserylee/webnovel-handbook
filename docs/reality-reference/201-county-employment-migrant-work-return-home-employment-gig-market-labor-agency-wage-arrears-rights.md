# 201. 县域就业、外出务工、返乡就业、零工市场、劳务中介与欠薪维权链专项

状态：已确认  
日期：2026-06-09  
适用对象：AI 写作 agent、审稿 agent、现实题材设定 agent、乡村/家庭伦理/县域商业/基层治理/职场/返乡创业项目

## 1. 本篇解决什么问题

县域和乡村题材里，“出去打工”常被写成一句背景交代：父母去了外地、哥哥在工地、女主进厂、男主返乡创业。可真实生活里，务工不是瞬移，而是一条由熟人介绍、劳务中介、零工市场、招工信息、合同或口头约定、考勤、工资发放、社保断续、欠薪维权、返乡再就业共同组成的链条。

本篇把县域就业、外出务工、返乡就业、零工市场、劳务中介、工程建设领域欠薪和劳动争议整理成 AI agent 可执行的叙事规则。

| 判断项 | 写作意义 | 常见误写 |
|---|---|---|
| 人从哪里找活 | 熟人、村里劳务经纪人、公共就业服务、零工市场、劳务中介、平台招聘的证据不同 | 主角想挣钱，下一秒就进厂上班 |
| 到哪里工作 | 本县园区、外省工厂、工地、餐饮家政、平台零工、返乡项目的制度环境不同 | 所有务工都像办公室职场 |
| 谁是真正用工主体 | 包工头、劳务公司、总包、分包、工厂、平台商家、个体老板可能分离 | 欠薪时只找“老板”吵架 |
| 工资怎么证明 | 合同、工牌、考勤、工资表、转账、聊天记录、实名制平台、专用账户都能做证据 | 只靠工人集体喊口号 |
| 返乡后怎么办 | 就近就业、技能培训、创业扶持、家庭照护、社保接续和债务压力都要回写 | 返乡就是躺平或开挂创业 |

## 2. 安全边界

1. 本篇只用于小说设定和审稿，不提供现实劳动维权法律意见、仲裁策略、讨薪组织方案、规避用工责任、规避社保、伪造劳动关系证据、恶意投诉或群体性事件操作方法。
2. 不写扣押证件、非法招工、黑中介、强迫劳动、暴力讨薪、堵路围堵、威胁伤害、极端讨薪的可执行细节。
3. 涉及未成年人就业、童工、性骚扰、工伤事故、职业病、传销、诈骗、非法拘禁、人身伤害时，必须联动安全、公安、劳动和未成年人保护专题。
4. 县域就业政策、返乡创业补贴、技能培训、社保缴纳、最低工资、工程欠薪治理和地方零工市场服务高度地方化、年度化。小说可写制度链和材料压力，不写具体可申领金额或现实操作指南。
5. 写农民工和零工时，不把他们写成单纯苦难符号。每个人都有技能、比较、面子、家庭算盘、城市经验、返乡犹豫和尊严。

## 3. 核心叙事链

```text
家庭现金压力或职业选择 -> 招工信息来源 -> 中介/熟人/公共就业/零工市场筛选 -> 用工主体和工作地点确认 -> 合同、口头约定或入场登记 -> 交通住宿和进场培训 -> 考勤、工时、绩效或计件 -> 工资发放、社保或劳务报酬 -> 欠薪、工伤、辞退、跑路或返乡 -> 投诉、监察、调解、仲裁、诉讼或欠薪治理机制 -> 工资到账、信用受损、家庭关系和下一次就业选择回写
```

更适合章节节奏的压缩链：

```text
一个招工群消息 -> 一个熟人担保 -> 一辆凌晨出发的大巴 -> 一张没看懂的工资规则 -> 一个月没到账的工资 -> 一份考勤或工资表证据 -> 一次回乡过年被欠薪卡住 -> 一场家庭和村庄面子崩塌
```

## 5. AI agent 先判六件事

### 5.1 判就业路径

| 路径 | 核心问题 | 可写冲突 |
|---|---|---|
| 熟人介绍 | 亲戚、同村人、老乡群、班组长牵线 | 信任强但证据弱，出事后碍于面子 |
| 劳务中介/人力资源服务机构 | 招聘、职业介绍、派遣、外包、收费和资质 | 黑中介、扣证件、虚假岗位、收费争议 |
| 公共就业服务 | 县乡公共就业服务机构、招聘会、岗位信息、就业援助 | 政策岗位和真实工资落差，登记和培训材料压力 |
| 零工市场 | 日结、短工、临时活、小时工、搬运、保洁、家政 | 活不稳定、工资当天结不清、伤害责任模糊 |
| 平台招聘/群聊招工 | 招工 APP、微信群、短视频招聘、直播招工 | 信息真假、地点变更、薪资话术和个人信息泄露 |
| 校企/技校实习 | 职校、技校、实习就业衔接 | 学习和劳动边界、实习工资、未成年学生保护 |
| 返乡就业 | 县域园区、合作社、民宿、电商、物流、农业企业 | 工资低于外地，但能照顾家庭 |
| 返乡创业 | 民宿、加工、电商、合作社、零售服务 | 政策不是万能，市场、贷款和用工压力并存 |

### 5.2 判工作场景

| 场景 | 叙事素材 | 风险 |
|---|---|---|
| 工程建设 | 总包、分包、班组、实名制、工资专用账户 | 欠薪、工伤、层层转包、谁发工资不清 |
| 制造工厂 | 进厂体检、宿舍、计件、加班、流水线 | 工时长、工资规则复杂、离职结算 |
| 餐饮服务 | 包吃住、提成、小店老板、临时工 | 休息少、工资口头约定、老板跑路 |
| 家政护理 | 雇主家庭、平台、老人孩子照护 | 边界不清、意外责任、住宿和尊严 |
| 快递物流 | 分拣、配送、装卸、司机、站点 | 罚款、丢件赔偿、车辆和工伤风险 |
| 农忙短工 | 摘果、插秧、采茶、装车、临时雇工 | 日结、口头约定、天气和伤害风险 |
| 县域产业园 | 本地就业、工厂招工、通勤和社保 | 离家近但工资低，家里期待和个人志向冲突 |
| 乡村项目 | 民宿、合作社、加工厂、直播仓配 | 亲戚雇工、熟人管理、项目亏损欠薪 |

### 5.3 判主体关系

| 主体 | 能写什么 | 不宜误写成 |
|---|---|---|
| 劳动者/农民工 | 求职、劳动、考勤、工资、维权、返乡选择 | 只有苦情没有技能和判断 |
| 包工头/班组长 | 组织工人、安排活、传工资消息 | 总是最终法律责任主体，具体要看用工链 |
| 施工总包/分包 | 工程款、工资专用账户、实名制管理 | 完全不知道现场工人是谁 |
| 劳务公司/派遣公司 | 招人、派遣、发工资、合同或协议 | 可以空手套人且无监管风险 |
| 人力资源服务机构 | 招聘、职业介绍、就业信息 | 能扣押证件、乱收费而无后果 |
| 工厂/店铺老板 | 直接管理、考勤、工资、辞退 | 口头一句话解决所有劳动争议 |
| 平台或用工 APP | 信息撮合、规则、结算、评价 | 完全隐身，不影响劳动者收入 |
| 公共就业服务机构 | 岗位信息、职业指导、培训、就业援助 | 私人中介或万能安排工作 |
| 人社/劳动监察 | 欠薪投诉、执法检查、责令支付 | 直播曝光后立即发钱的魔法机关 |
| 劳动仲裁/法院 | 劳动争议裁判路径 | 情绪宣泄场或综艺审判 |
| 工会/社区/村委 | 协助维权、服务农民工、信息传递 | 代替劳动者直接拿到工资 |

### 5.4 判收入和证据

| 收入证据 | 叙事用途 |
|---|---|
| 招工广告/群消息 | 工资承诺、岗位地点和招聘主体 |
| 身份证复印件/入场登记 | 人什么时候进场或入职 |
| 劳动合同/劳务协议 | 关系性质、工资、岗位、期限和责任 |
| 工牌/工作服/门禁记录 | 证明劳动者实际在场工作 |
| 考勤表/打卡记录 | 工时、出勤和计薪依据 |
| 计件单/派工单/施工日志 | 工程、工厂和零工工资核算 |
| 工资表/工资条 | 欠薪金额和支付周期 |
| 银行流水/微信支付宝转账 | 工资是否到账、谁发的钱 |
| 工资专用账户记录 | 工程建设欠薪关键证据 |
| 聊天记录/录音 | 承诺、催薪、安排工作和辞退证据，但不能万能 |
| 投诉受理单/监察文书 | 欠薪进入正式处置链 |
| 仲裁申请和裁决 | 劳动争议进入程序后的证据 |

### 5.5 判风险事件

| 风险 | 可能触发 | 叙事推进 |
|---|---|---|
| 岗位虚假 | 中介夸大工资、换地点、换岗位 | 从发财梦变成回不去的车票钱 |
| 扣证件或押金 | 黑中介、非法用工、住宿管理 | 联动公安和劳动保障监察风险 |
| 工资被拖欠 | 工程款未到、老板跑路、账目混乱 | 证据收集、投诉、监察、仲裁 |
| 工伤或受伤 | 工地、物流、农忙短工、机械作业 | 联动工伤、医疗、社保专题 |
| 社保断续 | 灵活就业、临时工、外包、离职 | 医保、养老、失业和家庭保障受影响 |
| 返乡失败 | 工资不高、项目亏损、家人不理解 | 家庭期待和现实市场冲突 |
| 欠薪过年 | 年关、车票、孩子学费、老人看病 | 情绪高点和群像冲突 |
| 零工受伤 | 没合同、日结、谁雇谁不清 | 关系性质和责任边界成为争点 |
| 女性务工风险 | 家政、工厂宿舍、夜班、骚扰 | 必须联动安全和职场保护 |
| 未成年务工 | 辍学、暑假工、黑中介 | 联动未成年人保护和控辍保学 |

### 5.6 判救济路径

| 路径 | 适合写什么 | 边界 |
|---|---|---|
| 内部协商 | 先找班组长、老板、HR、财务 | 不要让一句威胁自动发钱 |
| 村委/工会/社区协助 | 信息传递、陪同沟通、服务农民工 | 不是正式裁判机关 |
| 劳动保障监察投诉 | 欠薪、工资支付、用工违法等行政执法 | 需要主体、证据和管辖 |
| 劳动争议调解 | 争议前置化、节省成本 | 调解不等于强制万能 |
| 劳动仲裁 | 劳动关系、工资、解除、经济补偿等争议 | 不是公开审判，程序有材料和周期 |
| 法院诉讼/执行 | 仲裁后诉讼、执行工资款 | 判赢和拿到钱之间还有执行问题 |
| 欠薪失信惩戒 | 严重欠薪进入信用惩戒 | 不是每次拖一天工资都上名单 |
| 公安刑事线 | 拒不支付劳动报酬、诈骗、非法拘禁等涉嫌犯罪 | 不要把普通争议直接写成抓人 |

## 6. 关键制度素材转化

### 6.1 公共就业和零工市场：找活也有制度现场

县域就业不是只有熟人介绍。公共就业服务、招聘会、零工市场、线上岗位发布和技能培训都能成为剧情场景。

可用流程：

```text
就业需求 -> 求职登记或岗位信息获取 -> 职业指导/技能培训 -> 岗位匹配 -> 面试或现场挑工 -> 录用/派工 -> 工作反馈 -> 稳岗、转岗或再次求职
```

可写冲突：

| 冲突 | 叙事价值 |
|---|---|
| 公共岗位信息真实但工资低 | 稳定和收入之间的选择 |
| 零工市场日结活抢不到 | 尊严、体力和年龄焦虑 |
| 招聘会热闹但岗位不合适 | 政策宣传和个人困境落差 |
| 技能培训和就业脱节 | 证书有了，岗位没有 |
| 女性中年返岗 | 照顾家庭多年后重新进入劳动市场 |
| 残障或大龄就业困难 | 就业援助、公益性岗位和自尊冲突 |

### 6.2 外出务工：离乡不是单向出逃，而是家庭分工

外出务工常常是全家共同决策：谁出去、谁留守、谁照顾老人孩子、谁还债、谁种地、谁承担城市风险。

可用链条：

```text
家庭用钱压力 -> 亲戚或中介提供岗位 -> 交通进城 -> 入职或进场 -> 宿舍和班组生活 -> 工资寄回家 -> 家庭留守问题 -> 节假日返乡 -> 再次外出或返乡就业
```

可写冲突：

- 母亲外出做家政，女儿在寄宿学校以为自己被抛下。
- 父亲工地欠薪，春节不敢回村，因为答应给孩子买手机。
- 夫妻一人进厂一人留乡，钱多了，话少了。
- 老乡介绍的工作出了问题，村里人情从“帮忙”变成“你骗我”。

### 6.3 劳务中介和人力资源市场：信息差是最大戏眼

劳务中介可以是正规人力资源服务机构，也可以是黑中介或熟人型劳务经纪人。戏剧冲突来自“岗位信息是否真实、收费是否合规、证件是否被扣、工资承诺是否兑现、用工主体是否清楚”。

可写节点：

```text
看到招工信息 -> 咨询岗位 -> 确认费用和地点 -> 签协议或登记 -> 交通安排 -> 到达用工单位 -> 岗位落差 -> 退费或投诉争议
```

风险信号：

| 信号 | 写法 |
|---|---|
| 高薪包吃住无细节 | 让角色产生希望，也埋下陷阱 |
| 先交费用再上车 | 可写风险和后果，不教规避或诈骗 |
| 扣押身份证 | 直接进入违法风险，不要正常化 |
| 到地换岗位 | “说是电子厂，到了成了夜班分拣” |
| 工资规则模糊 | 基本工资、计件、满勤、住宿扣费混在一起 |
| 老乡背书 | 最痛的不是被骗，是熟人也说不清自己是不是帮凶 |

### 6.4 工程建设欠薪：不要只写包工头跑路，要写专用账户和责任链

工程建设领域是农民工欠薪高发场景，也最适合写证据链。不要把所有矛盾都压在包工头身上，要写总包、分包、劳务、班组、实名制、工资专用账户、工程款和人工费分离。

可用链条：

```text
进场实名登记 -> 班组安排工作 -> 考勤和计量 -> 总包/分包确认工资 -> 工资专用账户拨付 -> 银行代发或工资发放 -> 拖欠预警 -> 投诉监察 -> 责令支付、专用账户核查、失信惩戒或仲裁诉讼
```

可写冲突：

| 冲突 | 小说用法 |
|---|---|
| 工人没实名登记 | 干了活却难证明进场 |
| 班组记工本丢失 | 欠薪金额说不清 |
| 工程款和工资混同 | 老板说甲方没给钱，工人只认工资没到 |
| 专用账户有钱但名单错 | 系统登记和真实工人不一致 |
| 包工头垫钱失败 | 他也是中间受压的人，不一定是终极反派 |
| 年关讨薪 | 车票、学费、医疗和面子压力同时爆发 |
| 失信惩戒 | 不是普通争吵，而是信用和经营资格后果 |

### 6.5 零工和灵活就业：自由不等于没有风险

零工市场适合写“今天有活、今天有饭”的现实压力。灵活就业看似自由，但证据弱、保障薄、收入波动大。

可用场景：

| 场景 | 冲突 |
|---|---|
| 早市零工等活 | 年龄、体力、熟人优先、一天空等 |
| 家政钟点工 | 雇主评价、平台抽成、意外损坏、尊严边界 |
| 装卸搬运 | 日结工资、受伤、老板临时压价 |
| 外卖/同城配送 | 平台规则、罚款、交通安全、装备成本 |
| 农忙短工 | 天气、计件、包饭、收工结算 |
| 夜市摊帮工 | 口头约定、营业不稳定、城管和食品安全 |

写法提醒：灵活就业不是“没有老板”。要问谁派单、谁定价、谁评价、谁结算、谁承担风险。

### 6.6 返乡就业：回村不是失败，也不是开挂

返乡就业可以来自父母生病、孩子上学、婚姻变动、城市失业、工伤、欠薪、想创业、县域产业发展。它的戏剧张力在于“城市经验能不能在县域兑现”。

可用链条：

```text
城市工作受挫或家庭召回 -> 返乡观望 -> 县域岗位/园区/项目/创业机会 -> 工资和生活成本比较 -> 家庭照护重新分工 -> 技能再培训或创业尝试 -> 稳定就业、再次外出或双城奔波
```

可写冲突：

- 女主返乡照顾老人，发现县城工资只有城市一半，但孩子终于有人接送。
- 男主从大厂运营回村做电商，被亲戚认为“读那么多书就回来卖菜”。
- 县域产业园招人，年轻人嫌工资低，老人嫌他们不肯吃苦。
- 返乡创业项目雇了亲戚，亲戚要人情，主角要考勤，关系瞬间变质。

## 7. 常见剧情模型

### 7.1 年关工地欠薪

```yaml
scene_model: 工程建设欠薪
character:
  protagonist: 外出务工父亲或基层人社工作人员
  pressure: 春节前工资没到账，孩子学费和回家车票都等着这笔钱
employment_chain:
  - 进场登记
  - 班组记工
  - 工资表确认
  - 工资专用账户核查
  - 劳动监察投诉
  - 支付或追责
conflict_hook: 他不是不想回家，是空手回村比留在工地更难堪。
agent_note: 写证据和责任链，不要写暴力讨薪教程。
```

### 7.2 黑中介高薪招工翻车

```yaml
scene_model: 劳务中介虚假招聘
character:
  protagonist: 刚毕业的县城女孩
  pressure: 招工广告承诺高薪包吃住，到地后岗位、工资和住宿全变了
employment_chain:
  - 招工群消息
  - 中介登记
  - 交通到厂
  - 岗位落差
  - 退费争议
  - 投诉或求助
conflict_hook: 她以为自己离开小县城，第一步却是把身份证递给了不该信的人。
agent_note: 不写诈骗操作，只写识别风险、证据和后果。
```

### 7.3 零工市场父亲一天没被点走

```yaml
scene_model: 零工市场等待
character:
  protagonist: 中年父亲
  pressure: 清晨去零工市场等活，年轻人被挑走，他空等到中午
employment_chain:
  - 零工市场
  - 现场挑工
  - 日结工资承诺
  - 未被选中
  - 回家面对孩子补课费
conflict_hook: 他最怕的不是累，是别人看一眼就知道他不如年轻人值钱。
agent_note: 适合现实主义和家庭伦理，不要把零工写成笑料。
```

### 7.4 返乡就业和留守儿童重组家庭

```yaml

  protagonist: 外出务工母亲

  - 城市离职
  - 返乡求职
  - 县域园区面试
  - 工资和陪伴权衡
  - 孩子学校走访
  - 家庭关系修复或再冲突
conflict_hook: 城市工资高到能寄钱，县城工资低到能每天接她放学。
agent_note: 联动第200篇留守儿童和乡村教育。
```

### 7.5 返乡创业雇亲戚后欠薪

```yaml
scene_model: 乡村项目用工纠纷
character:
  protagonist: 返乡创业男主

  - 亲戚入职或帮忙
```

## 9. AI 写作检查问题

1. 角色通过什么路径找活：熟人、中介、公共就业、零工市场、平台招聘还是返乡项目？
2. 工作场景是什么：工地、工厂、餐饮、家政、物流、农忙短工、县域园区还是乡村项目？
3. 谁是真正用工主体？包工头、劳务公司、总包、分包、工厂、店铺、平台还是合作社？
4. 有没有至少一个就业证据：招工信息、合同、工牌、考勤、工资表、转账、聊天记录、入场登记？
5. 如果写欠薪，是否写了工资金额怎么计算、谁该支付、欠了多久、证据在哪里？
6. 如果写工程欠薪，是否写了实名制、班组、工资专用账户、总包分包或劳动监察？
7. 如果写零工，是否写了谁派活、谁结算、谁评价、受伤或拖欠谁负责？
8. 如果写返乡就业，是否写了城市收入和县域生活成本、家庭照护、社保接续或再次外出选择？
9. 是否避免暴力讨薪、伪造证据、恶意维权、黑中介教程和规避用工责任？
10. 事件结束后，是否回写工资到账、债务、孩子照护、老人医疗、社保、村里名声和下一次就业选择？

## 10. 审稿风险信号

| 风险信号 | 读者体感 | 修正方向 |
|---|---|---|
| 想打工就马上上班 | 悬浮 | 补招工来源、交通、入场、合同或口头约定 |
| 欠薪只找包工头吵 | 责任链太薄 | 补总包、分包、劳务、实名制、工资专用账户和监察 |
| 零工自由无风险 | 太轻 | 写活源、日结、受伤、评价、结算和保障薄弱 |
| 劳务中介全是坏人 | 脸谱化 | 区分正规机构、熟人经纪和黑中介 |
| 返乡创业不算工资 | 熟人剥削风险 | 补岗位、考勤、薪酬、项目现金流和亲情边界 |
| 劳动仲裁像公开审判 | 程序不真实 | 联动第141篇和第162篇，写材料、周期、裁决和执行 |
| 外出务工只为卖惨 | 人物扁平 | 写技能、比较、面子、城市经验和家庭决策 |
| 工资不到账无后果 | 钱没有重量 | 回写债务、学费、医疗、车票、社保和家庭关系 |

## 12. 最小可用规则

1. 写县域就业，必须先标明找活路径、工作场景和用工主体。
2. 写外出务工，必须写交通、住宿、工资规则、家庭留守或返乡节点中的至少三个。
3. 写劳务中介，必须区分正规人力资源服务、熟人经纪和黑中介风险。
4. 写零工，必须写谁派活、谁计酬、谁结算、是否日结和受伤或欠薪责任。
5. 写工程欠薪，必须写实名制、班组、工资表、专用账户、总分包或监察路径中的至少三个。
6. 写返乡就业，必须写收入、照护、社保、技能、县域岗位或再次外出选择。
7. 劳动争议至少保留一个证据：合同、考勤、工资表、流水、工牌、聊天记录、投诉受理单或仲裁材料。
8. 结尾必须回写工资到账、债务、家庭照护、社保连续性、村庄评价或下一次就业选择。
