# 162. 招聘入职、劳动合同、薪酬社保、绩效晋升、解雇离职与职场制度

适用对象：AI agents、网文作者、审稿 agent、都市职场 agent、行业文 agent、商战 agent、娱乐圈/公司线 agent、现实题材 agent、劳动争议/证据链 agent、连续性维护 agent

适用任务：设计、起草、续写、审稿或修订招聘、面试、背调、offer、入职、试用期、劳动合同、劳务/外包/实习、岗位职责、汇报线、薪酬工资、奖金提成、社保公积金、考勤打卡、加班调休、休假、绩效考核、KPI/OKR、晋升降职、转岗调岗、保密协议、竞业限制、离职、辞退、裁员、解雇、劳动仲裁、工伤、产假病假、职场歧视、办公室政治和任何“人在组织中通过劳动换取钱、职位、资源和未来机会”的剧情。

核心问题：AI 写职场时常把公司写成“老板一句话决定一切”：主角入职没有 offer 和岗位，工资不发也没后果，加班只为卖惨，绩效只是打脸数字，晋升没有职位空缺和预算，辞退只靠领导吼一声，劳动仲裁像公开审判，HR 只会坏笑或发通知。这会让职场文变成披着工牌的宫斗，也会让现实题材的制度可信度崩掉。

一句话原则：职场不是办公室背景板，而是“岗位 -> 合同 -> 工资 -> 考勤 -> 绩效 -> 权限 -> 责任 -> 风险 -> 离职余波”的制度链；每一次入职、晋升、加薪、背锅、裁员或离职，都要改变钱、权限、记录、关系、名声或下一次机会。

重要边界：本篇用于小说创作和审稿，不提供现实劳动法律建议、仲裁策略、规避社保、违法裁员、伪造证据、恶意维权或规避用工责任的方法。涉及真实劳动法、社保、工伤、休假、女职工保护、未成年工、外籍用工、竞业限制、劳动仲裁和裁员时，必须查最新官方资料，并联动第 20、111、120、141 篇。

## 2. 职场制度十线卡

AI agent 写任何职场、招聘、合同、绩效或离职情节前，先填这张卡。

```text
职场事项：招聘 / 面试 / offer / 入职 / 试用期 / 合同签订 / 薪酬发放 / 加班考勤 / 绩效考核 / 晋升 / 调岗 / 降职 / 停职 / 辞职 / 辞退 / 裁员 / 劳动争议 / 工伤 / 其他
组织类型：公司 / 事业单位 / 工作室 / 娱乐公司 / 医院 / 学校 / 律所 / 工厂 / 平台用工 / 外包 / 宗门工坊 / 商会 / 架空组织
劳动关系：劳动合同 / 劳务 / 实习 / 外包 / 临时工 / 平台接单 / 合伙 / 学徒 / 宗门任务 / 未确认
角色岗位：岗位名称、职责、级别、汇报对象、下属、预算、签字权、系统权限、客户资源
关键文书：简历、offer、劳动合同、员工手册、保密协议、竞业协议、考勤记录、工资条、绩效表、会议纪要、解除/离职证明
薪酬资源：基本工资、绩效、奖金、提成、补贴、社保、公积金、股权/期权、灵石/贡献点、项目分成
评价机制：KPI、OKR、项目验收、客户满意、销售额、工时、产量、质量、导师评价、排行榜、主观评价
风险争点：欠薪、加班、试用期、调岗降薪、背锅、工伤、歧视、骚扰、竞业、裁员、离职交接、证据丢失
权力接口：直属上级、HR、财务、法务、老板、客户、工会/员工代表、仲裁/法院、宗门执事、系统管理员
本场变化：钱、职位、权限、绩效记录、名声、项目归属、劳动关系、证据状态、台账回写
```

最小规则：只要写职场，就必须回答“这个角色靠什么岗位拿钱、谁评价他、谁能改变他的权限、记录会留下什么后果”。

## 3. 员工生命周期：从招聘到离职

SHRM 和 CIPD 的员工入职/员工生命周期资料都强调，招聘承诺、入职体验、管理支持和后续发展会影响员工融入、满意度和留任。迁移到网文中：一份工作不是“主角坐进办公室”，而是一条身份和资源链。

```text
招聘需求 -> JD/岗位预算 -> 简历面试 -> offer -> 背调/材料 -> 入职 -> 试用期 -> 日常绩效 -> 晋升/转岗 -> 争议/离职 -> 交接 -> 离职证明/行业名声 -> 下一份机会
```

### 3.1 每个节点都能写冲突

| 节点 | 可写冲突 | 爽点/压力 | 风险 |
|---|---|---|---|
| 招聘需求 | 岗位是真缺人还是背锅位 | 主角识破坑位 | 公司像随便招人 |
| 面试 | 能力、背景、薪酬、歧视、信息不对称 | 主角反问击中痛点 | 面试像打脸舞台 |
| offer | 薪酬、职级、试用期、入职时间、保密 | 谈判拿到条件 | offer 当录用万能凭证 |
| 入职 | 工牌、电脑、权限、合同、导师、团队规则 | 新人识别组织暗流 | 入职像逛办公室 |
| 试用期 | 目标不清、主管卡人、学习曲线 | 主角用结果过关 | 试用期随便辞退 |
| 日常绩效 | 项目、考勤、协作、客户评价 | 真实成绩入账 | KPI 只做数字 |
| 晋升转岗 | 空缺、预算、评审、派系、背书 | 权限和责任升级 | 晋升只有头衔 |
| 离职/辞退 | 通知、交接、工资、补偿、证明、竞业 | 主角带着证据走 | 被吼一句就失业 |
| 争议 | 调解、仲裁、证据、程序、执行 | 证据链反杀 | 仲裁像综艺审判 |

### 3.2 职场身份不是头衔，是权限包

一个职位至少包含：

- 工作内容。
- 汇报线。
- 预算或资源。
- 系统权限。
- 责任边界。
- 评价指标。
- 会议席位。
- 对外名片。
- 替代者或继任者。
- 出错后谁担责。

如果“升职”后这些都不变，只是称呼变好听，读者不会感到落袋。

## 4. 招聘、入职和试用期怎么写

### 4.1 招聘不是抽奖

有效招聘要回答：

- 为什么现在招人？扩张、离职、背锅、客户要求、项目启动、合规要求、领导派系。
- 谁批准岗位？老板、HR、预算、部门负责人、宗门执事、公会管事。
- 招来的人要解决什么问题？不是“很优秀”，而是缺技能、缺人手、缺资质、缺替罪羊。
- 候选人为什么愿意来？钱、平台、资源、逃离旧环境、人情、不得不。

### 4.2 面试是信息交换，不只是审判

面试里双方都在判断。

候选人看：岗位是否真实、领导是否靠谱、工资是否兑现、加班文化、成长机会、风险。

公司看：能力、稳定性、成本、团队匹配、背景、可管理性、风险。

小说用法：让主角在面试中发现公司矛盾，而不是只让面试官被主角惊艳。

### 4.3 入职第一天的可写物件

- 工牌或门禁。
- 电脑和账号。
- 工位和座位位置。
- 员工手册。
- 合同和入职材料。
- 组织架构图。
- 群聊邀请。
- 内网权限。
- 导师/带教人。
- 第一个任务。

这些东西能快速展示组织文化：是混乱、压迫、专业、家族化、派系化，还是虚伪体面。

### 4.4 试用期不能当“老板随便开人”

现实题材涉及试用期解除、试用期工资、试用期长度等内容时必须查最新法律。小说层面至少要写：

- 试用期目标是否清楚。
- 谁记录表现。
- 试用期是否被滥用。
- 通过/不通过会影响什么。
- 主角是否有证据证明工作成果。

## 5. 劳动合同、用工形态和责任边界

ILO 的 employment relationship 资料把雇佣关系理解为劳动者在特定条件下为雇主工作并取得报酬的关系。迁移到小说中：谁控制工作、谁支付报酬、谁承担风险，决定角色在制度中的位置。

### 5.1 常见用工形态

| 形态 | 典型特征 | 剧情功能 | 风险 |
|---|---|---|---|
| 劳动合同员工 | 固定岗位、管理控制、工资、社保、制度约束 | 稳定身份和权益 | 不查法就乱写 |
| 实习生 | 学习、低权力、转正期待 | 易被压榨或成长 | 写成免费劳动力合理化 |
| 外包/派遣 | 三方关系、身份尴尬、权限有限 | 归属冲突和背锅 | 公司责任消失太随便 |
| 劳务/自由职业 | 项目交付、报酬、合同边界 | 灵活和风险并存 | 当劳动关系逃避责任 |
| 平台接单 | 算法派单、评分、账号、提现 | 数据权力和不稳定收入 | 只写自由不写算法压力 |
| 合伙/股东 | 分红、控制权、债务、决策权 | 商战权力 | 和员工身份混淆 |
| 宗门/公会任务 | 身份牌、贡献点、任务等级、惩罚 | 架空职场 | 任务无限无合同边界 |

### 5.2 合同不是装饰

合同可以改变：

- 工资和奖金。
- 工作地点。
- 工作内容。
- 试用期。
- 保密义务。
- 竞业限制。
- 培训服务期。
- 调岗和出差。
- 解除条件。
- 争议路径。

好写法：合同条款在后文被具体触发。坏写法：签合同时写一堆，后文没人记得。

## 6. 薪酬、社保和劳动收入

### 6.1 工资不是一个数字

薪酬可拆成：

```text
基本工资 + 岗位工资 + 绩效 + 奖金 + 提成 + 补贴 + 加班费 + 社保公积金 + 股权/期权 + 项目分成 + 福利
```

叙事用法：

- 主角涨薪但绩效被卡，现金流仍紧。
- 高薪但没有社保或休假，风险转嫁给个人。
- 提成延迟发放，逼销售和公司对立。
- 奖金和项目署名绑定，导致团队内斗。
- 期权听起来美，但兑现条件很远。

### 6.2 工资发放能制造现实压力

有效细节：工资条、到账短信、扣款项、社保缴费、报销迟迟不批、年终奖规则、提成周期、项目奖金名单。

不要只写“她很穷/他年薪百万”。写“月底房租、社保断缴、报销压着、年终奖被延期、离职后奖金归属争议”，钱才会变成剧情压力。

### 6.3 社保、公积金、工伤和生育相关

现实制度会变化，必须查最新官方资料。小说中只需抓住叙事功能：

- 社保不是背景福利，而是医疗、养老、失业、工伤、生育等风险的制度入口。
- 工伤不是“受伤了公司赔不赔”的口水仗，而要有事故、申请、认定、证明和待遇链。
- 产假、孕期保护和女职工保护不是甜宠福利，而是工作安排、收入、岗位和歧视风险。
- 社保断缴、未缴、漏缴会改变现实压力。

## 7. 考勤、加班和休假

### 7.1 考勤是权力工具

考勤不只是打卡。它可以证明：

- 谁在现场。
- 谁加班。
- 谁被故意安排无法完成任务。
- 谁被排除出项目。
- 谁有不在场证明。
- 谁的时间被公司占用。

### 7.2 加班不能只卖惨

加班场景要写清：

- 为什么必须加班：客户、上线、审计、事故、领导拖延、资源不足。
- 谁决定加班。
- 有没有记录。
- 补偿或调休怎么处理。
- 加班后影响什么：睡眠、家庭、错误率、关系、健康、证据。

现实加班工资、调休和工作时间规则要查官方资料。小说不要输出规避加班费、伪造考勤或压榨劳动的操作建议。

### 7.3 请假和休假

病假、年假、婚假、产假、陪产、丧假、工伤休假、事假等都能制造真实冲突。

可写：

- 主角不敢请假，因为绩效要被扣。
- 老板口头批准，系统没有记录。
- 产假回来岗位被边缘化。
- 病假证明被质疑。
- 丧假和项目 deadline 冲突。
- 调休欠账变成离职争议。

## 8. 绩效、晋升和权力重排

### 8.1 绩效不是分数，是未来分配机制

绩效影响：

- 奖金。
- 晋升。
- 调薪。
- 裁员名单。
- 项目机会。
- 培训名额。
- 老板信任。
- 同事站队。
- 离职背调。

### 8.2 绩效场景五问

```text
指标是什么：销售额、交付质量、客户评价、工时、增长、成本、合规、团队协作。
谁设定：老板、HR、客户、总部、算法、宗门功勋堂。
谁记录：系统、上级、会议纪要、客户邮件、考勤、工单。
谁解释：上级、HR、评审委员会、派系、数据管理员。
结果改变什么：奖金、职位、项目、权限、裁员、名声、下一轮目标。
```

### 8.3 晋升要有空位和代价

晋升不是“你很优秀所以升职”。它需要：

- 组织需要这个职位。
- 有预算或权力空位。
- 有评审或老板拍板。
- 有竞争者。
- 有交接和下属。
- 有更高责任和更高风险。

如果主角升职后没有更难的目标、更贵的错误和更多敌意，晋升就没有进入故事系统。

---

## 9. 调岗、降职、辞退、裁员和离职

### 9.1 离职不是出门就完事

离职链条：

```text
触发 -> 沟通 -> 通知/申请 -> 交接 -> 权限回收 -> 工资结算 -> 社保/档案/证明 -> 竞业/保密 -> 行业名声 -> 下一份机会
```

### 9.2 辞退和裁员要有文书与理由

现实题材中，辞退、裁员、解除劳动合同、经济补偿等规则复杂且变化，必须查官方资料。小说层面至少要写：

- 谁提出解除。
- 解除理由是什么。
- 有没有证据和通知。
- 工资、补偿、奖金怎么结算。
- 权限何时回收。
- 离职证明怎么写。
- 角色是否保留证据。
- 后续求职和行业名声怎样变化。

### 9.3 调岗和降薪

调岗降薪不是一句“公司安排”。它会改变：职责、工资、地点、汇报线、绩效指标、尊严、未来晋升、离职选择。

可写冲突：明升暗降、边缘岗位、冷板凳、项目移交、被迫签字、拒绝调岗、工会/员工代表/仲裁介入。

---

## 10. 劳动争议和证据链

劳动争议不等于法庭爽文。它通常由材料、记录、程序和执行构成。

### 10.1 常见争点

- 确认劳动关系。
- 欠薪。
- 加班费。
- 未缴社保。
- 违法解除。
- 经济补偿或赔偿。
- 工伤待遇。
- 竞业限制。
- 培训服务期。
- 绩效奖金。
- 产假/病假/医疗期争议。
- 职场歧视或骚扰。

### 10.2 证据类型

| 证据 | 功能 | 风险 |
|---|---|---|
| 劳动合同 | 证明关系和条款 | 只签空白合同、条款缺失 |
| 工资流水/工资条 | 证明报酬 | 现金、拆分、代发难解释 |
| 考勤记录 | 证明出勤和加班 | 公司掌握数据，员工取证难 |
| 工作邮件/群聊 | 证明任务和通知 | 隐私和合规风险 |
| 会议纪要 | 证明责任分配 | 事后补写或没人签字 |
| 工牌/门禁/系统账号 | 证明劳动管理 | 离职后权限回收 |
| 绩效表 | 证明评价和奖金 | 指标主观或被改动 |
| 解除通知/离职证明 | 证明离职原因 | 措辞影响背调 |
| 工伤材料 | 证明事故和时间 | 医疗和申报节点需查证 |

### 10.3 争议程序的写法

可以写：协商、调解、仲裁、诉讼、执行，也可以写架空宗门裁决、公司纪律委员会、行业协会调解。

规则：

- 程序要有时间和成本。
- 不是谁哭得惨谁赢。
- 材料缺失会改变胜率。
- 仲裁/裁决结果要进入钱、职位、名声、行业关系和下一步行动。
- 不输出现实诉讼或仲裁操作建议，只写叙事结构。

## 11. 职场题材用法

### 11.1 都市职场爽文

重点：能力证明、流程反杀、项目落袋、岗位升级、公开记录。

规则：打脸不能只靠会议吵赢，必须让证据、客户、绩效、合同、预算或职位发生变化。

### 11.2 商战/创业

重点：合伙人身份、股权、员工工资、劳动成本、核心员工离职、竞业和保密。

规则：公司不是主角一个人的战斗力。员工工资、岗位和制度会决定公司能不能扩张。

### 11.3 娱乐圈/文娱公司

重点：经纪合同、艺人团队、绩效资源、商务提成、解约、劳动/劳务边界。

规则：艺人爆红后，不只涨粉，还会改变团队分工、合约谈判、分成和公司控制。

### 11.4 医院/律所/学校等专业机构

重点：执业资格、排班、带教、绩效、科室/团队资源、责任边界。

规则：专业岗位不是只看聪明，要看资格、记录、授权和风险责任。

### 11.5 平台经济/主播/自由职业

重点：账号、平台规则、提现、评分、流量、算法、客户评价、合同边界。

规则：没有老板不等于没有权力。平台规则和账号权限就是新的上级。

### 11.6 玄幻宗门/公会/系统文

重点：任务牌、贡献点、弟子籍、岗位、执事考核、资源发放、除名。

规则：宗门任务也可以按职场制度写：职责、指标、贡献、惩罚、晋升、离职/除名。

## 12. 常见失败模式与修法

| 失败模式 | 读者体感 | 修法 |
|---|---|---|
| 老板万能 | 职场像封建宫斗 | 加 HR、财务、法务、直属上级、客户、流程和记录 |
| 入职无手续 | 公司像咖啡馆 | 加 offer、合同、工牌、账号、导师、试用目标 |
| 工资空泛 | 钱没有压力 | 拆工资、绩效、奖金、社保、报销、到账时间 |
| 绩效假数字 | KPI 没意义 | 写指标来源、记录者、解释权和后续分配 |
| 晋升只有称呼 | 没有落袋 | 加预算、下属、权限、责任、敌意和新目标 |
| 辞退靠吼 | 不可信 | 加通知、理由、证据、交接、结算和离职证明 |
| 仲裁像综艺 | 程序悬浮 | 加材料、时间、成本、调解/仲裁路径和执行后果 |
| 加班只卖惨 | 低级苦情 | 写原因、决定者、记录、补偿、睡眠和错误率 |
| HR 脸谱化 | 工具人 | 给 HR 的 KPI、风险、信息边界和组织位置 |
| 同事只会八卦 | 群像空 | 给同事岗位、利益、风险和当前任务 |
| 现实制度胡写 | 出戏和风险 | 查最新官方资料，只转成剧情约束 |
| 维权教程化 | 合规风险 | 不提供现实操作步骤，只写叙事后果和证据类型 |

## 13. 审稿检查问题

### 13.1 职场制度可信度

1. 主角的岗位、职责、汇报线、工资来源和评价机制是否清楚？
2. 公司里谁有权招聘、调岗、发奖金、开除、审批预算和改变权限？
3. 入职、试用、晋升、离职是否有文书、记录和交接？
4. 工资、奖金、社保、报销、加班和休假是否影响现实压力？
5. HR、财务、法务、客户、同事是否有独立利益，而不是工具人？

### 13.2 剧情功能

1. 本场职场戏是否改变钱、职位、权限、绩效记录、名声或下一份机会？
2. 绩效、项目、会议、邮件、考勤是否留下可回收证据？
3. 晋升是否带来更大责任和更贵错误？
4. 离职是否影响行业名声、客户关系、竞业、工资结算和证明？
5. 职场冲突是否不只是吵架，而是资源、责任、流程或记录变化？

### 13.3 安全边界

1. 是否涉及现实劳动法、社保、工伤、产假、未成年工、就业歧视、劳动仲裁？
2. 如果涉及，是否查证最新官方资料？
3. 是否避免提供规避社保、违法裁员、伪造证据、恶意维权或规避用工责任的方法？
4. 是否涉及骚扰、歧视、报复、职场霸凌或心理伤害，需要联动第 20/105 篇？
5. 是否避免把过劳、无偿加班、孕产歧视、残障歧视写成理所当然？

## 14. 台账字段

```text
角色：
组织：
岗位/职级：
劳动关系：劳动合同 / 劳务 / 实习 / 外包 / 平台 / 合伙 / 架空任务
入职日期/阶段：offer / 入职 / 试用 / 正式 / 调岗 / 离职 / 争议
直属上级：
HR/财务/法务接口：
工资结构：基本工资、绩效、奖金、提成、补贴、社保公积金、股权/期权、其他
考勤与休假状态：
绩效指标与当前结果：
权限：工牌、门禁、系统、客户、预算、签字权、项目资料
关键文书：合同、员工手册、保密协议、竞业协议、绩效表、工资条、会议纪要、通知、离职证明
当前争点：欠薪、加班、调岗、晋升、背锅、辞退、竞业、工伤、歧视、其他
证据状态：
本次变化：钱、职位、权限、关系、名声、劳动关系、争议进度
下次回收点：
禁写边界：不提供现实违法操作或法律建议
```

最小回写：入职、转正、调薪、绩效、晋升、调岗、辞退、离职、仲裁、工伤、产假/病假、重大加班争议都必须写入台账。

## 15. 可直接给 AI agent 的提示词

### 15.1 职场制度场景设计提示词

```text
请设计一场职场制度场景。不要只写老板吼人或同事八卦。先填写职场制度十线卡：事项、组织类型、劳动关系、岗位、关键文书、薪酬资源、评价机制、风险争点、权力接口和本场变化。正文必须让合同、工资、考勤、绩效、权限、项目记录、HR/财务/法务或离职交接中的至少三项改变角色选择，并在结尾回写台账。
```

### 15.2 职场爽点审稿提示词

```text
请审稿这段职场爽点，重点检查：主角是否只是会议吵赢；证据是否能改变绩效、职位、预算、客户信任、工资或项目归属；HR、财务、法务、客户和同事是否有独立利益；晋升或离职是否有后续责任和台账。请按“问题 -> 读者体感 -> 结构修法 -> 需查证来源 -> 台账回写”输出。
```

### 15.3 劳动争议安全提示词

```text
请审稿这段劳动争议/辞退/欠薪/加班/工伤情节。不要提供现实法律建议或操作教程。请从叙事可信度检查：劳动关系是否清楚，合同和记录是否存在，争点是什么，证据类型是什么，程序成本是什么，结果如何改变钱、职位、名声、行业关系和下一步行动。涉及现实法律时标注需查最新官方资料。
```

## 16. 来源与吸收方式

- 中国《中华人民共和国劳动法》：吸收点是劳动合同、工作时间、休息休假、工资、劳动安全卫生、女职工和未成年工特殊保护、社会保险与劳动争议等制度；迁移为“职场不是老板口头规则，而是工资、时间、休假、安全和争议共同构成的制度场”。
- 中国《中华人民共和国劳动合同法》及人社部相关解读：吸收点是劳动合同订立、履行、变更、解除、终止、试用期、经济补偿等规则；迁移为“入职、试用、辞退、离职必须有文书、理由、记录和后果”。
- 中国《中华人民共和国社会保险法》与《社会保险经办条例》：吸收点是养老、医疗、工伤、失业、生育等社会保险与个人风险保障相关；迁移为“社保不是背景福利，而是医疗、生育、失业和工伤风险的制度入口”。
- 中国《工伤保险条例》：吸收点是事故伤害、职业病、工伤认定和待遇有流程；迁移为“职场受伤不能只靠老板赔不赔，要有事故、申报、认定、材料和后果”。
- 中国《女职工劳动保护特别规定》：吸收点是孕期、产假、生育医疗费用和女职工特殊保护有制度安排；迁移为“现代职场孕产不能写成单纯家庭问题，也会影响岗位、收入和保护边界”。
- 中国《劳动争议调解仲裁法》与《劳动人事争议仲裁办案规则》：吸收点是劳动争议可经协商、调解、仲裁和诉讼路径处理，争议范围包括合同、劳动报酬、社会保险、经济补偿、工伤等；迁移为“劳动争议不是会议吵赢，而是材料、程序、时间和执行”。
- ILO Employment relationship 资料：吸收点是雇佣关系是劳动者在特定条件下为雇主工作并取得报酬的关系，现实用工形态多样；迁移为“谁控制工作、谁支付报酬、谁承担风险，决定角色制度位置”。
- SHRM Onboarding 资料：吸收点是有效入职不只是表格和介绍，而是管理支持、关系建立、成长承诺和组织文化落地；迁移为“入职场景可以快速展示公司制度和暗流”。
- CIPD Induction / Talent management 资料：吸收点是入职、融入、发展和留任共同构成员工生命周期；迁移为“招聘承诺、试用期、绩效和晋升需要前后连贯”。
- U.S. Department of Labor Fair Labor Standards Act 资料：吸收点是最低工资、加班、记录保存和童工等劳动标准具有明确制度边界；迁移为“不同国家规则不同，但职场写作都需要工资、工时、记录和保护意识”。

## 17. 最小执行规则

如果时间不够，AI agent 至少执行以下 8 条：

1. 写职场先定岗位、汇报线、工资来源、评价机制和权限。
2. 招聘入职要有 offer、合同、工牌/账号、试用目标或架空等价物。
3. 工资要拆成基本工资、绩效、奖金、补贴、社保/福利或架空资源。
4. 绩效和晋升必须改变奖金、职位、权限、责任、敌意或下一轮目标。
5. 加班、请假、休假和考勤要影响睡眠、家庭、证据、工资或健康。
6. 离职、辞退、裁员要有通知、理由、交接、结算、权限回收和证明/名声后果。
7. 劳动争议要靠合同、工资条、考勤、邮件、会议纪要、绩效表等材料推动，不靠吵架取胜。
8. 现实劳动法、社保、工伤、女职工保护、未成年工、仲裁和裁员内容必须查最新官方资料，不输出违法操作建议。
