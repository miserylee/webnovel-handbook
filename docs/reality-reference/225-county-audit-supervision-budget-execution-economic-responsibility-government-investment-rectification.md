# 225 县域审计监督、预算执行审计、经济责任审计、政府投资审计、自然资源资产离任审计与整改销号专项

状态：已确认  
更新时间：2026-06-09  
适用对象：县域现实题材、基层治理、财政资金、政府投资、乡村振兴项目、国企事业单位、学校医院经费、自然资源生态保护、干部离任、审计整改、纪检巡察、反腐悬疑和公共资金追责剧情。

## 1. 本专题解决什么问题

AI 写审计时，常把审计写成“会计一翻账本，坏人立刻下线”的万能桥段：

- 审计员看一眼发票，就直接认定贪污。
- 审计报告一出，纪检、公安、法院全部自动接上。
- 预算执行审计、政府投资审计、经济责任审计、内部审计、财政监督和巡察混成一锅。
- 审计整改只写“责令整改”，不写清单、责任人、期限、销号、回头看和长期机制。
- 审计发现的问题全部被写成腐败，不区分管理不规范、绩效低效、制度漏洞、历史遗留、失职失责和违法犯罪线索。

本专题的目标是给 AI agent 一条可执行的审计叙事链：审计项目怎么立项、审计组怎么取证、审计报告怎么形成、问题怎么定性、意见怎么反馈、整改怎么推进、线索怎么移送、结果如何改变公共生活。

## 2. 一句话原则

审计不是万能判决书，而是用证据、程序和整改要求把公共资金、公共资产、公共资源和领导干部经济责任重新放回可核验轨道。

## 3. 标准叙事链

```text
公共资金、国有资产、政府投资、自然资源、政策落实或干部经济责任出现风险
-> 审计机关或内部审计机构纳入审计计划、专项审计、跟踪审计、经济责任审计、离任审计或整改回头看
-> 明确审计对象和范围：部门单位、项目、资金、企业、领导干部任期、自然资源资产、政策措施、预算执行或整改事项
-> 送达审计通知或启动内部审计：调取资料、进点会、承诺说明、数据接口、现场踏勘、访谈和函证
-> 审计实施：查账、查合同、查预算指标、查支付凭证、查工程量、查台账、查会议纪要、查资产资源、查绩效目标和服务对象反馈
-> 形成审计证据：工作底稿、取证单、问题事实、金额口径、责任链、被审计单位说明和审计组复核
-> 征求意见：审计报告征求意见稿、被审计单位反馈、补充材料、事实修正和争议保留
-> 出具结果：审计报告、审计决定、审计移送处理书、审计建议、整改要求、公告或向人大政府报告
-> 结果分流：责令整改、追回资金、调整账目、完善制度、主管部门处理、纪检监察移送、司法移送、组织人事使用、信用或考核影响
-> 整改销号：问题清单、责任清单、时限清单、整改材料、跟踪检查、对账销号、回头看、公开反馈、长效机制
-> 剧情回写：项目能不能继续、钱能不能追回、干部评价如何变化、群众信任是否修复、台账更新
```

## 4. 当前制度底座的小说转化

### 4.1 审计法与审计法实施条例

已确认规则：《中华人民共和国审计法》于 2021 年修正，修正后自 2022 年 1 月 1 日起施行；《中华人民共和国审计法实施条例》为审计监督范围、审计机关职责权限、审计程序和法律责任提供实施细化。

小说转化：

- 审计机关依法对财政收支、财务收支的真实、合法和效益进行审计监督。
- 县级审计机关不只是查账，也可能审计预算执行、部门单位财政财务收支、政府投资项目、专项资金、国有资产、自然资源资产、民生资金和领导干部经济责任。
- 审计不是随时被主角叫来的私人侦探。一般要有审计计划、专项安排、上级交办、领导干部经济责任审计安排、整改检查或案件线索移送等入口。
- 审计证据要支撑审计结论。发票、合同、银行流水、预算指标、工程量、会议记录、台账、现场照片、访谈记录和被审计单位说明都可能进入证据池。
- 审计发现问题后，可以提出处理处罚意见、作出审计决定、移送主管部门或纪检监察司法机关，但审计本身不等于刑事裁判。

风险：

- 把审计报告写成法院判决，会混淆权力边界。
- 把审计人员写成随意封账、抓人、审讯，会失真。
- 把审计对象全部写成贪腐，会抹掉审计对管理、绩效和制度漏洞的监督作用。

### 4.2 预算执行和财政收支审计

预算执行审计适合写县域财政到底有没有把钱按预算、用途、程序和绩效花出去。

小说转化：

- 审计会关注预算编制是否完整、预算批复和调整是否规范、专项资金使用是否合规、国库支付是否有凭证、结转结余是否合理、政府采购和绩效目标是否落实。
- 审计问题不一定是“钱被贪了”。也可能是资金沉淀、执行率低、预算调整不规范、项目绩效差、超范围支出、账实不符或制度漏洞。
- 审计报告可以推动人大监督、政府整改、财政部门追缴、主管部门完善制度和项目重新评估。

高质量写法：

- 审计发现一个乡镇“民生项目资金执行率 98%”，但服务对象满意度很低，进一步发现钱花在宣传和平台维护上，真正服务交付不足。
- 审计指出某专项资金长期沉淀，部门解释“等项目成熟”，财政追问为何当初入库和绩效目标设置不实。

### 4.3 政府投资项目审计

政府投资项目审计不是“工程师肉眼看质量”。它围绕政府投资和以政府投资为主的建设项目预算执行、竣工决算、资金管理、工程质量、绩效和建设运营情况开展。

小说转化：

- 道路、桥梁、污水管网、学校医院楼、园区基础设施、文旅停车场、冷链物流、安置房、老旧小区改造等，都可进入政府投资审计。
- 审计抓手包括立项批复、概算预算、招投标资料、合同、工程量清单、变更签证、监理记录、检测报告、竣工资料、付款凭证和项目绩效。
- 工程审计不是主角一脚踢出豆腐渣，而是用图纸、合同、工程量、现场复核和检测证据证明偏差。
- 竣工决算审计和审计整改会影响工程款支付、资产入账、责任追究和后续项目安排。

常见戏剧冲突：

- 低价中标后大量变更，审计关注变更签证是否真实、必要、程序合规。
- 项目已投入使用，但竣工资料不完整、资产未入账，后续维护无人负责。
- 审计不是不让付款，而是要求工程量和合同依据对上，避免违规支付。

### 4.4 经济责任审计

经济责任审计关注领导干部任职期间贯彻执行政策、重大经济决策、财政财务收支、国有资产资源管理、重大项目建设、内部管理和廉洁从政从业等经济责任履行情况。

小说转化：

- 经济责任审计常出现在干部离任、任中、换届、提拔、调任或问题线索集中时。
- 审计对象不是普通工作人员，而是党政主要领导干部、部门单位主要负责人、国有企事业单位主要领导人员等。
- 审计不是“谁签字谁全责”。要区分直接责任、主管责任、领导责任，也要看决策程序、分工、当时政策环境和后续纠偏。
- 经济责任审计结果可供干部管理监督、考核任免、奖惩问责使用，也可移送纪检监察、组织人事、主管部门等。

好用冲突：

- 前任留下烂尾项目，现任被群众骂，经济责任审计要拆清前任决策责任、现任整改责任和部门监管责任。
- 领导离任前突击通过几个项目，审计追踪重大经济决策、预算安排和资产处置过程。
- 被审计干部说“集体研究过”，审计要看会议记录、参会人员、不同意见和实际执行。

### 4.5 自然资源资产离任审计

自然资源资产离任审计适合连接生态环境、土地矿山、林地水资源、耕地保护、污染治理和干部责任。

小说转化：

- 审计关注领导干部任期内自然资源资产管理和生态环境保护责任履行情况。
- 县域可写抓手：耕地保护、矿山修复、河湖岸线、林地草地、水资源、污染治理、生态项目资金、违法占地、整改不实。
- 它不是环保执法的替身，而是从领导干部责任和资源资产变化角度审计。
- 问题线索可移送有关部门，责任不因调离、转岗、提拔或退休就自然消失。

高质量写法：

- 领导调任后，当地矿山修复项目账面完成，遥感和现场踏勘却显示复绿只在道路两侧；审计将资金、项目、资源变化和干部责任连起来。

### 4.6 内部审计

内部审计不是外部审计的替代品，但在学校、医院、国企、事业单位、平台公司、村集体经济组织和企业里很适合做早期预警。

小说转化：

- 内部审计关注本单位风险管理、内部控制、财务收支、资产管理、采购合同、建设项目、下属单位和整改落实。
- 内部审计人员要保持独立、客观、公正、保密，不能参与可能影响独立履职的工作。
- 内部审计发现的问题可以内部整改，也可能移送主管部门、审计机关、纪检监察或司法。
- 在小说里，内部审计适合做“有人先发现风险但被压住”的伏笔。

## 5. 审计发现问题的类型库

| 问题类型 | 可写素材 | 常见误写 | 正确写法 |
| --- | --- | --- | --- |
| 预算执行不规范 | 预算调整、超范围支出、资金沉淀 | 一律写成贪污 | 区分违规、低效和腐败 |
| 专项资金使用问题 | 救助、农业、环保、教育、文旅资金 | 专项资金随便挪用无后果 | 写用途限制、追回、整改和问责 |
| 政府投资问题 | 变更签证、工程量、竣工决算 | 主角肉眼断定工程造假 | 用合同、图纸、现场和检测互证 |
| 资产资源管理问题 | 国有资产闲置、集体资产流失 | 资产不见了就等于被偷 | 查登记、使用、处置、收益和责任 |
| 绩效低效问题 | 设备闲置、项目无人用、服务不到位 | 验收过就没有问题 | 写绩效目标和使用效果 |
| 重大决策问题 | 项目论证不足、收益测算乐观 | 领导拍板就全错 | 看决策程序、信息基础和后果 |
| 自然资源问题 | 耕地、矿山、林地、河道、生态修复 | 环保局一罚就完 | 连接资源变化、项目资金和干部责任 |
| 内控缺陷 | 一人管钱管章管合同 | 小错忽略 | 写制度漏洞如何放大风险 |
| 整改不实 | 纸面整改、重复整改、屡审屡犯 | 报告出来就结束 | 写清单销号和回头看 |
| 线索移送 | 涉嫌违纪违法、犯罪 | 审计直接定罪 | 审计移送证据材料，由相关机关处理 |

## 6. 常见角色与真实压力

| 角色 | 想要什么 | 怕什么 | 可写冲突 |
| --- | --- | --- | --- |
| 县审计局审计组长 | 证据扎实、定性准确、整改有效 | 被说影响发展、证据不足、扩大化 | 坚持事实，不把管理问题硬扣腐败帽子 |
| 审计人员 | 查清账、看懂项目、按期出报告 | 资料不全、专业跨度大、被围情绪 | 既要懂财务，也要下工地、问群众、看绩效 |
| 被审计单位负责人 | 保住单位评价、解释历史问题 | 问题被公开、责任被扩大 | 有些问题确实是前任遗留，有些是自己放任 |
| 财政部门 | 预算和支付合规 | 审计指出资金沉淀、违规拨付 | 既是被审计对象，也可能是整改推动者 |
| 项目单位 | 项目继续、工程款支付 | 审计卡款、整改拖延 | 认为审计“找茬”，却拿不出验收依据 |
| 施工或供应商 | 结算、尾款、信用 | 审减、暂停支付、移送 | 争议点在工程量和合同变更 |
| 领导干部 | 经济责任评价公正 | 离任后旧账被翻、前任问题牵连 | 想证明决策有依据，不只是甩锅 |
| 内部审计人员 | 提前发现风险 | 被打压、被边缘化 | 写一份内部审计报告，没人愿意看 |
| 纪检监察人员 | 接住移送线索 | 审计材料不等于违纪事实 | 从审计问题转化为责任调查 |
| 群众/服务对象 | 项目真有用、钱不白花 | 整改只在纸面 | 他们的使用反馈成为审计绩效证据 |

## 8. 高质量剧情用法

### 8.1 审计发现项目闲置

低质写法：审计员看到账本，发现村干部贪污，立刻移送。  
高质写法：审计组先查资金拨付和采购合同，账上没有明显贪污；现场却发现冷链设备长期闲置，电费记录、农户销售记录和绩效目标对不上。问题定性不是贪污，而是项目论证不足、绩效目标虚化、资产管理不到位和主管部门监管不力。

爽点：审计的锋利不只在抓坏人，也在让“合法但无效”的项目无法继续装成功。

### 8.2 政府投资工程审减争议

低质写法：审计把工程款砍掉一半，承包商认栽。  
高质写法：承包商主张变更工程量，审计要求提供设计变更、现场签证、监理确认和检测资料。施工方说“现场确实做了”，但资料缺失；建设单位说“当时赶工来不及补手续”。最后不是简单少付，而是区分可认工程量、程序瑕疵、责任主体和后续整改。

爽点：不是谁更强势，而是谁能让工程量、合同和证据对上。

### 8.3 领导干部经济责任审计

低质写法：干部一离任，审计查出一堆黑料。  
高质写法：经济责任审计围绕任期内重大项目、财政资金、国有资产处置和生态修复推进情况展开。一个项目是前任立项、现任续建、主管部门验收，群众把账全算到现任头上。审计拆清决策责任、执行责任、监管责任和整改责任，反而让人物关系变复杂。

爽点：审计不是替读者找一个坏人，而是把责任链拆到不能再躲。

### 8.4 自然资源资产离任审计

低质写法：环保罚款后，干部就没事了。  
高质写法：领导离任后，自然资源资产离任审计发现矿山修复账面完成、现场复绿不足，水土保持和资金使用存在偏差。环保执法只处理企业违法，审计进一步追问主管部门监管、领导决策和整改责任。

爽点：生态问题不是一张罚单，而是一份离任后仍然追得上的责任账。

### 8.5 审计整改回头看

低质写法：单位提交整改报告，事情结束。  
高质写法：审计整改回头看发现三类问题：已追回资金但制度没改，制度已发文但项目没修，项目已修但群众仍不能用。审计机关要求对账销号，主管部门被约谈，人大代表追问整改结果公开。

爽点：真正的结局不是报告写完，而是纸面整改经得起回访。

## 9. 与既有专题的分工

- 与 `223` 财政预算和公共资金专题：`223` 管资金类型、预算状态、国库支付、绩效和债务风险；本篇管审计如何监督这些资金并推动整改。
- 与 `222` 政府采购和项目验收专题：`222` 管采购交易和履约验收过程；本篇管审计如何复核合同、工程量、付款、绩效和整改。
- 与 `221` 纪检监察和巡察专题：本篇负责审计发现事实、形成审计报告和移送线索；`221` 负责违纪违法、廉洁风险、干部责任和处分问责。
- 与 `224` 统计数据专题：`224` 管统计口径和数据质量；本篇可使用统计数据做审计分析，但审计结论还要靠审计证据。
- 与 `209` 村务公开和集体经济专题：村级财务、集体资产和成员监督看 `209`；涉及审计监督、整改和移送时看本篇。
- 与 `207` 生态环境、`194` 土地宅基地、`195` 乡村项目、`201` 欠薪维权等业务专题：这些专题管业务事实；本篇管审计如何介入、查证和推动整改。

## 10. 禁写与降风险清单

AI agent 写作时，应避免下列内容：

- 详细描写做假账、虚开发票、隐匿账册、销毁凭证、串供、伪造合同、规避审计抽查或转移资金的操作方法。
- 把审计人员写成可以抓人、审讯、封门、判刑的万能执法者。
- 把审计报告当成刑事判决或纪检处分决定。
- 把所有审计问题都写成贪污腐败，不区分管理问题、绩效问题、制度漏洞和违纪违法线索。
- 把整改写成一句“已整改”，不写问题清单、责任主体、整改材料、销号和回头看。
- 泄露审计中接触的国家秘密、商业秘密、个人信息或未公开敏感资料。
- 把审计卡付款、审计核减、审计移送写成单纯刁难，而不写证据和规则。

如果剧情必须出现做假账、毁证、串供或抗拒审计，应把它写成风险、罪错或反派行为，并写出审计追查、责任移送和证据失效后果。

## 11. 审稿检查问题

1. 这场审计属于预算执行审计、财政财务收支审计、政府投资审计、经济责任审计、自然资源资产离任审计、专项资金审计、内部审计还是整改回头看？
2. 审计对象、审计范围和审计触发来源是否清楚？
3. 审计证据是否具体到预算指标、账簿、流水、合同、发票、会议记录、工程量、验收资料、现场核查、访谈或绩效反馈？
4. 是否有审计通知、审计实施、取证、征求意见、审计报告或审计决定等程序节点？
5. 问题定性是否区分真实合法效益问题、管理不规范、绩效低效、制度漏洞、失职失责和涉嫌违纪违法？
6. 责任是否区分直接责任、主管责任、领导责任、前任遗留、现任整改责任和部门监管责任？
7. 审计发现的问题是否正确分流到财政、主管部门、纪检监察、司法、组织人事或整改机制？
8. 审计整改是否有清单、责任、期限、材料、销号、回头看和公开反馈？
9. 是否存在做假账、毁证、规避审计等教程化内容或正向奖励？
10. 审计结果是否改变项目、资金、资产、干部评价、群众信任或后续预算，而不是只让反派下线？

## 13. 写作提醒

审计线真正好看的地方，不是审计员像神探一样秒破案，而是它逼所有人回答一个朴素问题：钱花了、项目建了、干部走了，但公共责任有没有兑现？

商业级县域审计线应同时写出三件事：

- 问题怎么被证据证明。
- 责任怎么被拆到个人、部门和制度。
- 整改后普通人的生活有没有真的改变。
