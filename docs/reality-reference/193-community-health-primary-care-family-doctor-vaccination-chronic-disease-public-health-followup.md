# 193. 社区医院、基层卫生服务、家庭医生、疫苗接种、慢病管理与公共卫生随访链专项

状态：已确认  
日期：2026-06-09  
适用对象：需要描写社区卫生服务中心、社区卫生服务站、乡镇卫生院、村卫生室、基层医疗、家庭医生签约、居民健康档案、基本公共卫生服务、预防接种、儿童保健、孕产妇健康管理、老年人健康管理、高血压糖尿病慢病随访、严重精神障碍患者管理、传染病报告、分级诊疗和双向转诊的 AI agents。

## 1. 本专题解决什么问题

基层医疗不是“三甲医院的低配版”，也不是角色家门口随叫随到的小诊所。它的叙事价值在于连续性：谁给居民建档，谁做健康教育，谁提醒老人复查，谁给孩子查接种证，谁发现慢病失控，谁把病人往上级医院转，谁在传染病和公共卫生事件里最先接触居民。

本专题把基层卫生服务写成可执行链路：

```text
辖区居民/家庭 -> 居民健康档案 -> 家庭医生签约或基层首诊 -> 基本医疗服务/健康咨询 -> 基本公共卫生服务项目 -> 重点人群管理（儿童/孕产妇/老年人/慢病/严重精神障碍） -> 预防接种和健康教育 -> 随访记录与风险评估 -> 异常发现 -> 转诊/上门/报告/家庭沟通 -> 医保和费用衔接 -> 隐私与数据保护 -> 纠纷、投诉、漏管、失访或公共卫生事件 -> 台账回写
```

写作原则：基层医疗最真实的戏，不在“医生神医秒治”，而在一个熟悉社区的医生如何长期看见家庭里的身体、钱、照护、隐私和关系压力。

## 3. 安全边界

本专题只用于小说创作、审稿和设定，不提供诊断、治疗、用药、疫苗选择、急救、心理干预、传染病防控或现实医疗建议。

禁止写成教程的内容：

- 如何自行诊断高血压、糖尿病、传染病、精神障碍或儿童发育问题。
- 如何调整药量、停药、替代治疗、延误就医或规避转诊。
- 如何伪造接种记录、健康档案、随访记录、传染病报告、医保材料或病历。
- 如何逃避传染病调查、隐瞒接触史、规避隔离或阻碍公共卫生处置。
- 如何泄露居民健康档案、精神障碍患者信息、儿童接种记录、孕产妇隐私或老人家庭信息。

可写方向：健康守门人、早期发现、转诊纠结、家属沟通、慢病长期管理、疫苗信任危机、基层资源有限、公共卫生责任和隐私保护。

## 5. 基层机构不是同一种地方

| 机构 | 叙事功能 | 常见场景 | 常见误写 |
| --- | --- | --- | --- |
| 社区卫生服务中心 | 城市居民首诊、慢病复诊、公卫管理、家庭医生团队 | 老人复诊、儿童接种、孕产随访、家医签约 | 写成三甲医院科室齐全 |
| 社区卫生服务站 | 更近的服务点，熟人社会强 | 量血压、换药、健康咨询、转中心 | 写成万能诊所 |
| 乡镇卫生院 | 乡村基层医疗、公卫、转诊和村医管理 | 农村慢病、孕产转诊、传染病报告 | 资源和距离消失 |
| 村卫生室 | 村民日常就医和公共卫生触点 | 乡村医生上门、老人失访、儿童接种提醒 | 写成无证神医点 |
| 家庭医生团队 | 长期签约、健康管理、转诊协调 | 老人、慢病、失能、孕产、儿童家庭 | 写成私人医生随叫随到 |
| 上级医院 | 疑难重症、急危重症、专科诊疗 | 双向转诊、出院回社区、检查结果回流 | 基层和医院完全不沟通 |

写作提醒：基层医疗的“熟”既是优势也是压力。医生认识你妈、知道你爸喝酒、知道你家孩子漏种疫苗，也可能知道不该随便被邻居知道的隐私。

## 6. 居民健康档案：基层医疗的故事账本

居民健康档案不是装饰性表格。它记录身份、家庭、既往病史、慢病、接种、随访、生活方式和重点人群管理，是基层剧情的连续性工具。

可写节点：

- 新搬来的租户没有建档，社区医生上门摸排时被误以为推销。
- 老人子女长期不在，健康档案里反而比家里人更清楚他的血压变化。
- 家庭医生发现老人多次未按时复诊，打电话才知道老人摔倒住院。
- 居民担心档案泄露，不愿透露精神病史或传染病接触情况。
- 社区卫生系统里名字、身份证、手机号录错，导致接种通知和随访通知错发。

证据功能：档案、随访表、接种记录、签约协议、转诊单、电话记录、微信通知都能成为后续纠纷和温情回收的证据。

## 7. 家庭医生签约：不是“私人医生”

家庭医生签约服务适合写长期关系，但不能写成高端私人医疗。它强调基层医生团队与居民之间的稳定服务关系、健康咨询、就诊指导、慢病管理、转诊和重点人群照护。

写作规则：

- 签约对象可以是个人，也可以围绕家庭展开，但服务内容、有效期、联系方式和服务包要写清。
- 家庭医生不是 24 小时免费上门医生，资源有限，任务很多，优先级要合理。
- 签约关系能制造信任：医生知道家里谁做主，谁隐瞒病情，谁怕花钱，谁照护老人。
- 签约关系也能制造冲突：居民以为签了约就能优先住院，医生解释转诊规则，家属觉得被推诿。
- 家庭医生和上级医院之间的转诊、用药、检查结果回传，是现实感关键。

高张力场景：

- 独居老人不接电话，家庭医生让网格员和居委会一起上门。
- 家属在群里质问“签约医生为什么不救我爸”，医生拿出多次随访未接记录。
- 年轻人嫌家庭医生打扰，直到母亲糖尿病指标失控才意识到这些电话不是形式主义。

## 8. 基本公共卫生服务：最适合写群像

基本公共卫生服务不是只有打疫苗。它覆盖健康档案、健康教育、儿童、孕产妇、老年人、慢病、严重精神障碍、传染病和卫生监督协管等多个方向。小说中可以把它写成社区里看似琐碎、实则保命的信息网络。

常见服务节点：

- 居民健康档案和健康教育。
- 0-6岁儿童健康管理、孕产妇健康管理和老年人健康管理。
- 高血压、2型糖尿病等慢性病患者健康管理。
- 严重精神障碍患者管理和转介衔接。
- 传染病及突发公共卫生事件报告和处理。
- 卫生计生监督协管、结核病患者健康管理、中医药健康管理等地方项目接口。

写作技巧：让不同角色因为同一张通知单交叉。比如儿童接种日、老人免费体检、慢病随访、传染病排查同一天发生，社区卫生服务中心像一个小型舞台。

## 9. 预防接种：信任、记录和追溯

疫苗接种戏不要只写“打一针”。要写接种单位、预约或通知、接种证、告知询问、禁忌核查、留观、接种记录、疑似异常反应、家长焦虑和公共卫生信任。

写作规则：

- 儿童接种通常会牵动接种证、学校托幼查验、家长请假、漏种补种和家庭意见分歧。
- 接种前要有告知、询问和核对；接种后要有留观和记录，不要让护士随手一扎就结束。
- 疫苗犹豫要写信息来源：家长群、短视频、旧事故、亲友经历、医生解释和官方记录。
- 疑似预防接种异常反应要写报告、调查、医学判断和沟通，而不是直接定性。
- 接种记录是强证据：批号、时间、接种单位、受种者、接种医生和系统记录都可能推动剧情。

## 10. 慢病管理：长期压力比急病更磨人

慢病管理适合写家庭现实：药费、复诊、饮食、运动、情绪、老人固执、子女异地、基层医生反复提醒。

高血压、糖尿病等慢病管理叙事节点：

- 建档和风险评估。
- 定期随访和指标记录。
- 用药依从性、生活方式、复诊检查。
- 异常指标的复查、转诊和上级医院衔接。
- 家庭照护者疲惫、老人不愿承认病情、年轻患者自我放任。
- 医生指标压力和真实服务之间的拉扯。

写作提醒：不要提供具体用药方案和治疗目标。小说只写“随访、异常、复诊、转诊、家庭沟通和后果”。

## 11. 严重精神障碍和心理支持：隐私优先

基层公卫可能涉及严重精神障碍患者管理和心理健康支持，但这类情节高敏感，必须联动第105篇和第159篇。

写作规则：

- 不污名化，不把患者写成天然危险或恐怖工具。
- 随访、服药、家属照护、社区支持和专业医疗衔接要写得克制。
- 隐私极重要，邻居、物业、业主群不能随意传播患者信息。
- 出现自伤、伤人风险或严重心理危机时，不写方法细节，转向保护、求助、专业转介和家属支持。
- 社区医生不是心理危机万能解决者，要写转介和多方协同。

## 12. 传染病报告和公共卫生事件

截至 2026 年，传染病防治法已经完成 2025 年修订。现实题材涉及传染病报告、监测、调查、学校托幼、社区随访和突发公共卫生事件时，不能只靠“居委会通知”。

写作规则：

- 医疗机构、疾控机构、学校、托育机构、社区和家庭之间可能形成联动。
- 传染病报告不是八卦传播，必须处理隐私、必要范围和专业分工。
- 社区随访可能让居民不满：有人觉得被打扰，有人害怕被邻居知道，有人拒绝配合。
- 公共卫生处置要避免污名化患者和家庭，不把传染病当道德审判。
- 群聊谣言、抢药、隐瞒行程、拒绝检测等情节要写后果，不给规避方法。

## 13. 分级诊疗和双向转诊

基层医疗的戏剧点经常在“该不该转、能不能转、转到哪里、谁接得住”。

可写冲突：

- 社区医生建议转三甲，老人舍不得钱和路费，坚持在社区拿药。
- 三甲医院出院小结要求社区随访，家属以为出院就结束。
- 家庭医生帮忙预约专科，患者觉得“还是要自己排队，签约有什么用”。
- 上级医院检查结果没有及时回到社区，随访医生只能问家属拍照发来。
- 基层医生怕漏诊，患者嫌他“胆小推人”。

写作原则：转诊不是甩锅，也不是绿色通道万能钥匙。它是资源分层和风险判断。

## 14. 隐私、熟人社会和健康数据

基层医疗最独特的风险，是医生和患者可能住在同一片社区。

高风险信息：

- 传染病、精神健康、孕产、避孕、未成年人健康、老人认知退化、家暴伤情、医保困难、低保信息。
- 居民健康档案、接种记录、随访记录、处方、转诊单、微信群通知、上门照片。
- 社区医生、居委会、网格员、物业之间的信息流转。

写作规则：

- 健康数据共享要有必要性和边界，不要让社区群随便知道某人病情。
- 上门随访和电话通知要考虑家属关系：有些信息不能让控制型伴侣、邻居或房东先知道。
- 医生一句无心的“你妈今天血糖又高了”，可能引爆家庭冲突。
- 隐私泄露要有后果：投诉、处分、关系破裂、患者失访、社区信任下降。

## 15. 纠纷处理与责任分流

基层医疗纠纷不要全部写成“去大医院闹”。先判断争议性质。

分流规则：

- 普通诊疗、转诊、用药、病历争议：接入第169篇医疗服务和病历链。
- 基本公共卫生随访、建档、漏管、失访、健康档案错误：接入基层卫生机构、卫生健康主管部门和内部整改链。
- 预防接种争议、疑似异常反应、接种记录错误：接入疫苗管理、疾控、接种单位和医学调查链。
- 传染病报告、疫情处置、隐瞒或泄露：接入传染病防治、疾控、公安或行政责任链。
- 健康数据泄露、隐私传播、社区群扩散：接入个人信息保护、数字证据、平台和民事责任链。
- 家庭医生签约服务不满：接入签约协议、服务包、机构投诉、医保或主管部门评价链。
- 上门服务、老人失联、家暴伤情、心理危机：接入社区治理、警务、心理健康和未成年人/老人保护链。

## 16. 类型文里的用法

| 类型 | 用法 | 爽点或痛点 |
| --- | --- | --- |
| 都市现实 | 社区医生、老人慢病、儿童接种、家庭医生 | 家门口的小事慢慢决定命运 |
| 乡村现实 | 村医、乡镇卫生院、转院距离和熟人社会 | 资源有限但关系很近 |
| 家庭伦理 | 老人不听医嘱、子女异地、家长疫苗焦虑 | 健康问题暴露家庭权力 |
| 医疗文 | 分级诊疗、转诊、基层和三甲衔接 | 医疗系统不是只有急诊和手术台 |
| 社区群像 | 体检日、接种日、健康讲座、随访电话 | 一张表串起很多家庭 |
| 悬疑 | 健康档案、接种记录、传染病报告、失访 | 记录里的空白比记录更有戏 |
| 灾害/公共卫生 | 社区排查、传染病报告、家庭隔离困境 | 保护和隐私的拉扯 |
| 重生逆袭 | 提前建档、及时转诊、阻止漏管失访 | 爽点来自连续照护而非神药 |

## 18. 写作提示词

### 18.1 生成基层随访场景

```text
请根据第193篇基层卫生与公共卫生随访链，生成一个社区医生上门或电话随访场景。必须包含：居民健康档案、家庭医生或公卫人员身份、重点人群类型、随访原因、家庭阻力、记录证据、下一步转诊或复诊安排。不要提供诊疗、用药或急救建议。
```

### 18.2 审稿社区医院情节

```text
请审稿以下社区医院/基层医疗情节，检查是否存在：把社区医院写成三甲、家庭医生私医化、健康档案缺失、随访无记录、疫苗接种流程悬浮、慢病管理变成用药教程、传染病报告被八卦化、隐私泄露无后果、AI 味过重。按“问题-影响-修改建议”输出。
```

### 18.3 设计儿童接种纠纷

```text
请设计一个儿童预防接种纠纷桥段。要求：写清接种证、预约或通知、家长焦虑来源、接种单位核对、留观或记录、家校/托幼查验接口和后续沟通。不要提供疫苗选择或医学判断建议。
```

### 18.4 设计慢病管理家庭冲突

```text
请设计一个慢病随访引发的家庭冲突。要求：基层医生只提供随访、复诊或转诊建议，不写具体治疗方案；冲突来自老人固执、子女异地、费用压力、饮食习惯或隐私泄露；结尾回写下一次随访和家庭关系。
```

### 18.5 设计公共卫生信息泄露桥段

```text
请设计一个基层公卫信息泄露桥段。要求：信息来源、传播路径、被泄露者损害、机构回应、证据链和整改后果都要清楚；不得公开真实诊断细节或把传染病、精神健康污名化。
```

## 19. 自查清单

- 是否区分社区卫生服务中心、站、乡镇卫生院、村卫生室和上级医院？
- 是否写出居民健康档案、签约协议、随访表、接种记录或转诊单？
- 家庭医生是否被写成私人医生？基层医生是否被写成神医？
- 基本公共卫生服务是否只剩打疫苗，还是覆盖儿童、孕产、老人、慢病、传染病等？
- 疫苗接种是否有告知、询问、核对、留观、记录和追溯？
- 慢病管理是否避免具体用药和治疗建议，只写随访、异常、复诊和转诊？
- 传染病报告是否保护隐私，避免群聊八卦和污名化？
- 严重精神障碍或心理危机是否联动第105、159篇，避免污名和方法细节？
- 分级诊疗和双向转诊是否有原因、材料和结果？
- 健康档案和随访数据是否处理个人信息保护和熟人社会风险？
- 结尾是否回写下一次随访、健康状态、家庭关系、费用、隐私或信任？

## 20. 最小可用规则

如果时间很紧，至少执行以下 8 条：

1. 先确认机构层级：社区、乡镇、村室还是上级医院。
2. 基层医疗必须有档案、签约、随访、接种、转诊或公卫记录之一。
3. 家庭医生不是私人医生，资源和服务边界要清楚。
4. 疫苗情节写记录和追溯，不写医学判断和接种建议。
5. 慢病情节写随访和转诊，不写用药教程。
6. 传染病和精神健康信息必须保护隐私，避免污名化。
7. 纠纷要分流到基层机构、卫健、疾控、医保、公安、法院或专业医疗链。
8. 结局必须改变健康档案、下一次随访、转诊路径、家庭关系或社区信任。
