# 237 县域自然灾害预警、转移避险、防汛抗旱、地质灾害、安置救助与灾后恢复专项

状态：已确认

适用对象：AI agents 在设计、起草、审稿或修订县域自然灾害、气象预警、暴雨洪涝、山洪、台风、干旱、地震、地质灾害、滑坡泥石流、防汛抗旱指挥、乡镇村社转移避险、避险安置点、灾情统计报送、救灾物资、受灾群众基本生活救助、倒损住房恢复重建、冬春救助、农业灾损、灾后恢复、舆情谣言和基层干部抗灾群像时使用。

关联文档：第 164 篇负责灾害事故场景生命周期和救援调度；第 236 篇负责安全生产事故和应急管理安全监管；第 208 篇负责水电气通信中断和民生保供；第 196 篇负责农业保险、贷款和灾后恢复资金；第 210 篇负责社会救助；第 225 篇负责审计监督和资金整改；第 221 篇负责纪检监察和群众身边腐败；第 190 篇负责无人机救援和空域数据；第 143 篇负责天气季节对叙事氛围和行动限制的影响。

本篇只补“自然灾害预警转移和救灾救助制度链”。不要把它写成第 164 篇的灾难大场面复刻，也不要把防汛抗旱写成单纯英雄救援。

## 1. 一句话定位

县域自然灾害桥段的可信底座是：

```text
风险季节和隐患点 -> 监测预警 -> 会商研判 -> 指令下达 -> 乡镇村社叫应和转移避险 -> 安置点运行 -> 灾情统计和上报 -> 救灾款物发放与公示 -> 基础设施和生产生活恢复 -> 倒损住房、农业损失、冬春救助和长期台账回写
```

写作目标：让 AI agent 把自然灾害写成“天气、地形、制度、家庭、资源和基层信任共同变化”的长链，而不是突然暴雨、主角救人、第二天一切恢复。

## 2. 先修正常见 AI 错误

| 常见写法 | 问题 | 正确处理 |
|---|---|---|
| 暴雨预警只是气氛音效 | 预警没有触发组织动作 | 写监测预警、会商研判、值班值守、叫应、转移和安置准备 |
| 村干部一喊，全村立刻转移 | 忽略人性和成本 | 写老人、病人、儿童、牲畜、财物、农田、行动不便者和不愿转移者 |
| 救灾物资凭空出现 | 资源链断裂 | 写储备、调拨、运输、接收、登记、公示、发放、监督和余缺调剂 |
| 灾情上报就是夸大或隐瞒 | 过度阴谋化 | 写初报、续报、核报、现场核查、口径统一和数据误差 |
| 转移避险等于撤出就安全 | 后半段缺失 | 写安置点饮水、食物、药品、隐私、卫生、无障碍、宠物牲畜和信息需求 |
| 防汛就是堤坝上喊口号 | 场景空泛 | 写水位、雨情、堤防巡查、险情处置、闸泵调度、蓄滞洪区、群众转移 |
| 地质灾害靠主角看山体裂缝秒懂 | 专业链缺失 | 写隐患点巡查、群测群防、气象风险预警、自然资源部门、专业调查 |
| 灾后重建一笔钱解决 | 后果太轻 | 写倒损住房、临时安置、维修重建、农业补种、保险理赔、贷款展期、冬春救助 |
| 基层干部天然正确或天然无能 | 人物扁平 | 写他们的信息不完整、资源不足、责任压力、群众信任和家庭牺牲 |
| 主角单刷救灾 | 机构失真 | 主角只能提供局部贡献，不能替代指挥体系、专业救援、村社组织和公共资源分配 |

## 3. 角色和机构分工

| 角色 | 叙事功能 | 可写动作 | 不能写成 |
|---|---|---|---|
| 县防汛抗旱指挥机构/应急指挥体系 | 统筹会商、调度、响应和协调 | 值班值守、会商研判、启动响应、调拨队伍物资、协调部门 | 一个领导拍脑袋全县动起来 |
| 应急管理部门 | 灾害应急、救助、物资、灾情统计和综合协调 | 组织救灾救助、灾情报送、物资调拨、救助审核、应急预案衔接 | 万能气象水利自然资源专家 |
| 气象部门 | 天气监测预报预警 | 发布暴雨、台风、高温、寒潮等预警，提供滚动预报 | 对具体滑坡点和水库调度全权定夺 |
| 水利部门 | 水情、雨情、山洪、水库河道和防汛工程 | 水位监测、山洪灾害防御、水库闸泵调度、堤防巡查技术支撑 | 只在灾后背锅 |
| 自然资源部门 | 地质灾害防治 | 隐患点排查、地灾气象风险预警、专业调查、避险建议 | 被村民随口替代专业判断 |
| 乡镇街道 | 属地组织和执行 | 叫应村社、组织转移、统计灾情、协调安置、巡查风险点 | 只会转发通知 |
| 村社区/网格员/灾害信息员 | 末端触达和第一手信息 | 敲门、电话、喇叭、微信群、登记特殊人群、反馈险情 | 一句话说服所有群众 |
| 消防救援、武警、民兵、社会救援队 | 专业和辅助救援 | 搜救、转运、排涝、道路清障、物资运输 | 各自为战、不受统一指挥 |
| 民政、卫健、教育、交通、住建、农业农村 | 专门民生领域 | 临时救助、医疗防疫、停课复课、道路抢通、房屋安全、农业灾损 | 全都隐身 |
| 受灾群众 | 行动选择、损失和关系压力 | 决定是否转移、保护家人、安置点生活、申报损失、重建 | 只负责哭或被救 |
| 村干部家属/基层干部家庭 | 基层牺牲和情绪账 | 家里也受灾，却还要值守、送物资、做群众工作 | 只当伟光正背景 |
| 媒体和自媒体 | 信息传播和舆情压力 | 辟谣、求助扩散、曝光问题、记录互助 | 一味煽动或一味正能量 |
| 纪检审计财政 | 灾后资金和廉洁监督 | 检查救灾款物、补助名单、工程重建和公示 | 灾后再无监督 |

## 4. 预警到转移避险链

```text
天气或水情地质风险出现
-> 气象/水利/自然资源监测预警
-> 县级会商研判和响应启动
-> 预警信息发布到部门、乡镇、村社和群众
-> 乡镇村社叫应重点人群和危险区域
-> 转移路线、车辆、安置点、物资和特殊人群准备
-> 入户劝导、强降雨临近、道路变化和临时决策
-> 人员转移、登记、安置和回流管控
-> 灾情核查、救助发放和恢复重建
```

写法要点：

1. “预警”要落到人。谁收到、谁转发、谁确认、谁敲门、谁不接电话、谁负责独居老人和留守儿童。
2. “转移”要有阻力。群众不愿走可能不是愚蠢，而是怕牲畜丢、怕房子被偷、怕老人折腾、怕误工、怕安置点不方便。
3. “叫应”要闭环。仅发微信群不够，要有人确认重点户是否知晓、是否转移、是否需要帮助。
4. “撤离路线”会变化。暴雨导致塌方、漫水桥、停电、通信中断，原本预案可能需要临场调整。
5. “回流”同样危险。水退一点就回家取东西，可能引发二次风险。

## 5. 防汛抗旱写法

防汛抗旱不是同一套场景换天气。防汛重在“短时高压、转移避险、工程调度、险情处置”；抗旱重在“长期资源分配、农业生产、饮水保障、产业和家庭现金流”。

| 类型 | 主要压力 | 可写材料 | 叙事后果 |
|---|---|---|---|
| 暴雨内涝 | 城区排水、地下空间、交通、停电 | 雨量、水位、泵站、积水点、交通管制 | 上班上学中断、商铺受损、抢险排涝 |
| 山洪 | 山区沟谷、夜间强降雨、预警转移 | 山洪预警、危险区名册、转移路线、叫应记录 | 老人不愿走、道路被冲断、村干部背责 |
| 江河洪水 | 水位、堤防、圩区、蓄滞洪区 | 水位站、巡堤记录、险情报告、启用补偿 | 转移规模大、财产损失、上下游利益冲突 |
| 台风 | 风雨潮叠加、停课停工、海边山边风险 | 台风路径、停航停运、避风渔船、广告牌树木 | 物流中断、旅游退订、城市管理压力 |
| 干旱 | 饮水、灌溉、养殖、森林火险 | 水源调度、送水记录、抗旱机井、农业灾损 | 家庭现金流、邻里用水纠纷、保险贷款 |

防汛桥段高质冲突：

- 堤防巡查发现管涌苗头，老村支书知道旧险工段，年轻干部有无人机图像但不懂历史水路。
- 县城地下商场积水，商户要抢货，消防和城管要求封控，店主担心贷款断供。
- 蓄滞洪区启用牵涉转移、补偿、庄稼、牲畜和“为什么牺牲我们这里”的情绪。
- 干旱时上游水库、下游灌区、养殖户、学校饮水和工业用水之间发生现实分配冲突。

## 6. 地质灾害写法

地质灾害常见于山区县、矿山周边、公路边坡、旅游民宿、学校道路、移民安置点和暴雨后山体。

核心链：

```text
地形和历史隐患 -> 巡查排查 -> 降雨或工程扰动 -> 地质灾害气象风险预警 -> 群测群防员/村干部发现变形迹象 -> 自然资源部门和专业技术人员核查 -> 临时避险转移 -> 封控和监测 -> 灾后调查、治理工程或搬迁避让
```

可写细节：

- 裂缝、落石、浑水、树木歪斜、房屋变形等只能作为风险信号，不能让主角凭直觉定性。
- 地质灾害治理不只是修挡墙，可能涉及监测、搬迁、避让、工程治理、道路改线、矿山修复和群众生计。
- 群测群防员不是专业专家，但他们熟悉本地山、水、路和人，是预警转移的关键末端。
- 暴雨夜“谁去叫醒最后一户”适合写基层人物高光，但要避免让无防护角色冒险作秀。

## 7. 安置点和救灾救助写法

安置点不是“大家睡体育馆”一句话。它是人性密度很高的群像场：

| 需求 | 可写细节 | 冲突 |
|---|---|---|
| 登记 | 姓名、户籍、家庭成员、特殊需求、原住址 | 漏登、重复登记、外来人口是否纳入 |
| 食物饮水 | 饮用水、热饭、婴幼儿和老人饮食 | 物资不足、口味、排队、优先顺序 |
| 医疗防疫 | 慢病药、孕妇、伤员、消杀、心理支持 | 药断了、隐私病史、传染病担忧 |
| 隐私和安全 | 男女分区、夜间照明、儿童看护、财物保管 | 拥挤、偷拍、谣言、失窃 |
| 信息 | 家属联络、灾情通报、返家时间、救助政策 | 谣言、焦虑、手机没电、信号差 |
| 无障碍 | 残疾人、老人、病人、轮椅和助听 | 安置点不适配、照护者崩溃 |
| 宠物牲畜 | 宠物、猪牛羊鸡、农机农具 | 感情、财产和公共卫生冲突 |
| 回流管理 | 何时能回家、谁评估安全 | 偷偷回去拿东西、二次灾害 |

自然灾害救助常见类型：

- 紧急转移安置和基本生活救助。
- 倒塌、损坏住房恢复重建或维修救助。
- 因灾遇难人员家属抚慰。
- 旱灾临时生活困难救助。
- 冬春临时生活困难救助。

写作要点：救助不是施舍。它需要申请或摸排、核查、民主评议或公示、资金物资发放、监督和后续复核。对 AI agent 来说，最有戏的是名单边缘的人：外来租户、离婚回村女性、没有正式户籍资料的老人、失联家属、承包地不在本村的人、灾前就困难但灾后更困难的人。

## 8. 灾情统计和救助资金链

```text
灾害发生
-> 村社初步摸排人员伤亡、房屋、农作物、基础设施、直接经济损失
-> 乡镇汇总上报
-> 县级核查、会商和续报
-> 必要时上级核查评估
-> 救灾资金物资下达
-> 分配方案、名单审核、公示和发放
-> 群众质疑、复核、投诉、审计纪检监督
-> 结余、追回、整改和长期台账
```

写作规则：

- 灾情数据有时间性：初报快但粗，续报不断修正，核报更接近真实。不要把第一份数据写成最终真相。
- 面积、损失、受灾人口、倒损房屋、转移安置人数都可能因统计口径不同产生争议。
- 救灾款物要避免“干部私吞”一招鲜。更真实的矛盾包括优先级、重复领取、信息不对称、群众误解、亲友关系、弱势群体漏保和名单公示压力。
- 审计、纪检、财政、应急和乡镇可以在灾后形成二次剧情：一袋米、一张表、一条群众留言都可能牵出责任链。

## 9. 场景入口表

| 场景 | 第一个问题 | 关键材料 | 冲突点 | 后果回写 |
|---|---|---|---|---|
| 暴雨红色预警 | 谁收到预警，谁负责叫应 | 预警信息、值班记录、重点户名册、转移路线 | 老人不愿走、道路快断、干部人手不够 | 安置、回流、问责或表扬 |
| 山洪转移 | 危险区是否有明确名册 | 山洪预警、村广播、敲门记录、车辆安排 | 夜间转移、留守儿童、牲畜财物 | 人员安全、财产损失、邻里互助 |
| 地质灾害隐患点 | 是否巡查到变形迹象 | 隐患点台账、巡查照片、专家意见、封控通知 | 民宿旺季不愿停业 | 避险搬迁、工程治理、赔偿争议 |
| 县城内涝 | 排水和交通如何中断 | 雨量、积水点、泵站记录、交通管制 | 商户抢货、地下空间封控 | 保险理赔、停业、责任争议 |
| 蓄滞洪区启用 | 谁被转移，补偿如何说明 | 启用指令、转移名单、财产登记、补偿政策 | 上下游公平感、庄稼被淹 | 补偿、重建、干部信任 |
| 干旱送水 | 谁优先用水 | 水源调度、送水记录、困难户名单、农业灾损 | 饮水与灌溉、工业与民生 | 农业保险、贷款展期、邻里纠纷 |
| 安置点运行 | 安置点是否满足基本生活 | 登记表、物资台账、医疗记录、志愿者名单 | 隐私、慢病药、儿童照看、谣言 | 群众评价、干部疲惫、关系升温 |
| 救灾物资发放 | 名单是否准确公正 | 接收单、发放表、公示、投诉记录 | 漏发、重复、亲疏质疑 | 复核、追回、纪检审计 |
| 倒损住房重建 | 房屋是否安全，谁有资格 | 房屋鉴定、申请、评议、公示、补助 | 老宅情感、宅基地、资金不足 | 迁建、修缮、家庭分裂 |
| 冬春救助 | 灾后几个月谁还困难 | 前期灾损、收入变化、救助名单、回访记录 | 灾后热度过去，困难仍在 | 长线生活压力和人物成长 |

## 10. 证据和材料清单

| 材料 | 用途 | 常见误写 |
|---|---|---|
| 气象预警信息 | 触发值守、叫应和转移准备 | 只当氛围背景 |
| 防汛抗旱会商记录 | 说明研判和响应依据 | 领导随口决定 |
| 值班值守记录 | 证明责任链和时间线 | 灾前所有人都睡着 |
| 重点人群和危险区名册 | 转移避险末端抓手 | 只喊“全村转移” |
| 转移路线和安置点清单 | 说明撤往哪里、怎么管 | 到了体育馆就完事 |
| 敲门、电话、微信群、广播记录 | 证明预警触达和叫应 | 发一条消息等于完成 |
| 水位、雨量、山洪、地灾监测资料 | 支撑专业判断 | 主角凭直觉判断洪峰或滑坡 |
| 灾情初报、续报、核报 | 反映灾情变化 | 第一份数据就是最终真相 |
| 物资接收、调拨和发放表 | 救灾资源链 | 物资凭空出现或凭空消失 |
| 受灾群众救助申请/摸排表 | 救助资格和生活困难证明 | 领导一句话给钱 |
| 民主评议和公示材料 | 防止名单争议 | 救助不需要公开 |

| 审计、纪检、投诉和复核材料 | 灾后监督 | 救灾款物发完就无后续 |

## 12. 可直接使用的桥段模板

### 12.1 暴雨夜叫醒最后一户

低质写法：村干部喊一声，老人全都上车，山洪来了大家安全。

高质写法：暴雨红色预警后，村里危险区名册显示还有三户没确认。独居老人手机关机，儿子在外地，老人舍不得屋后的牛和刚收的玉米。村干部不是热血冲锋，而是先安排转移车、通知邻居、联系兽医站或临时看护点，再和网格员上门。真正的冲突是：水声越来越近，老人不相信“以前几十年都没事”的经验会失效。

### 12.2 安置点里的一盒降压药

低质写法：受灾群众住进体育馆，志愿者送来盒饭，大家感动。

高质写法：安置点登记时发现多名老人没带慢病药，孕妇需要单独休息区，外来租户不知道能不能领物资，孩子因为手机没电联系不上母亲。干部和医生一边稳定秩序，一边补登记、找药、分区、辟谣。灾害爽点不在“大家鼓掌”，而在一个混乱场景里重新建立最基本的生活秩序。

### 12.3 地灾隐患点和民宿旺季

低质写法：女主看见山体裂缝，断定马上滑坡，老板被打脸。

高质写法：暴雨后民宿后山出现裂缝，群测群防员拍照上报，自然资源部门和技术人员到场核查，乡镇要求临时封控和转移游客。民宿老板刚接满暑期订单，觉得“是不是太夸张”。冲突不是老板坏，而是现金流、口碑、游客安全和专业风险判断之间的撕扯。

### 12.4 救灾物资名单争议

低质写法：村干部私吞物资，被主角当场抓住。

高质写法：物资发放表公示后，有人质疑某户房子没进水却领了棉被，另一户外来租住家庭受灾严重却不在名单。村干部拿出摸排表解释口径，但也承认临时转移人员统计漏了。主角帮助补证据、复核名单，而不是直接把所有干部写成贪腐。最后真正改变的是公示、复核和回访流程。

### 12.5 干旱送水和邻里冲突

低质写法：县里派水车，村民排队领水，旱灾结束。

高质写法：连续干旱后，村里优先保障人畜饮水，果园灌溉只能排后。养殖户怕牲畜死亡，老人家里水缸见底，合作社担心贷款到期。送水车路线、取水点、每户额度都成了矛盾。抗旱不是一场爆点，而是慢慢磨损家庭现金流和村庄信任的长线压力。

## 13. 审稿检查问题

1. 是否写清灾害类型、季节、地形和历史隐患，而不是“突然天灾”？
2. 是否有预警来源、会商研判、响应启动、值班值守和末端叫应？
3. 是否写出重点人群、危险区名册、转移路线、安置点和回流管理？
4. 群众不愿转移是否有真实生活成本，而不是被写成愚昧？
5. 安置点是否有饮水、食物、药品、隐私、卫生、无障碍、信息和秩序问题？
6. 防汛、抗旱、地质灾害是否分别写出专业部门和不同压力，而不是全部归给“应急局”？
7. 灾情数据是否区分初报、续报、核报和现场核查？
8. 救灾款物是否有接收、登记、发放、公示、复核和监督？
9. 灾后是否回写住房、农业、保险、贷款、学校复课、交通通信、冬春救助和基层信任？
10. 是否避免危险涉水、自行救援、私闯封控区、造谣、伪造灾情、截留物资等教程化内容？

## 14. 安全边界

- 不写危险涉水、闯入塌方区、私自返回危房、穿越封控线、靠近滑坡泥石流、私自打开闸门、破坏防汛设施、冲击救援现场、伪造灾情、截留物资、冒领救助、制造谣言的现实操作步骤。
- 不把转移避险写成胆小或丢脸，也不鼓励“主角逆行证明勇敢”。必要的逆行必须由专业人员在统一指挥下执行。
- 不泄露受灾群众隐私、死亡细节、未成年人身份和家庭财产清单。
- 不把救灾款物争议一概写成干部贪腐。可以写腐败和失职，但要进入纪检、审计、财政、应急和司法链。
- 不把灾后重建写成“一笔钱恢复原样”。灾后恢复有时间、资金、审批、施工、家庭分歧和心理后果。
