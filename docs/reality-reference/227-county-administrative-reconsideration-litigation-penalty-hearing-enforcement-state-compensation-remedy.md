# 227 县域行政复议、行政诉讼、行政处罚听证、行政强制、国家赔偿与行政救济专项

状态：已确认  
更新时间：2026-06-09  
适用对象：县域现实题材、政务服务、市场监管、生态环境、城管执法、自然资源、教育培训、社会救助、行政许可、行政处罚、行政强制、信息公开、征收补偿、基层信访分流、企业维权、群众救济和法治政府剧情。

## 1. 本专题解决什么问题

AI 写群众不服行政处理时，常把所有出口混成一句“去找领导”：

- 被罚款就上网曝光，第二天处罚撤销。
- 行政许可被拒，主角去信访大厅一闹，窗口马上办证。
- 行政处罚没有调查、告知、陈述申辩、听证、决定和送达，干部一句话就罚。
- 行政强制被写成执法人员当场砸门、搬东西、封店，没有法律依据、程序、文书和救济。
- 行政复议、行政诉讼、行政赔偿、国家赔偿、纪检举报、信访、12345、民事诉讼全混成一个万能“申诉”。

本专题的目标是给 AI agent 一条可执行的行政救济链：先判断是不是行政行为，再判断应走复议、诉讼、听证、申诉投诉、信息公开、国家赔偿还是其他路径，最后写清材料、期限、证据、决定、执行和后果。

## 2. 一句话原则

行政救济戏的爽点不在“闹一闹就改”，而在当事人把行政机关的事实、证据、依据、程序和裁量一步步拉回可审查轨道。

## 3. 标准叙事链

```text
行政机关作出或未作出某个行为
-> 判断是否属于行政行为：处罚、许可、确认、征收征用、强制、信息公开答复、不履行职责、行政协议、补贴救助决定、监管处理等
-> 拿到或确认文书：告知书、决定书、不予受理通知、责令改正、处罚决定、强制决定、信息公开答复、补偿决定、送达凭证
-> 初步审查四件事：主体权限、事实证据、法律依据、程序和裁量
-> 判断前置程序：是否有陈述申辩、听证、补正、内部复核、投诉举报、信息公开申请、复议前置或直接起诉空间
-> 选择路径：行政复议、行政诉讼、行政赔偿/国家赔偿、信访分流、纪检监察、行政执法监督、民事诉讼、仲裁或放弃争议
-> 准备材料：身份、行政文书、证据、申请书或起诉状、授权委托、期限证明、送达记录、损失材料、法定代表或利害关系证明
-> 受理审查：是否属于范围、是否超过期限、申请人是否有利害关系、被申请人/被告是否正确、请求是否明确
-> 审查办理：复议机构审理、听取意见、调解和解、作出复议决定；或法院立案、举证、开庭、审理、判决
-> 结果分流：维持、撤销、变更、确认违法、责令履行、重新作出、赔偿、驳回、调解、和解、执行或上诉
-> 后果回写：许可恢复、处罚撤销或减轻、强制解除、赔偿到账、信用修复、执法整改、部门公信力、人物关系和台账更新
```

## 4. 当前制度底座的小说转化

### 4.1 行政复议法：行政争议化解主渠道

已确认规则：新修订《中华人民共和国行政复议法》自 2024 年 1 月 1 日起施行。行政复议被明确作为化解行政争议的重要渠道。另需注意：修订后的《中华人民共和国行政复议法实施条例》已于 2026 年 4 月 29 日公布，将自 2026 年 7 月 1 日起施行；写 2026 年 7 月 1 日以后情节时，应按新条例理解。

小说转化：

- 行政复议不是“求上级领导开恩”，而是由行政复议机关依法审查行政行为是否合法、适当。
- 适合写：对行政处罚、行政许可、不予公开信息、征收补偿、社会救助决定、行政强制、行政不作为等不服。
- 复议可以纠正违法或不当行政行为，也可以推动行政机关履职、完善制度和改进行政执法。
- 复议申请要有申请人、被申请人、具体行政行为、复议请求、事实理由和证据材料。
- 复议决定可能维持、撤销、变更、确认违法、责令履行或重新作出行政行为，不是只有“赢/输”。

风险：

- 把复议写成信访接待或领导批示，会失真。
- 把复议机关写成可以随意替原部门重新审批、替法院裁判民事权利，也会混淆职责。
- 行政复议与行政诉讼关系要按事项判断，有些可以先复议再诉讼，有些可直接诉讼，有些有复议前置要求；不要一律写成任选。

检查问题：

- 当事人不服的是哪个具体行政行为？
- 被申请人是谁？复议机关是否找对？
- 复议请求是撤销、变更、确认违法、责令履行，还是赔偿？
- 是否保留行政文书、送达时间和证据材料？

### 4.2 行政诉讼法：民告官不是万能官司

行政诉讼处理的是公民、法人或者其他组织认为行政机关及其工作人员的行政行为侵犯合法权益而提起的诉讼。

小说转化：

- 行政诉讼不是民事诉讼。被告通常是作出行政行为的行政机关，而不是某个个人执法人员。
- 法院主要审查行政行为的合法性，也会涉及事实、证据、程序、裁量、履职和赔偿等问题。
- 行政机关对其行政行为负有举证责任，不能只靠“我们一直这么办”。
- 行政案件可以有一审、上诉、再审、执行，也可能有调解、行政机关负责人出庭应诉、司法建议等现实节点。
- 行政诉讼结果可能是撤销、确认违法、责令履行、变更、驳回诉讼请求、确认无效、赔偿等。

高质量写法：

- 商户不服城管处罚，不是去法院哭诉“我很难”，而是围绕执法主体、处罚依据、事实证据、程序告知、裁量是否过当和文书送达展开。
- 农户不服补偿决定，法院不直接替他谈价格，而是审查补偿决定程序、依据、评估、听证和救济告知是否合规。

### 4.3 行政处罚法：处罚不是干部一句话

行政处罚法为行政处罚的设定、实施主体、管辖、适用、普通程序、简易程序、听证、执行和监督提供基础。县域故事中，行政处罚常见于市场监管、城管、生态环境、交通运输、文化旅游、应急管理、农业农村、自然资源等领域。

小说转化：

- 处罚要有事实、证据、依据、程序和裁量。不能写成执法人员一生气就罚。
- 一般处罚链条：线索或检查 -> 立案或核查 -> 调查取证 -> 事先告知 -> 陈述申辩 -> 必要时听证 -> 处罚决定 -> 送达 -> 执行 -> 复议诉讼或信用修复。
- 听证不是所有处罚都有，但重大、较重或依法应当听证的处罚要给当事人机会。
- 陈述申辩不是形式。角色可以提出证据、解释事实、请求从轻减轻、指出程序或依据问题。
- 行政处罚不等于民事赔偿完成，也不等于刑事责任排除；需要按事项分流。

常见戏剧抓手：

- 处罚告知书遗漏事实或依据，成为复议诉讼突破点。
- 当事人错过陈述申辩或听证机会，后续救济更难。
- 同类违法处罚轻重差异过大，引出行政裁量基准和执法监督。

### 4.4 行政强制法：强制不是粗暴执法

行政强制包括行政强制措施和行政强制执行。它强调法定权限、法定程序、比例原则、书面文书和救济边界。

小说转化：

- 行政强制措施常见为查封、扣押、冻结等；行政强制执行可能包括加处罚款或滞纳金、划拨、拍卖处理、排除妨碍、恢复原状、代履行等。
- 行政强制不是行政处罚的同义词，也不是民间讨债。
- 执法机关不能为了“效率”随意破门、搬东西、断水断电、堵门羞辱。涉及强制应写权限、文书、告知、催告、现场记录、清点、保管和救济。
- 有些行政机关没有直接强制执行权，需要依法申请人民法院强制执行。
- 当事人抗拒执行，也不能让主角提供规避强制、转移财产或对抗执法教程。

高质量写法：

- 摊贩货物被扣，不是“城管抢东西”，而要写是否有扣押依据、扣押清单、保存条件、处理期限和救济告知。
- 违法建筑拆除不能一铲子结束，要写认定、告知、限期拆除、催告、强制执行决定、现场安全和后续复议诉讼风险。

### 4.5 行政许可法：不予许可和撤销许可也有救济

行政许可线常出现在办证、经营许可、施工许可、食品经营、旅馆民宿、办学培训、医疗养老、危险作业、环保排污、文旅演出等情节。

小说转化：

- 行政许可不是“熟人盖章”，而是申请、受理、审查、补正、现场核查、决定、送达、延续、变更、撤销或注销的链条。
- 不予受理、不予许可、逾期不答复、要求重复证明、撤销许可、吊销许可证，都可能进入复议或诉讼。
- 许可争议要区分材料不全、条件不符、部门互相推诿、法律依据变化、规划限制和裁量问题。
- 行政许可救济常与第 206 篇政务服务、信息公开和营商环境专题联动。

### 4.6 国家赔偿与行政赔偿

国家赔偿法为行政机关及其工作人员违法行使职权侵犯合法权益造成损害时的赔偿提供基础。

小说转化：

- 国家赔偿不是“主角受委屈就赔钱”，通常要有违法行政行为、损害结果、因果关系和赔偿请求程序。
- 行政赔偿可以与行政复议、行政诉讼或单独赔偿请求发生衔接。
- 常见素材：违法查封扣押造成经营损失、违法行政拘留、违法强制拆除、错误行政处理导致财产损害、行政机关不履行法定职责造成损害等。
- 赔偿之外，还可能有道歉、恢复名誉、返还财物、解除查封、信用修复、整改和追责。

风险：

- 把国家赔偿写成部门随口给封口费，会失真。
- 把赔偿金额写成情绪惩罚，而不写损失证明和法定标准，会显得悬浮。

### 4.7 行政执法“三项制度”

行政执法公示、执法全过程记录、重大执法决定法制审核，是严格规范公正文明执法的重要制度。

小说转化：

- 执法信息应有公示和记录，行政处罚、行政许可、行政检查、行政强制等行为会留下文书、音像、系统记录和案卷。
- 重大执法决定通常要经过法制审核，不能靠执法队员个人情绪作出。
- 行政执法全过程记录是证据链好素材：执法记录仪、现场笔录、询问笔录、照片、视频、送达回证、案卷目录。
- 当事人维权时，重点不是抢手机拍冲突，而是要行政文书、案卷材料、执法记录、证据目录和救济告知。

| 角色 | 想要什么 | 怕什么 | 可写冲突 |

## 8. 高质量剧情用法

### 8.1 摊贩不服城管处罚

低质写法：摊贩直播哭诉，处罚当天撤销。  
高质写法：摊贩拿到处罚告知后，发现处罚依据引用的地方规定不适用于临时便民摊区，且执法记录缺少现场劝导和整改机会。他先陈述申辩，要求查看案卷；处罚决定作出后，再按文书提示申请行政复议。复议机关审查事实、依据、程序和裁量后，决定撤销或责令重新作出。

爽点：不是谁哭得惨，而是谁把行政行为审查点抓住了。

### 8.2 民宿经营许可被拒

低质写法：女主找熟人，第二天证就办好。  
高质写法：民宿申请被不予许可，理由是消防和规划条件不符合。女主先补材料和申请信息公开，确认究竟是硬条件不符、材料缺失，还是部门互相推诿。若不服不予许可决定，再走复议或诉讼。最终她不是靠关系办证，而是改造房屋、调整经营范围或证明部门适用依据错误。

爽点：把经营压力和法治路径都写出来，比“找关系盖章”更耐看。

### 8.3 违法建筑强制拆除

低质写法：执法队半夜开铲车，主角挡在门口反杀。  
高质写法：争议先从违法建设认定开始，再到责令限期拆除、催告、陈述申辩、强制执行决定、现场公告和安全保障。当事人可围绕认定事实、规划许可、送达程序和强制权限申请复议或诉讼。拆除如果违法，后续可能涉及确认违法和国家赔偿。

爽点：不是硬碰硬，而是让每一步程序都经得起追问。

### 8.4 市场监管处罚听证

低质写法：老板在听证会上讲苦情，监管人员感动免罚。  
高质写法：听证会围绕违法事实、抽检报告、进货查验、标签问题、主观过错、整改情况和处罚幅度展开。老板可以陈述申辩，但不能用“我不容易”替代证据。听证笔录进入案卷，影响最终处罚决定和后续复议诉讼。

爽点：真正改变结果的是证据和裁量理由，不是煽情。

### 8.5 信息公开答复引出行政诉讼

低质写法：部门不公开，主角黑进系统拿材料。  
高质写法：主角申请公开某项目补助分配依据，部门答复“信息不存在”或“非本机关掌握”。主角通过预算公开、村务公开和其他部门答复发现矛盾，针对信息公开答复申请复议或提起行政诉讼。法院审查的是答复是否合法，而不是直接裁判补助该给谁。

爽点：合法路径也能产生信息反杀，不必靠偷资料。

## 9. 与既有专题的分工

- 与 `206` 政务服务专题：`206` 管办事、材料、补正、窗口和热线；本篇管对许可、不予受理、不履职或信息公开答复不服后的行政救济。
- 与 `213` 信访和综治专题：`213` 管群众诉求表达、信访和分流；本篇负责行政争议的复议、诉讼、听证、赔偿等法定路径，防止信访替代救济。
- 与 `211` 市场监管、`212` 城管执法、`207` 生态环境、`224` 统计执法等专题：这些专题管具体执法场景；本篇管不服行政行为后的通用救济链。
- 与 `204` 公共法律服务、人民调解和法院专题：`204` 管法律援助、人民调解、基层法院和执行大框架；本篇细化行政复议、行政诉讼和行政赔偿。
- 与 `221` 纪检监察专题：纪检管干部违纪违法和廉洁风险；本篇管行政行为合法性救济。行政机关错误不一定等于干部违纪。
- 与 `226` 人大政协专题：代表建议和政协提案可推动制度改进，但不能替代个案复议诉讼。本篇负责个案行政救济。

## 10. 禁写与降风险清单

AI agent 写作时，应避免下列内容：

- 详细描写伪造行政文书、伪造送达凭证、伪造证据、串供、威胁执法人员、暴力抗法、转移财产、规避强制执行或绕过法院执行的方法。
- 把行贿找关系撤销处罚、违规办证、提前拿内部案卷写成主角正向爽点。
- 把信访、12345、媒体曝光、人大建议、政协提案写成替代行政复议和行政诉讼的万能捷径。
- 把复议机关、法院、纪检监察、公安、市场监管、城管、生态环境等职责混成一个万能机构。
- 写行政机关口头罚款、口头强拆、口头拒证却没有任何文书和后果。
- 泄露行政执法案卷中的个人信息、商业秘密或未公开敏感资料。
- 把国家赔偿写成部门私下封口费，或把赔偿金额写成情绪惩罚。

如果剧情必须出现越权执法、暴力抗法、伪造证据或规避执行，应让它成为风险、罪错或反派行为，并写出程序失效、责任追究和人物代价。

## 11. 审稿检查问题

1. 当事人不服的是哪个具体行政行为，还是行政机关不作为？
2. 行政文书是否存在：告知书、决定书、通知书、强制决定、送达回证、责令改正文书或信息公开答复？
3. 争议点是否拆成主体权限、事实证据、法律依据、程序、裁量和损害后果？
4. 是否写了陈述申辩、听证、补正、告知、送达等前置程序？
5. 复议机关、被申请人、法院被告、承办部门是否找对？
6. 是否区分行政复议、行政诉讼、国家赔偿、信访、纪检、民事诉讼和投诉举报？
7. 行政强制是否写清强制措施和强制执行的区别、文书、催告、清点保管和救济？
8. 行政处罚是否有调查、告知、申辩、听证、决定、送达和执行，而不是干部一句话？
9. 是否存在伪造证据、暴力抗法、行贿找关系、规避强制执行等被叙事奖励的问题？
10. 救济结果是否改变证照、处罚、强制、补偿、赔偿、信用、经营和人物关系，而不是只赢一场口头争吵？

## 12. 可复用提示词

### 12.1 起草提示词

```text
请按“县域行政救济链”写这一场：先说明具体行政行为或行政不作为，再写行政文书、事实证据、法律依据、程序告知、当事人陈述申辩或听证机会，随后判断应走行政复议、行政诉讼、国家赔偿、信访分流还是纪检监察。不要把曝光、找关系或信访写成万能解决。重点写文书、期限、证据和救济结果如何改变现实。
```

### 12.2 审稿提示词

```text
请审查这段行政处罚、行政许可、行政强制、行政复议、行政诉讼或国家赔偿剧情是否可信。按“行政行为-文书状态-争议点-前置程序-救济路径-证据池-机关分工-结果类型-安全边界-剧情后果”逐项指出问题，并给出可替换情节。重点检查信访替代复议诉讼、处罚无程序、强制粗暴化、法院复议机关万能化和行贿伪证正向化。
```

### 12.3 听证会提示词

```text
请设计一场县域行政处罚听证。要求围绕违法事实、证据、依据、程序、处罚幅度和裁量理由展开；当事人的情绪可以出现，但不能替代证据。听证结果不必当场翻盘，但必须影响最终处罚决定、复议诉讼策略或执法整改。
```

## 13. 写作提醒

行政救济线真正好看的地方，不是把行政机关写成一推就倒的反派，而是让角色学会一件更难的事：把愤怒变成请求，把委屈变成证据，把权力拉回程序。

商业级县域行政救济线应同时写出三件事：

- 行政行为为什么会伤到人。
- 救济路径为什么慢但有力。
- 结果如何改变个体命运和基层执法方式。
