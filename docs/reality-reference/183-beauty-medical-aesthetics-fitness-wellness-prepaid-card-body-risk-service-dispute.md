# 183. 美容医美、健身康养、服务合同、预付卡与身体风险纠纷链专项

状态：已确认  
日期：2026-06-09  
用途：供 AI agents 在创作现代都市、女性成长、娱乐圈、豪门、职场、直播带货、健身私教、医美机构、生活美容、美甲美睫、康养会所、按摩推拿、瑜伽普拉提、减脂塑形、预付卡跑路、身体损害纠纷等中文商业网文时，处理身体服务的资质、宣传、合同、预付费、风险告知、操作记录、服务履行、损害后果、退费维权和监管处罚。

## 1. 本专题解决什么问题

美容健身康养线最容易被 AI 写成三类假：

- 把医美、生活美容、化妆品、健身私教、按摩康养混成一种“变美服务”，不区分医疗资质、身体风险、产品来源和服务合同。
- 把预付卡、会员卡、私教课、疗程套餐写成随便退，或写成店家一句“不退”就结束，不写合同、课时、储值、监管、资金链和维权路径。
- 把身体改变写成即时爽点：打一针就逆袭、私教三天练出身材、按摩治百病、减肥产品无风险，忽略医学、运动、安全和心理边界。

本专题把身体服务拆成一条可执行链：

`外貌/健康/社交需求 -> 服务类型识别 -> 经营主体和资质 -> 宣传承诺 -> 咨询评估 -> 合同与预付费 -> 产品/器械/人员 -> 服务操作或训练 -> 身体反应 -> 记录证据 -> 售后/退费/医疗处置 -> 投诉举报/调解/诉讼 -> 赔偿整改 -> 心理和关系余波 -> 台账回写`

商业级身体服务戏的核心不是“变美后所有人震惊”，而是“人物为什么愿意把身体交给某个服务系统，服务系统如何承诺、收费、操作、记录、解释风险，并在出事后承担或逃避责任”。

## 2. 与既有专题的分工

- 第 20 篇管内容合规、医疗健康、广告、AI 标识和平台风险总闸门。
- 第 71 篇管身体感、感官细节和疼痛疲劳的可见描写。
- 第 105 篇管身体羞辱、创伤、控制、非自愿和敏感内容安全边界。
- 第 131 篇管伤病、治疗、恢复、医疗可信度。
- 第 154 篇管卫生、洗浴、气味、身体护理和日常生活质感。
- 第 159 篇管残障、慢性病、无障碍和辅助设备。
- 第 165 篇管医美分期、预付费、发票、保险、贷款和资金流水。
- 第 169 篇管医院急诊、病历、医疗纠纷、陪护照护。
- 第 174 篇管照片视频、肖像授权、术前术后照、影像证据和隐私传播。
- 第 180 篇管一般消费者权益、虚假宣传、退费和平台售后。
- 第 181 篇管培训、私教考证、职业资格和课程退费。

本篇是“身体服务消费入口”：凡涉及美容院、医美机构、健身房、私教、瑜伽普拉提、康养按摩、预付卡、疗程套餐、身体损害和变美焦虑，先读本篇，再按具体风险转其他专题。

## 3. 安全边界

本专题是小说创作资料，不是医疗、医美、健身、营养、康复、心理、法律或消费维权建议。

AI 不得提供以下现实可操作内容：

- 医美注射、手术、激光、药物、麻醉、消毒、术后处理的操作步骤。
- 减肥药、激素、兴奋剂、极端节食、脱水、催吐、过度训练、危险塑形方法。
- 非法行医、黑医美、无证注射、非法购药、假货器械、规避监管和骗赔方法。
- 假冒医生、伪造病历、伪造同意书、伪造术前术后照片、伪造健身课时。
- 诱导办卡、虚假疗效、刷单控评、恶意退费、泄露顾客身体照和隐私。
- 对身材、容貌、残障、疾病、年龄、产后身体进行羞辱或歧视。

写医美事故、运动损伤、身体焦虑、饮食失调、自伤倾向、被迫变美、伴侣控制身材时，必须联动第 105、131、159 篇，明确安全、支持和后果，不把伤害写成爱情、逆袭或自律神话。

## 5. 先区分服务类型

| 类型 | 可写爽点 | 必写约束 |
| --- | --- | --- |
| 生活美容 | 护肤、美甲、美睫、造型、仪式感 | 不能越界做医疗美容，产品来源和卫生要写 |
| 医疗美容 | 外貌改变、公众形象、事故高压 | 医疗机构、医师、病历、知情同意、医疗风险 |
| 化妆品护理 | 品牌、成分、过敏、直播带货 | 化妆品不是药，不能写治疗疾病效果 |
| 健身私教 | 自律成长、身体力量、社交关系 | 体测、训练计划、动作风险、资质和运动损伤 |
| 瑜伽/普拉提 | 身体觉察、疗愈、长期改变 | 不神化疗效，不把疼痛强忍写成进步 |
| 康养按摩 | 放松、照护、老人/职场疲劳 | 医疗治疗宣传风险、卫生和身体边界 |
| 高危险性体育 | 游泳、滑雪、潜水、攀岩等 | 许可、安全设施、教练、救援和风险提示 |
| 线上课程 | 减脂课、体态课、直播训练 | 账号、回放、退款、动作纠错不足和身体风险 |
| 预付会员 | 店铺经营、关系绑定、跑路风险 | 合同、储值、剩余次数、退款和资金链 |

## 6. 生活美容和医疗美容的边界

生活美容主要是非侵入性的护理、修饰、清洁、造型等服务；医疗美容涉及医疗技术、药物、器械、创伤性或侵入性操作，必须有医疗机构和医务人员资质。

AI 写医美时至少检查：

- 机构是否是合法医疗机构，诊疗科目或服务范围是否匹配。
- 操作者是否具备医师或相应专业资格。
- 是否有面诊、病史询问、过敏史、禁忌症、风险告知和知情同意。
- 是否有病历、处方、治疗记录、材料批号、器械记录和术后随访。
- 广告是否夸大效果、使用绝对化承诺或制造容貌焦虑。
- 发生损害后是否及时就医、封存病历、保存照片、沟通和付款记录。

禁止把“美容院后屋打一针”“朋友带来的医生上门做项目”“直播间买材料自己操作”写成聪明省钱或逆袭捷径。若出现，只能写风险和后果。

## 7. 化妆品、美容产品和宣传

化妆品适合写品牌经营、直播带货、敏感肌过敏、明星代言和消费维权。它不是药品，不能默认写成治疗疾病。

写化妆品要区分：

- 普通化妆品和特殊化妆品。
- 国产、进口、备案或注册信息。
- 成分、生产批号、保质期、标签、说明书和使用方法。
- 试用装、赠品、套盒、小样、院线产品、跨境购。
- 过敏、刺激、假货、虚假功效、夸大宣传和成分争议。

可用剧情：

- 女主品牌因达人夸大“治痘治敏”被监管和消费者同时质疑。
- 反派用过期小样或假冒院线产品坑害顾客，主角从批号和进货单追出证据。
- 明星代言护肤品翻车，粉丝维权、品牌切割、平台下架和艺人形象同时受损。

## 8. 健身房、私教和运动损伤

健身戏的爽点不是三天变瘦，而是身体能力、习惯、边界和自我关系的长期改变。

AI 写健身服务时先问：

- 是普通健身房、私教工作室、瑜伽馆、普拉提馆、游泳馆、滑雪场、攀岩馆还是线上课程。
- 是否有合同、会员卡、私教课、体测、训练计划、剩余课时和退费条款。
- 教练是否宣传过度承诺：包瘦、包塑形、快速逆袭、产后修复、康复治疗。
- 客户是否有病史、旧伤、怀孕产后、残障、年龄、运动基础等风险因素。
- 是否进行动作指导、强度调整、器械安全、热身冷却和现场救助。
- 受伤后是否有监控、训练记录、医疗记录、教练聊天和场馆处置。

写作边界：

- 不提供具体训练处方、极限减脂方法或危险动作教程。
- 不把“疼到吐还继续练”写成励志。
- 不把教练身体接触和权力不对等写成天然暧昧，必须尊重边界和同意。
- 健身成果要用时间、训练记录、饮食生活变化和身体反馈累积，不靠旁白“一周后惊艳全场”。

## 9. 康养、按摩、推拿和养生服务

康养服务常见于都市女性、老人照护、职场疲劳、豪门会所和小城生活。它容易被写成万能治病，因此必须守边界。

可信写法：

- 明确是放松护理、生活服务、传统保健、康复训练、医疗治疗还是养老照护。
- 如果宣称治疗疾病、诊断病情或开具治疗方案，要进入医疗资质和医疗风险线。
- 按摩推拿要写身体边界、隐私、更衣、房间、同性/异性服务安排和监控争议。
- 老人、孕产后、慢病、残障客户要写风险告知和是否适合该服务。
- 店铺卫生、用具消毒、床单毛巾、精油产品和技师身份可作为细节。

禁止把“神奇养生馆治好绝症”“按摩排毒包治百病”写成正向商业承诺。玄学或架空设定可以有特殊规则，但现实题材必须区分保健、康复和医疗。

## 10. 预付卡、会员卡和店铺跑路

美容、健身、康养最常见纠纷是预付费。

预付费链：

1. 销售承诺：优惠、赠送、升卡、限时、转让、跨店使用。
2. 付款：储值卡、次卡、年卡、私教包、分期、医美贷。
3. 合同或小票：服务项目、次数、有效期、退费规则、开票主体。
4. 履行：预约、签到、核销、剩余次数、店铺变更、教练离职。
5. 争议：涨价、换店、停业、跑路、转让门店、单方改规则。
6. 处理：协商、平台、市场监管、消协、诉讼、报警、破产清算。

好用冲突：

- 女主办卡后店铺换老板，新老板不承认旧卡。
- 私教离职，剩余课时被强制转给不合适教练。
- 医美项目拆成疗程，消费者做了一半发现资质问题。
- 健身房预售年卡还没开业就停工，销售群解散。
- 康养会所给老人办高额储值卡，子女发现后追查话术和合同。

退费戏不要只写“我不满意”。要写服务剩余、合同条款、谁违约、资金去向、证据和投诉路径。

## 11. 广告、种草、代言和平台传播

美容健身行业高度依赖视觉和承诺，最容易虚假宣传。

风险承诺：

- “无风险”“无痛”“永久”“一次见效”“包瘦”“包治”。
- “三天瘦十斤”“产后修复必成”“逆龄二十岁”。
- “专家亲诊”“公立医院专家”“明星同款”“官方认证”。
- 术前术后对比图夸大、修图、盗图、样本不一致。
- 直播间临时承诺与合同不一致。

写作要点：

- 广告承诺必须留证：页面截图、直播回放、海报、聊天、合同、达人视频。
- 术前术后照片涉及肖像和隐私，联动第 174 篇。
- 明星和达人推广涉及广告代言、平台规则和粉丝消费，联动第 61、180 篇。
- 医疗广告需要审查和更严边界，不要把医生直播夸大疗效写成专业。

## 12. 身体损害和纠纷处理

身体服务纠纷的后果比普通商品退货更重。AI 必须写身体、心理、时间和证据成本。

证据链：

- 宣传页面、咨询聊天、合同、付款、发票、预付卡记录。
- 面诊或体测记录、风险告知、知情同意、病历或训练计划。
- 产品批号、器械信息、医生/技师/教练身份、操作记录。
- 术前术后照片、受伤照片、监控、现场证人、平台记录。
- 医院诊断、检查、病历、费用、误工、修复治疗记录。
- 后续沟通、退款方案、投诉受理、鉴定或调解材料。

处理路径：

- 普通消费退费：商家协商、平台介入、市场监管、消协、诉讼，联动第 180 篇。
- 医疗美容损害：医疗机构投诉、卫生健康部门、病历封存、医疗损害鉴定、诉讼，联动第 169 篇。
- 化妆品不良反应或质量问题：保存产品、批号、购买记录、就医记录、市场监管或药监线索。
- 健身运动损伤：场馆记录、教练聊天、监控、医疗记录、保险和民事责任。
- 高危险性体育：许可、安全设施、教练资质、救援记录和主管部门。

不要让“赔钱整回来”抹平一切。身体风险会影响工作、亲密关系、自我认同、公众形象和长期恢复。

## 13. 高危险性体育和场馆安全

游泳、滑雪、潜水、攀岩等高危险性体育项目不是普通健身打卡。写到这些，要检查许可、场地、教练、设备、天气、救援和保险。

场馆检查：

- 是否取得经营高危险性体育项目许可。
- 是否公示安全须知、开放时间、救援人员和设施。
- 是否进行风险提示、能力评估、装备检查。
- 是否有监控、签到、场地巡查、事故记录。
- 未成年人、初学者、老人、慢病者是否有特别安排。

写法边界：

- 不写危险项目规避安全员、私闯野场、无装备挑战为浪漫勇敢。
- 不让主角替代专业救援体系。
- 不把场馆免责声明写成完全免责魔法，具体责任要看合同、提示、管理和事故原因。

## 14. 类型文用法

- 女性成长文：变美不是迎合凝视，而是身体边界、选择权和职业压力的复杂谈判。
- 娱乐圈文：上镜、医美、身材管理、代言、术后隐私和粉丝审视可以形成高压线。
- 都市职场：健身私教、医美分期、形象焦虑、办公室评价和收入压力相互牵连。
- 商战文：医美机构、美妆品牌、健身房、康养会所、预付卡资金链和广告合规都是经营战场。
- 悬疑文：术前照、监控、预约记录、注射材料批号、健身房门禁、私教聊天都可成为线索。
- 家庭伦理：老人高额康养卡、产后修复焦虑、伴侣要求变美、孩子体态焦虑都能制造亲密关系冲突。
- 重生文：主角可以避开黑医美和预付卡跑路，但要用资质、合同和证据布局，不靠“我知道未来”空降反杀。
- 古代/架空：不套现代监管，但可保留“医馆/女医/香粉胭脂/武馆/养生馆/师傅资质/药材来源/身体风险/契约赔偿”的功能链。

## 15. 常见坏味道与修正

| 坏味道 | 读者体感 | 修正方法 |
| --- | --- | --- |
| 医美像普通美容 | 高风险悬浮 | 写医疗机构、医师、面诊、病历、同意书和风险 |
| 私教三天练出神身材 | 假 | 写长期训练、体测、疲劳、饮食生活和阶段成果 |
| 预付卡一句不退 | 维权空心 | 写合同、剩余次数、收款主体、店铺变更和投诉路径 |
| 减肥痛苦越大越励志 | 伤害美化 | 写安全、恢复、身体反馈和心理边界 |
| 术前术后照片随便发 | 隐私风险 | 联动第 174 篇，写授权、肖像和平台传播 |
| 康养馆包治百病 | 虚假医疗 | 区分保健、康复、医疗和广告风险 |
| 健身受伤无记录 | 证据弱 | 写训练记录、监控、医疗记录、教练聊天 |

## 17. 给 AI agent 的生成提示词

起草医美纠纷：

```text
请按第183篇“美容医美、健身康养、服务合同、预付卡与身体风险纠纷链专项”写一场医美纠纷戏。先填 body_service_scene_card，明确服务类型、机构和人员资质、宣传承诺、面诊评估、知情同意、付款合同、产品器械、操作记录、身体反应、证据链和处理路径。不要写医美操作步骤，也不要把黑医美写成捷径。
```

审稿健身私教线：

```text
请审稿本章健身私教线是否悬浮。检查：合同课时、体测评估、教练资质、训练记录、身体反馈、风险提示、受伤处置、退费路径和职业边界是否写清。指出“速成变美”“疼痛美化”“教练越界暧昧”“预付卡无证据”等风险。
```

写预付卡跑路：

```text
请写一场美容院或健身房预付卡跑路后的维权戏。必须写清办卡宣传、合同或小票、剩余次数、收款主体、店铺变更、员工处境、顾客证据、监管投诉和资金追回困难。不要让报警或投诉一秒解决所有问题。
```

写身体焦虑修复：

```text
请写一场角色面对外貌或身材焦虑的成长戏。不要用羞辱制造动力，不要提供危险减肥方法。要写角色的身体感受、社会压力来源、边界选择、可信支持和一个可持续的小行动。
```

## 18. 自检清单

- 服务是生活美容、医疗美容、化妆品护理、健身私教、康养按摩还是高危险性体育？
- 经营主体和操作者是否有相应资质或许可？是否存在无证或超范围服务？
- 宣传承诺是否有截图、海报、直播回放、聊天或合同证据？
- 是否区分医疗广告、普通广告、达人种草和消费者评价？
- 是否有咨询评估、病史/过敏/体测、风险告知和知情同意？
- 合同、预付卡、会员卡、私教课、分期贷和发票是否写清？
- 产品、器械、材料、批号、训练计划或操作记录是否可追溯？
- 身体损害是否有就医、病历、照片、监控、费用和恢复周期？
- 退费和投诉路径是否区分平台、市场监管、卫健、药监、体育主管、消协、诉讼和报警？
- 是否避免医美操作教程、危险减肥、过度训练、黑医美和假证资质细节？
- 场景结束后，身体、心理、钱、职业、亲密关系或公众形象是否发生变化？

## 19. 最小规则

如果 AI 只能记住五条：

1. 身体服务不是普通商品，先区分生活美容、医疗美容、化妆品、健身、康养和高危险性体育。
2. 医美必须写医疗机构、人员资质、面诊、风险告知、病历记录和损害处理，不写操作教程。
3. 健身和康养必须写评估、边界、身体反馈、记录和安全，不把痛苦或速成神话当励志。
4. 预付卡和私教课要写合同、剩余次数、收款主体、退费条款和店铺资金链。
5. 身体风险戏的后果要落到恢复、心理、工作、关系、隐私和证据，不只落到赔钱。

## 20. 主要来源

- 原卫生部《医疗美容服务管理办法》，用于医疗美容机构、医师、服务项目、病历和医疗美容管理底座。
- 国务院《化妆品监督管理条例》，用于化妆品注册备案、标签、功效宣传、生产经营和不良反应背景。
- 《医疗广告管理办法》，用于医疗广告审查、发布内容、医疗机构广告和医美宣传边界。
- 商务部《单用途商业预付卡管理办法（试行）》，用于美容、健身、康养预付卡、备案、资金管理和消费者风险背景。
- 《中华人民共和国消费者权益保护法实施条例》，用于预付式消费、经营者信息披露、退费和消费者权益保护。
- 《中华人民共和国体育法》和《全民健身条例》，用于体育活动、健身服务、体育设施和安全保障背景。
- 国家体育总局《经营高危险性体育项目许可管理办法》和《第一批高危险性体育项目目录公告》，用于游泳、滑雪、潜水、攀岩等项目许可和安全边界。
