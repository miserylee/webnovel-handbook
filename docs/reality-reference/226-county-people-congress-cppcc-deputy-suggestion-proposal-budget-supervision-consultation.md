# 226 县级人大、人大代表建议、政协委员提案、预算审查监督、协商议政与办理答复专项

状态：已确认  
更新时间：2026-06-09  
适用对象：县域现实题材、基层治理、公共服务、预算财政、民生项目、代表委员群像、政策协商、群众诉求、政府工作报告、审计整改、信访分流、公共议题推动和温和制度线剧情。

## 1. 本专题解决什么问题

AI 写县域公共事务时，常把群众诉求只写成三条路：上网曝光、信访接待、纪委介入。这样会让现实题材变窄，也容易把所有矛盾都推向极端对抗。

实际上，县域公共事务还有一条更温和、更制度化、也很适合写群像的路径：人大代表联系群众、提出建议批评和意见，人大常委会监督政府工作；政协委员通过提案、社情民意信息、专题调研和协商议政提出意见建议；预算审查监督把公共资金、政策和民生效果放到公开讨论中。

本专题解决 AI 的几个常见错误：

- 把人大代表写成能直接命令政府部门的“民间钦差”。
- 把代表建议、政协提案写成一交上去就必须立即执行的命令。
- 把人大、政协、信访、纪检、审计、法院、政府窗口混成一个万能机关。
- 把会议写成空洞口号，不写调研、材料、交办、承办、答复、督办和落实。
- 把预算监督写成“代表嫌贵”，不写预算、支出政策、债务、国有资产、审计整改和项目绩效。

本专题的目标是让 AI agent 能写出“问题如何从群众体感进入制度议程，再通过建议、提案、审议、监督和办理反馈改变现实”的叙事链。

## 2. 一句话原则

人大建议和政协提案不是万能按钮，而是把群众体感转化为可调研、可交办、可答复、可督办、可回访的公共议题。

## 3. 标准叙事链

```text
群众或行业出现公共问题
-> 判断事项性质：公共服务、预算资金、政策执行、行政执法、基层治理、民生设施、产业发展、司法监督、生态环保、历史遗留或个案救济
-> 找到合适渠道：人大代表走访、代表联络站、代表建议批评意见、人大常委会监督议题、政协委员调研、政协提案、社情民意信息、专题协商、部门办理或信访司法纪检分流
-> 调研取材：走访群众、查看现场、查公开预算和政策、听部门说明、收集数据、对比外地经验、区分个案和普遍问题
-> 形成文本：代表建议、议案、审议意见、执法检查意见、政协提案、社情民意信息、调研报告或协商发言
-> 审查交办：登记、审查、立案或转为意见建议，确定承办单位、主办协办、办理时限和重点督办安排
-> 承办办理：部门调查研究、联系代表委员、会商协办单位、提出解决措施、不能解决的说明原因和计划
-> 答复反馈：书面答复、面商沟通、满意度反馈、重新办理、持续跟踪、公开办理结果或纳入下一年度计划
-> 监督督办：人大常委会听取专项报告、执法检查、专题询问、预算审查、审议整改；政协重点提案督办、专题协商、民主监督
-> 结果落地：项目调整、预算安排、制度修订、服务优化、整改销号、政策解释、不能解决但形成下一步计划
-> 人物余波：代表委员信用、部门压力、群众信任、项目台账、下一轮议题
```

## 4. 当前制度底座的小说转化

### 4.1 地方组织法：县级人大和常委会不是装饰

《中华人民共和国地方各级人民代表大会和地方各级人民政府组织法》为地方人大、人大常委会和地方政府的组织与职权提供基础。县级人大及其常委会在县域故事里不是背景板，而是地方国家权力机关和常设机关，有审查批准计划预算、监督政府和相关机关工作、决定重大事项、任免有关人员等制度功能。

小说转化：

- 县人民代表大会通常通过会议审查政府工作报告、计划和预算报告，选举和决定有关人事，讨论地方重大事项。
- 县人大常委会日常开展监督，可听取专项工作报告、组织执法检查、开展专题询问、备案审查规范性文件、监督预算决算和国有资产管理等。
- 人大监督不是替政府办事，也不是替法院判案。它通过审议意见、检查报告、询问、督办、任免和公开监督推动国家机关依法履职。
- 写县域公共事务时，人大可以成为“慢但稳”的制度推进器：代表发现问题、常委会列入监督议题、政府部门报告整改、下一次会议再追问。

风险：

- 把人大代表或人大常委会写成直接调拨财政资金、命令法院改判、指挥纪委办案，会混淆职权。
- 把会议写成只鼓掌，会浪费制度戏剧张力。真正好写的是会前调研、报告审议、问题追问、整改反馈和群众回访。

### 4.2 代表法：人大代表建议批评意见的正确用法

已确认规则：《中华人民共和国全国人民代表大会和地方各级人民代表大会代表法》于 2025 年修改，后续写代表履职、代表建议和代表联系群众时，应按修改后的代表法框架理解。

小说转化：

- 人大代表依法参加本级人民代表大会会议和闭会期间履职，联系原选区或原选举单位和人民群众，反映群众意见要求。
- 代表可以提出议案、建议、批评和意见。小说里要区分：议案更强调会议议程和法定程序；建议批评意见更常用于推动具体工作改进和群众关切办理。
- 代表建议不是代表私人求情，也不是部门必须照办的命令。承办单位要研究办理并答复，能解决的抓紧解决，暂时不能解决的说明计划，确实不可行的说明情况。
- 代表建议最好来自调研，而不是饭桌一句话：现场、群众、数据、政策、预算、历史办理情况和可行方案都能增加可信度。

可写冲突：

- 代表提出公交线路优化建议，交通部门答复“暂无条件”，代表追问客流数据、老年人就医路线和预算安排，促成试运行线路。
- 代表建议修桥，财政部门说明本年度预算已排满，人大预算审查时追问重大民生项目排序，最终列入下一年度计划。
- 群众以为代表能立刻解决个人纠纷，代表把民事个案分流给调解、法院或法律援助，同时把共性问题形成建议。

### 4.3 监督法：专项报告、执法检查、专题询问和备案审查

《中华人民共和国各级人民代表大会常务委员会监督法》于 2024 年修改，为人大常委会依法行使监督职权提供基础。

小说转化：

- 常委会可以听取和审议政府、监察委员会、法院、检察院专项工作报告。
- 执法检查适合写食品安全、环境保护、未成年人保护、道路交通、安全生产、文物保护等法律实施情况，不是单纯走过场。
- 询问、专题询问、质询可以制造公开压力，但必须围绕法定监督程序和具体问题。
- 备案审查适合写县政府或乡镇规范性文件是否与上位法冲突，例如不合理收费、限制权利、过度处罚、村规民约与法律冲突等。
- 监督结果通常是审议意见、整改要求、报告反馈和持续跟踪，不是当场替行政机关作决定。

高质量写法：

- 人大常委会组织未成年人保护法执法检查，发现学校心理老师配备、乡镇儿童主任履职、校园欺凌处置和预算保障之间存在断点。
- 专题询问会上，部门不是被主角骂倒，而是在数据、预算、职责和整改时限上被连续追问。

### 4.4 预算审查监督：公共钱如何进入代表视野

人大预算审查监督与第 223 篇财政资金链天然联动。预算监督不只是看总数，更要看支出预算、重点支出、重大投资项目、部门预算、转移支付、政府债务、国有资产管理、审计查出问题整改和政策实施效果。

小说转化：

- 县人大审查批准本级预算和预算执行情况报告，人大常委会可监督预算执行、审查决算、听取审计工作报告和审计整改报告。
- 代表看预算不是“嫌贵”，而是追问：钱投向哪里、政策依据是什么、绩效目标是什么、群众受益如何、债务风险怎样、审计问题是否整改。
- 预算联网监督、预算公开、审计报告和绩效评价可以成为代表发现问题的线索。
- 预算监督适合写“项目不是没人重视，而是资金排序不同”的现实矛盾。

可写冲突：

- 老年食堂项目群众呼声高，但预算里只有建设费，没有运营补助，代表在预算审查中追问可持续性。
- 一项文旅项目连续追加预算，审计和绩效评价显示客流不足，人大常委会要求政府报告整改和下一步安排。
- 政府债务压力导致新项目暂缓，代表必须向群众解释为什么“有政策”不等于“今年有钱”。

### 4.5 人民政协：提案、协商议政和民主监督

《中国人民政治协商会议章程》明确人民政协主要职能包括政治协商、民主监督、参政议政。政协不是国家权力机关，也不是行政审批机关，但在县域故事里非常适合写跨界专家、行业人士、企业家、教师医生、基层工作者和社会组织的协商力量。

小说转化：

- 政协委员可以通过提案、社情民意信息、调研报告、协商发言、民主监督等方式反映问题、提出建议。
- 提案通常要经过审查立案，再交承办单位办理并答复；重点提案可以督办、协商办理和跟踪落实。
- 政协提案更适合写专业性、协商性和跨部门问题，例如养老服务、文旅品牌、县域医疗、民营经济、残疾人无障碍、非遗保护、营商环境等。
- 政协委员不能直接命令政府部门，但可以通过调研、协商、提案督办和社情民意让问题进入决策视野。

高质量写法：

- 一位医生委员提交基层急救能力提案，不是简单要求“多买救护车”，而是写清人口分布、乡镇卫生院夜班、转诊路径、医保支付和财政预算。
- 企业家委员提出优化营商环境提案，承办单位答复后还邀请委员现场查看办事流程，发现真正卡点是跨部门数据共享和材料重复提交。

## 5. 常见渠道与写作抓手

| 渠道 | 适合写什么 | 不能写成什么 | 关键证据 |
| --- | --- | --- | --- |
| 人大代表走访 | 群众诉求、民生痛点、政策效果 | 代表直接命令部门 | 走访记录、现场照片、群众名单、政策文件 |
| 代表建议批评意见 | 具体工作改进、共性问题办理 | 私人求情条 | 建议文本、交办单、承办答复、满意度反馈 |
| 人大议案 | 会议议程和重大事项 | 普通诉求万能升级 | 议案条件、联名、审查、会议处理程序 |
| 专项工作报告审议 | 政府部门工作接受监督 | 部门被骂完即整改 | 报告、审议意见、整改反馈 |
| 执法检查 | 法律法规实施情况 | 检查团替执法部门处罚 | 检查方案、点位、问题清单、整改报告 |
| 专题询问 | 公开追问重点问题 | 情绪质问秀 | 问题、数据、职责、时限、答复 |
| 预算审查监督 | 财政资金、重大项目、债务、绩效 | 代表嫌项目贵 | 预算报告、审计报告、绩效评价、债务数据 |
| 政协提案 | 专业建议、跨部门议题 | 行政命令 | 提案审查、立案、承办答复、督办记录 |
| 社情民意信息 | 快速反映苗头性问题 | 网络爆料替代品 | 来源、事实、建议、采用反馈 |
| 专题协商 | 多方沟通和方案完善 | 茶话会闲聊 | 调研报告、协商纪要、部门回应 |

## 6. 常见角色与真实压力

| 角色 | 想要什么 | 怕什么 | 可写冲突 |
| --- | --- | --- | --- |
| 县人大代表 | 反映群众、推动办理、保持公信力 | 被群众当万能、被部门敷衍 | 在个人诉求和公共建议之间取舍 |
| 乡镇人大主席团工作人员 | 组织代表活动、收集建议 | 形式化履职、材料空转 | 把走访从拍照变成问题清单 |

| 政协委员 | 提专业建议、凝聚共识 | 提案空泛、办理敷衍 | 把行业经验转成可办理文本 |
| 政协提案委工作人员 | 审查立案、交办督办 | 提案太散、承办扯皮 | 把一个大题拆成主办协办责任 |
| 政府承办部门 | 按时答复、解决问题 | 预算不足、职责交叉、被追问 | 答复写得漂亮，代表委员不满意 |
| 财政部门 | 预算合规、支出排序 | 所有人都要钱 | 在预算监督中解释钱从哪里来 |

| 主角 | 找到温和制度路径 | 被激进一方说太慢，被部门说添乱 | 学会用调研和文本推动现实变化 |

## 8. 高质量剧情用法

### 8.1 代表建议推动公交线路试运行

低质写法：人大代表给交通局打电话，第二天公交车开到村口。  
高质写法：代表走访发现老人去县医院要转三趟车，先收集就医频次、公交客流、村庄人口结构和道路条件，再提出优化线路建议。交通部门答复成本和客流不足，代表继续推动试运行方案：赶集日、就医日增加班次，三个月后用客流数据评估。

爽点：不是代表一句话解决，而是让群众需求变成可试点、可评估、可延续的公共服务方案。

### 8.2 政协提案推动基层急救能力建设

低质写法：政协委员提案要求建医院，县里马上批钱。  
高质写法：医生委员调研发现乡镇夜间急救空档，提案不是笼统“建医院”，而是建议优化值班、培训村医、配置 AED、明确转诊路线、接入 120 调度和财政保障。承办单位答复部分采纳，先在三个乡镇试点。

爽点：专业提案让一个大问题变成可操作的小闭环。

### 8.3 预算审查追问老年食堂

低质写法：代表嫌预算少，财政局加钱。  
高质写法：预算审查时，代表发现老年食堂项目只有改造费，没有运营补贴和送餐人力。财政部门说明本级财力紧张，民政部门提出社会力量参与，人大审查意见要求下一步报告运营绩效和服务覆盖。项目不是立刻大扩张，而是先解决“建了能不能开下去”。

爽点：钱不是凭热情追加，而是被问到可持续性。

### 8.4 执法检查发现纸面整改

低质写法：人大执法检查组一来，企业马上停业。  
高质写法：常委会组织环保执法检查，现场点位和台账显示企业整改完成，但群众夜间投诉仍多。检查组不替生态环境部门处罚，而是把夜间监测、投诉处理、企业整改和部门监管写进检查报告，推动政府专项报告和整改反馈。

爽点：监督不是替代执法，而是让执法效果接受公开追问。

### 8.5 提案办理不满意

低质写法：委员不满意，承办单位就道歉完事。  
高质写法：委员提出无障碍改造提案，承办部门答复“已按标准建设”，委员带着残障人士实地体验，发现坡道角度、盲道断点和公交站台高度都不适用。提案转为重点督办，承办单位重新踏勘、分阶段整改，并把新建项目纳入前置审查。

爽点：满意度不是表情包，而是用现场体验倒逼答复从文字变成行动。

## 9. 与既有专题的分工

- 与 `213` 信访、网格和综治专题：`213` 管群众诉求、信访接待和矛盾化解；本篇管共性问题如何通过代表建议、政协提案、人大监督和协商议政进入制度议程。
- 与 `206` 政务服务、12345 和信息公开专题：`206` 管窗口办理、热线转办和行政服务；本篇管代表委员和人大政协渠道推动部门改进工作。
- 与 `223` 财政预算专题：`223` 管财政资金规则；本篇管人大预算审查监督如何追问支出政策、重大项目、债务、国有资产和绩效。
- 与 `225` 审计监督专题：`225` 管审计发现和整改；本篇管人大听取审计工作报告、审计整改报告，并通过审议意见督促落实。
- 与 `221` 纪检监察专题：`221` 管违纪违法和廉洁风险；本篇不替代纪检，只在发现干部问题线索时分流。
- 与 `209` 村民自治专题：`209` 管村级会议、公示和村务监督；本篇管县级人大政协渠道如何把村级共性问题推向更高层级监督协商。
- 与 `204` 人民调解、法律援助和基层法院专题：个案权利救济、诉讼执行看 `204/141`；本篇管公共政策和部门履职问题，不替代司法。

## 10. 禁写与降风险清单

AI agent 写作时，应避免下列内容：

- 把人大代表写成能直接命令政府部门、财政、法院、纪委或公安的个人权力者。
- 把政协委员提案写成行政命令或审批决定。
- 把代表建议、政协提案当作私人求情、关系运作或绕过法定程序的捷径。
- 伪造代表委员身份、冒名提交建议提案、伪造办理答复或泄露未公开敏感材料。
- 把个案诉讼、行政处罚、刑事案件、纪检线索强行写成建议提案解决，绕开法定救济渠道。
- 写代表委员收礼、打招呼、干预评审采购、干预司法或干预纪检并获得正向爽感。
- 把办理答复写成“部门随便糊弄”但不写代表委员如何再沟通、督办、列入计划或分流。
- 泄露群众隐私、未成年人信息、商业秘密或未公开预算审计资料。

如果剧情必须出现越权干预、利益输送或虚假办理，应让它成为风险、罪错或反派行为，并写出监督、追责和公信力损害后果。

## 11. 审稿检查问题

1. 这件事适合人大代表建议、人大议案、人大监督议题、政协提案、社情民意信息，还是应走信访、政务服务、司法、纪检或行政复议？
2. 代表或委员是否做了调研，而不是只凭一句听说？
3. 建议或提案是否把个案转化为共性问题，并提出可办理、可评估的建议？
4. 承办单位、主办协办、办理时限、书面答复和反馈是否写清楚？
5. 如果不能立即解决，是否写了预算、权限、规划、法律限制或技术条件，而不是部门敷衍？
6. 人大监督是否落到专项报告、执法检查、专题询问、预算审查、备案审查、审议意见或整改跟踪？
7. 政协履职是否落到提案、调研、协商、民主监督和社情民意，而不是行政命令？
8. 是否把预算监督和第 223 篇财政资金链联动，追问支出政策、绩效、债务和审计整改？
9. 是否存在代表委员越权、收礼干预、伪造身份、泄露隐私或绕过法定程序的正向描写？
10. 办理结果是否改变了公共服务、预算安排、部门工作、群众信任或下一轮监督，而不是只写一纸答复？

## 12. 可复用提示词

### 12.1 起草提示词

```text
请按“县域人大政协监督协商链”写这一场：先判断群众问题是否适合代表建议、人大监督、政协提案或社情民意，再写调研取材、文本形成、审查交办、承办沟通、书面答复、满意度反馈、督办跟踪和现实后果。不要把人大代表写成能直接命令部门的人，不要把政协提案写成行政命令。重点写群众体感如何被转化为可办理的公共议题。
```

### 12.2 审稿提示词

```text
请审查这段人大代表建议、政协提案、预算审查、执法检查或专题询问剧情是否可信。按“渠道选择-调研证据-文本类型-承办单位-办理答复-监督督办-权限边界-隐私边界-剧情后果”逐项指出问题，并给出可替换情节。重点检查代表万能化、提案命令化、个案绕程序、预算监督空心化和办理答复无后续。
```

### 12.3 预算监督提示词

```text
请设计一场县人大预算审查或审议审计整改报告的剧情。要求至少追问三类问题：支出政策依据、项目绩效、政府债务或国有资产、审计整改、群众受益。财政或部门不能一句“没钱”结束，必须说明预算安排、支出排序、下一步计划和可监督节点。
```

## 13. 写作提醒

人大政协线最适合写“慢变量”：它不一定当场爆炸，却能让一个群众问题从抱怨、走访、文本、交办、答复、督办，慢慢变成公共决策的一部分。

商业级县域监督协商线应同时写出三件事：

- 群众问题如何被听见。
- 代表委员如何把体感变成可办理文本。
- 部门答复之后，现实有没有真的动一寸。
