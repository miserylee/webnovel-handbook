# 204 县域司法所、人民调解、公共法律服务、法律援助、人民法庭、诉前调解与执行链专项

状态：已确认  
更新日期：2026-06-09  
适用对象：需要写县城、乡镇、村庄、社区、家庭伦理、邻里冲突、债务纠纷、土地宅基地、婚姻家庭、赡养、消费合同、小额借贷、基层法治、诉前调解、人民法庭和执行难情节的 AI agents。

## 1. 这篇解决什么问题

很多 AI 写现实纠纷时，会在两个极端之间跳：

- 一吵架就报警，派出所一句话解决。
- 一翻脸就上法院，法官一锤定音，判完立刻清零。

县域现实里，大量纠纷既不是刑事案件，也不会一开始就进入完整庭审。它们会先经过亲友劝说、村委会或居委会协调、人民调解委员会、司法所、公共法律服务中心、法律援助、诉前调解、人民法庭、司法确认、判决调解书，最后才可能进入执行。

本篇专门补“报警和庭审之间的中间地带”，让 AI 写出纠纷如何从嘴上争吵，变成记录、协议、诉状、调解书、判决书和执行案件。

核心链条：

`纠纷触发 -> 当事人关系和风险判断 -> 证据与事实初步整理 -> 亲友/村居/行业组织先行沟通 -> 人民调解或司法所介入 -> 公共法律服务咨询 -> 是否需要法律援助 -> 调解协议或调解失败 -> 司法确认/诉前调解/起诉立案 -> 人民法庭或基层法院审理 -> 调解书/判决/裁定 -> 自愿履行或拒不履行 -> 申请执行 -> 财产查控、限制消费、失信惩戒或执行和解 -> 关系、信用、债务和乡土名声回写`

## 3. 安全边界

本篇用于小说创作和审稿，不提供法律意见，不替代律师、法律援助机构、法院、行政机关或公安机关的现实判断。

AI 写作必须遵守：

- 不教唆伪造证据、虚假诉讼、恶意串通、威胁证人、转移财产、规避执行或滥用投诉举报。
- 不把“找关系摆平司法所/法院/执行局”写成聪明正义。
- 不把人民调解写成强迫弱者让步。调解应强调自愿、平等、合法和尊重当事人真实意思。
- 不把法律援助写成任何人随便免费请律师的万能卡。它有法定范围、申请条件和机构审查。
- 不把信访写成替代诉讼、复议、仲裁和执行的万能通道；涉法涉诉事项要尊重法定程序。
- 不把判决等同于立即拿到钱。生效法律文书之后还可能有自动履行、申请执行、查控财产、执行和解、终结本次执行、恢复执行等后续压力。

## 4. 写作总原则

### 4.1 先分清纠纷性质，不要所有矛盾都报警或起诉

县域纠纷的第一步不是“谁嗓门大”，而是分流：

- 民事纠纷：借款、买卖、租赁、邻里、赡养、婚约财产、继承、损害赔偿。
- 家事纠纷：离婚、抚养、赡养、探望、分家析产、老人再婚、家庭财产。
- 行政争议：补偿、许可、处罚、信息公开、村务或部门决定相关争议。
- 劳动争议：欠薪、工伤、社保、劳动关系，通常先走劳动监察、调解仲裁或相关程序。
- 治安/刑事风险：打架伤害、威胁、诈骗、盗窃、故意损毁、家暴、限制人身自由，转入第 166 篇。
- 纯情绪冲突：亲戚旧账、面子、误会、舆论，有时需要调解，但不一定有法律请求。

AI 写作时要让角色先被迫回答：“这到底是什么纠纷？谁有权处理？要不要先保安全？有没有证据？是否适合调解？”

### 4.2 人民调解的戏剧性不是“和稀泥”，而是把情绪翻译成可履行条款

人民调解法确认人民调解委员会可以调解民间纠纷，调解协议对当事人具有法律约束力；调解不成时，应告知当事人依法通过仲裁、行政、司法等途径维护权利。

写作时，调解员不是万能老好人。他真正做的工作是：

- 把“他欺负我家”翻译成具体请求。
- 把“我没钱”拆成付款期限和分期安排。
- 把“以后别再来闹”写成停止侵扰、保持距离或赔礼道歉。
- 把“谁养老人谁得房”拆成赡养分工、居住安排、费用承担和后续继承另行处理。
- 把口头承诺变成签字、日期、金额、履行方式和违约后果。

好看的调解戏不是大家突然变善良，而是每个人都在一张桌子前发现：空话不能执行，羞辱不能写进协议，真正能落地的只有具体条款。

### 4.3 司法所和公共法律服务是“问路口”，不是魔法机关

县域司法所、公共法律服务中心和法律服务热线，适合写角色第一次把糊成一团的纠纷讲清楚：

- 该找派出所、法院、仲裁、行政复议、人民调解、市场监管，还是村委会？
- 有没有法律援助资格或其他公益法律服务资源？
- 需要准备哪些基本材料：身份证明、合同、借条、转账、聊天、照片、病历、调解记录？
- 什么能调解，什么必须转入其他程序？
- 风险是时效、证据缺失、主体不对，还是请求本身不清楚？

不要把司法所写成“帮主角出主意钻漏洞”的地方。它更像基层法治的导航站，帮人物从情绪进入程序。

### 4.4 法律援助适合写弱势角色的行动能力恢复

法律援助法强调国家建立法律援助制度，为经济困难公民和符合法定条件的其他当事人无偿提供法律咨询、代理、刑事辩护等法律服务。

小说中的法律援助价值，不是让角色突然有“免费律师外挂”，而是让原本没有资源的人第一次获得表达权：

- 老人追索赡养费。
- 留守母亲争取抚养费。
- 工伤劳动者、欠薪农民工整理材料。
- 未成年人、残障人士、受家暴者或经济困难当事人获得程序支持。
- 角色从“我不懂法，只会哭”转向“我知道自己可以提出什么请求”。

要写审查、材料、范围和等待，不要写成只要喊冤就自动配顶级律师。

### 4.5 人民法庭是县域纠纷的制度场，不是城市大法院缩小版

人民法庭贴近乡镇、农村和社区生活，它的叙事价值在于：法官熟悉当地关系，但仍要把熟人社会里的纠纷放进法律程序。

可写的矛盾：

- 法官知道两家是亲戚，但不能只按亲戚面子判。
- 村里都认为某种做法“惯例如此”，但法律后果并不一样。
- 当事人希望法庭替自己出气，法庭只能处理诉讼请求。
- 诉前调解能修复关系，但一方利用拖延逃债时就要进入审理或执行。
- 法庭巡回审判、就地调解、普法讲座能改变村里对纠纷的认知。

### 4.6 执行不是结尾，是第二场战争

很多网文把“判了”当作爽点终点。但现实纠纷常常在执行阶段才真正见血。

执行链可以写：

- 对方主动履行，关系还有余地。
- 对方没钱，判决变成纸面胜利。
- 对方有钱但拒不履行，进入财产查控、限制消费、失信惩戒等压力。
- 双方达成执行和解，分期付款却又断供。
- 家庭内部执行困难：赡养、探望、腾房、共有财产分割，比单纯转账更复杂。
- 执行结果回到村里名声、婚恋、信用、合作和亲戚关系。

爽点不是教主角如何逼债，而是让“拒不履行的人”付出制度化后果，同时让读者看到胜诉与拿到结果之间的距离。

## 5. AI 写作前的七项判断

### 5.1 纠纷性质判断

先判断纠纷属于哪类：

- 钱：借款、欠款、合同、赔偿、彩礼、赡养费、抚养费。
- 房地：宅基地、老宅、租赁、买卖、维修、邻地通行、承包地。
- 家庭：离婚、抚养、赡养、继承预热、老人再婚、分家析产。
- 邻里：噪音、漏水、停车、宠物、通道、围墙、树木、采光。
- 行政：补偿、许可、处罚、登记、信息公开、基层部门决定。
- 劳动：欠薪、工伤、社保、劳动关系。
- 安全：暴力、威胁、诈骗、家暴、限制自由，转入公安或保护路径。

性质不同，出口不同。不要把所有纠纷都塞进一个“调解室吵一架”。

### 5.2 当事人关系判断

县域纠纷的难点常常不是法条，而是关系：

- 亲戚之间：赢了官司可能输了族内关系。
- 邻居之间：判完还要继续抬头不见低头见。
- 村干部与村民：公共事务和私人恩怨混在一起。
- 返乡青年与本地熟人：外来知识和乡土规则冲突。
- 老人、妇女、未成年人、残障者：表达能力和资源不平等。
- 生意伙伴：合同不完整，靠熟人信用运行。

调解的价值常在于保留长期关系；诉讼的价值常在于切断不公平关系。AI 要判断角色要的是修复、止损、证明清白，还是拿回具体利益。

### 5.3 紧急风险判断

有些纠纷不能先调解：

- 存在人身危险、家暴、跟踪、威胁、限制自由。
- 有诈骗、盗窃、故意伤害、寻衅滋事等违法犯罪风险。
- 证据正在灭失，财产可能被转移。
- 未成年人、老人、残障者处于明显受害状态。
- 一方正在以“调解”为名继续控制或骚扰另一方。

这些情况不能写成“坐下来喝茶就行”，要接入公安、法院、民政、妇联、学校、医疗或其他安全出口。

### 5.4 证据判断

基层纠纷常见证据：

- 借条、欠条、合同、收据、发票、转账记录。
- 微信聊天、短信、电话录音、群公告、平台订单。
- 照片、视频、监控、维修单、医疗票据、伤情材料。
- 村委会证明、调解记录、会议记录、土地承包或宅基地材料。
- 证人、邻居、亲戚、村干部、物业、网格员。
- 既往履行记录：谁照护老人、谁还款、谁占用通道、谁修过房。

写证据时不要教人伪造，要写“证据让情绪有了边界”。没有证据的角色可以愤怒，但程序不会只靠愤怒前进。

### 5.5 渠道判断

常见渠道分工：

- 亲友、族亲、邻里：适合早期情绪缓和，但容易偏袒。
- 村委会、居委会、物业、行业组织：适合生活场景协调和事实说明。
- 人民调解委员会：适合民间纠纷自愿调解。
- 司法所、公共法律服务中心：适合咨询、分流、人民调解、法治宣传和基层服务。
- 法律援助机构：适合符合条件的当事人获得法律咨询、代理等帮助。
- 人民法庭/基层法院：适合诉前调解、司法确认、立案审理和裁判。
- 行政复议机关：适合对行政行为不服的争议。
- 信访渠道：适合反映诉求、意见和建议，但不能替代法定司法程序。

### 5.6 协议可执行性判断

调解协议或和解协议要能执行。至少写清：

- 谁向谁履行。
- 履行什么内容。
- 金额、物品、行为或停止行为是否明确。
- 时间、地点、方式是否清楚。
- 分期付款是否有节点。
- 是否需要道歉、修复、腾退、交付、探望或协助办理手续。
- 违约后如何处理。
- 是否需要司法确认或转入诉讼。

如果协议只写“双方互谅互让，以后不得再闹”，读者会觉得假，因为它无法处理真正的利益。

### 5.7 结果回写判断

纠纷解决不是“签字完结”。要回写：

- 谁真的履行了。
- 谁失去信用。
- 谁在村里名声改变。
- 谁因为调解暴露旧账。
- 谁因为执行被限制消费、影响生意、婚恋或合作。
- 谁从纠纷中学会保留合同和转账记录。
- 哪段亲情、邻里关系或商业关系再也回不去。

## 6. 可复用剧情模型

### 6.1 老宅通道纠纷从村口吵到人民法庭

适用：乡村邻里、宅基地、返乡现实。

剧情骨架：

`邻居砌墙占通道 -> 双方在村口争吵 -> 村干部先协调 -> 调解时发现老宅边界和多年口头约定不清 -> 司法所建议整理宅基地、照片、证人和历史通行事实 -> 人民调解失败 -> 人民法庭诉前调解或立案 -> 判决/调解明确通行和修复 -> 执行后两家仍要继续做邻居`

重点：不是谁骂赢，而是谁把“祖上就这么走”变成可被确认的事实和请求。

### 6.2 借钱不还，赢了官司才知道执行更难

适用：现实向、创业、亲戚借贷、爽文反击。

剧情骨架：

`熟人借钱 -> 没写清借期和利息 -> 多次催讨只剩情绪 -> 人民调解尝试分期 -> 对方签了协议又不履行 -> 起诉或司法确认 -> 生效文书 -> 申请执行 -> 查控发现对方名下财产少 -> 限制消费/执行和解 -> 主角明白程序不是魔法，但比私下闹更有边界`

### 6.3 留守母亲第一次走进法律援助中心

适用：女性成长、家庭伦理、抚养费/赡养费/家暴后续。

剧情骨架：

`角色长期被说“家事别外扬” -> 孩子抚养费或老人赡养费断供 -> 她带着零散聊天记录去公共法律服务中心 -> 工作人员帮她分清请求和材料 -> 判断是否符合法律援助条件 -> 她第一次把哭诉整理成事实时间线 -> 对方发现她不再只靠亲戚劝说`

爽点是行动能力恢复，不是机构替她开挂。

### 6.4 调解员夹在宗族面子和法律底线之间

适用：基层群像、乡土伦理、轻现实。

剧情骨架：

`两家亲戚因彩礼/赡养/土地撕破脸 -> 宗族长辈要求“按老规矩” -> 年轻调解员坚持自愿合法 -> 一方试图用舆论逼弱者签字 -> 调解员暂停调解并告知诉讼、援助或安全路径 -> 最后协议少了漂亮话，多了具体履行条款`

### 6.5 行政补偿争议不能只靠上访

适用：征收补偿、村务公开、基层治理。

剧情骨架：

`村民不满补偿或行政处理 -> 先在村群里吵 -> 有人鼓动集体闹事 -> 主角去公共法律服务中心咨询 -> 分清村务、民事合同、行政行为和信访事项 -> 可复议的走行政复议，可诉讼的准备材料，可调解的先调解 -> 角色从“喊冤”转向“选择正确路径”`

注意：信访可写，但不能把它写成替代行政复议、诉讼和执行的万能捷径。

### 6.6 赢了判决却输在拒不履行的现实压力

适用：商战小额纠纷、家庭借贷、合同违约。

剧情骨架：

`主角胜诉 -> 对方朋友圈继续炫富却不还钱 -> 申请执行 -> 法院要求提供财产线索，同时系统查控 -> 对方被限制高消费或纳入失信后生意受影响 -> 对方提出执行和解 -> 主角选择一次性少拿、分期等待或继续执行 -> 人物成长在于理解“正义也要落到钱、账户和履行期限上”`

## 7. 场景素材库

### 7.1 司法所或公共法律服务中心

可用细节：

- 墙上贴着人民调解、法律援助、公证、行政复议、普法宣传的指引。
- 当事人一进门先说“我要告他”，工作人员问“你要他做什么”。
- 主角拿出塑料袋里的借条、发票、病历和聊天截图。
- 工作人员先排除人身安全风险，再判断是调解、援助、仲裁、复议还是诉讼。
- 角色第一次意识到，讲得惨不等于请求清楚。

### 7.2 人民调解室

可用细节：

- 一张长桌，两边坐当事人，中间是调解员和记录本。
- 亲戚在门外等，随时想冲进来帮腔。
- 调解员把“他家欺负人”写成“赔偿维修费多少元，于某日前支付”。
- 一方不肯签，调解员终止调解并告知其他救济路径。
- 签字那一刻不一定和解，但至少把下一步写清楚。

### 7.3 人民法庭

可用细节：

- 法庭离县城法院很远，门口停着电动车和三轮车。
- 法官熟悉村名、路名、地块俗称，但要求当事人拿证据。
- 诉前调解室里，法官助理或特邀调解员先听双方陈述。
- 当事人以为“法官知道我是好人”，法官只问“你的诉讼请求是什么”。
- 巡回审判或就地调解让村里人第一次看见法律怎么处理旧规矩。

### 7.4 执行阶段

可用细节：

- 申请执行人拿着生效判决，以为当天就能拿钱。
- 执行法官问有没有财产线索，角色才发现自己从没留意对方账户、车辆、经营收入。
- 对方被限制消费后，婚礼、出差、生意合作都受影响。
- 执行和解分期付款，第一次到账比庭审胜诉更有真实冲击。
- 执行不顺时，主角要在继续追、接受和解、等待恢复执行之间选择。

## 8. AI 自查清单

写完基层纠纷解决情节后，用这些问题检查：

- 是否先判断纠纷性质，而不是所有问题都报警或起诉？
- 是否区分亲友劝说、村居协调、人民调解、司法所、公共法律服务、法律援助、诉前调解、法院审理和执行？
- 调解是否体现自愿、平等、合法，而不是逼弱者忍让？
- 角色的请求是否具体：要钱、道歉、腾退、修复、停止侵害、履行合同还是确认权利？
- 证据是否能支撑请求，而不是只靠哭诉、舆论或主角光环？
- 法律援助是否有适用条件和材料审查？
- 人民法庭是否写出基层关系和法定程序之间的张力？
- 判决、调解书或司法确认之后，是否写出履行和执行，而不是判完清零？
- 行政复议、信访、诉讼、调解是否没有混成一个万能“找政府”？
- 是否避免了伪造证据、转移财产、规避执行、虚假诉讼和滥用投诉的可模仿细节？

## 10. 禁写和风险

- 禁止写伪造证据、虚假诉讼、假调解、恶意串通、转移财产、藏匿收入、规避执行的具体方法。
- 禁止把调解员、法官、律师、司法所工作人员写成随意站队或收礼摆平的人，除非剧情明确是违法腐败风险，并设置后果。
- 禁止把“弱势一方被迫签字”写成圆满调解。
- 禁止把信访、曝光、堵门、拉横幅写成比法定程序更正义的万能爽点。
- 禁止把法律援助写成无条件无限服务。
- 禁止把执行阶段写成主角私力讨债、羞辱、威胁或暴力逼迫。
- 禁止把基层群众写成“不懂法的笑料”。更好的写法是：他们懂人情，但第一次学会把人情转成可执行的规则。

## 11. 来源与依据

- 《中华人民共和国人民调解法》提供人民调解委员会、人民调解员、调解原则、调解协议和调解不成后的救济路径依据。
- 《中华人民共和国法律援助法》提供法律援助制度、服务形式、政府保障、申请和规范边界。
- 中办、国办《关于加快推进公共法律服务体系建设的意见》提供覆盖城乡、便捷高效、均等普惠的公共法律服务体系框架，明确整合法律援助、人民调解、律师、公证、司法鉴定、仲裁、司法所等资源。
- 《中华人民共和国民事诉讼法》及 2023 年修改决定，为起诉、调解、司法确认、审理、裁判和执行提供基础程序框架。
- 最高人民法院关于人民法庭高质量发展的意见，为人民法庭服务乡村振兴、基层治理和群众司法需求提供现实依据。
- 最高人民法院一站式多元纠纷解决和诉讼服务体系资料，为诉前调解、多元解纷、诉讼服务中心和在线调解提供现实素材。
- 失信被执行人名单、限制消费和执行公开相关最高人民法院资料，为执行难、信用惩戒和判后履行压力提供现实依据。
- 《中华人民共和国行政复议法》和《信访工作条例》用于区分行政争议、信访事项和涉法涉诉程序，避免把所有基层诉求混成“找政府”。
