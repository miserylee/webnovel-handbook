# 150. 金钱货币、价格尺度、收入支出、债务账本、生活成本与经济可信度

状态：已确认

适用对象：AI agents、网文作者、审稿 agent、连续性维护 agent、经营/商战/玄幻升级/古言权谋/都市现实题材 agent

核心问题：AI 写小说时常把钱写成随手数字：一会儿一两银子买半座城，一会儿主角刚穷困潦倒又随手包下酒楼；灵石、金币、稿费、打赏、房租、债务、税粮、赔偿和经营收入没有统一尺度，导致阶层失真、爽点变廉价、经营线像报表、债务没有压力、资源争夺没有代价。商业级长篇需要把钱当成行动资格、社会信用、阶层标识、时间压力和连续性资产来管理。

---

## 1. 一句话原则

已确认：钱不是数字装饰，而是“谁承认它有价值、能买到什么、对谁贵、花掉后少了什么、欠下后谁来追”的叙事系统。

任何重要金额至少要回答三个问题：

1. 这个数对当前角色意味着什么？
2. 它能换来什么行动资格或社会位置？
3. 支付、获得、借入或失去后，后文状态如何改变？

如果一个金额既不改变选择，也不改变关系、资源、身份或风险，就不要详细写数字。

## 3. 金钱十线卡

AI agent 设计任何重要金钱、价格、债务、报酬、经营或交易剧情前，先填这张卡。

```text
货币名称：银两、铜钱、灵石、金币、信用点、积分、贡献点、人民币等。
价值承认者：国家、宗门、商会、平台、系统、银行、黑市、地方势力、读者打赏者。
最小单位/大额单位：日常小额怎么付，大额怎么保存或转移？
购买力锚点：一顿饭、一晚住宿、普通日薪/月薪、房租/地租、一件常用工具、一匹马/一辆车/一件低阶法器分别多少钱？
角色收入：主角、配角、底层、精英、权贵、反派组织各自稳定收入来源是什么？
固定支出：房租、伙食、税费、学费、药费、路费、维护费、供奉、赡养、团队工资。
流动资金：当前手里能立刻花的钱是多少？资产能否快速变现？
信用与债务：谁愿意赊账，利息或人情是什么，什么时候必须还？
价格波动：稀缺、季节、战争、灾荒、封锁、拍卖、黑市、平台热度会怎样改变价格？
支付痕迹：账本、票据、收据、转账记录、欠条、契约、灵契、系统日志、见证人。
后文状态：钱进账/出账后，行动资格、关系、风险、身份或台账发生什么变化？
```

## 4. 购买力锚点：先定“日常三件套”

不要一开始就写宏大的王朝财政或宗门经济。先定三个读者能理解的锚点：

1. 普通人一天或一个月能挣多少。
2. 普通人吃住一天要花多少。
3. 主角当前目标要花掉多少个“普通生活单位”。

示例换算方式：

```text
一枚下品灵石 = 普通外门弟子半月生活费。
一间下等客房 = 一天饭钱的三倍。
一瓶低阶疗伤丹 = 普通散修两个月净收入。
一张去主城的船票 = 主角当前现金的四成。
```

这样写的好处是：读者不需要背货币表，也能立刻知道“贵不贵”。

## 5. 金额的四种叙事功能

### 5.1 生存金额

用于制造底层压力：吃饭、住宿、药费、路费、房租、欠债、学费、赎身费、保证金。

写法：金额不必巨大，但要卡在角色刚好难受的位置。比如差半个月房租、差一味药钱、差返程路费。

风险：如果穷只写成形容词，角色又从不为钱改变选择，贫穷就会变成装饰。

### 5.2 身份金额

用于展示阶层、面子、资格、圈层门槛：入场券、会费、礼金、聘礼、嫁妆、拜师礼、贡品、会员费、学区房、贵宾席。

写法：同一笔钱对不同角色意义不同。权贵随手赏的银票，可能是伙计一年的积蓄。

风险：只写“豪掷千金”会很快疲劳。真正的爽点来自这笔钱改变了谁的态度、门槛、舆论或行动资格。

### 5.3 交易金额

用于谈判、买卖、雇佣、赎人、买线索、黑市、拍卖、签约、分成。

写法：价格要和稀缺性、风险、替代品、信息差、付款时限和违约成本绑定。

风险：交易双方如果都没有底价、让步理由和失败后果，议价会变成嘴炮。

### 5.4 战略金额

用于长期经营、势力建设、战争、赈灾、项目投资、宗门资源、公司现金流、平台运营。

写法：重点不是报表数字，而是钱如何转化为人手、物资、时间、地盘、声望、制度或下一轮行动资格。

风险：数字上涨不等于爽点。读者要看到增长带来的新能力和新压力。

## 6. 收入、支出与现金流

### 6.1 收入要分“稳定收入”和“一次性横财”

稳定收入：工资、俸禄、租金、店铺流水、任务报酬、分成、稿费、打赏订阅、宗门月例、平台补贴。

一次性横财：赏金、遗产、拍卖所得、秘境收获、系统奖励、赔偿、分赃、卖掉珍品。

已确认规则：一次性横财不能自动解决长期现金流。主角拿到一大笔钱后，要回答是否能持续覆盖固定支出、维护费、团队开销和下一轮投入。

### 6.2 支出要分“活着就要花”和“选择后才花”

活着就要花：吃住、交通、药、租金、税费、维护、通讯、服装、工具折旧。

选择后才花：雇人、请客、行贿、投资、升级装备、扩大店铺、买情报、赎人、送礼、参赛报名。

如果角色长期没有固定支出，世界会显得像免费游戏大厅。

### 6.3 现金流比总资产更能制造压力

角色可以“很有钱但没现金”：地契、法宝、房产、股票、股份、灵矿份额、应收账款、被封存的嫁妆，都不一定能立刻支付药费或路费。

可写压力：

- 有资产但不能卖。
- 卖了会暴露身份或损害长期利益。
- 买家压价。
- 账期没到。
- 银票/银行卡/储物戒被冻结。
- 大额灵石难以找零。

## 7. 债务不是“欠钱”两个字

债务必须包含六个要素：

```text
本金：最初欠多少。
利息/代价：钱、物、人情、服务、名声、身体风险、政治站队。
期限：什么时候必须还。
抵押/担保：房契、法器、身份、亲族、队友、商誉、契约、魂灯。
催收者：债主、宗门、钱庄、平台、黑市、亲戚、系统、家族长辈。
违约后果：失信、罚息、失去房产、被逐出、名声毁损、被迫站队、剧情危机。
```

写法：债务最好让角色改变选择，而不是只当背景悲惨。

示例：

```text
主角不是“欠了很多钱”，而是“七日后要还三十两，否则母亲押在药铺的银簪会被转卖，而那支银簪是认亲线索”。
```

## 8. 价格如何变动

价格不稳定，是制造剧情的好工具。

可用变动原因：

- 稀缺：丹药材料少、粮食歉收、灵矿封山、热门演唱会票少。
- 风险：黑市、战区、禁运、远途运输、违禁物、匿名交易。
- 时间：临近考试、秘境开启、节日、灾后、船期前、拍卖夜。
- 信息差：卖家不知道真价值、买家知道未来会涨、谣言推高价格。
- 权力：官价、垄断、税卡、宗门贡献兑换、平台分成、家族内部价。
- 信任：熟人赊账、陌生人加价、主角声誉好拿到折扣。
- 质量：真伪、年份、品相、耐久、售后、绑定限制、是否可转卖。

审稿时要问：这个价格为什么是现在这个数？有没有人能反驳、压价、囤货、套利或造假？

## 9. 货币与支付方式

### 9.1 货币要有“承认范围”

国家货币在本国好用，边境可能折价；宗门贡献点在宗门内好用，外面可能不能买米；平台积分能换流量，不一定能付房租；灵石在修士间通用，凡人可能只认银钱。

写法：换地图、换阶层、换组织时，货币承认范围可以制造真实摩擦。

### 9.2 支付方式改变剧情痕迹

现金：即时、匿名性强，但携带风险和找零问题明显。

账本/欠条：能赊账，但留下证据，也可能被篡改或追讨。

银票/钱庄/银行/转账：便携，便于大额交易，但依赖机构信用和身份验证。

数字支付/平台余额：方便，但有记录、冻结、限额、风控和隐私风险。

灵契/系统扣款：可以强制结算，但要写清触发规则和争议空间。

## 10. 阶层钱感：同一笔钱有不同重量

AI agent 写钱时，必须按人物阶层调整反应。

```text
底层角色：先想到饭、房租、药费、路费、欠债期限。
普通职场/学生：先想到工资、奖学金、月供、生活费、机会成本。
小商人/经营者：先想到现金流、库存、租金、人工、回款周期。
权贵/豪门：先想到面子、圈层门槛、礼数、政治成本和资产配置。
修士/冒险者：先想到丹药、装备、传送费、秘境资格、补给和疗伤成本。
反派组织：先想到收买、人手、情报、灭口、洗白成本和风险预算。
```

同一千两银子，可以是主角救命钱、商会试探钱、权贵面子钱、反派诱饵钱。不要让所有人用同一种震惊语气看钱。

## 11. 题材用法

### 11.1 玄幻/仙侠/升级流

重点：灵石、贡献点、丹药、法器、秘境资格、传送费、材料价格、等级通胀。

规则：

- 灵石不是万能钱。它可以分品级、纯度、地区承认、携带风险和兑换损耗。
- 宗门贡献点不是现金，通常只能换宗门内资源。
- 高阶资源贵，不只因为稀有，也因为会改变行动资格。
- 主角升级后，低阶钱可能贬值，但低阶生活成本仍然影响身边人。
- 拍卖场价格要有底价、竞争者、预算上限和后续现金流后果。

### 11.2 古言/权谋/官场

重点：银两、铜钱、俸禄、田产、铺面、嫁妆、聘礼、赎身、税粮、赈灾款、赏赐。

规则：

- 银钱、田产、人情和身份常常互相转化，但不能随便等价。
- 嫁妆、聘礼、赏赐、俸禄不仅是钱，也是身份和家族权力。
- 真实朝代货币制度复杂，涉及具体朝代时必须查证，不要把影视里的“银票”“一锭银子”随意套用。
- 赈灾、税粮和库银必须联动文书、人手、运输、地方执行和贪腐风险。

### 11.3 都市/现言/商战

重点：工资、奖金、房租、房贷、股权、合同款、违约金、品牌代言、直播打赏、平台分成、债务。

规则：

- 现实金额会被读者拿生活经验校验，不能随手写。
- 工资和房租最容易暴露作者不懂生活成本。
- 霸总/豪门花钱要写“钱改变了什么”，不要只写黑卡无限刷。
- 商战金额必须连接合同、回款、现金流、债务、舆论、法务和执行能力。

### 11.4 经营/种田/基建

重点：投入、产出、库存、人工、维护、税费、折旧、回款、扩张资金。

规则：

- 经营爽点来自“瓶颈被解决后循环变强”，不是数字变大。
- 收入增长必须带来新成本：租金、人手、原料、仓储、运输、竞争、税费、管理。
- 主角不能只赚不花。每一轮扩张都应有钱、人、物、地、名声或制度成本。

### 11.5 娱乐圈/直播/文娱

重点：稿费、版权、平台分成、打赏、代言、演出费、票房、宣发费、违约金。

规则：

- 公开数字会改变舆论和圈层位置。
- 打赏和榜单不是纯爽点，要进入平台规则、粉丝关系和资源分配。
- 版权、歌词、真实平台规则、合同和税务需要查证；正文避免提供现实规避教程。

## 12. 金钱场景写法

### 12.1 付钱场景

不要只写“他掏出银票”。要写钱如何改变场景。

```text
付款目标：买什么/解决什么问题？
付款阻碍：不够、找零、限额、身份验证、对方不收、货币不通用。
旁人反应：谁震惊，谁嫉妒，谁开始估价主角？
付款痕迹：收据、账本、欠条、转账记录、见证人。
后续变化：主角失去多少钱，获得什么资格，被谁盯上？
```

### 12.2 赚钱场景

赚钱不能只写“到账”。要写收入来源是否可持续。

```text
收入来源：卖货、任务、赏金、工资、稿费、分成、打赏、租金、投资、战利品。
收入门槛：技术、名声、渠道、资质、风险、平台规则、前期投入。
收入成本：材料、时间、人力、税费、平台抽成、维护、风险。
净收益：扣掉成本后真正能自由使用多少？
后续用途：还债、投资、买药、买装备、请人、扩大店铺、维持团队。
```

### 12.3 砍价/议价场景

议价不是吵价，而是信息和底线的碰撞。

```text
卖方底价：低于多少不卖，为什么？
买方上限：高于多少就不买，为什么？
替代方案：不成交各自怎么办？
信息差：谁知道真价值，谁误判？
让步代价：降价换什么，涨价补什么？
成交痕迹：现付、分期、赊账、欠人情、附加条件。
```

### 12.4 借钱/欠账场景

借钱场景的核心是信任定价。

```text
借款理由：为什么非借不可？
借款对象：为什么找这个人？
信任依据：关系、抵押、担保、名声、威胁、共同利益。
还款期限：什么时候还？
利息代价：钱、人情、服务、站队、秘密、名声。
违约后果：谁受损，谁来追，怎样影响后文？
```

## 13. 常见失败模式

- 随手数字：金额只是看着很大，没有购买力锚点。
- 穷人不为钱改变选择：角色说穷，但吃住行医从不受影响。
- 富人只会砸钱：豪门/神豪没有圈层、风控、舆论、法律和资源调度。
- 灵石万能：所有资源都能用灵石买，导致秘境、宗门、贡献点和稀缺资源失效。
- 横财清零长期压力：主角赚一笔后再也没有生活成本和现金流问题。
- 债务无催收：欠钱只制造背景惨，没人追、没有期限、没有抵押、没有违约后果。
- 经营报表化：只写营业额增长，不写成本、库存、回款、维护和竞争。
- 税费消失：古代/现代/架空社会中，钱大量流动却没有税、抽成、手续费、贡赋或平台分成。
- 价格不随稀缺变化：灾荒时粮价不动，战时药价不动，热门资源不涨价。
- 现实金额失真：都市文工资、房租、房价、合同款和赔偿额脱离读者常识。

## 14. 审稿检查问题

AI 审稿或修订时，逐条检查：

- 这笔钱有没有购买力锚点？读者知道它大概贵不贵吗？
- 这个价格对当前角色是轻松、肉疼、掏空、借债还是根本不可能？
- 货币是否在当前地区、组织、阶层被承认？
- 角色的稳定收入和固定支出是否匹配其生活方式？
- 主角获得横财后，是否仍有维护费、团队开销、投资需求或新风险？
- 债务是否有本金、利息、期限、抵押、催收者和违约后果？
- 交易是否有底价、上限、替代方案和付款痕迹？
- 经营线是否区分收入、成本、利润、现金流和库存？
- 钱的进出是否进入台账，并影响后文行动资格？
- 涉及现实贷款、投资、证券、税务、合同、工资、平台收益、房价房租时，是否需要查最新官方或可靠资料？

## 15. AI 写作提示词片段

### 15.1 设计钱感

```text
请为这个小说项目建立最小金钱尺度。
要求：不要写宏观经济论文，只给 AI 起草章节必需的购买力锚点。至少包括普通日薪/月薪、一顿饭、一晚住宿、一次长途交通、常用药/低阶丹药、普通武器/工具、主角当前目标成本，并说明每个金额对底层、普通人、富人/修士分别意味着什么。
```

### 15.2 修订随手数字

```text
请检查正文中的金额是否随手。
对每个重要金额标注：购买力锚点、角色承受度、支付后状态变化、是否需要进台账。没有叙事功能的数字请改成相对表达或删除。
```

### 15.3 写交易场景

```text
请写一段交易/议价场景。
要求：双方都有底价和上限；至少出现一个信息差或替代方案；成交条件不能只有钱，还要包含付款方式、交付时间、风险或后续人情；结尾写清交易后谁获得了什么、谁失去了什么。
```

### 15.4 写债务压力

```text
请把“主角欠钱”改写成可推动剧情的债务压力。
要求：写清本金、期限、利息/代价、抵押或担保、催收者、违约后果，以及主角因此必须改变的下一步行动。
```

## 17. 安全与真实性边界

- 本专题只用于小说创作，不提供现实投资、贷款、税务、证券、合同、逃债、洗钱、诈骗、规避监管或违法交易建议。
- 涉及现实金融产品、平台收益、纳税、借贷、劳动报酬、房价房租、合同违约和法律后果时，必须查最新官方或专业资料。
- 历史货币、银两、铜钱、纸币、钱庄、票号、税制和俸禄随朝代地区差异很大。真实历史写作必须查证，不要把不同朝代制度混写成通用古风。
- 幻想货币可以自设，但一旦建立购买力和承认范围，后文必须持续执行。

## 18. 来源与吸收方式

- Encyclopaedia Britannica, “Money”：吸收货币作为交换媒介、价值尺度和价值储藏的基础功能，用于“货币为什么被承认”的叙事规则。
- Bank of England, “What is money?” 与 “What are interest rates?”：吸收钱的功能、信用货币和利率作为借贷价格的基础解释，用于债务、信用和时间成本。
- Federal Reserve Education/FAQ 与 U.S. Bureau of Labor Statistics CPI 资料：吸收货币供给、价格变化、通胀和消费者价格指数相关基础概念，用于价格波动和购买力锚点。
- IMF 关于 inflation 的解释资料：吸收通胀会降低购买力、价格普遍上升影响生活成本的判断，用于灾荒、战争、资源涌入和平台流量经济中的价格变化。
- Reedsy、World Anvil 等世界构建资料：吸收经济、货币、贸易、阶层和社会规则应服务人物与情节的思路，用于把经济设定压缩成可执行的小说工具。
- 中国钱币博物馆、中国国家博物馆、故宫博物院等公共资料入口：用于提醒真实历史货币、钱币、银两、纸币、钱庄和礼俗消费需要按朝代和地区查证。

## 19. 最小执行规则

已确认：AI agent 只要写到重要金额，最少必须交代“三件事”：

1. 这笔钱大概能买什么。
2. 对当前角色是轻松、肉疼、掏空还是不可能。
3. 进账、出账或欠账后，后文状态改变什么。

如果三件事都没有，这个金额应改成相对表达、压缩或删除。
