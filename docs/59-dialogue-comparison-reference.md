# 对话样本对比参考书

状态：已确认  
用途：给 AI agents 作为“失败初稿 / 用户改稿 / 差异分析 / 可迁移规则”的对照训练材料，专门用于降低对白、动作和现场反应中的 AI 味。

## 使用方法

写稿或审稿时，如果出现“台词不像人说话”“短句太碎”“动作太直球”“现场反应模板化”“职场对峙像审讯台”，先读本文，再修改正文。

使用顺序：

1. 先看 Agent 初稿，识别 AI 味是怎样出现的。
2. 再看用户改稿或通过版，观察人写后的口语颗粒、动作选择和关系压力。
3. 对照差异分析，提取本次章节能直接使用的改法。
4. 改稿时不要机械套句，只迁移“说话方式、动作逻辑、关系压力和遮掩方式”。

硬性要求：样本对比必须成对保存。只保存通过版，会让后续 agent 只能模仿结果，看不见自己最容易错在哪里。

## 总原则：自然不是短，也不是白

用户确认的核心经验：不要写“漂亮句子”，先写“人真的会这么说、这么接”。

去 AI 味不是把句子全部改短、改白、改克制。真正自然的句子通常有这些特征：

1. 说话人带着身份、关系和目的开口。
2. 台词里有试探、找补、客气、推脱、自保或半句绕法。
3. 动作像人物当下会做的事，不像作者在摆剧情机关。
4. 情绪不靠“空气静了”“笑容僵住了”这类模板句翻译。
5. 关键处可以锋利，但锋利之前要有真实关系里的铺垫。

## 案例 1：散文化物象硬压情绪

场景：现实向家庭关系，欠账与遗物冲突。  
问题类型：想用物象写情绪，但句子像散文，不像网文现场。

### Agent 初稿问题句

```text
她把伞收了，水滴砸在门槛上。
```

### 用户反馈

`水滴砸在门槛上`这个表达很生硬，看得出想表达情绪，但是像在写散文似的。

### 通过方向

```text
她在门口把伞收好，站了会儿才开口：“哥，钱我不跟你要了，真不要了。”

表哥刚松一口气，她看向桌边那个盒子。

“但我妈那个镯子，你得给我。”
```

### 差异分析

1. 初稿把情绪压到“水滴”上，作者用力太明显。
2. 通过版把情绪放回人物动作：收伞、站住、先退一步、再指向镯子。
3. `钱我不跟你要了，真不要了`像亲戚之间真的会说的话，有重复和让步。
4. `但我妈那个镯子`比抽象控诉更具体，关系和痛点都落在物件上。

### 可迁移规则

现实、家庭、熟人冲突里，优先写生活动作、站位、停顿、手里东西和关系物件，不要用散文化物象替人物用力。

## 案例 2：台词不要写成审判腔

场景：亲戚欠账，人物想拿回母亲遗物。  
问题类型：台词意思对，但不像熟人会说的话。

### Agent 初稿倾向

```text
“你把我妈的镯子还我。”
```

### 用户反馈

台词不够口语化。

### 更自然方向

```text
“但我妈那个镯子，你得给我。”
```

### 差异分析

1. `你把我妈的镯子还我`像判词，准确但硬。
2. `但我妈那个镯子`有真实口语里的指代和绕法。
3. `你得给我`保留了关系里的硬度，比“还我”更像熟人之间压着火说话。

### 可迁移规则

熟人对话不要只追求语义准确。保留称呼、停顿、指代词、重复和半句转折，往往更有人味。

## 案例 3：短句不等于自然

场景：同上。  
问题类型：为了去 AI 味，把句子切得太碎，反而更像机器在模仿克制。

### Agent 错误倾向

```text
“哥，钱……算了。”
“镯子给我。我妈那个。”
```

### 用户反馈

有时候短句并不会很贴切，而且 AI 味会显得有点重。

### 通过版

```text
她在门口把伞收好，站了会儿才开口：“哥，钱我不跟你要了，真不要了。”

表哥刚松一口气，她看向桌边那个盒子。

“但我妈那个镯子，你得给我。”
```

### 差异分析

1. 错误倾向把“克制”误解成断裂，读起来像故意做留白。
2. 通过版的句子并不短，但有口语拖带和情绪转向。
3. 先说钱不要，再说镯子要，人物策略比单纯短句更重要。

### 可迁移规则

自然口语常常是“自然拖带 + 半句转折 + 具体指向”，不是一串短句。短句只能在角色真的说不下去、怕露怯、被打断或情绪断线时使用。

## 案例 4：功能性动作不要直球报功能

场景：职场对峙，角色手里可能有录音或证据。  
问题类型：动作直接暴露剧情机关，像作者在向读者说明功能。

### Agent 初稿问题句

```text
陈屿没急着接，先把手机录音关了。
```

### 用户反馈

句子有点直球，过于简洁了当，不太像人写的。

### 更自然方向

```text
陈屿没去碰那份合同，拇指在手机侧边停了一下，最后把屏幕朝下扣在文件夹旁。
```

或者：

```text
陈屿看了眼亮起的手机屏幕，没解释，只把它往文件夹下面推了半寸。
```

### 差异分析

1. `把手机录音关了`是在报告功能，不是在写人。
2. 改法保留了读者能感知到的遮掩动作，但不立刻说明“这是录音”。
3. `没去碰合同`和`手机扣下`同时服务人物态度和后续证据回收。

### 可迁移规则

录音、截图、证据、转账、监控、合同等功能性动作，先写成角色自然会做的小动作。除非剧情要求读者立刻知道，否则不要直写“打开录音”“关掉录音”“截图保存”。

## 案例 5：合同数字与附件二

场景：职场对峙，老板试图用合同数字发难。  
问题类型：原句能推进剧情，但老板问法太准，主角反击太直，不够像真实办公室里的试探和反压。

### Agent 初稿

```text
老板把合同推回来：“小陈，这数不对吧？”

陈屿没接那份合同，拿起杯子喝了一口，才说：“刘总，要不您再看看附件二。”

老板皱了下眉：“附件二怎么了？”

“我上午补过一版。”陈屿把杯子放回去，“您签的是旧的。”
```

### 用户改稿

```text
老板把合同推回来：“小陈，你瞅瞅这数好像不太对？” 说完盯着坐在对面的陈屿。

陈屿没接那份合同，只拿起杯子喝了一口，才说：“刘总，要不您再看看附件二。”

老板皱了下眉：“附件二怎么了？”

“我上午补过一版数据，”陈屿把杯子放回桌上，接着说到“您可能没注意到，您签的是旧的。”
```

### 差异分析

1. `这数不对吧`太像作者直接让老板发难；`你瞅瞅这数好像不太对`更像上位者装作随口一问。
2. `说完盯着坐在对面的陈屿`不花，但有现场，压迫感来自具体动作。
3. `只拿起杯子喝了一口`比普通喝水更强调陈屿故意不碰合同。
4. `我上午补过一版数据`比`我上午补过一版`更贴近工作语境。
5. `您可能没注意到，您签的是旧的`保留表面客气，实际完成反压。

### 可迁移规则

职场对峙里，上位者常常不会一上来直接定罪，而是用“你瞅瞅”“好像”“是不是你跟的”来试探和甩锅。主角反击也不要像法庭陈述，要带一点客气、一点绕、一点让对方自己踩进去的空间。

## 案例 6：客户报价甩锅

场景：客户报价出了问题，领导想确认责任归属。  
问题类型：原句过于平直，像审讯台词；用户改稿增强了动作力度、自保语气和职场绕法。

### Agent 初稿

```text
刘总把打印出来的表往桌上一放：“这个客户，昨天不是你跟的吗？”

陈屿看了一眼，没急着说话，先把手机上的消息划掉。

“是我跟的。”他说，“不过报价是下午三点改的，我那会儿已经去仓库了。”

刘总抬头看他：“谁改的？”

陈屿顿了顿：“您让我问，还是让我现在说？”
```

### 用户改稿

```text
刘总把打印出来的表往桌上一甩：“这个客户，昨天是你跟的吗？”

陈屿看了一眼，没急着说话，先把手机上的消息划掉。

“是我跟的。”他说，“不过报价是下午三点改的，和我可没关系，我那会儿已经去仓库了。”

刘总抬头看着他，声音里夹着一点怒气：“谁改的？”

陈屿顿了顿，微微笑了一下：“您是让我去问问，还是让我现在就说？”
```

### 差异分析

1. `往桌上一放`动作太平；`往桌上一甩`直接带出领导的不满和甩锅压力。
2. `昨天不是你跟的吗`像审讯；`昨天是你跟的吗`更像真实职场里先确认责任归属。
3. `我那会儿已经去仓库了`只是解释；`和我可没关系，我那会儿已经去仓库了`先自保再解释，更像被甩锅的人会说的话。
4. `刘总抬头看他`信息不足；`声音里夹着一点怒气`补出压迫，不需要写“气氛紧张”。
5. `您让我问，还是让我现在说`略像金句；`您是让我去问问，还是让我现在就说`多了职场表面客气和一点绕，更像人说话。

### 可迁移规则

职场对白要保留礼貌外壳和实际压力。老板不是法官，主角也不是侦探；他们都要在办公室关系里绕半句、留半句，这样才有人味。

## 审稿快查表

| 命中问题 | 常见 AI 写法 | 优先改法 |
| --- | --- | --- |
| 散文化物象 | 水滴砸在门槛上、风把沉默吹散 | 改成人物动作、站位、手里物件 |
| 审判腔对白 | 你必须把东西还我 | 加称呼、指代、半句转折和口语颗粒 |
| 短句切碎 | 我不要了。镯子给我。我妈的。 | 恢复自然拖带，保留说话人的气口 |
| 功能动作直球 | 他关掉录音、她截图保存 | 改成自然遮掩动作，后文回收功能 |
| 模板反应 | 屋里静了两秒、笑容僵住 | 改成人物找补、继续装正常或反问 |
| 审讯台职场戏 | 谁改的？我知道是谁。 | 改成试探、甩锅、自保、客气反压 |

## 案例 7：报价追问与第三人指向

场景：客户报价出错，领导继续追问“你不知道？”  
问题类型：主角容易过早显得掌控证据链，导致对白像侦探摊牌，而不像办公室里的自保和递压力。

### Agent 初稿

```text
刘总盯着他：“不知道？你跟的客户，你不知道？”

陈屿把报价单转回来，看了眼页脚：“我只知道我发出去那版是九八。这个一零二，是后来改的。”
```

### 用户改稿

```text
刘总盯着他：“不知道？你跟的客户，你不知道？”

陈屿把报价单转回来，看了眼页脚：“我只知道我发出去那版是九八。这个一零二，明显是后来改的。”
他敲了敲脑门，眼神瞟向刚刚走进公司的宋丁，示意到：“要不您去问问他。”
```

### 差异分析

1. Agent 初稿只完成了“证明这版不是我发的”，但没有把现场关系继续往前推。
2. 用户改稿里的`明显是后来改的`更像人在被追问时加强语气，不是冷冰冰陈述事实。
3. `敲了敲脑门`让陈屿不是立刻点名，而是做出一个像在想、又像故意提醒的动作。
4. `眼神瞟向刚刚走进公司的宋丁`把压力递给第三人，场面从二人对峙变成办公室现场，不再像单线审讯。
5. `要不您去问问他`比直接说“是宋丁改的”更自然，保留了职场里的遮掩和反压。

### 可迁移规则

1. 主角不要过早像侦探一样摊完整证据链。先说自己知道的边界，再把压力递给现场里的相关人。
2. 指向第三人时，不要直接判罪；用眼神、停顿、随口建议和“要不您问问他”这类话，让对方自己接住。
3. 办公室对峙要尽量增加现场感：有人进门、有人看表、有人停手，场面才不像两个人在念台词。
## 案例 8：女频现实向亲情压迫

场景：女主发现自己寄回家的钱被母亲转给弟弟。  
问题类型：Agent 初稿把委屈写成了利落反问，用户改稿保留了话说到一半、母亲不耐烦和现实困境，更像真实家庭冲突。

### Agent 初稿

```text
许晚把转账记录递过去：“妈，这三千不是给他的。”

母亲看了一眼，又把手机推回来：“你弟那边急用，先周转一下。”

许晚没吵，只问：“那我房租呢？我也先周转一下？”
```

### 用户改稿

```text
许晚把转账记录递过去：“妈，这三千不是给他的。你怎么……”

母亲看了一眼，又把手机推回来，有点不耐烦道：“你弟那边急用，先周转一下怎么了 。你作为姐姐帮帮弟弟不是应该的吗？”

许晚沉默了一会儿：“我连房租都付不起了，你们想考虑过吗。”
```

### 差异分析

1. Agent 初稿的`那我房租呢？我也先周转一下？`太利落，像精心设计的反击句，委屈不够真实。
2. 用户改稿里的`你怎么……`保留了话说不下去的状态，人物不是马上组织好语言。
3. 母亲的台词更长、更不耐烦，带出家庭压迫常见的道德绑架：`你作为姐姐帮帮弟弟不是应该的吗？`
4. `许晚沉默了一会儿`比直接反问更贴近现实冲突，委屈先压住，再说出真实困境。
5. `我连房租都付不起了`让冲突落到生存压力，不只是讲道理。

### 可迁移规则

1. 女频现实向亲情冲突，不要把女主写得太会反击。真实委屈常常是话说到一半、沉默一下，再挤出一句现实困境。
2. 家庭压迫方的台词可以更长、更烦、更理所当然，不要为了简洁删掉道德绑架的黏腻感。
3. 反击句不要只追求爽，要先让读者看到角色为什么难受、为什么说不出口。
## 案例 9：玄幻考核现场的热闹感

场景：宗门考核，测灵石裂开，师兄想把责任推给主角。  
问题类型：Agent 初稿过于平直，把玄幻考核写得像普通问责；用户改稿补出了执事惊慌、旁人作证和主角当众回应的现场感。

### Agent 初稿

```text
执事看着裂开的测灵石：“刚才谁碰过？”

旁边的师兄立刻说：“周临碰过。”

周临看了他一眼：“我是碰过。可我碰的时候，它还没裂。”
```

### 用户改稿

```text
执事看着裂开的测灵石，眼珠子都瞪出来了：“这……这个，刚才谁碰过？”

旁边的师兄立刻说：“周临碰过，我们都亲眼看见的。”

周临看了他一眼，往前站了一步，抬起头说到：“我是碰过。可我碰的时候，它可还没裂。”
```

### 差异分析

1. Agent 初稿过于冷静，缺少玄幻考核现场该有的夸张反应和围观压力。
2. `眼珠子都瞪出来了`虽然直白，但很适合爽文现场，比克制描写更有热闹感。
3. `这……这个`让执事的震惊进入台词，而不是旁白说明。
4. `我们都亲眼看见的`比`周临碰过`更像甩锅者拉群众作证，压力更大。
5. `往前站了一步，抬起头`让主角不是坐着解释，而是当众接招。
6. `它可还没裂`里的`可`有口语力度，比纯陈述更像当场反驳。

### 可迁移规则

1. 玄幻升级流的考核、擂台、宗门现场，不要写得过分克制；围观、惊呼、作证、站出来这些动作能制造爽文现场感。
2. 配角甩锅时，可以让他多拉一句“我们都看见了”，把个人指控变成群体压力。
3. 主角反击不一定要金句，先让他在众目睽睽下站出来，气势就有了。
## 案例 10：甜宠轻喜的可爱夸张和暧昧缓冲

场景：男主嘴硬说只是顺路送女主。  
问题类型：Agent 初稿有轻松感，但互动太直，缺少甜宠轻喜里常见的可爱夸张、动作缓冲和语气弹性。

### Agent 初稿

```text
林小满看了眼导航：“你不是说顺路吗？”

程述没看她：“嗯，绕一点也是顺。”

“绕了二十公里。”

他停了两秒：“那可能是我方向感不好。”
```

### 用户改稿

```text
林小满看了眼导航，地图上的路线像是画了个大问号：“你不是说顺路吗？”

程述没看她，捏了捏方向盘，随口说到：“嗯，只是绕了一点点。”

“你确定是一点点？我看是绕了二十公里吧。”

他停了两秒，微微一笑：“那可能是我方向感不好吧。”
```

### 差异分析

1. `地图上的路线像是画了个大问号`带出轻喜感，夸张但贴合导航场景。
2. `捏了捏方向盘`比单纯`没看她`更有嘴硬和心虚的小动作。
3. `只是绕了一点点`比`绕一点也是顺`更像嘴硬狡辩，也更可爱。
4. `你确定是一点点？我看是绕了二十公里吧`保留女主吐槽，不是冷冰冰报数字。
5. `微微一笑`和句尾`吧`让男主的回答不那么硬，形成暧昧缓冲。

### 可迁移规则

1. 甜宠轻喜的对白要有“可爱夸张 + 小动作 + 句尾缓冲”，不要只靠聪明反问。
2. 男主嘴硬时，不要写成强辩；让他有一点被看穿后的松动，甜感才出来。
3. 女主吐槽要像顺口接话，适当加`你确定`、`我看`、`吧`这类口语颗粒。
## 案例 11：悬疑刑侦的审讯现场感

场景：证人声称自己没进过房间，但门把手照片暴露了问题。  
问题类型：Agent 初稿只完成了线索揭示，像谜题问答；用户改稿加入烟、灯光、嗓音和盯视，让场景变成审讯现场。

### Agent 初稿

```text
老周把照片放到桌上：“你说你没进过屋。”

男人看了一眼：“是没进过。”

老周点了点门把手那块：“那这枚油印，怎么在里面这一侧？”
```

### 用户改稿

```text
老周把照片放到桌上：“你说你没进过屋？”语气很随意。
说完他点了支烟，一缕烟气在昏暗的灯光下直直升起。

男人看了一眼，没有任何表情：“是没进过。”沙哑的声音像是有一口老痰卡在喉咙。

老周猛吸了一口烟，点了点照片上门把手那块：“那这枚油印，怎么在里面这一侧？”
说完他把烟掐了，死死的盯着男人的眼睛。
```

### 差异分析

1. Agent 初稿太干净，只像在展示推理点，缺少审讯室的空气和压力。
2. `语气很随意`和后面的点烟形成反差，老周不是急着逼问，而是在等对方自己稳不住。
3. `一缕烟气在昏暗的灯光下直直升起`给场景加了视觉锚点，让读者进入审讯空间。
4. `没有任何表情`和`沙哑的声音像是有一口老痰卡在喉咙`让证人有了身体质感，不只是“男人回答”。
5. `猛吸了一口烟`、`把烟掐了`、`死死的盯着男人的眼睛`把线索揭示变成压迫动作，读者能感觉到审问开始收网。
6. 悬疑刑侦可以比职场戏更允许氛围描写，但氛围必须服务审讯压力，不能只为了好看。

### 可迁移规则

1. 悬疑刑侦不要只写“问题 + 回答 + 证据揭示”，要写出审讯室里的节奏：放松、停顿、试探、突然收紧。
2. 物件细节要参与压迫，比如烟、灯、照片、桌面、门把手，而不是单纯做装饰。
3. 证人不能只是提供信息的嘴，要有表情、嗓音、手部动作或身体反应。
4. 关键证据抛出前，可以先让侦查者显得随意；证据抛出后，再用动作和视线收紧场面。
## 案例 12：轻松日常异能的异常发现节奏

场景：主角第一次发现自己能听见猫说话。  
问题类型：Agent 初稿让主角太快接受异常，缺少普通人发现怪事时的检查、怀疑和反应过程。

### Agent 初稿

```text
楼下那只橘猫蹲在垃圾桶边，盯着沈嘉手里的包子。

沈嘉刚咬一口，就听见一个声音说：“别吃了，肉馅不新鲜。”

他停住，低头看猫。

橘猫舔了舔爪子：“看我干什么，我昨天就吐过。”
```

### 用户改稿

```text
楼下那只橘猫蹲在垃圾桶边，盯着沈嘉手里的包子。

沈嘉刚咬一口，就听见一个声音说：“别吃了，肉馅不新鲜。”

他停住，扫视了一下四周，空无一人。
眼神疑惑地转向了旁边的橘猫，歪了歪头，好像在问：是你在说话？

橘猫舔了舔爪子：“嗯，你没听错，是我说话，这包子我昨天就吃过，之后就吐了。”
```

### 差异分析

1. Agent 初稿里`低头看猫`太快，主角像立刻知道猫会说话，少了第一次遭遇异常的真实反应。
2. 用户改稿先让沈嘉`扫视了一下四周，空无一人`，这一步很重要，普通人会先找人，不会直接认定猫开口。
3. `眼神疑惑地转向了旁边的橘猫，歪了歪头`让主角的怀疑变成可见动作。
4. `好像在问：是你在说话？`虽然直白，但在轻松日常里能帮助读者迅速进入喜剧误会。
5. 橘猫直接说`嗯，你没听错，是我说话`，把荒诞感落地；猫不神秘，反而理直气壮，轻松感更强。
6. `这包子我昨天就吃过，之后就吐了`比单纯`我昨天就吐过`信息更完整，也更好笑。

### 可迁移规则

1. 第一次出现异能、怪事或非现实设定时，不要让主角立刻接受。先写普通人的检查：看四周、找人、怀疑自己、再看异常对象。
2. 轻松日常里的非人角色不要装神秘，越理直气壮、越像普通邻居吐槽，喜剧感越强。
3. 荒诞设定要用生活细节落地，比如包子、垃圾桶、昨天吃过、吐了，而不是只写“震惊”。
## 案例 13：古言宅斗的礼数、吞吐和见证压力

场景：嫡母当众扣下庶女月例银，庶女借祖母和嬷嬷反压。  
问题类型：Agent 初稿虽然有信息反击，但太直接，缺少宅斗场景里的礼数、吞吐、被逼着说和旁人见证。

### Agent 初稿

```text
夫人把账册合上：“你院里这个月用度不少，月例先缓一缓。”

沈知棠低头行礼：“母亲说的是。只是祖母那边昨日还问，女儿怎么连药钱都支不出来。”

屋里几个嬷嬷都不说话了。
```

### 用户改稿

```text
夫人把账册合上：“你院里这个月用度不少，月例先缓一缓。”

沈知棠低头行礼：“母亲说的是。只是…………”
“只是什么？”
“既然母亲问了，我也不便瞒着。祖母那边昨日还问，女儿怎么连药钱都支不出来。”沈知棠擦了擦眼角，眼神瞟向旁边的几个嬷嬷。

屋里几个嬷嬷都低着头不敢说话。
```

### 差异分析

1. Agent 初稿让沈知棠直接把祖母抬出来，反击太顺，不像宅斗里谨慎递话。
2. 用户改稿先用`只是……`吞住半句，制造“不是我要说，是你逼我说”的姿态。
3. `只是什么？`让嫡母主动追问，等于自己给了沈知棠开口的台阶。
4. `既然母亲问了，我也不便瞒着`很关键，表面守礼，实际把责任推回对方。
5. `擦了擦眼角`和`眼神瞟向旁边的几个嬷嬷`把委屈和见证压力同时写出来。
6. `低着头不敢说话`比单纯`不说话了`更符合内宅等级关系，嬷嬷知道话不该接。

### 可迁移规则

1. 古言宅斗的反击不要太直，要先做礼数、吞吐和推辞，让对方主动追问。
2. “既然您问了”“我也不便瞒着”“原不该说”这类话，可以制造表面恭顺、实际反压。
3. 宅斗场面要利用旁人见证：嬷嬷、丫鬟、婆子、账房都能成为压力来源。
4. 情绪动作不要过猛，擦眼角、低头、瞟一眼，往往比大哭大怒更符合内宅戏。
## 案例 14：末世生存的直接压力和自保反应

场景：小队物资被偷，队长怀疑新人。  
问题类型：Agent 初稿能交代冲突，但压力推进太平；用户改稿补出了扫视、确认式问话和新人慌张解释。

### Agent 初稿

```text
赵衡把空背包扔到地上：“谁动过？”

没人说话。

他看向新来的男孩：“你昨晚守后门。”

男孩脸白了：“我没拿。我连压缩饼干放哪儿都不知道。”
```

### 用户改稿

```text
赵衡把空背包扔到地上：“谁动过？”

扫视了一圈，没人说话。

他看向新来的男孩：“你昨晚守后门，是吧。”

男孩脸唰的一下白了，连忙解释道：“我……我没拿。我连压缩饼干放哪儿都不知道。”
```

### 差异分析

1. `扫视了一圈`让队长的压迫变成动作，不只是旁白说没人开口。
2. `你昨晚守后门，是吧`比`你昨晚守后门`更像逼对方承认事实，压力更足。
3. `脸唰的一下白了`虽然直白，但符合末世生存里高压下的身体反应。
4. `连忙解释道`和`我……我没拿`让新人有恐惧感，不像冷静陈述。
5. 末世场景里，物资就是命，人物反应可以更直接、更急，不必过度含蓄。

### 可迁移规则

1. 末世生存的紧张感来自资源压力和即时后果，句子可以直，但动作和反应要跟得上。
2. 逼问时使用确认式尾巴，比如`是吧`，能制造被迫承认的压力。
3. 被怀疑的人不要太稳，尤其是新人；结巴、急解释、脸色变化都能体现生存压力。
## 案例 15：青春暗恋先理清场面逻辑

场景：男生每天提前替女生占图书馆靠窗位置。  
问题类型：Agent 初稿动作链不清，读者看不懂笔记本是谁的、座位是谁占的、男生问话的对象和意图。用户改稿先补清逻辑，再保留暗恋嘴硬。

```text
晚自习前，林栀又在靠窗的位置看见那本蓝色笔记本。

她坐下没多久，周野拎着水从旁边经过：“没人吧？”

林栀看着那本本子：“你每天都放这儿？”

周野脚步没停：“顺手。”
```

### 用户反馈

这个片段逻辑不通顺，看不懂。

```text
林栀到图书馆时，靠窗的位置还空着，桌角压着一本蓝色笔记本。
```

```text
林栀到图书馆时，靠窗的位置还空着，桌角放着一本蓝色笔记本。
```

2. 用户改稿用`这些天几乎每天都在那儿`补出连续性，让林栀能合理意识到周野一直在占座。
3. `今天来得挺准时`比`你来得挺准`更自然，像男生嘴硬时随口找的话。
4. `你每天都提前来？`比`你每天都先来？`更准确，直接指向占座行为。

6. 青春暗恋的暧昧不是靠漂亮句子，而是靠“他做了很多，但嘴上不承认”。

1. 青春暗恋先保证动作链清楚：谁的东西、放在哪里、为什么能被认出来、对方为什么此刻出现。
2. 暧昧不能建立在读者看不懂的留白上。逻辑清楚之后，少说一点才是甜。
3. 嘴硬台词要配逃避动作，比如转身要走、低头收东西、把水放下就走。

## 案例 17：历史架空官场的迟疑、威压和具体数目

场景：小吏发现粮仓账册对不上，县丞想压下去。  
问题类型：Agent 初稿过于凝练，冲突成立但官场压迫和粮荒风险不够具体；用户改稿补出县丞迟疑、小吏为难、具体数目和威严语气。

### Agent 初稿

```text
县丞把账册合上：“这事先别往外说。”

顾砚看着桌上的米粒：“大人，仓里少的不是几袋米。”

县丞皱眉：“你想说什么？”

顾砚低声道：“再压两天，城东粥棚就该断锅了。”
```

### 用户改稿

```text
县丞把账册合上，想了想：“这事先别往外说。”

顾砚看着桌上的米粒，有一点为难道：“大人，仓里少的可不是几袋米，是整整五十担啊。”

县丞皱眉：“你想说什么？” 语气威严，不容置疑。

顾砚只好低声道：“再压两天，怕是城东的粥棚就该断锅了。”
```

### 差异分析

1. `想了想`让县丞不是机械压事，而是权衡后选择压下去，更像官场人物。
2. `有一点为难道`很重要，小吏不是硬顶上司，而是在职责和压力之间开口。
3. `可不是几袋米，是整整五十担啊`给出具体量级，读者立刻知道事情严重。
4. `语气威严，不容置疑`虽然直白，但在架空官场样本里能明确上下级压力。
5. `只好低声道`补出顾砚被压住后的姿态，比单纯低声更有被迫开口感。
6. `怕是`比直接`就该`更有古代口语感，也保留谨慎语气。

### 可迁移规则

1. 历史架空官场戏不要只追求凝练，要写出上级权衡、下级为难和等级压迫。
2. 民生风险需要具体数目支撑，比如五十担、三日粮、两仓空账；没有量级，危机容易发虚。
3. 小吏顶撞上官时，不要写成现代职场硬怼；用`为难`、`只好`、`怕是`保留身份压力。
## 案例 18：现代婚恋争执的生活动作和情绪外露

场景：妻子发现丈夫把共同存款借给前女友。  
问题类型：Agent 初稿太冷静、太像精致反问；用户改稿加入手机、扣屏、连忙解释、叉腰和连续语气，更像真实夫妻争执。

```text
许知夏把银行短信放到桌上：“这二十万，是怎么回事？”
```

```text
许知夏把手机放到桌上，屏幕上正显示的银行短信：“这二十万，是怎么回事？”

许知夏连连点头，手叉在腰上：“好好好，那你说我们下个月的房贷，怎么办？”
```

### 差异分析

1. Agent 初稿的`所以我们下个月的房贷，也先借一下？`太像设计好的冷笑话，不够像夫妻吵架。
2. `屏幕上正显示的银行短信`让证据可见，冲突不是抽象质问。
3. 丈夫`连忙把手机扣上`很有生活感，也暴露心虚。
4. `过段时间就还了`是典型现实争执里的搪塞话，不能省掉。
5. `连连点头`和`手叉在腰上`让许知夏的火气外露，比克制点头更像真实家庭场景。
6. `好好好，那你说……怎么办？`比精致反问更口语，也更有压不住的怒气。

### 可迁移规则

1. 现代婚恋争执不要过度冷感，尤其涉及钱、前任、房贷时，人物往往会有明显生活动作和情绪外露。
2. 证据要落在手机短信、转账记录、扣屏、聊天记录这类生活物件上。
3. 争吵台词要保留搪塞、重复和情绪词，比如`过段时间就还了`、`好好好`、`那你说怎么办`。
4. 精致反问适合少量使用，不能替代真实夫妻争执里的急、慌和火气。
## 案例 19：游戏竞技队麦的火气和稳队节奏

场景：排位赛关键团前，辅助误开团导致团灭，队友开始甩锅。  
问题类型：Agent 初稿队麦过于文明、冲突太轻；用户改稿加入爆粗、附和、辅助自知理亏和主角复盘后稳队，更像真实游戏竞技场景。

### Agent 初稿

```text
屏幕灰下去的那一秒，队麦里安静了一下。

打野先开口：“谁让你上的？”

辅助没说话。

林舟看着回放：“别吵。他技能交早了，但对面闪也没了。下一波还能打。”
```

### 用户改稿

```text
屏幕灰下去的那一秒，队麦里安静了一下。

打野先开口骂道：“MD！谁让你上的？会不会玩？！”
法师也附和道：“我真是服了。”

辅助没说话，他也知道这波团灭确实是自己的问题。

林舟闭着眼闪回了一下刚才这波团战。
睁开的眼睛依然坚定，他说到：“别吵。虽然他技能交早了，但对面闪也没了。下一波还能打。问题不大，别内讧。”
```

### 差异分析

1. Agent 初稿把队麦写得太克制，真实团灭后往往先有骂声和情绪宣泄。
2. `MD！谁让你上的？会不会玩？！`比单纯`谁让你上的`更符合排位赛火气。
3. `我真是服了`是常见队友附和，能让压力从单人指责变成队内情绪扩散。
4. `他也知道这波团灭确实是自己的问题`补出辅助沉默的原因，避免沉默变成空镜头。

6. `问题不大，别内讧`是竞技文很实用的稳队话，主角要解决的不只是战术，还有队伍心态。

### 可迁移规则

1. 游戏竞技队麦不要写得太文明；失误、团灭、掉点后会有骂声、附和和甩锅。
2. 主角稳队前要先有复盘动作或判断依据，不要直接输出正确结论。
3. 竞技文的爽感不只来自操作，还来自主角能压住队友情绪、抓住下一波机会。
4. 队内台词可以更短、更急、更口语，比如`会不会玩`、`我真是服了`、`问题不大`、`别内讧`。
## 案例 20：仙侠虐恋的追问、撑不住和照顾转场

场景：女主替男主挡雷劫后，男主以为她是为了宗门任务。  
问题类型：Agent 初稿停在一句克制痛感上，情绪有了但场面没有继续推进；用户改稿加入男主重复追问、女主撑不住和男主扶住，关系从误解转向照顾。

### Agent 初稿

```text
谢无尘看着她袖口的血：“宗门让你来的？”

云岫靠在石壁上，笑了一下：“你觉得是，那就是吧。”

他皱眉：“我问你话。”

云岫闭上眼：“别问了，我疼。”
```

### 用户改稿

```text
谢无尘看着她袖口的血：“宗门让你来的？”

云岫靠在石壁上，笑了一下：“你觉得是，那就是吧。”

他皱眉，又问了一遍：“是不是宗门让你来的？”

云岫闭上眼：“别问了，我疼。”说完作势要倒下。
谢无尘连忙过来扶住：“罢了，先回去养伤。”
```

### 差异分析

1. Agent 初稿的`我问你话`偏硬，像普通质问；用户改稿让男主重复原问题，执念更清楚。
2. `又问了一遍`让读者感到男主在意答案，不只是摆姿态发火。
3. `别问了，我疼`如果停在这里，会像漂亮虐句；加上`作势要倒下`后，身体伤势接住了台词。
4. `连忙过来扶住`让男主的真实关心露出来，不需要旁白解释他心疼。
5. `罢了，先回去养伤`符合仙侠语境，也把误解暂时压下，留下后续情绪账。

### 可迁移规则

1. 仙侠虐恋不要只停在一句克制痛感，台词后要有身体代价和关系动作承接。
2. 男主误解女主时，可以重复同一个问题，表现执念和不甘，比换成硬质问更有情绪。
3. 照顾动作要及时出现：扶住、输灵力、带回去、压住伤口，让虐感落到身体和关系上。
4. `罢了`、`先回去养伤`这类话能把当下冲突暂时封住，给后续误解发酵留空间。
## 案例 21：民俗悬疑的土味留白和半截话

场景：主角夜里守灵，发现棺材底下有新抓痕。  
问题类型：Agent 初稿有异常点，但民俗味和邪性不足；用户改稿通过念念有词、半截话和口语留白，让恐怖感更贴近乡土现场。

### Agent 初稿

```text
三叔把香灰往门口撒了一圈：“今晚谁都别睡死。”

陈砚蹲在棺材边，手电往下一照，木板底下全是新抓痕。

他抬头：“三叔，昨天不是说人走得挺安稳吗？”
```

### 用户改稿

```text
三叔把香灰往门口撒了一圈，嘴里念念有词：“今晚谁都别睡死，有的忙。”

陈砚蹲在棺材边，手电往下一照，发现木板底下全是新抓痕。

他抬头问到：“三叔，昨天不是说人走得挺安稳吗？看着不太像啊……”
```

### 差异分析

1. `嘴里念念有词`给三叔加了民俗操作感，不需要解释他念的是什么。
2. `有的忙`比单纯`别睡死`更有乡土老手的判断，也带一点“不明说”的邪性。
3. `发现木板底下全是新抓痕`比直接抛出结果稍慢一点，更符合手电照过去后的视觉发现。
4. `看着不太像啊……`不直接说“这人没死透”或“有问题”，而是留半截，让恐怖从话外冒出来。
5. 民俗悬疑的对白要像村里人压着声音说事，不能像侦探在公布线索。

### 可迁移规则

1. 民俗悬疑的恐怖感常来自半截话、老手判断和不解释的仪式动作。
2. 不要急着说破异常。用`有的忙`、`不太像啊`、`这事不对`这类土味口语留白，更有现场感。
3. 仪式细节不必全解释，香灰、门槛、手电、棺木、抓痕这些物件自己会带气氛。
## 案例 22：娱乐圈爽文的顺口阴阳

场景：女主被同剧组女二抢妆造，经纪人劝她忍一忍。  
问题类型：Agent 初稿有反讽，但略像设计好的机巧句；用户改稿更像化妆间里被气到后的顺口阴阳。

### Agent 初稿

```text
化妆间里，林乔的礼服被挂到了别人架子上。

经纪人压低声音：“她今天有品牌方来，先让她穿一次。”

林乔看着镜子里卸了一半的妆：“行啊，礼服让了，红毯也替她走吗？”
```

### 用户改稿

```text
化妆间里，林乔的礼服被挂到了别人架子上。

经纪人压低声音，，解释道：“她今天有品牌方来，先让她穿一次。”

```

### 可迁移规则

1. 娱乐圈、职场、婚恋这类现实场景里，阴阳怪气优先写成顺口话，不要过度打磨成机巧句。
2. `要不……得了`、`行啊`、`那干脆`这类口语结构，常比漂亮反问更有人味。
3. 经纪人、助理、品牌方等中间人要写出“解释/劝/压低声音”的姿态，冲突才有行业现场感。

```text
陆沉把滤芯拆下来，指尖蹭出一层黑灰。
```

```text
队长把名单贴到墙上，沈禾从头看到尾，没看见自己的名字。
```

## 案例 26：修仙轻松流的包袱要接真实压力

场景：灵宠偷吃丹房废丹，结果意外进阶。  
问题类型：Agent 初稿为了制造笑点，接了一个现场没有铺垫的动作前提；用户指出句子听着不正常，最终勉强通过“先别急着算账”。

### Agent 初稿

```text
丹房门口，黑毛鸡蹲在药渣堆里，打了个饱嗝。

管事脸都绿了：“它吃了三炉废丹？”

沈砚看着黑毛鸡头顶冒出来的一撮金毛，沉默了一下：“要不……先别让它吐？”
```

### 用户反馈

`“要不……先别让它吐？”`这个句子听不正常。

### 修正版 / 勉强通过版

```text
丹房门口，黑毛鸡蹲在药渣堆里，打了个饱嗝。

管事脸都绿了：“它吃了三炉废丹？”

沈砚看着黑毛鸡头顶冒出来的一撮金毛：“管事，要不您先别急着算账？”
```

### 差异分析

1. `先别让它吐`的问题是前文没人说要让鸡吐，动作前提凭空出现，读者会觉得怪。
2. 前一句真正的压力是`它吃了三炉废丹`，潜台词是赔丹、追责、算账。
3. `先别急着算账`顺着管事的压力来，虽然不算特别惊艳，但逻辑成立。
4. 轻松流包袱不能只追求意外，必须接住现场已经存在的矛盾。

### 可迁移规则

1. 包袱要接真实压力，不要凭空制造一个前文没有出现的动作或目标。
2. 修仙轻松流里，丹房、灵宠、废丹这类场景的天然压力通常是赔钱、责罚、算账、闯祸、意外收益。
3. 如果一句笑点读起来不正常，先问：前一句到底给了什么压力？这句有没有接住那个压力？
### 前提修复版

用户进一步给出修复方式：不是`先别让它吐`这句一定不能用，而是需要先补出动作前提。

```text
丹房门口，黑毛鸡蹲在药渣堆里，打了个饱嗝。

管事脸都绿了：“它吃了三炉废丹？也不怕毒死，干净让它吐出来。”

沈砚看着黑毛鸡头顶冒出来的一撮金毛，沉默了一下：“要不……先别让它吐？”
```

补充差异：

1. 管事先说`赶紧让它吐出来`，动作目标已经出现，后文`先别让它吐`就不再突兀。
2. 同一句台词能不能用，取决于前文有没有给它铺好压力和动作前提。
3. 轻松流包袱可以保留，但要么改台词接现有压力，要么补前文让包袱成立。

补充规则：

如果某句包袱“听着不正常”，不要只判断这句本身，要检查它缺的是不是前置条件。能补前提，就补前提；补不了，再换台词。
