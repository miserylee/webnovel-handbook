# 56. 卷间转场与新地图/新阶段启动专项

长篇时间跳跃与状态压缩入口：

- 如果需要写“几天后、几个月后、几年后”、压缩训练/赶路/调查/备赛/项目推进/伤后恢复，或卷间存在未写时间，先看 `docs/workflows/57-long-timeskip-state-compression.md`。该专题负责判断哪些内容能跳、哪些必须写成场景，并用状态差额、关系债、伏笔回响、对手变化和台账回写防止追读断裂。

## 1. 核心判断：卷间转场是承接，不是重开

下一卷开头不是新的第一章。它可以有新地图、新规则、新人物、新对手，但必须让读者感觉：

- 我还在看同一本书。
- 主角带着上一卷的成果和代价进入新阶段。
- 旧问题没有消失，而是换了更高层级的表现。
- 新地图不是背景板，而是立刻改变主角行动。
- 新卷第一单元有明确目标和回报方向。

一句话规则：

> 新卷第一场景必须带入旧卷至少两个资产，并让新阶段规则通过一个具体行动压力露面。

旧卷资产包括：

- 资源。
- 资格。
- 名声。
- 伤势。
- 债务。
- 人情。
- 关系变化。
- 秘密暴露。
- 对手仇恨。
- 活跃伏笔。
- 规则认知。
- 身份变化。

## 2. 进入新卷前的硬门槛

进入新卷启动前，必须先确认 `docs/advanced-craft/55-volume-ending-settlement-next-hook.md` 的卷末到下一卷交接包已经完成。

如果以下任一项不成立，不要直接写新卷第一章：

- 本卷目标没有回答。
- 主角状态变化不清。
- 支线和伏笔没有清账。
- 下一卷目标候选不清。
- 下一卷第一场景触发器不清。
- 下一卷和本卷结果没有因果关系。

允许进入新卷的最低条件：

- 有卷末结果。
- 有下一卷行动压力。
- 有下一卷第一场景触发器。
- 有必须带入的新旧资产清单。
- 有第一单元回报候选。
- 有不能空降的内容清单。

## 3. 新卷启动五件套

### 3.1 旧卷资产带入

新卷第一场景至少带入两个旧卷资产。

带入方式：

- 资源被要求兑现。
- 资格带来新责任。
- 名声引来审视或挑战。
- 伤势影响行动。
- 人情债被索取。
- 关系变化改变站位。
- 伏笔在新地图出现回响。
- 对手损失后开始新反制。

### 3.2 新阶段目标

新卷开头必须让读者知道这一阶段要看什么。

目标要具体：

- 完成一次考核。
- 通过新地图入场门槛。
- 接下新项目或案件。
- 修复或升级一段关系。
- 守住刚拿到的资源。
- 追查真相半开的高层线索。
- 对抗上一卷对手的规则反扑。

不要写成：

- 继续变强。
- 去新地方看看。
- 等幕后人出现。
- 先适应新环境。

### 3.3 新规则压力

新地图或新阶段必须有规则差异。

规则差异要通过行动出现，不要先百科解释。

有效展示：

- 主角按旧规则行动，立刻碰壁。
- 旧资源在新地图贬值或被重新定价。
- 旧名声在新圈层变成负担。
- 新组织要求主角交换、站队或担责。
- 新地图对能力、身份、资源或关系有门槛。

### 3.4 第一单元阻力

新卷不能只给新目标，还要给第一单元阻力。

阻力可以是：

- 入场考核。
- 审查。
- 竞争者。
- 资源门槛。
- 关系条件。
- 规则限制。
- 舆论误读。
- 对手残余势力。

### 3.5 第一单元回报

新卷前 3-8 章内必须有一个小回报方向。

回报可以是：

- 拿到正式入场资格。
- 通过第一关。
- 建立新关系交换。
- 识别新对手打法。
- 发现旧伏笔在新地图的新含义。
- 让旧资产在新规则里找到新用法。

## 4. 新卷前五章建议拍表

这里的“前五章”指下一卷开头的第 1-5 章，不是全书第 1-5 章。

| 新卷章节 | 核心任务 | 必须出现 | 禁忌 |
| --- | --- | --- | --- |
| 新卷第 1 章 | 承接卷末余波，触发新阶段 | 旧卷资产、行动压力、新目标苗头 | 重新介绍主角或世界 |
| 新卷第 2 章 | 进入新规则，不灌设定 | 旧方法碰壁或被重定价 | 纯地图说明 |
| 新卷第 3 章 | 第一阻力正面出现 | 新对手/规则/门槛影响行动 | 新人物只负责讲解 |
| 新卷第 4 章 | 主角调整策略 | 用旧资产找新打法 | 直接开新外挂 |
| 新卷第 5 章 | 第一单元小回报和更大压力 | 小胜/小败/代价胜，明确后续目标 | 五章仍在适应环境 |

因果链：

```text
旧卷结算
-> 余波触发新目标
-> 新地图/新规则压主角
-> 第一阻力显形
-> 主角用旧资产换新策略
-> 第一单元小回报
-> 新卷卷目标进一步清楚
```

## 5. 新地图启动规则

新地图不是风景，也不是设定说明书。新地图必须给行动增加差异。

新地图至少回答：

- 这里和旧地图的权力结构有什么不同？
- 主角旧资源在这里值不值钱？
- 这里的入场门槛是什么？
- 这里的对手如何识别主角？
- 这里有什么不能照旧解决的问题？
- 这里能产出什么新回报？

推荐写法：

- 先写主角带着旧卷结果进入新场景。
- 立刻让一个规则或人物让主角旧经验失效。
- 通过冲突展示新地图，而不是通过旁白说明。
- 用第一单元回报让读者理解新地图价值。

坏写法：

- 新地图开头连续介绍地理、组织、等级、历史。
- 主角到了新地方，前文人物和资产暂时消失。
- 新地图只是换皮，冲突逻辑不变。
- 新地图过大，第一单元目标过虚。

## 6. 旧资产重新定价

卷间转场最有价值的动作，是让旧资产在新阶段被重新定价。

| 旧资产 | 新阶段可能变化 | 好处 |
| --- | --- | --- |
| 资源 | 在新地图能换入场券，也可能被觊觎 | 旧回报不浪费 |
| 名声 | 在旧圈层有用，在新圈层变成审查对象 | 避免无脑一路爽 |
| 资格 | 带来新责任和更高门槛 | 让胜利继续产生压力 |
| 关系 | 旧盟友提供帮助，也带来牵连 | 让人物关系不断线 |
| 秘密 | 在新阶段更容易暴露 | 增加长期压力 |
| 伤势/代价 | 影响新阶段行动 | 让卷末代价不作废 |
| 伏笔 | 在新地图出现新含义 | 让世界变厚 |
| 对手仇恨 | 残余势力反制 | 让对手线延续 |

检查问题：

- 上一卷最重要的三个结果，在新卷开头是否有用？
- 上一卷最贵的代价，在新卷开头是否仍然影响行动？
- 上一卷最重要的关系变化，在新卷开头是否改变站位？

## 7. 新卷开头的信息投放

新卷开头需要新信息，但不能灌设定。

信息投放顺序：

1. 先给行动压力。
2. 再给主角不理解或误判的规则。
3. 让规则造成小后果。
4. 用人物冲突补必要信息。
5. 在第一单元回报后再补结构性解释。

允许解释：

- 当前行动必须知道的信息。
- 主角判断错误会付代价的信息。
- 读者理解爽点或危机必须知道的信息。

暂缓解释：

- 完整历史。
- 全部势力表。
- 全地图等级。
- 所有新人物背景。
- 未来很久才用的规则。

## 8. 时间跳跃和转场压缩

卷间可以跳时间，但不能跳掉因果。

允许时间跳跃：

- 旧卷余波已经交接。
- 关键代价没有被跳没。
- 新卷第一场景能显示这段时间带来的变化。
- 台账记录了期间发生的状态变化。

不允许时间跳跃：

- 用跳时间逃避支线清账。
- 用跳时间抹掉关系冲突。
- 用跳时间让伤势、债务、舆论、对手反制自动消失。
- 用跳时间直接换到新地图，旧卷资产断线。

时间跳跃后必须补：

- 主角状态。
- 关系状态。
- 资源状态。
- 对手状态。
- 伏笔状态。
- 新阶段行动压力。

## 9. 新卷第一场景触发器

下一卷第一场景不能泛泛开始。它需要触发器。

触发器类型：

| 类型 | 示例 | 适用 |
| --- | --- | --- |
| 资格触发 | 主角凭上一卷资格进入考核/组织/项目 | 升级、职业、玄幻 |
| 代价触发 | 上一卷代价到期，必须处理 | 复仇、情绪线、权谋 |
| 对手触发 | 对手残余或背后势力反制 | 爽文、都市、权谋 |
| 关系触发 | 关系质变后必须共同承担 | 女频、群像 |
| 真相触发 | 真相半开指向新地点/人物 | 悬疑、系统、世界观 |
| 资源触发 | 新资源被争夺、要求兑现或出现限制 | 经营、升级、行业 |
| 规则触发 | 新权限、新身份或新制度开始约束主角 | 系统、职业、组织 |

触发器必须包含：

- 旧卷来源。
- 当前行动压力。
- 本章结果方向。

## 10. 题材对应表

| 类型 | 新阶段启动重点 | 第一单元阻力 | 高危错误 |
| --- | --- | --- | --- |
| 玄幻升级 | 新资格、新地图、新资源层级 | 考核、门槛、资源争夺 | 新地图等级百科 |
| 都市逆袭 | 项目、资本、家族、舆论升级 | 合同、审查、竞品、人脉 | 旧胜利被忘记 |
| 女频情绪线 | 关系质变后的现实压力 | 家族、身份、旧伤、共同任务 | 关系重置成暧昧初期 |
| 重生复仇 | 旧节点改变后的新时间线 | 蝴蝶效应、提前行动 | 前世信息继续全知 |
| 悬疑探案 | 阶段真相指向新案件层级 | 证据缺口、证人、规则限制 | 只开新案不接旧案 |
| 系统机制 | 新权限和新限制同时出现 | 冷却、代价、观察者、误判 | 只发新功能 |
| 职业行业 | 更高项目和更高责任 | 客户、流程、事故、竞品 | 行业规则只作背景 |
| 群像团队 | 团队重组或新任务 | 分工、利益、信任、外部压力 | 队伍关系回到零 |

## 11. 常见失败模式

### 11.1 新卷重新开书

表现：

- 重新介绍主角。
- 重新介绍世界。
- 旧卷人物、资源、代价暂时消失。
- 新卷目标和旧卷结果无关。

修法：

- 第一场景带入两个旧资产。
- 用旧卷结果触发新目标。
- 先写行动压力，再补新阶段信息。

### 11.2 换地图灌设定

表现：

- 连续解释地理、势力、等级、历史。
- 主角没有行动压力。
- 读者不知道为什么要继续看。

修法：

- 用冲突展示规则。
- 先给入场门槛。
- 让旧方法碰壁。
- 五章内给第一单元小回报。

### 11.3 旧代价自动消失

表现：

- 上一卷伤势、债务、秘密、关系裂痕、新仇恨都没影响新卷。

修法：

- 写旧代价在新地图的新后果。
- 让旧代价影响第一单元选择。
- 台账中保留代价状态。

### 11.4 新敌人空降

表现：

- 新敌人和旧卷没有因果。
- 新敌人只靠头衔压人。

修法：

- 让新敌人因旧卷结果注意主角。
- 或让新敌人掌握新地图规则，卡主角入场。
- 或让旧对手残余势力引出新敌人。

### 11.5 新卷目标过虚

表现：

- “探索新世界。”
- “继续变强。”
- “寻找真相。”

修法：

- 改成第一单元可验证目标。
- 写出失败代价。
- 设计前 3-8 章小回报。

## 12. 质量分级

### 高质量

- 新卷第一场景承接旧卷结果。
- 至少两个旧资产被带入。
- 新阶段目标具体。
- 新规则通过行动压力展示。
- 第一单元阻力明确。
- 前 3-8 章有小回报方向。
- 旧代价没有消失。
- 新地图、新敌人或新阶段和旧卷有因果。

### 可用但需加强

- 新卷承接旧卷，但旧资产只带入一个。
- 新目标清楚，但第一单元回报偏虚。
- 新规则出现，但主要靠解释。
- 旧代价存在，但影响行动不够。

### 不合格

- 新卷像重新开书。
- 换地图先灌设定。
- 旧卷资产断线。
- 下一卷目标空降。
- 新敌人和旧卷无因果。
- 五章后仍不知道新阶段要争什么。

不合格时，不要继续批量写新卷。先重做卷间转场。

## 13. 新卷启动闸门

新卷开头 3-5 章完成后，必须判断是否允许继续批量推进。

允许继续的条件：

- 旧卷至少两个资产已经进入新卷行动。
- 新阶段目标清楚。
- 新规则已经通过冲突展示。
- 第一单元阻力已经出现。
- 旧代价至少影响一次选择。
- 第一单元小回报方向明确。
- 新卷和旧卷存在因果连续。

暂停修订的信号：

- 新卷第一章像新书第一章。
- 新地图只靠说明。
- 旧卷资产没有出现。
- 主角没有行动压力。
- 新卷目标说不清。

闸门结论模板：

```markdown
新卷启动闸门：

- 旧卷带入资产：
- 旧卷带入代价：
- 新阶段目标：
- 新规则压力：
- 第一场景触发器：
- 第一单元阻力：
- 第一单元小回报候选：
- 旧卷和新卷因果：
- 结论：可继续新卷 / 先补旧资产 / 先补新规则压力 / 先补第一单元目标 / 先删设定灌输 / 重做卷间转场
```

## 14. AI agent 执行流程

### 14.1 设计新卷前

先读取：

- `docs/advanced-craft/55-volume-ending-settlement-next-hook.md`
- `docs/advanced-craft/53-after-30-first-volume-structure.md`
- `docs/storycraft/28-worldbuilding-setting-exposition.md`
- `docs/core-writing/07-continuity-ledger.md`

再填：

- 卷末到下一卷交接包。
- 旧卷必须带入资产。
- 旧卷必须保留代价。
- 新阶段目标。
- 第一场景触发器。
- 新规则压力。
- 第一单元回报候选。

### 14.2 起草新卷开头时

每章先写任务句：

```text
本章任务：因为【旧卷结果/代价】，主角带着【旧资产】进入【新阶段目标】，但【新规则/新阻力】让旧方法受限，本章结尾落下【新状态/小回报/下一章钩子】。
```

### 14.3 审稿新卷开头时

按顺序检查：

1. 第一场景是否承接旧卷。
2. 是否带入至少两个旧资产。
3. 旧代价是否影响行动。
4. 新阶段目标是否具体。
5. 新规则是否通过冲突展示。
6. 是否有第一单元阻力。
7. 是否避免地图百科。
8. 五章内是否有小回报方向。
9. 新卷是否和旧卷有因果连续。

- 新地图或新规则是否通过冲突展示？
- 主角旧方法是否遇到限制或重定价？
- 是否避免连续地图百科？
- 新卷前 3-8 章是否有小回报方向？

- 是否没有把关系线重置？

## 17. 来源与迁移说明

本专题吸收的是通用叙事资料、系列续作资料、场景转场资料和网文连载经验，经中文网文连载场景重组后使用。

- Helping Writers Become Authors 关于 scene-sequel、reaction、dilemma、decision 的资料：上一场景结果应自然推动下一场景目标，迁移为卷末余波触发新卷第一场景。
- Helping Writers Become Authors 关于系列结构和续作开篇的资料：续作或新阶段既要承接既有角色与代价，也要快速建立新目标，迁移为“承接，不重开”。
- Story Grid 关于 scene、progressive complication、crisis、climax、resolution 的资料：新阶段第一场景也必须有目标、冲突、选择、结果，迁移为新卷前五章拍表。
- Jane Friedman 关于系列写作和多故事线管理的资料：续作需要承接已有读者期待，同时避免重新解释过多背景，迁移为旧资产带入和信息投放规则。
- Reedsy 关于 chapter transitions、opening chapters 和 story setting 的资料：转场要保持因果，设定要通过行动和冲突展示，迁移为新地图启动规则。
- Writer's Digest 关于 chapter transitions 和 sequel hooks 的资料：转场、钩子和续作开头都应服务故事推进，迁移为第一场景触发器和不能空降清单。
- 中国作家网关于网络文学连载和读者订阅习惯的访谈资料：连载作品需要持续黏性和有效推进，迁移为新卷前 3-8 章必须给小回报方向。

迁移到单本小说项目时，不要整段复制本专题。应复制“卷间转场与新阶段启动卡”，并填入旧卷交接输入、新阶段目标、新卷前五章拍表、新地图信息投放和新卷启动闸门结论。


## 与 63 号专题的衔接：跨卷旧资产需要重新进入读者视野

新卷启动时，旧物、旧债、旧关系、旧规则、旧敌人、旧承诺如果要在新阶段继续发挥作用，应调用 `docs/storycraft/63-foreshadowing-callback-reader-memory-management.md` 评估其记忆温度。

跨卷旧资产默认按“休眠”或“冷”处理，除非上一卷结尾刚刚强回响。新卷前几章可以用短场景、行动后果、关系互动或低强度日常把旧线重新放回读者视野，再让它参与新目标、新地图或新冲突。
