# 105. 敏感内容、创伤知情写法与读者伤害防护专项

状态：已确认

适用范围：章节或设定涉及创伤、心理危机、自伤自杀、性暴力、亲密关系权力差、未成年人、校园欺凌、家庭暴力、极端羞辱、暴力血腥、精神控制、创伤回忆、受害者叙事、救赎线、反英雄越界、现实案件改编、暗黑题材、虐恋或任何可能引发读者强烈不适、二次伤害、模仿风险、平台风险和价值观争议的内容。

核心结论：敏感内容不是“不能写”，但不能用猎奇、低俗、爽点、反转噱头或道具化受害者的方式写。AI agent 必须先判断内容必要性、目标读者、平台边界、伤害风险、叙事立场、人物主体性、细节尺度、后果呈现、求助/支持路径和读者预期，再决定删除、降级、隐去、转述、预警、延后或重构。

最小链路：内容必要性 -> 风险类型 -> 目标读者 -> 平台/合规边界 -> 视角选择 -> 细节尺度 -> 主体性与后果 -> 情绪补偿 -> 预警方式 -> 审稿闸门。

---

## 1. 为什么需要独立专题

第 20 篇已经提供内容合规和平台风险总闸门，但它更偏“能不能发、会不会违规”。实际写作时，AI agent 还需要回答更细的问题：

- 敏感事件是否必须进入正文，还是可以移到背景、台账或角色过去？
- 需要写到什么细节程度，才能服务人物和剧情，又不变成猎奇？
- 受害者是否有主体性，还是只被当成主角成长、反派作恶或爽点反弹的工具？
- 自伤自杀、性暴力、未成年人伤害、霸凌和心理危机是否被浪漫化、合理化或教程化？
- 黑暗题材、虐恋和反英雄是否有清楚读者契约，而不是包装背刺？
- 内容提示如何帮助读者选择，而不是剧透、恐吓或把风险当营销噱头？

本篇补的是“如果题材必须涉及，怎样写才更安全、更可信、更尊重人物和读者”。

## 3. 来源吸收与迁移边界

### 3.1 SAMHSA 创伤知情原则

SAMHSA 的创伤知情方法强调认识创伤影响、识别创伤迹象、将创伤知识融入实践，并主动避免再创伤。其原则包括安全、可信和透明、同伴支持、协作、赋权/声音/选择，以及文化、历史和性别议题。

迁移规则：小说不是治疗服务，但创作可以借鉴这些原则：让人物有安全需求、声音、选择、支持系统和恢复路径；不要把创伤角色写成单一标签或永远只负责痛苦。

### 3.2 WHO / Samaritans / 988：自杀和自伤呈现

WHO、Samaritans、988 Lifeline 等资料都强调，自杀和自伤相关内容需要避免详细方法、浪漫化、单因解释和奖励式后果，并应鼓励求助、希望和恢复路径。

迁移规则：网文中如果涉及自伤自杀，不能把它写成解决问题、证明爱情、换取团圆、制造爽点或教程化细节。应转向复杂原因、阻止与支持、现实后果、恢复过程和求助资源。

### 3.3 RAINN：同意、性暴力和亲密关系权力差

RAINN 关于性暴力和亲密关系性暴力的资料强调，不能同意、无法同意、被强迫或被权力压迫的情形需要被识别。不同司法辖区定义会变化，本知识库不做法律判断，只迁移为创作边界。

迁移规则：亲密关系张力可以有冲突和欲望，但不能把胁迫、控制、羞辱、无法拒绝、权力压迫或未成年人不当关系包装成浪漫爽点。

### 3.4 未成年人网络保护和儿童权利资料

国家网信办、教育部和 UNICEF 相关资料都强调保护未成年人身心健康、隐私、尊严和免受伤害。UNICEF 的儿童报道原则还强调避免让儿童遭受羞辱、报复、重新痛苦或身份暴露风险。

迁移规则：涉及未成年人时，AI agent 应按更高标准处理：不色情化、不猎奇化、不教程化危险行为、不把儿童痛苦当主角爽点工具，不让未成年人被识别、羞辱或消费。

### 3.5 内容提示与读者选择

内容提示/触发提示在研究和出版实践中存在争议，但其稳定价值是帮助读者了解内容范围并作出选择。NetGalley、教育写作中心和读者社区资料都强调提示应简洁、具体、不过度剧透。

迁移规则：中文网文可根据平台、题材和读者预期使用“内容提示/排雷说明/阅读提示”。提示不能替代正文质量，也不能把敏感内容当营销噱头。

## 4. 概念定义

### 4.1 敏感内容

定义：可能涉及现实伤害、心理危机、身体伤害、性和权力边界、未成年人、歧视、群体创伤、极端暴力、羞辱、隐私或高模仿风险的内容。

### 4.2 创伤知情写法

定义：写作时意识到创伤可能改变人物的注意力、判断、身体反应、关系边界和恢复路径，并避免把创伤简化成标签、卖惨、反转工具或永久缺陷。

### 4.3 二次伤害

定义：文本、评论、作话或剧情安排让受害者再次被羞辱、责怪、消费、曝光、怀疑或工具化。

### 4.4 细节尺度

定义：敏感事件在正文中呈现到什么程度。尺度包括：不写、暗示、后果呈现、角色回忆、旁人转述、低细节场景、完整场景。高风险内容默认不写可模仿步骤和刺激性细节。

### 4.5 内容提示

定义：在简介、作话、章节前或正文外，用简洁语言告知读者可能出现的敏感内容，帮助读者选择是否继续阅读。

## 5. 五个总原则

### 5.1 必要性原则

敏感内容必须回答：

- 它是否改变人物选择、关系、风险或主题？
- 是否有低风险替代表达？
- 是否只是为了刺激、反转、虐点或点击？
- 删除它后，故事是否仍然成立？

如果只是为了加重口味，不进入正文。

### 5.2 主体性原则

受伤害人物不是道具。必须有：

- 自己的欲望、恐惧和判断。
- 至少一个选择或表达机会。
- 事件后的持续影响，而不是剧情用完就丢。
- 不被旁白轻易定性为“活该”“软弱”“麻烦”。

### 5.3 非教程化原则

不得提供现实可模仿的伤害、自伤、自杀、犯罪、逃避监管、侵犯隐私、制毒赌博、暴力实施或操控步骤。

写结果、影响、阻止、求助、后果和恢复，不写操作指南。

### 5.4 非浪漫化原则

不得把伤害写成：

- 爱的证明。
- 成长捷径。
- 英雄牺牲的唯一路径。
- 让他人后悔或团圆的手段。
- 主角获得爽点的装饰。

### 5.5 后果原则

敏感事件必须有后果：心理、身体、关系、法律、社会、经济、名誉、信任或价值选择。不能“下一章清零”。

## 6. 六类高风险内容写法

### 6.1 自伤、自杀和心理危机

禁止：

- 写具体方法、步骤、工具和可复制细节。
- 把自杀写成合理解决方案或浪漫结局。
- 用单一原因解释心理危机。
- 让自伤自杀直接换来爱情、团圆、道歉或胜利。
- 把心理危机当作操控他人的剧情按钮。

推荐：

- 写复杂压力、求助困难、旁人的支持或失误、角色的恢复路径。
- 写人物在危机前后的关系和行动变化。
- 写阻止、陪伴、求助、治疗、危机缓解和长期恢复，而不是刺激性现场。
- 正文外可根据平台和地区提供简洁求助资源，但不要把热线当剧情道具。

### 6.2 性暴力、胁迫和亲密关系权力差

禁止：

- 把非同意、无法拒绝、权力胁迫写成浪漫。
- 用受害过程制造低俗刺激。
- 用“他其实爱她”洗白控制、跟踪、羞辱或侵犯边界。
- 把受害者的痛苦当成 CP 升温工具。
- 把未成年人卷入不当亲密关系。

推荐：

- 明确叙事立场：伤害就是伤害。
- 将重点放在边界、后果、选择、支持和恢复。
- 如果写权力差恋爱，必须给弱势方真实选择权、拒绝权、资源和后果保护。
- 亲密关系张力可来自价值冲突、误解、保护欲、目标冲突和昂贵选择，而不是胁迫。

### 6.3 未成年人伤害和校园/家庭暴力

禁止：

- 色情化、猎奇化、消费化未成年人。
- 详细描写未成年人可模仿的危险行为、欺凌步骤、犯罪方法或自伤方式。
- 用儿童痛苦推动成人主角爽点，随后忽视儿童后续处境。
- 暴露可识别身份、隐私或羞辱性细节。

推荐：

- 保护未成年人视角和尊严。
- 写可信成年人、制度、同伴、老师、监护或救助资源的介入与限制。
- 让后果进入后文，不把儿童伤害当一次性情绪燃料。
- 如果必须写校园/家庭压迫，重点写权力结构和逃离路径，不写可模仿细节。

### 6.4 暴力、血腥和犯罪

禁止：

- 为猎奇堆叠血腥细节。
- 写可复制犯罪手法、规避侦查或伤害教程。
- 把酷刑、虐杀、霸凌或私刑包装成纯爽点。
- 让受害者只承担“刺激读者”的功能。

推荐：

- 写暴力的代价和后果，而不是技术步骤。
- 将画面焦点放在人物选择、风险、救援、证据、关系变化和心理影响。
- 悬疑可写调查和因果，不写可执行教程。
- 反英雄越界要有代价、边界和读者契约。

### 6.5 群体、文化、疾病和身份议题

禁止：

- 用刻板印象制造反派、笑点或异域猎奇。
- 把某个群体写成天生危险、低等、愚昧或神秘工具。
- 把疾病、残障、心理状态当作道德缺陷或反派标签。

推荐：

- 写具体人物，不写群体标签。
- 区分个人行为、制度压力和文化背景。
- 涉及真实群体时优先查证，避免单一来源。
- 让人物有多个维度，不只围绕伤痛或身份存在。

### 6.6 极端羞辱、虐主和黑暗压迫

禁止：

- 连续羞辱无反弹抓手。
- 为虐而虐，用恶心和压迫替代冲突设计。
- 让读者长期只感到无力、厌恶和被消耗。

推荐：

- 每次压迫都要换来信息、资源、决心、同盟、证据、策略或关系变化。
- 黑暗题材要提前建立读者契约。
- 长低谷必须有微回报、反弹方向或情绪利息。
- 调用第 68、79、101 篇判断读者是否期待反弹而不是弃书。

## 7. 内容尺度分级

| 等级 | 定义 | 处理 |
| --- | --- | --- |
| S0 不进入正文 | 仅作为背景或台账事实 | 用一句后果替代事件过程 |
| S1 暗示 | 读者知道发生过，但无现场细节 | 适合高风险内容和开篇快速交代 |
| S2 后果呈现 | 写创伤后的身体、关系、资源和选择变化 | 推荐默认方案 |
| S3 低细节场景 | 写场景压力和人物选择，不写刺激性细节 | 需审稿和内容提示 |
| S4 高强度场景 | 直接呈现严重伤害或心理危机 | 极高风险，通常不建议；必须合规审查 |
| S5 禁止场景 | 可模仿自伤/犯罪步骤、未成年人性化、低俗猎奇等 | 不写，重构 |

默认策略：高风险内容从 S2 开始设计；如果 S2 已能完成叙事功能，不升级到 S3/S4。

## 8. 视角选择规则

### 8.1 受害者视角

优点：有主体性、能写真实感受。

风险：容易过度贴近痛苦，造成二次伤害或猎奇。

处理：控制细节，写选择、感知边界、求助路径和后续恢复。

### 8.2 旁观者视角

优点：可降低直接刺激，突出救援、见证、制度问题。

风险：受害者被工具化，旁观者抢走故事。

处理：旁观者不能替受害者定义全部意义；后文要回到受害者状态。

### 8.3 加害者视角

优点：可揭示权力、操控和犯罪逻辑。

风险：美化、共情错位、教程化、读者误读。

处理：极慎用；不得提供可复制细节；叙事立场和后果必须清楚。

### 8.4 事后视角

优点：可聚焦后果、恢复和关系变化。

风险：事件本身被轻描淡写。

处理：通过行为、回避、关系边界、资源损失和选择变化体现重量。

## 9. 内容提示规则

内容提示的目标是帮助读者选择，不是恐吓、营销或免责声明。

### 9.1 何时提示

建议提示：

- 自伤自杀或心理危机。
- 性暴力、亲密关系胁迫或严重边界侵犯。
- 未成年人伤害。
- 家庭暴力、校园欺凌、虐待。
- 高强度暴力、血腥、酷刑。
- 强虐、黑暗题材、反英雄越界。

### 9.2 怎么提示

推荐：

```text
阅读提示：本卷涉及校园霸凌后果、心理危机和家庭压力，相关内容以低细节方式呈现，介意者请谨慎阅读。
```

避免：

```text
超刺激黑暗虐杀大场面来了，胆小勿入。
```

### 9.3 提示不能替代改稿

如果内容本身猎奇、低俗、教程化或价值观错误，不能靠提示解决。先改正文。

## 11. 审稿闸门

敏感内容审稿必须先看高层风险，再看文笔。

### 11.1 一票暂停项

出现以下情况，暂停后续生产：

- 可模仿的自伤、自杀、犯罪或伤害步骤。
- 未成年人色情化、猎奇化或消费化。
- 非同意、胁迫或控制被包装成浪漫爽点。
- 受害者被责怪、羞辱或工具化，文本无批判立场。
- 暴力、酷刑或血腥细节以猎奇为主要卖点。
- 真实案例未去身份化，可能造成隐私、名誉或二次伤害。
- 内容提示和包装承诺明显欺骗读者。

### 11.2 常规检查

- 这段内容是否必要？
- 是否可以降低尺度仍完成叙事功能？
- 角色是否有主体性？
- 伤害是否有后果？
- 是否把严重事件写成单一原因或简单治愈？
- 是否提供了支持系统、求助路径或恢复可能？
- 是否与目标读者和类型承诺匹配？
- 是否会成为第 101 篇定义的强毒点？

---

## 12. 常见错误与修法

| 错误 | 风险 | 修法 |
| --- | --- | --- |
| 用创伤制造人设深度 | 角色只剩伤痛标签 | 补欲望、能力、关系、选择和恢复路径 |
| 用受害推动主角成长 | 受害者工具化 | 给受害者后续状态、声音和选择 |
| 用自杀换团圆或后悔 | 奖励化、危险误导 | 改成求助、阻止、长期关系修复 |
| 用强迫写浪漫 | 边界侵犯被美化 | 改成明确选择权、拒绝权和尊重 |
| 用未成年人痛苦冲击读者 | 猎奇和平台风险 | 降低细节，写保护、后果和制度问题 |
| 用血腥替代恐惧 | 低俗刺激 | 写未知、代价、选择和后果 |
| 用作话对线敏感反馈 | 二次伤害读者信任 | 简洁说明、尊重反馈、必要时修订 |
| 用提示替代正文责任 | 读者仍被背刺 | 先改正文，再做提示 |

## 13. 多 agent 分工

### 13.1 策划 agent

负责：在立项和大纲阶段识别敏感内容，判断是否必要，决定尺度上限和预警策略。

输出：敏感内容风险表。

### 13.2 起草 agent

负责：按尺度写正文，不越权扩写刺激细节，不把风险内容写成教程或爽点。

输出：低风险正文方案。

### 13.3 审稿 agent

负责：检查主体性、后果、叙事立场、内容尺度、平台风险和读者伤害风险。

输出：删除/降级/保留/合规复查建议。

### 13.4 反馈 agent

负责：保留读者原话，区分目标读者雷点、非目标读者口味、真实伤害反馈和平台风险信号。

输出：风险聚类和修订优先级。

### 13.5 知识库维护 agent

负责：只把稳定安全规则写入知识库。不得保存现实伤害细节、真实受害者隐私或高风险桥段复述。

## 14. 最小规则

时间很少时，至少执行：

1. 高风险内容先问是否必要，不必要就删。
2. 能写后果就不写过程，能暗示就不详写。
3. 不写可模仿的自伤、自杀、犯罪和伤害步骤。
4. 不把强迫、控制、羞辱或无法拒绝写成浪漫。
5. 不色情化、猎奇化或消费化未成年人。
6. 受害者必须有主体性和后续状态。
7. 严重伤害必须有后果，不能下一章清零。
8. 内容提示帮助读者选择，不能替代正文责任。
9. 目标读者强烈不适时，进入第 101 篇毒点评估。
10. 平台、法律、医疗和心理健康相关判断正式发布前必须重新确认。
