# 212 县域城市管理、市容环卫、流动摊贩、占道经营、夜市、停车与公共空间执法链专项

状态：已确认  
更新日期：2026-06-09  
适用对象：负责县城现实题材、基层治理、返乡创业、夜市经济、小摊小店、社区矛盾、街区改造、城管执法、环卫、公共空间争议和舆情单元的 AI agents。

## 1. 本篇解决什么问题

AI 写“城管、摊贩、占道经营、市容整治、垃圾分类、违停、夜市治理”时，常见失真有三种：

- 把城管写成只会抢东西、罚款和吼人的脸谱反派。
- 把摊贩写成绝对弱者，因此任何占道、油烟、噪声、堵路、卫生和消防风险都被合理化。
- 把公共空间治理写成领导一句“创城”“整治”，缺少投诉来源、法律依据、证据固定、劝导整改、处罚程序和民生安置。

本篇把县域城市管理写成一条可执行的叙事链：

```text
谋生/经营/通行/生活需求
-> 公共空间属性与时间段
-> 摊点、门店、车辆、垃圾、广告牌或施工行为出现
-> 居民、商户、学校、医院、司机、环卫或游客产生体感影响
-> 12345、社区、街道、城管、交警、市场监管、住建环卫等入口分流
-> 劝导、告知、限期整改、现场检查、拍照取证、询问记录
-> 简易程序或普通程序处理，必要时行政处罚、行政强制措施或移送
-> 复查、整改、投诉复核、复议诉讼、舆情回应、摊点规划和民生回写
```

一句话：城管戏的核心不是“谁嗓门大”，而是公共空间里谁能用、怎么用、影响谁、凭什么管、管到什么尺度。

## 3. 当前规则底座

### 3.1 城市管理执法不是“临时队伍随便管”

《城市管理执法办法》用于理解城市管理执法主管部门、执法范围、执法行为规范、执法协作和监督。写作时要把“执法人员、辅助人员、街道社区、市场监管、公安、交警、环卫、市政、住建”等角色拆开。

### 3.2 行政处罚要受程序和比例约束

《中华人民共和国行政处罚法》和国务院关于规范罚款设定与实施的文件提醒：罚款不是基层治理的万能按钮。处罚要有事实、证据、依据、程序和过罚相当。

写作影响：

- 不要让城管因为“看不顺眼”就罚。
- 不要把罚款写成创收工具而不呈现监督风险。
- 不要把“教育劝导、责令改正、限期整改、轻微违法不罚或首违不罚”等柔性治理空间全部抹掉。

### 3.3 摊点和灵活就业要分开看

国务院关于支持多渠道灵活就业的政策强调因地制宜、规范有序、清理不合理限制，支持个人经营和灵活就业。县城小摊、便民市场、临时摊点、夜市、早市可以写成就业和城市活力，但不能因此忽视交通、消防、卫生、噪声、学校医院周边秩序和公平竞争。

## 4. 先判断：这件事到底归谁管

城市管理题材最容易混部门。AI agent 每场戏先做分流判断。

| 事项 | 常见主管或参与方 | 写作重点 |
| --- | --- | --- |
| 占道经营、店外经营、乱设摊点 | 城管、街道、社区、市场主办方 | 空间属性、时间、通行、安全、劝导整改 |
| 食品摊点卫生、食品安全 | 市场监管、城管协同 | 经营许可/备案、食品来源、后厨或摊点卫生 |
| 缺斤少两、价格欺诈 | 市场监管 | 计量、价格、消费者投诉，不要全推给城管 |
| 机动车违停、交通秩序 | 交警、城管按地方分工 | 道路属性、停车位、交通影响、电子取证 |
| 生活垃圾分类、清运、乱倒垃圾 | 环卫、城管、社区、物业 | 分类投放、收运频次、责任主体、处罚或整改 |
| 建筑垃圾、装修垃圾乱倒 | 城管、住建、环卫、物业 | 产生者、运输者、处置点、台账和违法倾倒 |
| 违法建设、私搭乱建 | 规划自然资源、住建、城管、乡镇街道 | 权属、规划、历史形成、拆除程序和安全风险 |
| 噪声扰民 | 生态环境、公安、城管、社区按声源分流 | 时间段、声源、持续性、证据和调解 |
| 阻碍执法、打砸伤人 | 公安 | 不把暴力冲突写成普通口角 |

## 5. 城管剧情卡

写任何城管、市容、摊贩或公共空间冲突前，先填这张卡。

```yaml
urban_management_scene_card:
  scene_goal: "用公共空间治理制造民生、秩序、经营或舆情冲突"
  space_type: "城市道路/人行道/学校门口/医院周边/市场外溢/广场/夜市街区/老小区门口/景区街面/桥下空间/公园入口"
  actor_type: "流动摊贩/固定门店/早餐车/夜市摊/水果车/外卖骑手聚集点/装修业主/建筑垃圾运输者/违停车主/环卫工/居民/执法队员"
  behavior: "占道经营/店外经营/乱停乱放/乱倒垃圾/广告牌外摆/私搭乱建/噪声油烟/排队拥堵/施工围挡/共享单车堆积"
  legal_or_policy_basis: "城市管理执法办法/市容环卫条例/城市道路管理/生活垃圾分类/地方摊点管理/行政处罚法/待确认"
  harm_or_impact:
    - "通行受阻"
    - "消防或急救通道风险"
    - "噪声、油烟、异味、垃圾"
    - "学校医院周边秩序"
    - "公平竞争或门店投诉"
    - "居民休息和公共卫生"
  evidence:
    - "现场照片或视频"
    - "投诉工单或12345记录"
    - "执法记录仪、询问记录、整改通知"
    - "摊点位置、时间、占用面积"
    - "垃圾、油污、噪声、拥堵或安全隐患记录"
    - "地方临时摊点政策或夜市规划"
  response_path: "劝导/告知/责令改正/限期整改/简易程序处罚/普通程序调查/暂扣或强制措施/移送公安或其他部门/调解/复议诉讼"
  livelihood_layer: "失业、返乡、老人补贴家用、孩子学费、门店租金、淡旺季、平台订单、摊位费"
  aftermath: "整改复查/摊点入市/夜市规范化/舆情回应/罚款争议/执法监督/邻里关系/经营台账回写"
```

缺少 `space_type`，城管戏就会悬浮。缺少 `harm_or_impact`，整治就像面子工程。缺少 `livelihood_layer`，摊贩就会变成符号。缺少 `response_path`，执法就会变成吵架。

## 6. 公共空间的四个判断

### 6.1 这个地方能不能摆

同样是卖早餐，学校门口、人行道、菜市场内、便民摊区、夜市规划街、医院急救通道旁，风险完全不同。

要写清：

- 道路、人行道、广场、市场、社区门口、商场外摆区、景区街面、桥下空间的管理边界。
- 是否有地方划定的便民摊点、临时疏导点、夜市街区、早市时段。
- 是否影响盲道、消防通道、急救车辆、公交站、学校医院周边秩序。

### 6.2 什么时间摆

时间决定冲突强度。

可写节点：

- 早高峰早餐摊与通勤、上学、清扫保洁冲突。
- 夜市摊与居民休息、油烟噪声、垃圾清运冲突。
- 节假日景区摊贩与游客通行、价格投诉、食品安全冲突。
- 创城、赛事、检查、招商活动前的集中整治压力。

### 6.3 谁被影响

公共空间冲突不是“执法者 vs 摊贩”两个人的事。

被影响者包括：

- 旁边交租金的门店老板。
- 每天推轮椅、婴儿车或骑电动车经过的居民。
- 负责清扫油污和垃圾的环卫工。
- 需要急救通道的病人和救护车。
- 被噪声影响休息的老人、学生和夜班工人。
- 想买便宜早点、也不想道路被堵的普通市民。

### 6.4 有没有替代方案

高质量城管剧情要写“治理后的去处”。

可写替代方案：

- 引导到便民市场、疏导点、夜市摊区。
- 规定经营时段、摊位边界、垃圾清理和油污处理要求。
- 低成本摊位、临时摊位抽签、公益摊位、农产品直销点。
- 对老人、残障者、失业者、返乡人员等群体联动就业服务。
- 对拒不整改、严重影响安全或多次违法者依法处理。

## 7. 执法流程写法

### 7.1 日常巡查和投诉进入

真实入口常常不是“领导突然想整治”，而是：

- 12345 多次投诉学校门口拥堵。
- 环卫工反映夜市后油污垃圾清不完。
- 门店老板投诉门前被流动摊贩长期占用。
- 医院急救通道被私家车和摊车堵住。
- 县里创建文明城市、迎接节会或景区客流高峰，提前排查风险。

### 7.2 劝导、告知和整改

好写法：

```text
执法队员先说明问题 -> 指出影响和依据 -> 要求退回规定线内或清理现场 -> 记录摊主身份和时间地点 -> 给出整改期限或引导点位 -> 复查仍不改再进入处罚程序
```

不要把每次执法都写成“上来就扣车”。扣押、暂扣、强制清除、罚款等都需要更强的事实和程序支撑。

### 7.3 取证和程序

可写细节：

- 执法记录仪开启，现场拍照记录摊位位置、占用面积、时间和影响。
- 出示执法证件，说明事项和依据。
- 询问摊主、门店、居民、环卫或市场管理方。
- 制作现场笔录、整改通知或行政处罚文书。
- 当事人可以陈述、申辩；对处罚不服可依法救济。

读者会感到可信的地方，不是专业名词堆满，而是“每一个动作都在把争吵变成可核对的事实”。

## 8. 摊贩不是符号：五种可写动机

| 摊贩类型 | 表层行为 | 深层压力 | 写作风险 |
| --- | --- | --- | --- |
| 失业中年人早餐车 | 早高峰占道卖包子豆浆 | 房贷、孩子学费、找不到稳定工作 | 只写可怜，不写卫生和通行影响 |
| 老人水果摊 | 小区口摆摊补贴家用 | 养老金低、子女不在身边 | 把老人当免罚护身符 |
| 返乡青年夜市摊 | 用短视频引流，摊位外扩 | 想创业但不懂规则 | 把烟火气写成无成本自由 |
| 农户直销车 | 进城卖菜卖瓜 | 农产品价格低、进市场成本高 | 忽视检疫、计量、道路秩序 |
| 门店外摆 | 店内面积小，想多卖 | 租金高、竞争激烈 | 把交租门店和流动摊贩对立得过于简单 |

商业级写法：让摊贩既有谋生的痛，也有必须面对的公共责任。

## 9. 城管执法人员也不是工具人

城管角色的真实压力：

- 上级考核、投诉工单、舆情和群众满意度同时压在身上。
- 执法过轻，被居民骂“不作为”；执法过重，被拍成短视频骂“欺负人”。
- 编制人员、协管人员、街道社区和外包服务之间职责不同。
- 同一条街今天保民生、明天迎检查、后天办活动，尺度很难稳定。
- 面对老人、小孩、残障者、失业者等弱势群体时，合法性之外还有社会效果。

好人物写法：

- 新人执法队员第一次遇到老人摊贩，既想讲法，也怕自己冷血。
- 老队员知道谁家困难，但也记得那次急救车被堵的后果。
- 队长不是反派，他最怕的是“前脚柔性执法，后脚出事故没人承担”。

## 10. 高价值场景

### 10.1 学校门口早餐摊

```text
早餐摊方便学生和家长
-> 上学高峰人车混杂
-> 环卫投诉油污，家长投诉堵路，学生喜欢便宜热乎
-> 城管和学校、交警、街道协调
-> 引导摊位移动到侧街或便民点
-> 摊主担心客流断掉
-> 主角设计排队线、垃圾桶、限时经营或摊位轮换
```

冲突点：便利和安全不是天然一致。

### 10.2 夜市烟火气与居民睡眠

```text
县里打造夜经济
-> 摊位越来越多，音乐、油烟、垃圾和停车外溢
-> 居民多次打12345
-> 商户说夜市带动就业和客流
-> 城管、环卫、市场监管、公安、街道联合调整时段和边界
-> 主角必须在“热闹”和“能过日子”之间找尺度
```

冲突点：网红打卡的一小时，可能是楼上居民每天凌晨的崩溃。

### 10.3 建筑垃圾乱倒

```text
老小区装修潮
-> 装修队夜里把建筑垃圾倒在城郊空地
-> 村民发现扬尘和堵沟
-> 城管、环卫、住建追查运输车和装修来源
-> 业主说自己已经付了清运费
-> 装修公司、运输者和物业开始甩锅
```

冲突点：城市更新和垃圾末端治理经常不是同一个叙事层级。

### 10.4 执法短视频舆情

```text
摊贩与执法人员冲突被剪成短视频
-> 评论区站队
-> 完整执法记录仪显示前面已有多次劝导和整改通知
-> 摊贩确有困难，但也反复占用急救通道
-> 县里既要调查执法是否规范，也要回应民生安置
```

冲突点：短视频只截取一段，小说要写完整事实和情绪后果。

### 10.5 便民摊区分配

```text
县里设便民摊区
-> 摊位有限，谁能进、多少钱、谁抽签、谁管理成了新矛盾
-> 老摊贩、人情关系、返乡青年、残障家庭和市场内商户都不满意
-> 公示、抽签、投诉、监督和运营成本成为剧情线
```

冲突点：治理不是把人赶走，而是重新分配稀缺空间。

## 11. 舆情写法：不要只写“网友反转”

城管和摊贩冲突特别容易被短视频切片。写舆情时至少补四层：

- 第一层：公开视频看到什么，没看到什么。
- 第二层：现场此前是否有投诉、劝导、整改和风险。
- 第三层：当事人是否有困难、是否有替代路径。
- 第四层：执法是否规范，处罚是否过当，是否有人借题发挥。

不要用“完整视频一出，全网道歉”偷懒。更好的余波是：

- 当事人得到临时帮扶，但也要遵守摊区规则。
- 执法队完善记录和沟通方式。
- 县里重新划定夜市边界和清运频次。
- 主角意识到，正义不是站队，而是让规则能保护更多人。

## 12. 常见错误与修复

| 错误写法 | 读者体感 | 修复方式 |
| --- | --- | --- |
| 城管一来就抢秤、掀摊、罚款 | 脸谱化、价值观粗糙 | 补投诉来源、劝导记录、执法依据和程序 |
| 摊贩可怜，所以占消防通道也没问题 | 情绪绑架、公共风险消失 | 写谋生压力，也写通行和安全后果 |
| 夜市只有烟火气 | 悬浮美化 | 补垃圾、油烟、噪声、厕所、停车、环卫和居民睡眠 |
| 创城等于形式主义 | 单薄 | 写考核压力、公共服务改善、基层负担和过度整治之间的张力 |
| 罚款解决一切 | 行政程序缺失 | 加入责令改正、教育劝导、轻微不罚、复查、申辩、救济 |
| 所有部门都叫城管 | 制度混乱 | 按事项分流到市场监管、交警、公安、住建、生态环境、街道社区 |
| 摊区规划一出大家都满意 | 假 | 写摊位数量、租金、抽签、公示、老摊贩流失和门店竞争 |

## 13. 安全边界

不得提供以下现实可操作内容：

- 如何逃避城管巡查、规避处罚、伪造摊位许可或投诉材料。
- 如何改装摊车、遮挡车牌、逃避电子取证或阻碍执法。
- 如何倾倒建筑垃圾、隐匿运输轨迹、规避清运处置。
- 如何组织暴力抗法、围堵执法人员或煽动网暴人肉。
- 如何伪造执法视频、恶意剪辑、造谣或曝光个人隐私。

可以写：

- 合法摆摊、便民摊区、夜市治理、柔性执法和整改复查。
- 摊贩谋生、居民权益、执法监督和公共空间再分配的矛盾。
- 违法后果、证据链、程序和救济，但不写规避方法。

## 14. 审稿检查问题

写完城管或摊贩剧情后，逐项检查：

- 这个空间是道路、人行道、市场、广场、小区门口、学校医院周边还是夜市街区？
- 行为具体是什么：占道、外摆、垃圾、油烟、噪声、违停、违建、广告牌还是施工围挡？
- 谁投诉或谁受影响？有没有写通行、安全、卫生、睡眠、公平竞争或公共资源？
- 是否区分城管、市场监管、公安、交警、住建、环卫、街道社区？
- 执法前是否有劝导、告知、整改、取证或复查？
- 如果罚款，是否有事实、证据、依据、程序和过罚相当？
- 摊贩是否有谋生压力？这种压力有没有被写成违法免责牌？
- 居民、门店、环卫工、学生、老人等第三方是否有真实感？
- 是否写了替代方案：便民市场、限时摊点、夜市规范化或就业支持？
- 舆情是否只截取片段？有没有写完整事实、调查和余波？

## 15. 可直接使用的桥段提示词

### 15.1 早餐摊治理

```text
请写一场县城学校门口早餐摊治理戏。必须写清摊贩谋生压力、学生家长便利、道路拥堵或安全隐患、投诉来源、城管/交警/学校/街道分工、劝导整改、替代点位和情绪余波。不要把城管写成纯反派，也不要把占道风险洗白。
```

### 15.2 夜市舆情

```text
请设计一个夜市整治引发短视频舆情的小副本。要求写清夜市经济收益、居民噪声垃圾投诉、执法记录、摊贩困难、评论区站队、完整事实调查、整改方案和主角选择。不要用“全网反转”草草解决。
```

### 15.3 建筑垃圾乱倒

```text
请写一场建筑垃圾乱倒追责戏。必须写清装修业主、装修队、运输者、物业、环卫或城管的责任边界，证据包括监控、车牌、清运合同、倾倒地点和投诉工单。不要提供规避清运监管的方法。
```

## 16. 一句话原则

县域城管戏的商业张力，不在“谁把谁制服”，而在一条街上同时存在谋生、通行、卫生、安全、体面、舆论和规则；主角的价值，是让这些互相冲突的需求被看见、被排序、被安放。
