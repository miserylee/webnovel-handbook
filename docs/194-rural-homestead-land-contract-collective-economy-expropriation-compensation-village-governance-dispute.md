# 194. 农村宅基地、土地承包、征收补偿、村集体经济与乡村治理纠纷链专项

状态：已确认  
日期：2026-06-09  
适用对象：AI 写作 agent、审稿 agent、现实题材设定 agent、乡村/家族/年代/重生创业/基层干部线项目

## 1. 本篇解决什么问题

农村土地相关剧情最容易被写成“村里不同意”“亲戚抢地”“拆迁发财”这类粗糙梗。真实冲突通常不是单一财产争夺，而是身份、程序、亲缘、村庄权力、政策窗口和地方习俗叠在一起。

本篇把农村土地和村集体治理拆成可执行的叙事链，帮助 AI agent 判断：

| 判断项 | 写作意义 | 常见误写 |
|---|---|---|
| 争的是哪种地或权益 | 宅基地、承包地、经营权、集体收益、征地补偿的规则不同 | 把祖宅、承包田、分红、拆迁款混写成“我家的地” |
| 角色是什么身份 | 村民、集体经济组织成员、承包户、外嫁女、返乡创业者、外来投资者、村干部的权利不同 | 只看户口，不看成员资格和承包关系 |
| 程序卡在哪里 | 申请、会议、公示、合同、审批、登记、公告、补偿、仲裁、诉讼都能制造剧情节点 | 用一句“村长拍板”替代全部流程 |
| 谁能代表集体 | 村委会、村民小组、农村集体经济组织、乡镇政府、部门各有边界 | 把村长写成土地所有者 |
| 冲突如何升级 | 家庭争执、村内协调、乡镇调解、仲裁、行政复议、诉讼、信访舆情的路径不同 | 直接黑化或直接上法院，缺少中间压力 |

## 2. 安全边界

1. 本篇只用于小说设定和审稿，不提供法律意见、维权方案、征地补偿测算或具体案件判断。
2. 不指导伪造合同、伪造会议记录、虚构成员身份、骗取补偿、非法占地、强拆、暴力阻工或规避执法。
3. 涉及宅基地、征收补偿、成员资格、土地流转、集体收益分配时，必须提示“地方细则会影响结果”，不要把全国性原则写成所有村都一样。
4. 写基层治理时避免把村干部整体脸谱化。真实冲突常来自职责不清、信息不透明、历史遗留、亲缘压力和利益分配，而不只是“坏村长”。
5. 写女性成员、外嫁女、离婚丧偶女性、入赘、再婚、服役、就学、外出务工等身份变化时，不要默认其自动失去集体权益。

## 3. 核心叙事链

```text
人物身份 -> 家庭户与集体成员资格 -> 地块或权益类型 -> 历史来源与证据 -> 村民小组/村集体初步意见 -> 村级会议或公示 -> 乡镇/部门审批或备案 -> 合同、证书、登记或公告 -> 收益、使用、流转、征收或建房行为 -> 异议、举报、调解、仲裁、复议、诉讼 -> 结果落地 -> 人情关系和村庄秩序回写
```

写作时不要从“争钱”直接跳到“反派被抓”。更有商业张力的路线是：

```text
一个小手续缺口 -> 一个亲戚翻旧账 -> 一个村干部要求走程序 -> 一个公示栏引爆全村议论 -> 一个证据缺失让主角被动 -> 一个政策节点逼大家站队 -> 一个补偿或分红方案让旧怨变现
```

## 5. AI agent 先判三件事

### 5.1 判地或权益

| 类型 | 核心问题 | 可写冲突 |
|---|---|---|
| 宅基地 | 谁有资格申请、是否一户一宅、面积标准、审批手续、建房规划、旧宅处置 | 返乡建房被拦、兄弟分户争宅基地、老人旧屋翻建、公示期被举报 |
| 农房 | 房是家庭生活空间，宅基地是集体土地使用权益 | 买卖农房、继承旧屋、翻建审批、拆迁安置、外村人购房风险 |
| 承包地 | 家庭承包关系、承包合同、确权登记、承包期稳定、收回调整限制 | 外出务工后地被别人种、离婚后承包收益争议、老人去世后家庭内部分配 |
| 土地经营权 | 承包方把经营权出租、入股、托管或流转给他人 | 流转合同压价、老板跑路、改变农地用途、规模经营失败 |
| 集体收益 | 集体经营、资产出租、项目分红、福利分配 | 外嫁女分红、服役大学生资格、村干部亲属优先、名单公示引爆 |
| 征地补偿 | 公共利益、公告程序、补偿安置、社保、地上附着物、青苗、成员分配 | 高铁/产业园征地、补偿名单漏人、祖坟迁移、抢种抢建、村民代表会议争议 |
| 集体经营性建设用地 | 集体依法用于经营性建设，和宅基地、承包地不同 | 村企合作、招商、租赁、入市收益、项目失败后的债务和分红 |

### 5.2 判人

| 角色身份 | 叙事权限 | 典型误区 |
|---|---|---|
| 本集体经济组织成员 | 可能涉及承包、宅基地申请、收益分配、征地补偿分配 | 不能只用“户口在不在村”一刀切 |
| 承包户家庭成员 | 承包经营权常以户为单位延续 | 不要把每块承包地都写成个人私产 |
| 外嫁女、离婚女性、丧偶女性 | 容易成为成员资格、分红、补偿争议核心 | 不要默认出嫁就没有权利 |
| 入赘或迁入人员 | 要看成员确认规则、村规章程、地方规定和实际生活关系 | 不要默认迁入就一定能分红或建房 |
| 外出务工、就学、服役人员 | 可能长期不在村但身份未必消失 | 不要把“没在村住”直接写成“没资格” |
| 返乡创业者 | 可能有情感归属、项目需求和身份尴尬 | 不要让其凭钱直接买下全村土地 |
| 外来投资者 | 主要通过租赁、合作、土地经营权流转、项目合同进入 | 不要写成能随意购买集体土地所有权 |
| 村干部或集体经济组织负责人 | 负责组织、协调、公开、公示、执行程序 | 不要写成个人拥有审批权和所有权 |
| 乡镇干部 | 常在审批、调解、监督和政策落地中出现 | 不要让乡镇替代所有村民自治程序 |

### 5.3 判程序卡点

| 程序节点 | 剧情功能 | 适合制造的压力 |
|---|---|---|
| 申请 | 主角提出建房、承包、流转、补偿、分红诉求 | 资格被质疑、材料缺失、亲属反对 |
| 村民小组或村级讨论 | 村庄公共意见浮出水面 | 宗族、邻里、旧账、人情债集中爆发 |
| 公示 | 信息从私下变成公开 | 名单、面积、补偿金额、资格排序被围观 |
| 乡镇审批或部门联审 | 村内意见进入行政流程 | 材料不合格、规划冲突、耕地红线、历史违建 |
| 合同或协议 | 利益被写进纸面 | 口头承诺失效、签字人资格、显失公平、代签争议 |
| 登记、证书或确权资料 | 证明力增强 | 老证缺失、档案错误、名字写错、兄弟共户 |
| 征收公告和补偿方案 | 外部项目改变村庄秩序 | 抢种抢建、提前泄密、分配名单、安置房排序 |
| 调解、仲裁、复议、诉讼 | 冲突进入正式救济 | 证据、期限、管辖、亲情破裂、舆论反噬 |

## 6. 关键制度素材转化

### 6.1 宅基地：不是“祖上私有土地”，而是集体土地上的居住权益

原则：农村宅基地通常围绕本集体成员、户、建房需求、面积标准、规划和审批展开。它适合写“居住权利”和“家庭身份”的冲突，而不是简单写成可自由买卖的私有土地。

用法：

| 场景 | 正确写法方向 | 风险写法 |
|---|---|---|
| 返乡建房 | 先写是否具备申请资格、是否已有宅基地、是否符合规划、村级审查和乡镇审批 | 主角回村直接圈地盖别墅 |
| 老宅翻建 | 写原宅基地、旧房安全、规划许可、邻里采光排水、是否超面积 | 老人一句“祖宗留下的”就无视审批 |
| 兄弟分家 | 写户的认定、原宅基地分配、父母居住、旧屋权属、村内名额 | 直接按兄弟人数平均切土地 |
| 外村人买农房 | 写合同风险、宅基地资格限制、后续翻建和确权困难 | 外来老板买下整片宅基地开发民宿 |
| 征收旧宅 | 写宅基地、房屋、附着物、安置人口和补偿方案的多层次冲突 | 只写“拆迁款一到账，全家暴富” |

冲突钩子：

- 主角母亲一直住在老宅，但宅基地批准书上是已故爷爷名字。
- 堂哥先一步在老屋旁堆材料，制造“已经开工”的事实。
- 村里公示建房名单，主角家被备注“已有一处宅基地”。
- 返乡女主想翻建母亲旧房，被亲戚用“你嫁出去了”阻拦。
- 民宿项目要盘活闲置农房，但村民担心以后不能给儿子建房。

### 6.2 承包地：家庭承包关系稳定，不等于任何人可以随意处置

原则：承包地的戏要抓住“集体所有、家庭承包、承包期稳定、农地农用、权益可流转但有边界”。

用法：

| 场景 | 可写细节 | 读者体感 |
|---|---|---|
| 外出务工 | 地被亲戚代种多年，补贴、租金、确权名字引发争执 | 现实感强，适合家族旧账 |
| 离婚 | 承包地以家庭户为单位，婚姻变化牵动收益和居住安排 | 比单纯争房更复杂 |
| 老人去世 | 承包户内成员继续经营还是家庭内部分配收益 | 适合继承专题联动 |
| 流转给大户 | 合同、租金、期限、用途、毁约、恢复地力 | 适合创业和乡村振兴线 |
| 项目失败 | 老板拖欠租金，地被撂荒或改作他用 | 现实题材反转点 |

不要写：

- 村干部随口收回某家承包地奖励别人。
- 主角把承包地卖给城市公司永久开发。
- 流转合同完全不写期限、用途、价款和恢复责任，却能支撑百万级剧情。

### 6.3 土地经营权流转：商业爽点来自“合同和用途边界”

土地经营权流转适合写返乡创业、农业公司、合作社、直播助农、文旅项目、预制菜基地和现代农业园区。

可用流程：

```text
农户意愿 -> 村集体或服务平台协调 -> 流转价格谈判 -> 合同签订 -> 用途约束 -> 备案或管理 -> 租金支付 -> 种植养殖投入 -> 风险事件 -> 合同履行或解除 -> 土地恢复与赔偿
```

可写冲突：

| 冲突 | 戏剧价值 |
|---|---|
| 价格过低 | 老农觉得被占便宜，年轻人觉得反正荒着也是荒着 |
| 期限过长 | 短期钱和长期控制权冲突 |
| 用途变更 | 说种粮，实际建仓库、停车场、露营地 |
| 老板跑路 | 租金欠付，地里设施烂尾 |
| 土地质量下降 | 退地时谁负责恢复 |
| 村干部背书 | 口头承诺和合同文字不一致 |
| 农户反悔 | 市场行情上涨后要求涨租或收回 |

### 6.4 集体经济组织：村庄里的“钱袋子”和身份引擎

农村集体经济组织法把农村集体经济组织、成员权利、财产经营和收益分配变成很适合小说的制度素材。

核心写法：

| 元素 | 小说功能 |
|---|---|
| 成员确认 | 决定谁能参与分红、补偿、福利和部分集体权益 |
| 章程 | 把村里的“老规矩”转成纸面规则，也给反转留下证据 |
| 成员大会或代表会议 | 让利益从私下谈判变成集体表决 |
| 财产经营 | 村厂、鱼塘、山林、商铺、光伏、停车场、文旅项目都可入戏 |
| 收益分配 | 小钱是福利，大钱是爆点 |
| 妇女权益 | 外嫁、离婚、丧偶、户无男性等情形都能成为现实冲突核心 |

成员资格冲突的写法：

```text
角色身份变化 -> 村里形成旧口径 -> 集体收益或征地补偿变大 -> 名单公示 -> 当事人发现被排除 -> 村干部解释“以前都这样” -> 法律、章程、地方细则和实际生活事实发生冲突 -> 调解或诉讼压力升级
```

适合人物线：

| 人物 | 内在压力 |
|---|---|
| 外嫁女 | 不是只争钱，而是争“我是不是这个村的人” |
| 服役青年 | 国家身份和村庄身份之间的错位 |
| 外出务工者 | 对家乡有贡献但长期缺席村会 |
| 离婚女性 | 婚姻解体后被两个村同时推开 |
| 入赘女婿 | 在家庭劳动贡献和集体成员资格之间尴尬 |
| 村会计 | 掌握账目，知道每个家庭的痛点 |
| 年轻村干部 | 想依法公开，但要面对宗族和长辈压力 |

### 6.5 征地补偿：不要只写“赔多少钱”，要写“公共利益、程序和分配”

农村征地线天然有商业张力，因为外部项目会把平静村庄里看不见的身份差异一次性变现。

可用流程：

```text
项目风声 -> 预公告或摸底 -> 土地现状调查 -> 社会稳定风险评估 -> 补偿安置方案 -> 公告和听取意见 -> 协议签订 -> 补偿支付和安置 -> 交地 -> 后续就业、社保、迁坟、复垦、矛盾回流
```

可写冲突：

| 冲突点 | 叙事用法 |
|---|---|
| 消息提前泄露 | 抢种、抢建、抢迁户口，制造悬疑和道德灰区 |
| 地类认定 | 一块地到底是耕地、宅基地、林地、建设用地，影响补偿和用途 |
| 地上附着物 | 青苗、果树、养殖棚、坟墓、机井、围墙都能成为细节爆点 |
| 补偿分配 | 土地补偿费、安置补助、青苗补偿、房屋补偿、集体留成分配不同 |
| 安置人口 | 户籍、居住、婚姻、出生、死亡、服役、就学都可能影响名单 |
| 社保安置 | 失地后的长期生活问题，比一次性补偿更有现实厚度 |
| 村内表决 | 公共方案和个体不满冲突 |
| 交地 | 外部施工进场，村内矛盾变成可见对抗 |

写法提醒：征地剧情里，最有价值的不是“拆迁暴富”，而是“谁被看见、谁被遗漏、谁替谁签字、谁掌握消息、谁把旧怨包装成程序问题”。

## 7. 村民自治和村务公开：基层戏的舞台

村民委员会不是土地所有者本人，但它是农村公共事务、群众自治和日常协调的重要舞台。农村集体经济组织则更直接涉及集体资产经营和收益分配。两者在现实中常相互配合，也容易在小说中被混写。

| 主体 | 适合写什么 | 不宜误写成 |
|---|---|---|
| 村党组织 | 方向、组织、协调、基层治理核心 | 个人任性拍板一切 |
| 村民委员会 | 公共事务、村务公开、自治协调、调解 | 土地和集体资产的私人老板 |
| 村民小组 | 更贴近宅基地、承包地和组级集体资产 | 没有任何程序的私下小圈子 |
| 农村集体经济组织 | 发包土地、宅基地事项、经营集体资产、分配收益 | 普通公司或村干部个人账户 |
| 乡镇政府 | 宅基地审批、监督指导、调解和政策落地 | 替村民会议决定所有分配 |
| 农业农村部门 | 承包、流转、宅基地管理等职责线索 | 万能审批部门 |
| 自然资源部门 | 国土空间规划、用地、耕地保护、土地管理 | 解决所有家庭纠纷 |
| 法院、仲裁机构 | 正式纠纷解决 | 情绪宣泄场 |

村务公开的戏剧用法：

```text
过去靠口头规矩 -> 项目带来大额收益 -> 村里必须公开名单和账目 -> 某个名字、金额、签字或日期异常 -> 主角从“听说”转入“证据” -> 村庄关系重新站队
```

## 8. 常见剧情模型

### 8.1 返乡女主翻建老宅

```yaml
scene_model: 返乡翻建老宅
character:
  protagonist: 外嫁后离婚的女主
  pressure: 母亲独居旧房漏雨，女主想翻建养老
land_or_right: 宅基地和农房
procedure_chain:
  - 查旧宅来源和批准资料
  - 向村里提交翻建或建房申请
  - 村里初审和公示
  - 邻居提出采光排水异议
  - 亲戚以外嫁身份阻拦
  - 乡镇审批卡住面积或规划
conflict_hook: 公示栏上女主名字被划掉，旁边写着“已外嫁”
agent_note: 重点写身份羞辱、母女居住安全和程序证据，不要只写泼妇吵架。
```

### 8.2 重生男主流转土地创业

```yaml
scene_model: 返乡创业流转土地
character:
  protagonist: 重生返乡的农业创业男主
  pressure: 需要连片土地做高标准种植
land_or_right: 土地经营权流转
procedure_chain:
  - 与农户谈意愿和价格
  - 村集体协调边角地和撂荒地
  - 合同约定期限、租金、用途和恢复责任
  - 备案或村内台账
  - 投入设施和种苗
  - 市场波动后部分农户反悔
conflict_hook: 男主知道未来行情会涨，但不能用低价长期锁死村民利益，否则爽感会变成剥削感。
agent_note: 商业爽点应来自技术、组织和信用，不要来自坑农户。
```

### 8.3 高铁征地牵出家族旧账

```yaml
scene_model: 征地补偿家族冲突
character:
  protagonist: 在外工作的孙女
  pressure: 祖宅和承包地进入征收范围
land_or_right: 宅基地、农房、承包地、征收补偿
procedure_chain:
  - 村里摸底登记
  - 公告补偿方案
  - 叔伯争抢户主签字权
  - 祖母被要求按手印
  - 孙女发现名单缺少母亲权益
  - 调解不成进入行政或诉讼路径
conflict_hook: 亲戚说“你妈嫁出去二十年了”，但祖母养老和旧宅修缮一直由母亲承担。
agent_note: 把证据、照护贡献、身份羞辱和补偿名单同时写出来。
```

### 8.4 村集体分红名单风波

```yaml
scene_model: 集体收益分配
character:
  protagonist: 新任村会计或返乡大学生
  pressure: 光伏项目第一次分红，名单公开后全村炸锅
land_or_right: 集体经济组织收益分配
procedure_chain:
  - 集体经济组织形成分配方案
  - 成员或代表会议讨论
  - 名单和金额公示
  - 外出务工、外嫁、离婚、服役人员提出异议
  - 旧章程和新法律口径冲突
  - 乡镇介入指导
conflict_hook: 分红金额不大，但它证明了“谁还算这个村的人”。
agent_note: 这类戏的情绪核心是身份承认，不只是钱。
```

### 8.5 村企合作项目烂尾

```yaml
scene_model: 村企合作烂尾
character:
  protagonist: 年轻村干部
  pressure: 前任签下的文旅项目烂尾，租金拖欠，土地用途被质疑
land_or_right: 集体经营性资产、土地经营权或建设用地使用
procedure_chain:
  - 查合同和会议记录
  - 查实际用地和规划手续
  - 查租金支付和收益分配
  - 村民要求追责
  - 企业要求继续延期
  - 上级部门和法院路径介入
conflict_hook: 老人只记得当年“大家都同意”，但会议记录上少了关键签字。
agent_note: 反派可以不是单个坏人，而是一串当年为了发展留下的模糊承诺。
```

## 9. 类型化用法

| 类型 | 用法 |
|---|---|
| 现实主义 | 用土地、成员资格和村务公开写基层治理的复杂性 |
| 乡村振兴 | 用流转、合作社、集体经济、文旅项目写发展和分配的平衡 |
| 重生创业 | 让主角用信息差、组织能力和合同意识破局，但不要坑农户 |
| 家族伦理 | 用宅基地、祖宅、承包地、征地补偿让旧亲情变成可见利益 |
| 女性成长 | 外嫁女、离婚女性、丧偶女性争取成员权益和居住安全很有现实情绪 |
| 基层干部 | 程序、公示、调解、舆情、上级考核和村内人情是天然压力锅 |
| 悬疑 | 地块档案、旧合同、会议记录、签字、坟墓迁移、抢种抢建可以做线索 |
| 年代文 | 土地制度变化要按历史阶段另查资料，不能直接套用 2026 年规则 |

## 10. AI 写作检查问题

1. 这一场争的是宅基地、农房、承包地、经营权、集体收益还是征收补偿？有没有混用概念？
2. 角色是村民、集体经济组织成员、承包户成员、外来投资者还是村干部？他凭什么主张权利？
3. 村干部说话前，有没有村级讨论、公示、会议、审批、合同或登记这些可见流程？
4. 如果写“外嫁女没有资格”，有没有把它作为冲突观点，而不是作者默认结论？
5. 如果写征地补偿，有没有写公告、调查、补偿方案、安置、地上附着物和分配名单？
6. 如果写土地流转，有没有合同期限、价款、用途、恢复责任和农户自愿？
7. 如果写返乡创业，主角的爽点是否来自创造价值，而不是利用信息差压低农民权益？
8. 如果写村务公开，公开的是什么：名单、金额、会议记录、合同、账目还是补偿方案？
9. 如果冲突升级，有没有先经过村内协调、乡镇调解、仲裁、复议或诉讼中的合理路径？
10. 这一场结束后，村庄人情、家族关系和基层治理台账有没有发生变化？

## 11. 审稿风险信号

| 风险信号 | 读者体感 | 修正方向 |
|---|---|---|
| 所有土地都叫“我家的地” | 概念糊，现实读者出戏 | 先标明宅基地、承包地、经营权或集体收益 |
| 村长一句话决定所有补偿 | 像短剧反派，不像基层治理 | 增加会议、公示、乡镇、部门和文件痕迹 |
| 外嫁女天然失权 | 价值观和现实规则风险高 | 写成争议观点，并引入成员资格和妇女权益线 |
| 征地只写暴富 | 缺少公共利益、程序和后续生活 | 增加安置、社保、迁坟、交地、就业和家庭撕裂 |
| 返乡创业者随便买地 | 规则错误 | 改成流转、租赁、合作、托管、入股或项目合同 |
| 村民全是愚昧阻挠 | 人物扁平，容易冒犯 | 给每方真实利益：养老、子女建房、风险恐惧、历史亏欠 |
| 法院一出场全部解决 | 戏剧断裂 | 写证据、周期、执行、亲情破裂和村庄回流 |
| 政策口号压过人物 | 像宣传稿 | 把政策落到一张名单、一枚手印、一块界桩、一笔租金 |

## 13. 最小可用规则

1. 宅基地写“资格、户、审批、规划、面积、旧宅”，不要写成自由买卖私地。
2. 承包地写“集体所有、家庭承包、承包期稳定、农地农用”，不要让村干部随手收回。
3. 土地经营权流转写“自愿、有偿、合同、用途、期限、恢复”，不要写成永久卖地。
4. 集体收益写“成员资格、章程、会议、公示、账目”，不要只写村长分钱。
5. 征地补偿写“公告、调查、方案、安置、名单、异议路径”，不要只写拆迁暴富。
6. 外嫁、离婚、丧偶、就学、服役、务工等身份变化要当成冲突点细写，不要默认自动失权。
7. 每个土地场景至少出现一个可见证据：合同、名单、公告、批准书、界桩、会议记录或账本。
8. 每个基层治理场景至少保留一层人情后果：亲戚不来往、村里站队、老人受压、干部被夹在中间。
