## 2. 推荐读取链路

### 2.1 开篇和前十章

```text
22 开篇留存 -> 29 类型承诺 -> 14 机制 -> 15/25 关系情绪 -> 21 对手阻力
```

适合：开书、第一章、前三章、前十章方向和早期留存问题。

### 2.2 中段防水和卷结构

```text
23 中段卷结构 -> 21 对手系统 -> 25 情绪线 -> 63 伏笔回收 -> 62 日常过场防水
```

适合：写到中段开始重复、支线发散、日常水、敌我升级不足。

### 2.3 悬疑反转和信息控制

```text
26 悬疑反转 -> 63 伏笔读者记忆 -> 28 设定投放 -> 07 连续性台账
```

适合：线索太直、反转突兀、读者忘伏笔、设定解释影响节奏。

### 2.4 情绪线和人物关系

```text
15 关系类型 -> 25 情绪线 -> 27 喜剧轻松感 -> core-writing/37 角色声音
```

适合：CP 不甜、队友情不成立、宿敌没火花、群像关系不黏。

### 2.5 结尾和完本

```text
24 结尾风险 -> 63 伏笔回收 -> 25 情绪结算 -> market-serialization/104 完本复盘
```

适合：最后一卷、完本、番外前的情绪和坑位结算。

## 3. 常见误用

- 不要把 `14` 的机制写成万能外挂；机制必须有成本、限制、误判空间和可升级阻力。
- 不要把 `25` 的情绪线写成互相解释感情；关系要通过选择、亏欠、让步、误会和代价推进。
- 不要把 `26` 的反转写成作者突然翻牌；读者回看时应该觉得线索存在，只是当时被误导。
- 不要把 `27` 的喜剧写成插科打诨；梗点最好来自人物缺点、关系错位和场景压力。
- 不要把 `28` 的世界观说明堆在开头；设定必须压在人物行动、冲突和回报里展示。
- 不要用 `62` 给日常场景找借口；低强度场景也必须改变关系、信息、情绪或下一步选择。
